5. rumus past perfect tense kalimat verbal dan nominal


verbal past perfect : (+) had + v3. (-) had not + v3. (?) had + s + v3? nominal past perfect : (+) had been + noun. (-) had not been + noun. (?) had + s + been + noun?Kalimat verbal:
1. Kalimat aktif:
    Kalimat positif: Subjek + Had + Kata Kerja III + Komplemen
    Kalimat negatif: Subjek + Had + Not + Kata Kerja III + Komplemen
    Kalimat interogatif: Had + Subjek + Kata Kerja III + Komplemen
2. Kalimat pasif:
    Kalimat positif: Subjek + Had + Kata Kerja III + By + Objek 
    Kalimat negatif: Subjek + Had + Not + Kata Kerja III + By + Objek
    Kalimat Interogatif: Had + Subjek + Kata Kerja III + By + Objek

Kalimat nominal:
    Kalimat positif: Subjek + Had + To Be III (Been) + Bukan Kata Kerja
    Kalimat negatif: Subjek + Had + Not + To Be III (Been) + Bukan Kata Kerja
    Kalimat interogatif: Had + Subjek + To Be III (Been) + Bukan Kata Kerja
