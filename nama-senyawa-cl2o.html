23. nama senyawa dari Cl2O, Cl203,Al2O3 berturut turut adalah​


Jawaban:

diklorin monoksida, diklorin trioksida, Aluminium oksida

Jawaban:

Tata nama senyawa:

Cl2O = Diklorin Monoksida

Cl2O3 = Diklorin Trioksida

Al2O3 = Aluminium Oksida

Penjelasan:

Tata nama senyawa nonlogam + nonlogam:

Awalan ( jumlah atom ) nama non logam + Awalan ( jumlah atom ) nama non logam + -ida

Untuk jumlah atom = 1 pada unsur non logam pertama tidak perlu diberi awalan, cukup nama unsur tersebut

Awalan sesuai jumlah atom:

1 = mono

2 = di

3 = tri

4 = tetra

5 = penta

Tata nama senyawa nonlogam + nonlogam:

Nama logam + Nama unsur non logam + -ida
