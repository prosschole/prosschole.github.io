Penjelasan dengan langkah-langkah:

PECAHAN BIASA => PECAHAN DESIMAL

[tex]1. \: \frac{24}{50} = \frac{24 \times 2}{50 \times 2} = \frac{48}{100} = 0.48[/tex]

[tex]2. \: \frac{3}{5} = \frac{3 \times 2}{5 \times 2} = \frac{6}{10} = 0.6[/tex]

[tex]3. \: \frac{3}{8} = \frac{3 \times 125}{8 \times 125} = \frac{375}{1.000} = 0.375 [/tex]

PECAHAN BIASA => PERSEN (%)

[tex]1. \: \frac{4}{5} = \frac{4 \times 20}{5 \times 20} = \frac{80}{100} = 80\%[/tex]

[tex] 2. \: \frac{10}{20} = \frac{ 10 \times 5 }{20 \times 5} = \frac{50}{100} = 50\%[/tex]
