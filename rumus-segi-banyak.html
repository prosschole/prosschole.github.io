Jawab:

   K = 2p+2l

3. L = 1/2 x a x t

   K = s+s+s

4. L = a x t

   K = 2 (a + b)

Jawaban:

1.Untuk menghitung luas persegi, maka rumus yang digunakan adalah L = s x s. Dengan keterangan L adalah luas, sedangkan s adalah sisi. Misalnya sebuah persegi memiliki sisi dengan panjang 5 cm, maka untuk menghitungnya, L = 5 cm x 5 cm dari rumus L = s x s.

2.Rumus Persegi Panjang

Nama Rumus

Luas (L) L = p x l

Keliling (K) K = 2 x ( p + l )

Panjang (p) p = L ÷ l p = (K ÷ 2) – l

Lebar (l) l = L ÷ p l = (K ÷ 2) – p

3.Rumus segitiga untuk mencari luas bangun adalah 1/2 x alas x tinggi, untuk mencari keliling segitiga dapat ditentukan dengan menjumlahkan panjang dari masing-masing sisi pada segitiga.

4.Pengertian Jajar Genjang atau Jajaran Genjang

Nama Rumus

Keliling (Kll) Kll = 2 × (a + b)

Luas (L) L = a × t

Sisi Alas (a) a = (Kll ÷ 2) - b

Sisi Sisi Miring (b) a = (Kll ÷ 2) - a
