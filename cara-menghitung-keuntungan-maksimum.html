3. Cara menghitung keuntungan maksimum menggunakan metode linear programming


Cara menghitung keuntungan maksimum menggunakan metode linear programming ada empat tahapan. Tahapan tersebut diantaranya menyederhanakan soal dengan mengubahnya kedalam bentuk model matematika, membuat fungsi dan pertidaksamaan, menggambar grafik dan menentukan titik potong, dan menghitung nilai fungsi berdasarkan titik potong.

Pembahasan

Metode linear programming adalah suatu metode untuk menyelesaikan masalah dengan menggunakan persamaan maupun pertidaksamaan linear dengan banyak penyelesaian, dengan memperhatikan syarat-syarat agar memperoleh hasil yang maksimum/minimum. Secara umum, masalah program linear dapat diselesaikan dengan langkah-langkah sebagai berikut.

Mengubah persoalan kedalam bentuk model matematika. Hal ini bertujuan untuk menyederhanakan soal agar lebih mudah dipahami.Membuat sistem pertidaksamaan dan fungsi tujuan berdasarkan model matematika.Menggambar grafik dan menentukan titik potongnya.Menghitung nilai fungsi tujuan berdasarkan titik potong yang ditentukan.Berdasarkan nilai fungsi tujuan ini, maka program linear telah diselesaikan.Pelajari lebih lanjut

4. Pak Ahmad adalah seorang pedagang sapi dan kambing, pada saat menjelang hari raya Idul Adha Pak Ahmad membeli sapi dan kambing di Sidoarjo untuk dijual kembali di Surabaya. Modal yang dimiliki Pak Ahmad sebesar Rp260.000.000,00. Namun Pak Ahmad mengalami kebingungan membeli berapa ekor sapi dan kambing agar mencapai keuntungan maksimum. Oleh karena itu, Pak Ahmad bertanya kepada anaknya yang bernama Hanif.Pak Ahmad:“Hanif, bapak bingung harus membeli berapa ekor sapi dan kambing agar keuntungan bapak banyak”.Hanif:“Kenapa bingung pak, hanif bisa bantu menghitung keuntungan yang akan bapak peroleh”.Pak Ahmad:“Kalau gitu bantu bapak ya, kandangnya hanya muat tidak lebih dari 41 ekor, dan rencananya harga jual untuk sapi Rp22.000.000,00 dan kambing Rp.5.500.000”.Hanif:“Eh pak maaf ternyata Hanif lupa cara meghitungnya”.Pak Ahmad:“Kamu itu gimana toh nif, katanya bisa”.Hanif:“Maaf pak, hanif lupa”.Mari bantu Pak Ahmad dan Hanif untuk menentukan banyak sapi dan kambing yang harus dibeli agar mencapai keuntungan maksimum​


Misal:

x = jumlah sapi

y = jumlah kambing

*) Karena kandang hanya muat 41 ekor, maka:

x + y ≤ 41 ... (1)

*) Karena modal Rp 260.000.000,00, maka:

Rp 20.000.000,00 x + Rp 4.000.000,00 y ≤ Rp 260.000.000,00

5x + y ≤ 65 ... (2)

*) x ≥ 0 dan y ≥ 0

Untuk pertidaksamaan 1, garis batas memiliki titik (0,41) dan (41,0).

Untuk pertidaksamaan 2, garis batas memiliki titik (0,65) dan (13,0).

Sehingga terbentuk daerah himpunan seperti gambar terlampir.

Sehingga titik terluar adalah (0,0), (0,41), (13,0), dan (6,35).

Karena harga jual:

sapi = Rp 22.000.000,00

kambing = Rp 5.500.000,00

Maka hasil penjualan:

1) Untuk (0,0):

Rp 22.000.000,00 (0) + Rp 5.500.000,00 (0)

= 0

2) Untuk (0,41):

Rp 22.000.000,00 (0) + Rp 5.500.000,00 (41)

= Rp 225.500.000,00

3) Untuk (13,0):

Rp 22.000.000,00 (13) + Rp 5.500.000,00 (0)

= Rp 286.000.000,00

4) Untuk (6,35):

Rp 22.000.000,00 (6) + Rp 5.500.000,00 (35)

= Rp 132.000.000,00 + Rp 192.500.000,00

= Rp 324.500.000,00

Sehingga untuk mencapai keuntungan maksimum, Pak Ahmad bisa membeli 6 sapi dan 35 kambing.
