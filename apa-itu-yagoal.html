30. Diketahui kubus ABCD.EFGH. Jika panjang diagonal sisi AC =\/18cm .panjang diagobal ruang AG adalah​


Jawab:

Penjelasan dengan langkah-langkah:

Panjang AC = diagonal sisi = √18 = 3√2

Kalau cara cepet, jika diagonal sisi 3√2, maka panjang masing2 sisi nya 3, dan diagonal ruangnya 3√3.

*catatan.

Misal, suatu kubus panjang sisi a cm.

Diagonal sisi = a√2 cm

Diagonal ruang = a√3 cm

Cara panjang :

(Diagonal sisi)² = sisi² + sisi²

(√18)² = s² + s²

18 = 2s²

9 = s²

S = √9 = 3 cm

(Diagonal ruang)² = (diagonal sisi alas)² + sisi²

*diagonal sisi alas = diagonal sisi AC

AG² = AC² + s²

AG² = (√18)² + 3²

AG² = 18 + 9

AG² = 27

AG = √27 = √9×3 = 3√3 cm

Semoga membantu
