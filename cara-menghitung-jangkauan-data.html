2). Nilai rata-rata – pada suatu kelompok yaitu jumlah nilai data kemudian dibagi dengan banyaknya data tersebut, biasanya nilai rata-rata digunakan untuk mengevaluasi data sehingga lebih cepat dan menggambarkan seluruh data serta tidak dapat digunakan untuk menentukan nilai data tertentu di antara sekelompok data, sebagai contoh misalnya nilai rata-rata budi yaitu 85 jadi tidak bisa disimpulkan bahwa nilai untuk pelajaran Matematika Budi lebih dari 75.
