Ini adalah masalah ketika akun user yang kamu pakai tidak memiliki akses Administrator.

Penjelasan:

Cara penyelesaian:

Pertama-tama, buka File Explorer.Buka ke path: PC > Local Disk (C:) > Users.Klik kanan ke folder User Profile kamu folder dan klik Properties.Klik tab Security, di bawah Group atau area nama user, klik nama user kamu dan klik Edit.Klik 'Full control check box' di bawah 'Permissions for authenticated users' dan klik tanda 'Apply' kemudian 'OK'.Lalu, klik 'Advanced' di bawah tab Security.Klik 'User profile' kamu di bawah 'Permission entries' dan klik 'Edit', klik 'Allow' seluruh 'permissions for SYSTEM' dan klik 'OK'.Klik kotak cek opsi 'Replace all child object permissions entries with inheritable permission entries from this object'.Klik tombol Apply > OK untuk menyimpan perubahan.Restart/mulai ulang laptop kamu dan cek apakah sudah bisa memasang setup tersebut.

9. Cara menginstal ulang laptop windows 7 tanpa CD room atau sebagainya dan tanpa menghilangkan data yang ada ?


Install ulang tanpa CD Room/Flashdisk bisa, tapi localdisk C otomatis terformat.
Carannya, download file .iso dari windows 7(32bit pake 32bit, 64bit pake 64bit, 32bit pakr 64bit atau sebaliknya gak bisa), taruh di localdisk D atau E.
Nanti akan muncul Window Install Windows
Tekan Install Now
Nanti layar Install Windows akan jadi full screen
Setelah itu akan ada pilihan mengenai Update, pilih saja yang Do Not ...
Setelah itu centang I Accept ....
Akan muncul pilihan lagi yakni Upgrade atau Custom(advance)
Pada pilihan ini anda disuruh untuk memilih option ingin Upgrade atau Custom. pilih Custom
Setelah itu muncul lagi Window untuk memilih Local Disc
Jika anda sudah sampai tahap ini, anda harus mencari Local Disk C: tempat system. Pastikan yang anda pilih adalah Local Disc C: . Jangan sampai salah. Unuk melihat mana C: , anda bisa melihat size dari Local Disc masing masing.
Baca : Nah disini mungkin letak perbedaan instal ulang dengan CD atau flashdisk. Jika anda pernah instal ulang menggunakan media tersebut, distep ini harusnya kita melakukan pemformatan pada Local Disc C:. Tapi disini berbeda, karna menginstal ulang menggunakan cara ini, kita membutuhkan file system mungin.
Setelah itu tekan next dan tinggal tunggu saja
Pada step ini anda tinggal menunggu saja prosesnya. Dan mungkin akan melakukan restart. Jika restart nanti akan disururuh Force Restart dan Cancel, pilih saja Force Restart. Setelah itu, akan tinggal diam diam dan menunggu proses selanjutnya.

27. Bagaimana cara menganti motherboard tanpa instal ulang Windows 10?


Jawaban:

Bisa

Penjelasan:

1.) Hidupkan PC.

2.) Setelah powering up komputer Anda, tekan "F8" berulang kali. Muncul pilihan pilih aja "Safe Mode", di antara pilihan lain.

3.) Pilih "Safe Mode".

4.) Ketika komputer Anda telah boot ke layar desktop, layar Anda akan memiliki pemberitahuan bahwa Anda berada dalam safe mode. Jika Anda mendapatkan pesan yang memungkinkan Anda untuk "... klik OK untuk melanjutkan", klik ok.

5.) Berikutnya, anda perlu masuk ke "Device Manager", dan uninstall semua driver / perangkat lunak yang berhubun den motherboard lama Anda.

6.) Klik pada menu "Start"

7.) Klik "Control Panel" (Jika dalam "View Category", klik "Performance and Maintenance", kemudian klik "System" icon. Jika dalam "Classic View", double klik "System" icon.)

8.) Klik pada "Hardware" tab.

9.) Klik pada "Device Manager".

11.) Klik kanan semua sub kategori untuk perangkat keras yang terdaftar di bawah, dan klik "Uninstall". Ketika "Uninstall" tidak bisa, pilih "Disable". Kategori-kategori yang harus dipilih dan subkategori dihapus (atau dinonaktifkan) hak paten nya dibawah ini harus clear :

"IDE ATA / ATAPI Controllers"

"Network adapter"

"Sound Driver dan Game Controller"

"Universal Serial Bus Controller"

"VGA Adapter"

Setelah kelompok perangkat keras di atas dihapus atau dinonaktifkan, sekarang saatnya untuk menghapus perangkat lunak yang berhubun den hardware / driver yang dihapus.

12.) Pilih menu "Start", kemudian klik "Control Psayal." Berikutnya, klik da "Add or Remove Programs." Uninstall software APAPUN yang berhubun den perangkat keras yang akan diti. (Misalnya, jika Anda memiliki NVIDIA kartu vga onboard pada motherboard lama Anda, dan Anda melihat software untuk: "NVIDIA Control Psayal", HAPUS Jika anda tidak yakin apa yang harus dihapus SKIP LANGKAH INI, dan hubungi produsen yang berpengalaman,. atau costumer service (sekarang lah tinggal bikin thread di kaskus udah dibantu)

13). Selanjutnya, saatnya untuk mematikan komputer anda dan mempersiapkan diri untuk menganti motherboard baru. Sekarang perangkat lunak lama dan driver tidak boleh bertentan den perangkat keras baru, dan driver yang akan diinstal setelah pengtian.

Sumber : https://kevinsource.blogspot.com/2013/04/ganti-motherboard-tanpa-install-ulang.html?m=1

29. cara menginstal laptop


Apabila kamu ingin menginstall laptop kamu , maka kamu harus memikirkan terlebih dahulu OS apa yang akan kamu install
1. Windows ( Disarankan Windows 7 atau yang lebih tinggi )
2. Linux

Cara instalasinya melalui dua cara yaitu :
1. Melalui CD/DVD
2. Melalui FlashDisk ( minimal 4GB )

Cara menginstall via FLASHDISK :
1. Masukkan CD
2. Masuk ke Bios Setting
3. Pilih Boot
4. Lalu Ubah Entry DVD ke urutan yang paling atas , begitu juga dengan install via flashdisk.
5. LAlu tekan Save and Exit ( biasanya F10 )
6. Apabila berhasil maka akan masuk ke menu instalasi
