1. jenis jenising sandhangan panyigeging wanda yaiku​


Jawaban:

sudah tertera di buku tulis saya

Penjelasan:

SEMOGA MEMBANTU!!!

Jenis jenising sandhangan panyigeging wanda yaiku :

Sandhangan WignyanSandhangan LayarSandhangan CecakSandhangan Pangkon

Pembahasan :

Sandhangan aksara yaiku tetengger kang dienggo ngewahi utawa muwuhi unineng aksara utawa pasangan. Sandhangan ing aksara Jawa kapantha dadi telu, salah sijine yaiku sandhangan panyigeg wanda. Sandhangan panyigeg wanda yaiku sandhangan digunakake kanggo sesigeging wanda utawa suku kata. Sandangan Panyigeg wanda keperang dadi papat, yaiku :

Sandhangan Wignyan, digunakake kanggo ngganti swara h ( ꦃ). Contoh : ꦲꦸꦪꦃꦥꦸꦠꦶꦃ (Uyah Putih)Sandhangan Layar, digunakake kanggo ngganti swara r ( ꦂ). Contoh : ꦏꦧꦂ (Kabar)Sandhangan Cecak, digunakake kanggo ngganti swara ng ( ꦁ). Contoh : ꦲꦧꦁ (Abang)Sandhangan Pangkon, digunakake kanggo ngganti aksara mati ( ꧀). Contoh : ꦥꦶꦏ꧀ꦤꦶꦏ꧀ (Piknik)

Rincian jinis sandhangan panyigeg wanda ana ing lampiran.

3. Jlentrehna apa sing kasebut sandhangan panyigeging wanda lan sandhangan wyanjana


⏺️ Sandhangan panyigeg wanda yaiku sandhangan ing digunakake kanggo ngganteni aksara sigeg.

⏺️ Sandhangan wyanjana yaiku sandhangan sing digunakake kanggo sesulihe aksara sigeg.

Pembahasan

Jinising sandhangan iku ana 3 :

1. Sandhangan swara yaiku sandhangan sing digunakake kanggo swara, i, u, e, é, o.

>> sandhangan swara cacahe ana 5 :

wulu = unine i ➡️ ꦷsuku = unine u ➡️ ꦸpepet = unine e ➡️ ꦼtaling = unine é ➡️ ꦺtaling tarung = o ➡️ ꦴꦺ

2. Sandhangan panyigeg wanda yaiku sandhangan sing digunakake kanggo ngganteni aksara sigeg, kajaba aksara sigeg mau aksara H, R lan ng. Amarga aksara sigeg mau wis duwe tandha dhewe dhewe.

>> Sandhangan panyigeg wanda ana 4 :

wignyan = paten H ➡️ ꦃlayar = paten R ➡️ ꦂcecak = paten ng ➡️ ꦁpangkon = saliyane paten H, R lan ng ➡️ ꧀

3. Sandhangan wyanjana yaiku sandhangan sing digunakake kanggo sesulihe aksara sigeg.

>> Sandhangan wyanjana ana 3 :

Cakra = sesulih ra ➡️ ꦿkeret = sesulih cakra lan pepet ➡️ ꦽꦽpengkal = sesulih ya ➡️ ꦾ

7. Apa bae sing klebu sandhangan panyigeging wanda wangsulan


Apa bae sing klebu sandhangan panyigeging wanda?

cecaklayarwignyanpangkon

_______________________________________

Penjelasan:

Sandhangan panyigeg wanda yaiku sandhangan sing digunakake kanggo nyigeg (mateni) wanda ing pungkasane tembung utawa tembung ing pungkasane gatra.

Jinise sandhangan sigeg

⚫ Cecak

Yaiku sandhangan sing digunakake kanggo sigeg ng awujud ( ꦁ )

→ Tuladhane:

layang = ꦭꦪꦁkanggo = ꦏꦁꦒꦺꦴmaling = ꦩꦭꦶꦁ

⚫ Layar

Yaiku sandhangan sing digunakake kanggo sigeg r awujud ( ꦂ )

→ Tuladhane:

sabar = ꦱꦧꦂpasar = ꦥꦱꦂgarwa = ꦒꦂꦮ

⚫ Wignyan

Yaiku sandhangan sing digunakake kanggo sigeg h awujud ( ꦃ )

→ Tuladhane:

sawah = ꦱꦮꦃpanah = ꦥꦤꦃwahyu = ꦮꦃꦪꦸ

⚫ Pangkon

Yaiku sandhangan sing digunakake kanggo sigeg saliyane ng, r, lan h awujud ( ꧀ )

→ Tuladhane:

macan = ꦩꦕꦤ꧀kodok = ꦏꦺꦴꦢꦺꦴꦏ꧀lawas = ꦭꦮꦱ꧀

29. ing ngisor iki tulisen nganggo basa Jawa nganggo panyigeging Wanda yaiku...​


Jawaban:

a. ꧋ ꦧꦥꦏ꧀ꦠꦶꦤ꧀ꦢꦏ꧀꧈

Jawaban Opsi A✔

Penjelasan:

Mangerteni panyigeg wanda. Panyigeg wanda kalebu jinising sandhangan ing aksara Jawa, kang nduweni fungsi kanggo nyigegi tembung ing saben wanda utawane mateni aksara.

Sandhangan Panyigeg Wanda cacahe ana 4, antarane⤵️

Wignyan = ꦃ sigeg "h"Layar = ꦂ sigeg "r"Cecak = ꦁ sigeg "ng"Pangkon = ꧀ sigeg "paten"

_________________________

Njelntrehake ukara aksara Jawa sing nganggo panyigeg wanda kanthi trep.

a. ꧋ ꦧꦥꦏ꧀ꦠꦶꦤ꧀ꦢꦏ꧀꧈ ( bapak tindak ) ✅aksara ba, aksara pa, aksara ka (sigeg), + pasangan ta + wulu, aksara na (sigeg), + pasangan da + wulu, aksara ka + pangkon.

b. ꧋ ꦲꦏꦸ ꦭꦒꦶ ꦠꦸꦫꦸ꧉ ( aku lagi turu ) ❎aksara ha, aksara ka + suku, aksara la, aksara ga + wulu, aksara ta + suku, aksara ra + suku.

c. ꧋ ꦱꦼꦱꦸꦏ꧀ꦲꦏꦸ ꦩꦺꦭꦸ꧉ ( sesuk aku melu ) ❎aksara sa + pepet, aksara sa + suku, aksara ka (sigeg), + pasangan ha, aksara ka + suku, aksara ma + taling, aksara la + suku

d. ꧋ ꦱꦶꦠꦶ ꦱꦏ ꦱꦭꦠꦶꦒ꧉ (siti saka Salatiga ) ❎aksara sa + wulu, aksara ta + wulu, aksara sa, aksara ka, aksara sa, aksara la, aksara ta + wulu, aksara ga.

a. ꧋ꦧꦥꦏ꧀ꦠꦶꦤ꧀ꦢꦏ꧀꧉⚫

d. ꧋ꦱꦶꦠꦶꦱꦏꦱꦭꦠꦶꦒ꧉

__________________Pembahasan

Sing diarani sandhangan panyigeg wanda yaiku aksara kang mapane ana ing sak mburine aksara sigeg utawa huruf mati.
