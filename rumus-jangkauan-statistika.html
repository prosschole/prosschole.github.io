1. Soal statistika Mohon dijawab beserta cara /rumus. Menentukan : Jangkauan, Simpangan rata², Simpangan baku, ragam (variansi). ​


Diketahui:

Data:

2, 2, 4, 5, 5, 6, 6, 7, 7, 8, 8, 5, 6, 9, 4, 4, 9, 10, 7, 6

Ditanyakan:

a. Jangkauan

b. Simpangan Rata-Rata

c. Ragam

d. Simpangan Baku

Jawab:

Kelompokkan data tersebut.

Nilai -> Frekuensi

2 -> 2

4 -> 3

5 -> 3

6 -> 4

7 -> 3

8 -> 2

9 -> 2

10 -> 1

-------- +

n = 20

a. Jangkauan:

R = Xmaks - Xmin

= 10 - 2

= 8

b. Simpangan Rata-Rata:

1) Cari rata-rata data tersebut.

[tex]x = \frac{(2.2) + (4.3) + (5.3) + (6.4) + (7.3) + (8.2) + (9.2) + (10.1)}{2 + 3 + 3 + 4 + 3 + 2 + 2 + 1} \\ = \frac{4 + 12 + 15 + 24 +21 + 16 + 18 + 10 }{20} \\ = \frac{120}{20} \\ = 6[/tex]

2) Cari nilai mutlak dari selisih tiap nilai dengan rata-rata.

* |2-6| = |-4| = 4

* |4-6| = |-2| = 2

* |5-6| = |-1| = 1

* |6-6| = |0| = 0

* |7-6| = |1| = 1

* |8-6| = |2| = 2

* |9-6| = |3| = 3

* |10-6| = |4| = 4

3) Kalikan dengan masing-masing frekuesi.

* 2. 4 = 8

* 3. 2 = 6

* 3. 1 = 3

* 4. 0 = 0

* 3. 1 = 3

* 2. 2 = 4

* 2. 3 = 6

* 1. 4 = 4

-------- +

34

4) Tentukan simpangan rata-rata.

SR = 34/n

SR = 34/20

SR = 1,7

c. Ragam:

1) Tentukan kuadrat selisih dengan rata-rata.

* (2-6)² = (-4)² = 16

* (4-6)² = (-2)² = 4

* (5-6)² = (-1)² = 1

* (6-6)² = (0)² = 0

* (7-6)² = (1)² = 1

* (8-6)² = (2)² = 4

* (9-6)² = (3)² = 9

* 10-6)² = (4)² = 16

---------- +

51

2) Tentukan nilai ragam.

S² = 51/n

S² = 51/20

S² = 2,55

d. Simpangan baku = √Ragam

= √S²

= S

= √2,55

= (1/10)√255

* Maaf rumus akan ada di foto karena tidak bisa saya ketik.

Keterangan:

SR = Simpangan Rata-Rata

S² = Ragam / Variasi

S = Simpangan Baku

x = Rata - Rata

xi = Nilai data ke-i

fi = Frekuensi data ke-i

n = Total frekuensi

17. Rumus median mengenai statistika yang mudah dan cepat


Rumus median mengenai statistika yang mudah dan cepat adalah Median = Bb + p[(½n - F)/f]

Pembahasan :

median = nilai tengah = Q2

Letak Q2 :

= 2/4 (n+1)

= ½(n+1)

nilai Q2 atau median :

Median = Bb + p[(½n - F)/f]

keterangan :

Bb = Batas bawah yg mengandung median

* jika bulat mk kurangi 0,5

⇨ contoh : Bb = 60 - 0,5

* jika dibelakang koma ada 1 angka mk kurangi 0,05

⇨ contoh : Bb = 60,5 - 0,05

p = panjang kelas (ujung bawah ke 2 — ujung bawah ke 1)

n = jumlah frekuensi data

F = frekuensi kumulatif sebelum ditambah frekuensi yg mengandung median

f = frekuensi yg mengandung median

23. Apa rumus tentang Peluang dan Statistika..?


Peluang :

Pada suatu percobaan terdapat n hasil yang mungkin dan masing-masing berkesempatan sama untuk muncul. Jika dari hasil percobaan ini terdapat k hasil yang merupakan kejadian A, maka peluang kejadian A ditulis P ( A ) ditentukan dengan rumus : 
P (A) = k/n

Statiska ada di gambar :peluang = banyak kejadian jng muncul / banyak kejadian yg mungkin
