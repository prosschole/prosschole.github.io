9. pengen banyak nguasai bahasa Perancis sih Bahasa perancisnya pengen


pengen → vouloir

contoh kata-katanya pengen

saya pengen belajar → je veux apprendre

saya pengen tidur → je veux dormir

saya pengen makan → je veux manger

semoga membantu

Bahasa Prancisnya pengen adalah vouloir.

Pembahasan

Pengen dalam bahasa Prancis adalah vouloir. Kata "vouloir" jika berada di dalam sebuah kalimat present dapat memiliki konjugasi karena adanya penambahan subjek seperti je, tu, vous, il, elle, ils, elles, on, dan nous.

Konjugasi atau perubahan kata vouloirJe + vouloir => Je veux.Tu + vouloir => tu veux.Vous + vouloir => Vous voulez.Il + vouloir => Il veut.Elle + vouloir => Elle veut.Ils + vouloir => ils veulent.Elles + vouloir => Elles veulent.On + vouloir =>Nous + vouloir => Nous voulons.Contoh kalimat dengan kata "vouloir"Elle veut que je réponde à toutes ses questions (dia mau saya menjawab semua pertanyaannya.Vous voulez que je traduise le dictionnaire? (kamu mau saya menerjemahkan kamus itu).Je veux un livre (saya mau sebuah buku.Nous voulons une voiture (kami mau sebuah mobil).
