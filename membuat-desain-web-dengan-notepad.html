1. cara membuat desain web yang menarik dgn menggunakan notepad


1. Lakukan survey

Sebelum membuat website, Anda harus melakukan survey dengan mesin pencari (search engine) tentang contoh website yang sudah ada dan website pesaing bisnis yang sudah muncul. Anda pelajari website tersebut dengan seksama, jika perlu Anda tulis poin-poin penting yang menjadi andalan dari website tersebut. Hal ini akan sangat membantu Anda dalam menentukan isi website yang akan ditampilkan.

2. Resume hasil survey

Dari hasil survei tersebut, tulislah keinginan Anda tentang isi website yang akan dibuat secara detail. Catatan tersebut menjadi acuan Anda dalam memilah-milah bagian yang penting dan yang tidak penting untuk website. Jika Anda memiliki anggaran terbatas, maka  poin-poin yang penting dapat didahulukan untuk ditampilkan di website.

3. Membuat sketsa website

Anda buatlah sketsa tata letak halaman / website (coretan di atas kertas menggunakan pensil untuk membuat kotak-kotak yang dinamai). Anda bayangkan tentang tata letak layout website yang akan dibuat. Anda susun bagian yang akan ditonjolkan (eye-catching) pada atas website terlebih dulu dan yang tidak penting pada bagian website yang lain. Usahakan membuat pula halaman lain jika masing-masing halaman terdapat perbedaan bentuk.

4. Susunlah kata kunci

Anda susun kata kunci (keyword) yang akan digunakan sebagai acuan pencarian. Pikirkanlah secara seksama tentang isi website Anda untuk menentukan kata kunci yang tepat agar pengguna dapat menemukan Website Anda secara cepat berdasarkan kata kunci yang dimasukkan. Jika memiliki beberapa kata kunci, urutkan kata kunci tersebut berdasarkan prioritas. Misal: datakom, lintas, buana, penerbit, buku, bermutu.

5. Periksa data kembali

Anda Periksa kembali data-data dalam website sebelum di-launching. Hal ini untuk memudahkan Anda dalam perbaikan website sebelum diletakkan secara on-line.

Sebagai acuan Anda dalam membuat website berikut ini beberapa tips membuat website lebih menarik dan unik yaitu :

Desain web yang responsive untuk segala jenis perangkat misalnya laptop, handphone, tablet dan lain-lain.Pemilihan typography yang sesuai misalnya dari fontMenu dan konten yang statis atau fixed, fungsinya agar saat di scroll ke bawah menu tetap pada tempatnya.Hindari desain web yang kompleks, pilihlah yang minimalisMulai menghindari flashPilih background yang full page dan gambar berkualitas tinggiMasukkan presentasi video pada websiteBagi Anda yang ingin websitenya terlihat profesional maka Anda harus punya logo yang profesional jugaJauhkan website dari sesuatu yang tidak penting

17. bagaimana cara membuat layout desain web dengan adobe photoshop ?


1. Buka aplikasi Photoshop, buat layer baru dengan klik File => New atau Ctrl + N
2. Buat ukurannya dengan width 1000 pixel dan Height 600 pixel, Resolusi 72 pixel/inch
3. Klik toolbox dan ganti warna foreground color dengan biru (#0038c6) dan background color putih
4. Masih di toolbox, pilih Gradient Tool (G) dan pada keyboard tekan Shift, lalu tekan mouse mulai dari atas dan lepaskan mouse jika sudah berada paling bawah,
5. Buatlah garis bantu / Guide dengan, caranya pilih View => New Guide,
6. Lalu membuat Header, caranya pada layer klik Create a new layer lalu pada toolbox klik Rectangle Tool lalu klik kanan mouse dan ubah foreground color menjadi putih lalu pilih Paint Bucket Tool (G),
7. selesai
8. Buat garis bantu / Guide lagi untuk membuat kotak logo dan kotak iklan
9.  Lalu masukkan logo dan kotak iklan yang sudah anda buat terlebih dahulu,
10. Buat garis bantu untuk membuat menu navigasi
11. ntuk membuat kotak menu navigasi, caranya sama dengan nomor 5, ubah warna foreground color menjadi biru dan background color menjadi putih. Lalu klik Rectangle Tool, klik kanan mouse lalu pilih Make Selection dan berikan gradasi. Untuk membuat warna gradasi seperti pada nomor 3.
12. selesai
13.  Buat teks dengan cara pada pallet layer, klik Create a new layer dan pada toolbox klik Horizontal Type Tool (T), lihat gambar dibawah cara menempatkan teks di kotak menu navigasi
14. Buat garis bantu lagi untuk membuat kotak sidebar kiri, content / post blog dan kotak sidebar kanan, lihat caranya dibawah sesuai nomor urut. Untuk membuat kotaknya seperti pada nomor 5.
15. selesai
16. Buat garis bantu lagi untuk membuat Footer
17.  Copy kotak menu navigasi pada nomor 10, caranya klik kotak menu navigasi dan tekan Ctrl+J, lalu move ke bawah. Lalu pada menu diatas silahkan klik Edit => Transform => Rotate 180°
18. Dan hasil akhir dari desain website dengan photoshop
19. Selanjutnya potong dengan Slice Tool (K) bagian-bagian seperti body, header, logo, kotak iklan, menu navigasi, sidebar kiri, post blog, sidebar kanan dan footer.
20. Lalu pilih File => Save for Web and Devices

28. apa pengertian desain web


Web Design adalah jenis desain grafis yang ditujukan untuk pengembangan dan styling obyek lingkungan informasi Internet untuk menyediakan dengan fitur konsumen high-end dan kualitas estetika. Definisi yang ditawarkan memisahkan desain web dari pemrograman web, menekankan fitur fungsional dari sebuah situs web, serta desain posisi web sebagai semacam desain grafis.
