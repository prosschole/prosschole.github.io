14. jelaskan fungsi dari address bar


Fungsi dari address bar ialah

Menunjukkan website yang sedang dikunjungiMelakukan Quick Search ( penelusuran cepat ) Menunjukkan Certifikat dan cookies browser ( lihat lampiran 1 ) pada beberapa browserMemberikan fasilitas untuk menjadikan bookmark halaman serta membuka pengaturan website pada beberapa browser ( lihat lampiran 2 )

PembahasanAddress Bar

Address bar yang terdapat pada paling atas browser ( lihat lampiran 1 ), berfungsi sebagai menunjukkan website yang saat ini sedang dikunjungi. Jika kalian membuka halaman ini menggunakan browser maka akan terdapat tulisan "https://brainly.co.id/tugas/1841308". Address bar pada google chrome, mozilla firefox serta microsoft edge memiliki lebih banyak fitur daripada browser lainnya.

Fitur Address Bar

Address bar memiliki berbagai fitur, fitur terlengkap saat ini berada di google chrome, mozila firefox dan microsoft edge. Ketiga peramban tersebut memiliki fitur untuk memberitau cookies dan certifikat web, serta membuka permission yang diizinkan oleh website. Peramban tersebut juga memiliki fasilitas bookmark dan translate ( khusus google ) seperti yang tertera pada lampiran 2.

20. Jelaskan fungsi tombal navigasi dan address bar


tombol navigasi terdapat pada Navigation Toolbar, terdiri dari tombol:
Next: tombol yang berfungsi untuk menuju website yang sebelumnya telah diakses
Back:tombol yang berfungsi untuk kembali ke website yang sebelumnya telah diakses
Stop: tombol yang berfungsi untuk menghentikan proses loading suatu web
Refresh: tombol untuk me-refresh (me-loading kembali) tampilan web yang diakses
Home: tombol untuk menampilkan halaman awal web

sedangkan address bar berfungsi untuk menampilkan alamat halaman web yang sedang diakses.
Address bar sifatnya bisa diedit, sehingga bila kita sedang membuka web browser, kita dapat mengubah alamat situs web yang diinginkan.
