untuk tuas jenis 1
Lw + Lf = L
→ Lw = L - Lf

F • Lf = w • Lw
F • Lf = w • (L - Lf)
F • Lf = w • (L - Lf)
F • Lf = w • L - w • Lf
F • Lf + w • Lf = w • L
(F + w) • Lf = w • L
Lf = [w / (F + w)] • L ✓

Lw = L - Lf
Lw = L - [w / (F + w)] • L
Lw = [F / (F + w)] • L ✓
