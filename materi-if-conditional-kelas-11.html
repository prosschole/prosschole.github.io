Pembahasan

Apa itu Conditional?

Conditional tenses digunakan untuk berspekulasi tentang apa yang bisa terjadi, apa yang mungkin terjadi, dan apa yang kita harapkan akan terjadi. Dalam bahasa Inggris, sebagian besar kalimat menggunakan kondisional mengandung kata jika atau if. Banyak bentuk kondisional dalam bahasa Inggris digunakan dalam kalimat yang mencakup kata kerja dalam salah satu bentuk lampau. Penggunaan ini disebut sebagai "masa lalu yang tidak nyata" karena kita menggunakan bentuk lampau tetapi kita sebenarnya tidak mengacu pada sesuatu yang terjadi di masa lalu. Ada lima cara utama membangun kalimat kondisional dalam bahasa Inggris. Dalam semua kasus, kalimat-kalimat ini terdiri dari klausa if dan klausa utama. Dalam banyak kalimat kondisional negatif, ada konstruksi kalimat setara menggunakan "kecuali" bukannya "jika".

9. materi conditional type 1


In a Type 1 conditional sentence, the tense in the 'if' clause is the simple present, and the tense in the main clause is the simple future.

If clause (condition)
If + simple present
If this thing happens

Main clause (result)
simple future
that thing will happen.

As in all conditional sentences, the order of the clauses is not fixed. You may have to rearrange the pronouns and adjust punctuation when you change the order of the clauses, but the meaning is identical.

EXAMPLES

If it rains, you will get wet.
You will get wet if it rains.
If Sally is late again I will be mad.
I will be mad if Sally is late again.
If you don't hurry, you will miss the bus.
You will miss the bus if you don't hurry.
FUNCTION
The type 1 conditional refers to a possible condition and its probable result. These sentences are based on facts, and they are used to make statements about the real world, and about particular situations. We often use such sentences to give warnings. In type 1 conditional sentences, the time is the present or future and the situation is real.

EXAMPLES

If I have time, I'll finish that letter.
What will you do if you miss the plane?
Nobody will notice if you make a mistake.
If you drop that glass, it will break.
If you don't drop the gun, I'll shoot!
If you don't leave, I'll call the police.
In type 1 conditional sentences, you can also use modals in the main clause instead of the future tense to express the degree of certainty, permission, or a recommendation about the outcome.

EXAMPLES

If you drop that glass, it might break.
I may finish that letter if I have time.
If he calls you, you should go.
If you buy my school supplies for me, I will be able to go to the park.

21. Perbedaan mixed conditional dan If conditional


║MIXED CONDITIONAL VS IF CONDITIONAL║

[tex]Explanation[/tex]

Mixed Conditional dikatakan sebagai mixed time . Baik mixed conditional maupun if conditional, kedua nya sama-sama menjabarkan dampak/konsekuensi dalam suatu kondisi tertentu.

Dalam Mixed conditional, terdapat perbedaan struktur tenses (waktu ) yang ditempatkan pada –if clause –nya dengan tenses yang ada di main clause -nya (result clause). Ini merupakan hasil campuran  tenses ( mixed time ) yang diterapkan dari seluruh type pada  if conditional.  

→ If conditional terbagi dalam 3 kategori utama  terstruktur :

1. Conditional 1 => untuk menyatakan keinginan yang mungkin akan terwujud segera apabila syaratnya sudah terpenuhi saat ini { Real present conditional  }  

Pattern=> If+S+verb 1(s/es)+C , S+will+verb 1 + C

2. Conditional 2 => untuk menyatakan keinginan yang tidak mungkin  terwujud karena persyaratannya tidak  mampu untuk dipenuhi saat ini juga atau nanti-nanti { Unreal present conditional }

Pattern=>If+S+verb 2 + C , S+ would +verb 1 + C  

3. Conditional 3 => untuk menyatakan keinginan yang tidak terwujud di waktu lampau karena persyaratannya tidak terpenuhi { Unreal past conditional }  

Pattern=> If+S+had+verb 3+C , S+would +have+verb 3 + C

Sedangkan, dalam Mixed conditional  merupakan hasil campuran dari semua  kategori type –if conditional yang telah disebutkan di atas. Untuk mengetahui perbedaan mixed conditional dan –if conditional lebih jelas, dengan mencermati tenses yang digunakan dalam kondisi fakta nya.  

ANALYSIS DATA

1. Fact  : The students use Brainly application now  because they got the best mark yesterday

Conditional : If the students hadn’t gotten the best mark yesterday, they wouldn’t use Brainly application now.

Result ↑↓ :

Fakta pertama disimpulkan bahwa terdapat campuran tenses dalam makna kondisi faktanya.  

Maka,  verb “ hadn’t gotten “ merupakan penjabaran “got ” dan ini menggunakan  struktur –if clause dalam –If conditional tipe ke 3 untuk menyatakan kejadian lampau (past) tidak nyata. Sedangkan     “wouldn’t use” merupakan  penjabaran “ use”  dan ini menggunakan struktur –result clause  dalam –if conditional tipe ke 2 untuk menyatakan kejadian terkait  sekarang  (present  ) tidak nyata .

2. Fact  : The students got the best mark because they used Brainly application yesterday.

Conditional : If the students hadn’t gotten the best mark, they  wouldn’t have used Brainly application yesterday.

Result ↑↓ :

Fakta kedua disimpulkan bahwa tidak terdapat campuran tenses karena ini termasuk kategori terstruktur dari  type 3 conditional  untuk menyatakan kondisi fakta di waktu lampau (past).

“hadn’t gotten” merupakan penjabaran “got” dan “wouldn’t have used” merupakan penjabaran “used.” Bentuk kata kerja dalam If clause dan main clause sama-sama menyatakan kejadian terkait waktu lampau { past} tidak nyata,  jika dilihat tenses dari makna kalimat kondisi faktanya. Maka, ini termasuk kalimat type 3 conditional

Dari hasil analysis, terlihat jelas perbedaan antara mixed conditional dan –if conditional . Pada mixed conditional menggunakan pola tenses yang berbeda (campuran) dalam makna fakta nya dan terdapat kolaborasi  struktur kata kerja pada –if clause nya maupun main clause –nya yang diterapkan dari masing-masing  tipe conditional baik tipe 3 maupun tipe 2

Sedangkan –if conditional menggunakan pola tenses yang sama dalam makna faktanya dan terstruktur berdasarkan pola masing-masing tipe –if conditional.

30. Jelaskan materi conditional sentence type 3!


digunakan ketika result/consequence (hasil) dari condition (syarat) tidak ada kemungkinan terwujudkarena condition-nya harus sudah dipenuhi di masa lalu. Bagian conditional clause (if + condition) menggunakan subordinate conjunction “if”dan past perfect tense, sedangkan bagianmain clause (result) menggunakan perfect modal (modal + have + past participle). Kalimat Syarat Hasilnya If + Past Perfect Tense Past Future Perfect Tense If + S + had + Verb 3 S + would + have + Verb 3 Atau bisa juga dibalik menjadi: Hasilnya Kalimat Syarat Past Future Perfect Tense If + Past Perfect Tense S + would + have + Verb 3 If + S + had + Verb 3 contoh kalimat: If I had known that you were in hospital, I would have visited you. (Jika saja aku tahu bahwa kamu ada di rumah sakit, aku akan mengunjungimu).Conditional sentence type 3 = past impossible/unreal
yaitu pengandaian yang tidak mungkin menjadi nyata karena berlawanan dengan kondisi di masa lampau
Aturan :
a. Main clause (induk kalimat) : Past future perfect tense
b. If clause (anak kalimat) : Past perfect tense
c. Really meaning : SImple past tense

for example
1. I would have called her if I had known her name
the fact : I didn't call her because I didn't know her name
2. If you had given her money, She would have gone to school
the fact : you didn't give her money so she didn't go to school
