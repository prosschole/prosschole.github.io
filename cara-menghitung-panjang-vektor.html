3. vektor a dan b di bidang dimensi 2,

lal = 3, lbl = 5, la+b| = 7

Misal vektor a = (x₁, y₁), vektor b = (x₂, y₂), maka,

Dari besar vektor a diperoleh,

lal = 3

√(x₁²+y₁²) = 3

x₁²+y₁² = 9 {persamaan i}

Dari besar vektor b diperoleh,

√(x₂²+y₂²) = 5

x₂²+y₂² = 25 {persamaan ii}

Untuk a+b diperoleh,

a+b = (x₁, y₁) +(x₂, y₂) = ((x₁+x₂), (y₁+y₂))

Dari besar vektor a+b diperoleh,

√((x₁+x₂)²+(y₁+y₂)²) = 7

(x₁+x₂)²+(y₁+y₂)² = 49

x₁² + 2x₁x₂ + x₂² + y₁² + 2y₁y₂ + y₂² = 49

x₁² + y₁² + 2x₁x₂ + x₂² + y₂² + 2y₁y₂ = 49 {persamaan iii}

Kemudian persamaan i dan ii disubstitusikan ke persamaan iii,

9 + 2x₁x₂ + 25 + 2y₁y₂ = 49

2x₁x₂ + 2y₁y₂ = 15 {persamaan iv}

Untuk menentukan la-bl, terlebih dahulu dicari a-b sbb,

a-b = (x₁, y₁) - (x₂, y₂) = ((x₁-x₂), (y₁-y₂))

Untuk besar vektor a-b diperoleh,

|a-b| = √((x₁-x₂)²+(y₁-y₂)²)

|a-b| = √(x₁² - 2x₁x₂ + x₂² + y₁² - 2y₁y₂ + y₂²)

|a-b| = √(x₁² + y₁² + x₂² + y₂²- 2x₁x₂ - 2y₁y₂)

|a-b| = √(x₁² + y₁² + x₂² + y₂²- (2x₁x₂ + 2y₁y₂)) {persamaan v}

Kemudian persamaan i, ii, dan iv disubstitusikan ke persamaan v untuk memperoleh |a-b| sbb,

|a-b| = √(x₁² + y₁² + x₂² + y₂²- (2x₁x₂ + 2y₁y₂))

|a-b| = √(9 + 25 - 15)

|a-b| = √19

15. 1.Diketahui titik k (3,7) titik L (8,2) carilah vektor KL?2.Bagaimana cara menentukan panjang vektor?3.Bagaimana menghitung panjang vektor pada titik k (3,7) L (8,2)?4.Hitunglah vektor satuan dari titik k (3,7) titik L (8,2)?5.Bagaimana cara menentukan vektor satuan?​


1. [tex]K(3,7) \:dan \:L(8,2) [/tex]

[tex]\begin{align}\vec{OK}&=\vec{k}=\binom{3}{7}\\ \vec{OL}&=\vec{l}=\binom{8}{2} \end{align} [/tex]

Maka,

[tex]\begin{align}\vec{KL}&=\vec{OL}-\vec{OK}\\ &=\binom{8}{2}-\binom{3}{7} \\ &=\binom{5}{-5} \end{align} [/tex]

2. Misal, [tex]\vec{a}= a_x\vec{i }+a_y\vec{j}+a_z\vec{k} [/tex], dengan [tex]a_x [/tex],[tex]a_y [/tex], dan [tex] a_z [/tex] adalah komponen [tex]\vec{a} [/tex] terhadap sumbu-[tex]x[/tex], sumbu-[tex]y[/tex], dan sumbu-[tex]z[/tex] Maka:

[tex]|\vec{a}|=\sqrt{a_x^2+a_y^2+a_z^2} [/tex]

3. Panjang vektor [tex] \vec{AB} [/tex] sama dengan panjang ruas garis [tex]AB [/tex], maka:

[tex]\begin{align}AB=|\vec{AB}|&=\sqrt{(x_2-x_1)^2+(y_2-y_1)^2} \\ &=\sqrt{(8-3)^2+(2-7)^2} \\ &=\sqrt{5^2+(-5)^2} \\ &=\sqrt{25+25} \\&=\sqrt{50} \\&=5\sqrt{2}\end{align} [/tex]

4. [tex]\vec{KL}=5\vec{i }-5\vec{j} [/tex]

[tex]\begin{align}\hat{KL}&= \frac{\vec{KL}}{|\vec{KL}|}\\&= \frac{\binom{5}{-5}}{5\sqrt{2}}\\ &=\frac{1}{5\sqrt{2}}\binom{5}{-5} \\&=\binom{\sqrt{2}}{-\sqrt{2}}\end{align}[/tex]

5. Misal, ada vektor [tex]\vec{a} [/tex]. Maka vektor satuan [tex]\vec{a} [/tex] ([tex]\hat{a} [/tex]) dapat dirumuskan:

[tex]\begin{align}\hat{a}= \frac{\vec{a}}{|\vec{a}|} \end{align}[/tex]
