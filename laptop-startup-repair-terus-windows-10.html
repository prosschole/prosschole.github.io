pembahasan :
mengapa laptop terkena startup repair? ini terjadi karena ada masalah pada laptopmu. startup repair terjadi ketika ada kerusakan / masalah pada file sistem Windows. bisa saja itu termodifikasi, terhapus, ataupun korup. startup repair juga berfungsi untuk mengambil tindakan ketika komputermu mengalami korupnya file dengan mengganti file yang lama menjadi file yang baru tanpa harus ada instal ulang :D... ataupun karena komputer kena virus, minta dicekkan RAMnya.

startup repair dilakukan dengan :
1. menggunakan CD (jika tidak dapat dilakukan secara otomatis)
2. secara otomatis (jika diketahui file yang ada sudah korup)

jika kamu sudah melakukan startup repair tapi ga ampuh juga, bawa ke layanan pengguna laptop atau lakukan instal ulang. kalau instal ulang ga bisa juga, tanyakan kepada teknisi komputer untuk mengambil tindakan penggantian hardware

kesimpulan :
startup repair adalah perangkat yang berfungsi untuk mendiagnosis masalah yang terjadi pada siatem operasi Windows dan memperbaikinya secara otomatis

kode soal : 11

kode kategorisasi : 12.11


startup repair merupakan fitur yang berfungsi untuk memperbaiki suatu kesalahan yang terjadi pada sistem operasi windows. fitur ini akan berjalan secara otomatis jika ada kerusakan dengan file sistem windows dan memperbaikinya. startup repair memiliki penyebab seperti kerusakan file sistem windows, kesalahan registri, dll tergantung kepada permasalahan.


jika sering terjadi startup repair, kamu dapat :

1. menunggunya hingga selesai karena jika tidak, kamu tidak akan mendapat kode errornya. kode ini berfungsi untuk menampilkan kesalahan yang sedang terjadi dan pemecahannya

2. jika sudah coba namun tak berhasil, kerusakannya bisa saja melalui hardware seperti kejadian yang saya alami yaitu Hdd sudah bad sector dan masuk ke startup repair secara terus menerus. jika kamu menemukan gejala yang tidak biasa sebelum startup repair, tolong beritahu saya di kolom komentar.

3. jika memungkinkan lakukan instal ulang agar sistem operasinya kembali seperti semula / agar fresh. tapi jika instal ulang juga masih kena, kerusakan ada di hardware dan perlu diganti

Cara mengatasi Startup Repair yang Gagal

Note :

Kesabaran yang tinggi, sangat diperlukan untuk mengatasi problem ini.

Semua link artikel yang disarankan, bisa dibaca di blog ini.

Jika Anda menggunakan Windows 8, silahkan baca artikel Menggunakan Automatic Repair Windows 8.

Atau artikel Cara Memperbaiki Automatic Repair Windows 8 Gagal, atau Win 8 Gagal Booting.  

Kita mulai dengan cara yang paling mudah dahulu, dibawah ini.

1. Me-restart Komputer.

Kadang suatu problem bisa muncul hanya secara "kebetulan (temporary spike)".

Jika seperti ini, (biasanya) hanya dengan mematikan PC, menunggu beberapa saat, dan lalu meng-ON kan kembali, maka semuanya kembali normal. Jika belum melakukannya, cobalah hal ini.

Jika tidak membawa hasil, lanjutkan dibawah ini.

 

2. Jalankan Last Known Good Configuration - (LKGC).

Restart PC, masuk ke Advanced Startup Option Win7-XP (Advance Option Menu).

Untuk pengguna Windows 8 baca Startup Settings Winddows 8 untuk memilih opsi repair yang tersedia.

Pilih menu Last Known Good Configuration. Silahkan baca artikel tsb.

Dengan booting pada Last known good configuration, sistem akan mencoba loading menggunakan catatan konfigurasi terakhir (system log) yang tidak bermasalah. Jika berhasil, maka semuanya akan beres, tetapi jika gagal, lanjutkan dengan langkah di bawah.  

3. Jalankan System Restore.

Jika langkah diatas belum berhasil, dan sebelum waktu Anda "habis" untuk membaca uraian dibawah, maka silahkan coba dahulu artikel ini : Mengatasi Startup Repair Gagal menggunakan System Restore (Recovery Environment). Siapa tahu cocok dengan kasus yang sedang Anda hadapi.

Jika Anda belum mengenal sifat System Restore, sebaiknya membaca artikel Mengenal System Restore Win 7-XP, atau Mengenal System Restore Windows 8.

Tentang cara penggunaan System Restore,silahkan membaca artikel Menggunakan System Restore Win7-XP.

Jika Anda pengguna Windows 8, baca artikel Menggunakan System Restore Windows 8.  

4. Jalankan SFC.

Menjalankan System File Checker (SFC) saat booting, juga merupakan cara yang patut untuk dicoba, untuk memperbaiki file-sistem yang rusak. Apa dan bagaimana tentang SFC, silahkan membaca artikel tsb.

5. Pasang Hard drive di PC (desktop) lain.

Jika tersedia pc-desktop lain (OS-nya sama), lakukan step berikut :

Lepas harddisk-yang bermasalah, pasang sebagai hdd ke-2 di pc-lain tsb.

On-kan pc seperti biasa, jika muncul prompt untuk tekan (biasanya) F1, tekan F1. Semoga pc bisa booting normal dan loading windows sampai selesai.

Mungkin proses booting agak sedikit lebih lama, karena sedang mendeteksi hdd yang bermasalah tsb.

Jika dirasa terlalu lama (15 menit lebih), hampir dipastikan terjadi kerusakan fisik pada hdd tsb.

Dalam hal ini, mungkin harus lakukan format hdd dan clean instal Windows, atau pertimbangkan ganti hdd baru.

Buka Explorer, pada hdd yang bermasalah cari partisi yang berisi Windows . Disini pasti bukan C:\.

Klik-kanan partisi tsb - Properties - tab Tools - klik Check Now... - centang Automatically fix file system errors - klik Start.

Tunggu sampai selesai, proses ini butuh waktu cukup lama; perhatikan saja LED-Hdd (merah), jika terkadang berkedip itu tanda proses berjalan normal. Jika diminta restart ikuti saja. Setelah semua selesai, matikan pc, lepas hdd ke-2, pasang kembali ke pc/laptop anda, lalu On-kan. Berdoalah agar semua berjalan normal. Jika cara ini gagal, silahkan lanjutkan ke step berikut di bawah ini (dengan sabar :).

18. Cara membuka file startup di windows 8

21. apakah fungsu dari "startup repair" berikut ini


Startup Repair adalah sebuah system recovery tool buatan Microsoft dalam Windows 7, untuk memperbaiki beberapa masalah penyebab kegagalan loading (gagal booting) Windows.
Startup Repair (diharapkan) bisa memperbaiki :
masalah sekitar File-sistem yg rusak atau hilang.
Startup Repair tidak bisa memperbaiki :
masalah sekitar hardware dan instalasinya, dan
masalah personal data yg hilang.
Meski demikian tool ini bisa menjadi pilihan pertama jika menghadapi problem gagal booting Windows. Terutama bagi user yg belum paham seluk-beluk file-sistem Windows.

1. Ram
