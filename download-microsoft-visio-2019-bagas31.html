18. perbedaan power designer, visual paradigm dan microsoft visio


JAWABAN DAN PENJELASAN

Power Designer adalah alat pemodelan perusahaan kolaboratif yang diproduksi oleh Sybase dan dapat berjalan di bawah Microsoft Windows sebagai aplikasi asli atau dalam lingkungan Eclipse melalui sebuah plugin.

Visual Paradigm (Pradigma Visual) adalah alat KASUS UML yang menyediakan pembuatan laporan dan kemampuan rekayasa kode termasuk pembuatan kode.

30. cara instal microsoft office 2003 dari download an


Misalkan hasil download an adalah file bertipe .iso:
- gunakan program winrar untuk membuka file tersebut
- dobel klik file bernama setup.exe untuk memulai proses instal
- akan tampil jendela petunjuk instalasi (klik terus tombol next)
- bila ada instruksi mengisikan serial number, silakan isikan serial numbernya
- tunggu prosesnya hingga berjalan sampai selesai
- bila sudah selesai, klik tombol Finish
