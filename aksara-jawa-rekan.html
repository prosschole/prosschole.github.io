Pembahasan

Aksara rekan yaiku aksara jawa kang digunakake kanggo mbiyantu pemenuhan kebutuhan ing panulisan lan ukara ukara sing asale saka basa liya.

Aksara rekan cacahe ana 5 :

Kha = ꦏ꦳Da = ꦢ꦳Ga = ꦒ꦳Ja = ꦗ꦳Pa = ꦥ꦳

Jlentrehne aksara jawa

1. Khalid = ꦏ꦳ꦭꦶꦢ꧀

Khalid = aksara rekan Kha, aksara La + wulu, aksara Da + pangkon.

2. Dzat murni = ꦢ꦳ꦠ꧀ꦩꦸꦂꦤꦶ

dzat = aksara rekan Dza, aksara Ta (sigeg)murni = pasangan aksara Ma + suku + layar, aksara Na + wulu.

3. Hukum ghibah = ꦲꦸꦏꦸꦩ꧀ꦒ꦳ꦶꦧꦃ

hukum = aksara Ha + suku, aksara Ka + suku, aksara Ma (sigeg)ghibah = pasangan aksara rekan Gha, + wulu, aksara Ba + wignyan.

4. Zakat fitrah = ꦗ꦳ꦏꦠ꧀ꦥ꦳ꦶꦠꦿꦃ

zakat = aksara rekan Za, aksara Ka, aksara Ta (sigeg)fitrah = pasang aksara rekan Fa + wulu, aksara Ta + cakra + wignyan.

5. Fakir miskin = ꦥ꦳ꦏꦶꦂꦩꦶꦱ꧀ꦏꦶꦤ꧀

fakir = aksara rekan Fa, aksara Ka + wulu + layarmiskin = aksara Ma + wulu, aksara Sa (sigeg), aksara Ka + wulu, aksara Na + pangkon.

1. Sedane Khusnul khotimah

ꦱꦺꦢꦤꦺꦏ꦳ꦸꦱ꧀ꦤꦸꦭ꧀ꦏ꦳ꦺꦴꦠꦶꦩꦃ

2. Dzikir kudu dilakoni

ꦩꦸꦱ꧀ꦭꦶꦩ꧀ꦲꦶꦏꦸꦏꦸꦝꦸꦗ꦳ꦏꦠ꧀

ꦒ꦳ꦺꦴꦥ꦳ꦸꦂꦱꦼꦏꦺꦴꦭꦃꦲꦶꦁꦏꦺꦤꦺ
