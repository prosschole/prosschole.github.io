14. Untuk melakukan koneksi antara program VB dengan sebuah database yang dibuat di MYSQL diperlukan...


MySql ODBC Connector adalah cara agar database pada MySQL bisa terhubung dengan program yang dibuat menggunakan VB. Untuk menggunakan connector tersebut, kita bisa mengaktifkannya pada menu Administrative Tools. Apabila database MySQL kita tidak dihubungkan menggunakan connector tersebut maka database MySQL kita tidak akan terbaca.

Connector MySQL ini merupakan software yang cukup penting, dengan adanya konektor ini, database MySQL kita dapat terhubung dengan aplikasi yang sedang kita buat juga,

Pembahasan

MySQL merupakan salah satu dari sekian banyak database yang sering digunakan. Untuk menginstall MySQL, kita bisa menginstallnya sendiri atau menginstall software yang terdapat MySQL didalamnya seperti aplikasi XAMPP. Selain dari MySQL, beberapa database yang sering digunakan adalah

SQL ServerOracle DatabaseMongoDBMariaDBSAP HanaFirebirdPelajari lebih lanjut

Demikian pembahasan mengenai program VB yang menggunakan database MySQL, agar MySQL dapat terhubung dengan program VB biasanya menggunakan connector. Untuk mempelajari lebih lanjut mengenai pemrograman komputer dapat dipelajari dengan link di bawah ini

26. cara mengurutkan nomor file handle pada microsoft excel 2007 secara cepat dan otomatis


Dengan AutoFill
1. Masukkan angka yang diinginkan pada lembar Excel, misalnya angka 1 di sel B2.
2. Lalu klik titik tebal yang ada disebelah kanan bawah sel yang terseleksi (Fill Handle) dan drag ke bawah sesuai keinginan.
3. Selanjutnya klik kotak kecil yang muncul disamping kanannya (Auto Fill Options).
4. Selanjutnya pada kotak Auto Fill Options pilih perintah Fill Series.

Cara membuat nomor urut otomatis di Ms Excel

Ketikkan angka 1 pada sel A1 dan angka 2 pada sel A2.

Blok sel A1 dan A2.

Arahkan mouse pada pojok kanan bawah pada sel A2 sehingga pointer berubah menjadi tanda plus (+).

Klik kiri dan tahan menggunakan mouse kemudian drag atau seret kebawah sampai pad baris yang diinginkan.

Maka nomor urut akan tampil.
