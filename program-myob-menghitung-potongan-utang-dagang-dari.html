Materi : Langkah Membuat Kartu Utang Dagang

Daftar rekanan (customer/supplier) dapat disusun dan dibuat di MYOB dalam menu 'Lists – Cards' selain itu dapat pula dari menu utama dengan membuka menu 'Command Centre – Card File – Card Lists'

Langkah-langkah membuat kartu utang dagang:
a. Klik 'New' di dalam tampilan jendela 'Card File'
b. Isi kotak dialog
• Card Type : pilih 'Supplier'
• Name : isikan nama supplier/vendor
• Card Id : isi kode supplier
• Address
• Phone
Untuk termin, kebijakan pengenaan pajak serta kode NPWP dapat diisikan dalam menu Buying Details.
c. Klik OK

21. Utang dagang sebesar Rp 3.175.000,00 diskon 3% Hitung besarnya potongan yang di berikan..​


Jawaban ada dibawah ini.

Pembahasan

Diketahui :

Utang = Rp. 3.175.000Diskon = 3%

Ditanya : Jumlah diskon?

Penjelasan

Jumlah diskon = Diskon × Utang

Jumlah diskon = 3% × 3.175.000

Jumlah diskon = 3/100 × 3.175.000

Jumlah diskon = 3 × 31.750

Jumlah diskon = Rp. 95.250,-

25. Langkah langkah yang tepat dalam melakukan pencatat pelunasan utang dalam program myob adalah


Langkah dalam melakukan pencatat pelunasan utang dalam program MYOB:

Dari Commaned Center klik Sales, klik Receive paymentsPilih Deposit to Account: akun penampung pembayaran piutangPada costumer pilih nama piutang, pada amount received tulis nominal piutang, pada memo tulisan pembayaran nama piutang, pada transaksi tanggal, isikan cicilan piutang pada kolom discount, lalu klik Tab, klik pada total applied, pastikan tertulis total piutang, lalu klik TabKlik Record.Muncul konfirmasi untuk print, pilih print letter jika tidak ingin langsung di print.

Pembahasan

MYOB (Mind Your Own Business) adalah aplikasi komputer akuntansi yang digunakan untuk mengelola dan mengolah informasi akuntansi. Bisa dikatakan bahwa MYOB merupakan aplikasi akuntansi yang dapat digunakan oleh berbagai usaha kecil maupun besar.

Fungsi aplikasi MYOB:

Sebagai alat untuk melakukan tugas akuntansiMengurangi waktu penagihanMenarik lebih banyak perhatian ke perusahaan kita sebagai penggunaMembantu proses jual beli dengan melacak piutang, pembayaran, pajak, kutipan dan faktur hanya dengan sekali klik dari tombol yang tersedia.Mengelola barang yang dijual menjadi lebih mudah.

Dalam bidang akuntansi, aplikasi MYOB memang sangat berguna untuk pengelolaan keuangan, khususnya bagi pebisnis atau perusahaan. Aplikasi ini memudahkan pengelolaan dan pengolahan informasi akuntansi.

Namun, MYOB memiliki beberapa kelemahan diantaranya format yang tidak sesuai dengan format bahasa Indonesia. Karena itu, kita harus terlebih dahulu menyelesaikan proses untuk menggunakannya.

28. Bagaimana cara masuk pada program MYOB?

Berikut langkah-langkah untuk membuka program Myob Accounting 18 melalui sistem operasi windows 7

1. Klik tombol Start pada Windows Taskbar

2. Pilih All Program, drag scrolbar ke bawah lalu klik pada folder Myob Acconting v18 lalu pilih Myob Accounting v 18

3. Jendela welcome to MYOB Accounting ditampilkan. Terdapat lima tombol untuk mengakses program Myob, berikut fungsi dari kelimatombol tersebut :

Ø open, digunakan untuk membuka file yang telah dibuat

Ø Create, untuk membuat file atau data perusahaan yang baru

Ø Explore, untuk membuka file contoh yang telah disediakan oleh Myob

Ø What’s New, digunakan untuk melihat fitur-fitur baru pada Myob

Ø Exit, digunakan untuk mengakhiri atau menutup program Myob

4. Di dalam kotak dialog Open, pilih lokasi penyimpanan file misalkan di dalam Local Disk (C:)\Accounting 18

5. Pilih nama file (missal Mutiara Funiture.myob), lalu klik tombol open.

6. File Mutiara Furniture akan terbuka
