26. Cara mengatasi vcruntime140.dll not found windows 10 gimana ya?#Windows 10#64Bit​


Jawaban:

Instal Visual C++ 2015

menggunakan aplikasi yang bisa kamu temukan dengan mudah di internet, ini kalau kamu kesusahan dengan eror tersebut. Untuk mengunduh dan memasangnya, Anda bisa melakukan cara-cara berikut ini:

Unduh aplikasi visual c++ redistributable 2015 di laman resi Mirosoft, dan pastikan kamu mengunduhnya sesuai dengan jenis sistem operasi dengan versi yang sama dan sesuai dengan Windows yang dimiliki.

Instal sesuai dengan tata cara dan instruksi yang ditampilkan

jika sudah lakukan restart pada perangkat komputer yang kamu gunakan.

Penjelasan:

semoga membantu

Jawaban:

Solusi 1 : Download dan copy file VCRUNTIME140.DLL di direktori Windows

Download File ( cari digoogle " VCRUNTIME140.DLL " )

Jika sudah Ekstrak file VCRUNTIME140.DLL

Copy File VCRUNTIME140.DLL di direktori Windows :

Jika Anda menggunakan Windows 64 Bit  C:\Windows\SysWOW64\

Solusi 2 : Install Ulang Microsoft Visual C++ Redistributable Packages ( VC Redist )

Download Microsoft Visual C++ (cari digoogle " download vc_redist.x64 " )

( download yang 64 bit  )

Lanjutkan dengan proses instalasi

Restart Komputer Anda

Selesai

Di komputer kamu itu file vcruntime bisa saja tidak ada, atau kamu belum install aplikasi dari microsoft atau ada yang kurang atau bisa saja windows kamu tidak diupdate, jadi untuk mengatasinya download filenya ada disolusi 1  atau download software vc redist pada solusi 2.
