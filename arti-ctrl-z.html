6. Jelaskan apa fungsi masing-masing perintah Ctrl A Ctrl C Ctrl S Ctrl Z


Ctrl + A Untuk Memilih / Memblok Semua Yang Ada Di Lembar Kerja

Ctrl + C Untuk Mencopy

Ctrl + S Untuk Menyimpan / Save Dokumen

Ctrl + Z Untuk Membatalkan Perintah / Undo

Mata pelajaran : TI

Tema : Keyboard Shortcut

Kode soal : 11

Jelaskan fungsi masing-masing perintah CTRL + A, CTRL + C, CTRL + S dan CTRL + Z!

CTRL + A = Digunakan untuk mem-blok semua huruf.

CTRL + C = Digunakan untuk meng-copy huruf atau gambar.

CTRL + S = Digunakan untuk menyimpan file atau dokumen.

CTRL + Z = Digunakan untuk undo, mengembalikan apa yang sebelumnya sudah dituliskan.


7. Tuliskan perbedaan antara ctrl+z dan ctrl+y​


Jawaban:

CTRL + Z ((Undo) berfungsi untuk membatalkan perintah sebelumnya)

Ctrl + Y = fungsi shortcut redo/membalikan tindakan, ini kebalikannya dari undo.

Pertanyaan:

Tuliskan perbedaan antara Ctrl + Z dan Ctrl + Y

Jawaban dan penjelasan:

Ctrl + Z = fungsi shortcut undo/pembatalan tindakan kembali ke sebelumnya

Ctrl + Y = fungsi shortcut redo/membalikkan tindakan, ini kebalikannya undo

Jadi, perbedaan antara Ctrl + Z dan Ctrl + Y adalah undo dan redo
