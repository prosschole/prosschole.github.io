18. ragam atau varians data 3,2,5,3,4,6,4,5,4 sama dengan...


Ragam atau varians data 3,2,5,3,4,6,4,5,4 sama dengan 1,33

Pembahasan

Ragam atau Varians adalah perbandingan antara jumlah dari kuadrat selisih setiap data dan rata-ratanya dengan banyaknya data. Rata-rata adalah perbandingan antara jumlah data dengan banyaknya data. Jadi untuk menentukan ragam adalah kita cari rata-rata dari data tersebut, kemudian cari selisih setiap data dengan rata-ratanya lalu kita kuadratkan. Kemudian kita jumlahkan hasilnya baru dibagi dengan banyaknya data.

Rumus rata-rata = jumlah data / banyak data Rumus simpangan rata-rata = ∑[tex]\frac{(xi - x)}{n}[/tex] Rumus Ragam (Varians) = ∑[tex]\frac{(xi-x)^{2} }{n}[/tex] Rumus Simpangan Baku  Sb = √R

===========================

Analisis soal

Ragam atau varians data 3,2,5,3,4,6,4,5,4 sama dengan...

Jawab:

Langkah 1. Menghitung nilai rata - rata.

Rata - rata dari 3,2,5,3,4,6,4,5,4 adalah

Mean = [tex]\frac{3+2+5+3+4+6+4+5+4}{9}[/tex]

Mean = [tex]\frac{36}{9}[/tex]

Mean = 4

Langkah 2. Menghitung nilai ragam (varians).

Varians = ∑[tex]\frac{(xi - x)^{2} }{n}[/tex]

Varians = [tex]\frac{(3-4)^{2 + (2-4)^{2} + (5-4)^{2} + (3-4)^{2} + (4-4)^{2} + (6-4)^{2} + (4-4)^{2} + (4-5)^{2} + (4-4)^{2} } }{9}[/tex]

Varians = [tex]\frac{(-1)^{2} + (-2)^{2} + (1)^{2} + (-1)^{2} + (0)^{2} + (2)^{2} + (0)x^{2} + (1)^{2} + (0)^{2} }{9}[/tex]

Varians = [tex]\frac{1 + 4 + 1 + 1 + 0 + 4 + 0 + 1 + 0}{9}[/tex]

Varians = [tex]\frac{12}{9}[/tex]

Varians = 1,33

20. ragam (varians) dari data 4,3,5,6,4,7,8,7,9,8,6,5,6,5,7,6 adalah


Kategori Soal : Matematika - Statistika 
Kelas : XI (2 SMA)
Pembahasan :
Diketahui data 3, 4, 4, 5, 5, 5, 6, 6, 6, 6, 7, 7, 7, 8, 8, 9.
Banyaknya data adalah
n = 16
Rata-rata adalah
x = (3 + 4 + 4 + 5 + 5 + 5 + 6 + 6 + 6 + 6 + 7 + 7 + 7 + 8 + 8 + 9)/16
= 96/16
= 6
Ragam adalah
S² = 1/16[(3 - 6)² + (4 - 6)² + (4 - 6)² + (5 - 6)² + (5 - 6)² + (5 - 6)² + (6 - 6)² + (6 - 6)² + (6 - 6)² + (6 - 6)² + (7 - 6)² + (7 - 6)² + (7 - 6)² + (8 - 6)² + (8 - 6)² + (9 - 6)²]
= 1/16[(-3)² + (-2)² + (-2)² + (-1)² + (-1)² + (-1)² + 0² + 0² + 0² + 0² + 1² + 1² + 1² + 2² + 2² + 3²]
= 1/16[9 + 4 + 4 + 1 + 1 + 1 + 1 + 1 + 1 + 4 + 4 + 9]
= 1/16[40]
= 40/16
= 5/2
= 2,5

Simpangan baku adalah
S = √S² = √2,5
