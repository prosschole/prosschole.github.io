Pembahasan

S₂₀ = 610

U₅ = 14

Jawab:

Rumus jumlah n suku deret aritmatika adalah:

Sn = [tex]\frac{n}{2}[/tex] (2a + (n-1)b)

Sedangkan rumus suku ke- n deret aritmatika adalah:

Un = a + (n - 1)b

Maka:

Sn = [tex]\frac{n}{2}[/tex] (2a + (n-1)b)

S₂₀ = [tex]\frac{20}{2}[/tex] (2a + (20-1)b)

S₂₀ = 10(2a + 19b)

610 = 10(2a + 19b)

2a + 19b = [tex]\frac{610}{10}[/tex]

2a + 19b = 61......................1)

Un = a + (n - 1)b

U₅ = a + (5 - 1)b

a + 4b = 14.......................2)

eliminasi pers 1) dan 2)

2a + 19b = 61  |× 1|

a + 4b = 14     |× 2|

2a + 19b = 61

2a + 8b = 28     -

       11b = 33

         b = [tex]\frac{33}{11}[/tex]

         b = 3

Subtitusikan nilai b ke pers. 2)

a + 4b = 14

a + 4(3) = 14

a + 12 = 14

a = 14 - 12

a = 2

Maka diperoleh nilai a = 2, dan b = 3.

U₁₀ = a + (10 - 1)b

U₁₀ = a + 9b

U₁₀ = 2 + 9(3)

U₁₀ = 29

∴ Jadi suku ke- 10 deret aritmatika tersebut adalah 29.

216 = 10n - 4
