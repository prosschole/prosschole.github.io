13. Tolong bantu kak materi Limit trigonometri ​


lim x → 0 (x² - 36)(sin 3x)/(x³ + 5x² - 6x) = 18

Pendahuluan

Limit artinya batas, sedangkan dalam matematika Limit Fungsi merupakan salah satu sistem kalkulus untuk menentukan batas nilai variabel suatu fungsi mendekati suatu titik, baik didekati dari kiri maupun kanan.

⟨⟨ Sifat Limit Fungsi ⟩⟩[tex]\rm\lim\limits_{x\to a}k=k[/tex][tex]\rm\lim\limits_{x\to a}x=a[/tex][tex]\rm\lim\limits_{x\to a}\left(k\cdot f(x)\right)=k\cdot\lim\limits_{x\to a}f(x)[/tex][tex]\rm\lim\limits_{x\to a}\left(f(x)+g(x)\right)=\lim\limits_{x\to a}f(x)+ \lim\limits_{x\to a}g(x)[/tex][tex]\rm\lim\limits_{x\to a}\left(f(x)-g(x)\right)=\lim\limits_{x\to a}f(x)-\lim\limits_{x\to a}g(x)[/tex][tex]\rm\lim\limits_{x\to a}\left(f(x)\cdot g(x)\right)=\lim\limits_{x\to a}f(x)\cdot \lim\limits_{x\to a}g(x)[/tex][tex]\rm\lim\limits_{x\to a}\frac{f(x)}{g(x)}=\frac{\lim\limits_{x\to a}f(x)}{\lim\limits_{x\to a}g(x)}~,dengan~g(x)\neq 0[/tex][tex]\rm\lim\limits_{x\to a}\left(f(x)\right)^n=\left(\lim\limits_{x\to a}f(x)\right)^n~,dengan~n=bilangan~bulat[/tex][tex]\rm\lim\limits_{x\to a}\sqrt[n]{f(x)}=\sqrt[n]{\lim\limits_{x\to a}f(x)}~,dengan~f(x)\geq 0[/tex]

⟨⟨ Sifat Limit Fungsi Trigonometri ⟩⟩[tex]\rm\lim\limits_{x\to 0}\frac{\sin x}{x}=\lim\limits_{x\to 0}\frac{x}{\sin x}=\bf 1[/tex][tex]\rm\lim\limits_{x\to 0}\frac{\tan x}{x}=\lim\limits_{x\to 0}\frac{x}{\tan x}=\bf 1[/tex][tex]\rm\lim\limits_{x\to 0}\frac{\sin ax}{bx}=\lim\limits_{x\to 0}\frac{ax}{\sin bx}=\bf\frac{a}{b}[/tex][tex]\rm\lim\limits_{x\to 0}\frac{\tan ax}{bx}=\lim\limits_{x\to 0}\frac{ax}{\tan bx}=\bf\frac{a}{b}[/tex][tex]\rm\lim\limits_{x\to 0}\frac{\sin ax}{\sin bx}=\lim\limits_{x\to 0}\frac{\tan ax}{\tan bx}=\bf\frac{a}{b}[/tex][tex]\rm\lim\limits_{x\to 0}\frac{\sin ax}{\tan bx}=\lim\limits_{x\to 0}\frac{\tan ax}{\sin bx}=\bf\frac{a}{b}[/tex]

Pembahasan

[tex]\begin{aligned}\rm\lim\limits_{x\to 0}\frac{(x^2 -36)\sin 3x}{x^3 -5x^2 -6x}&=\rm\lim\limits_{x\to 0}\frac{(x^2 -6^2)\sin 3x}{x(x^2 -5x -6)}\\&=\rm\lim\limits_{x\to 0}\frac{\cancel{(x+6)}(x-6)\sin 3x}{x\cancel{(x+6)}(x-1)}\\&=\rm\lim\limits_{x\to 0}\frac{(x-6)\sin 3x}{x(x-1)}\\&=\rm\lim\limits_{x\to 0}\frac{x-6}{x-1}\cdot\lim\limits_{x\to 0}\frac{\sin 3x}{x}\\&=\frac{0-6}{0-1}\cdot 3\\&=\frac{-6}{-1}\cdot 3\\&=6\cdot 3\\&=\underline{\underline{\bf 18}}\end{aligned}[/tex]

■■■■■■■■■■■■■■■■■■■■■■■■■■■■

Kesimpulan

Nilai dari [tex]\rm\lim\limits_{x\to 0}\frac{(x^2 -36)\sin 3x}{x^3 -5x^2 -6x}=\bf 18[/tex]
