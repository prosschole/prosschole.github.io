8. gambarkan dan jelaskan fungsi ic 7432 sebagai gerbang logika or​


Penjelasan:

IC 7432

Gerbang ini sudah terkemas dalam IC tipe 7432. Sama dengan gerbang AND, gerbang OR hanya memiliki 2 buah input dan 1 output, sehingga dibutuhkan 2 gerbang untuk menjadikan 3 input dan 1 output. Operasi gerbang :

Pada output akan berharga 1 (indicator menyala) jika salah satu atau semua dari inputnya diberi masukan sebesar Vcc. Sebaliknya jika semua input diberi masukan dari ground atau terlepas, maka output akan berharga 0 (indicator tidak menyala. Dengan melihat tabel pada data percobaan, akan didapat persamaan pada output, yaitu :

Y = A + B + C

Komponen pembangun IC TTL(transistor-transistor logic) adalah sesuai dengan namanya IC ini berisi beberapa transistor yang digabungkan sehingga membentuk dua keadaan (ON/FF).

18. pengertian gerbang logika yang gerbang AND


Gerbang AND memerlukan 2 atau lebih Masukan (Input) untuk menghasilkan hanya 1 Keluaran (Output). Gerbang AND akan menghasilkan Keluaran (Output) Logika 1 jika semua masukan (Input) bernilai Logika 1 dan akan menghasilkan Keluaran (Output) Logika 0 jika salah satu dari masukan (Input) bernilai Logika 0. Simbol yang menandakan Operasi Gerbang Logika AND adalah tanda titik (“.”) atau tidak memakai tanda sama sekali. Contohnya : Z = X.Y atau Z = XY.

Penjelasan:

Perbedaan OR dan XOR hanya terjadi ketika input-inputnya sama-sama bernilai BENAR/TRUE/1.

B OR B = B

B XOR B = S

XOR (exclusive-OR) bersifat eksklusif, artinya XOR bernilai BENAR jika dan hanya jika SALAH SATU inputnya bernilai BENAR. Sedangkan gerbang logika OR akan menghasilkan output BENAR jika semua inputnya bernilai BENAR.

30. jelaskan pengertian gerbang logika xor


Gerbang logika xor adalah gerbang yang biasa digunakan untuk membuat suatu rangkaian atau menghitung operasi logika. Biasanya ini lebih dipelajari di Sistem Digital. Gerbang ini hanya bernilai TRUE ketika salah satu bernilai TRUE. Contoh :

0 0 = disini hasilnya false

1 0 = disini hasilnya true

0 1 = disini hasilnya true

1 1 = disini hasilnya false

Ingat :

1 = TRUE

0 = FALSE
