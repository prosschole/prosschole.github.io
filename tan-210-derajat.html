22. nilai dari tan 240 deajat -tan 210 derajat adalah


nilai dari tan 240 deajat -tan 210 derajat adalah ....

pembahasan :

rumus trigonometri lanjutan untuk tan
tan ( 90° - θ ) = cot θ
tan ( 90° + θ ) = - cot θ
tan ( 180° - θ ) = - tan θ
tan ( 180° + θ ) = tan θ
tan ( 270° - θ ) = cot θ
tan ( 270° + θ ) = - cot θ
tan ( 360 - θ ) = - tan θ
tan ( 360 + θ ) = tan θ
tan ( -θ ) = - tan θ
tan ( n.360 - θ ) = - tan θ
tan ( n.360 + θ ) = tan θ

tan 240 berada di kuadran 3
tan berada di kuadran 3 nilainya positif
tan 240 = tan (180 + 60)
              = tan 60
              = √3

tan 210 berada di kuadran 3
tan berada di kuadran 3 nilainya positif
tan 210 = tan (180 + 30)
             = tan 30
             = 1/3 √3

sehingga tan 240 - tan 210
= √3 - 1/3 √3
= 3/3 √3 - 1/3 √3
= 2/3 √3

jadi hasil dari tan 240 - tan 210 = 2/3 √3
