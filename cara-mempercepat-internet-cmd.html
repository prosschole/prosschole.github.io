11. Bagaimana cara mempercepat internet dengan hack dns (10x lebih cepat) ?


Jawaban:

Cara mempercepat koneksi internet By : @Mr.exe

Buka command promtCek IP router dengan melakukan perintah ipconfig /all (Biasa ip nya 192.168.0.1 atau 192.168.1.1)Lakukan perintah berikut pada cmd tersebut

ping -l 55000 192.168.1.1 -t

Penjelasan:

55000 adalah paket pengiriman yang akan membuat kecepatan internet menjadi lebih cepat dari pada sebelumnya maksimal "65500"192.168.1.1 adalah alamat IP router

Catatan

Resiko melakukan ping pada router ini akan membuat router menjadi panas yang membuat router memaksa kinerja dari pada standart nya.

cara menggunakan CMD atau Command Prompt yaitu dengan cara

Cara pertama penggunaan CMD yaitu: untuk pengguna Windows 10, anda dapat membuka CMD melalui pencarian di Start Menu, kemudian ketikkan CMD.

Cara kedua adalah: Anda dapat menekan tombol Windows+R pada Keyboard. Selanjutnya ketikkan CMD pada kolom search tersebut dan tekan Enter.

Cara ketiga : Atau anda dapat membuka CMD melalui Address Bare di File Explorer dengan mengetikkan kata yg sama seperti sebelumnya yaitu "CMD".

15. jelaskan bagaimana cara mempercepat akses internet


1. Mengubah Setting Bandwidth Windows
Secara default, bandwidth yang diberikan windows sebesar 20%, jadi untuk mengatasi ini atau menjadikannya 100% adalah dengan menyingkirkan batasan tersebut dengan cara:

Klik Start pada windowsPilih Run dan ketikkan gpedit.msc pada kotak kosong yang disediakan lalu klik "OK"Pilih Computer Configuration - Administrative Templates - Network - QoS Packet SchedulerSelanjutnya doble klik Limit Reservable BandwidthPilih menu Enable dan isi bandwidth limit dengan nilai 0.Klik apply dan restart komputer
2. Mengganti DNS
Selain ubah setting bandwidth cara mempercepat internet yang umum digunakan oleh pemula adalah mengganti DNS standar dengan open DNS. Walupun peningkatannya tidak terlalu signifikan tapi lumayanlah bisa naik sekitar 20%. Caranya:

Klik Start pada windowsPilih Control Panel - Network and Internet - Network and Sharing Center - Local Area ConnectionPilih Properties kemudian Pilih Internet Protocol Version 4 - propertiesUbah DNS menjadi Preferred DNS Server: 208.67.222.222 dan Alternate DNS Server: 208.67.220.220Klik "OK"Selain milik Open DNS, sobat juga bisa menggunakan DNS milik Google

Preferred DNS Server: 8.8.8.8Alternate DNS Server: 8.8.4.4
3. Copy Paste kode dibawah ini kedalam notepad dan simpan dengan nama "fastloading.reg" kemudian jalankan file tersebut.
REGEDIT4
[HKEY_LOCAL_MACHINESYSTEMCurrentControlSetServicesT cpipParameters]
“SackOpts”=dword:00000001
“TcpWindowSize”=dword:0005ae4c
“Tcp1323Opts”=dword:00000003
“DefaultTTL”=dword:00000040
“EnablePMTUBHDetect”=dword:00000000
“EnablePMTUDiscovery”=dword:00000001
“GlobalMaxTcpWindowSize”=dword:0005ae4c
4. Mempercepat Koneksi Internet bagi pengguna koneksi LAN
Berikut ini cara untuk mempercepat koneksi LAN :

Buka registry editor (start >> run >> ketik regedit)
Masuk ke HKEY_LOCAL_MACHINE\Software\Microsoft\Windows\Curr entVersion\Explorer\RemoteComputer\NameSpace dan DELETE key {D6277990-4C6A-11CF-8D87-00AA0060F5BF)
Tutup registry editor dan restart windows.

5. Mempercepat browsing dengan DNS cache
Buka notepad dan copy paste kode di bawah ini :
[HKEY_LOCAL_MACHINESYSTEMCurrentControlSetServic es|DnscacheParameters]
“CacheHashTableBucketSize”=dword:00000001
“CacheHashTableSize”=dword:00000180

“MaxSOACacheEntryTtILimit”=dword:0000012d

Double click file ini di windows explorer, tekan “yes”.
