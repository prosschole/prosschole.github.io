1. suatu polinon jika di bagi oleh (x-2) dan sisanya 8. jika di bagi (x+4) bersisa 4. tentukan pembagian jika polinom tersebut dibagi (x^2 + 2x - 8)saya mentok nih..ada yg bisa bantu ?


f(x) dibagi ( x - 2 ) sisa 8 ---> f(2) = 8
f(x) dibagi ( x + 4 ) sisa 4 ---> f(-4) = 4
f(x) dibagi ( x^2 + 2x - 8 ) ---> sisa = s(x) = ax + b

Pembagi ---> x² + 2x - 8 = ( x - 2 ) ( x + 4 )
Pembuat nol pembagi ---> 2 dan -4

f(2) = a(2) + b = 8 ---> 2a + b = 8 ... (1)
f(-4) = a(-4) + b = 4 ---> -4a + b = 4 ... (2)

Pers (1) dan (2) :
2a + b = 8
-4a + b = 4
--------------- (-)
6a = 4 ---> a = 2/3

Pers (1) :
2a + b = 8 ---> a = 2/3
2( 2/3 ) + b = 8 
b = 8 - 4/3 = 20/3

Jadi, sisa pembagian s(x) = ax + b = 2/3 x + 20/3

kain polos = x

kain motif = y

3y = 75.000

y = 75.000/3

didapat,

1m kain polos (x) = 40.000

1m kain motif (y) = 25.000

18. Abi membeli kain untuk membuat seragam, abi membeli 1 3/4 meter kain untuk baju, 3/4 meter kaim untuk celana dan 4/7 untuk kain sorban berapa meyer kain yang abi beli?


Untuk menghitung jumlah meter kain yang dibeli Abi, kita perlu menghitung jumlah kain untuk masing-masing jenis pakaian terlebih dahulu. Jumlah kain untuk baju adalah 1 3/4 meter, jumlah kain untuk celana adalah 3/4 meter, dan jumlah kain untuk sorban adalah 4/7 meter.

Untuk menghitung jumlah keseluruhan kain yang dibeli Abi, kita perlu menambahkan ketiga jumlah kain tersebut. Karena ketiga jumlah kain tersebut dinyatakan dalam bentuk campuran, maka kita perlu menyederhanakan bentuk campurannya terlebih dahulu.

Untuk menyederhanakan jumlah kain untuk baju, kita perlu mengkonversikan bentuk campurannya menjadi bentuk pecahan sederhana. Untuk melakukan itu, kita perlu menemukan faktor persekutuan terbesar (FPB) dari 3 dan 4, yaitu 1. Dengan demikian, jumlah kain untuk baju dapat dituliskan sebagai 1 1/4 meter.

Untuk menyederhanakan jumlah kain untuk celana, kita perlu menemukan faktor persekutuan terbesar (FPB) dari 3 dan 4, yaitu 1. Dengan demikian, jumlah kain untuk celana dapat dituliskan sebagai 3/4 meter.

Untuk menyederhanakan jumlah kain untuk sorban, kita perlu menemukan faktor persekutuan terbesar (FPB) dari 4 dan 7, yaitu 1. Dengan demikian, jumlah kain untuk sorban dapat dituliskan sebagai 4/7 meter.

Setelah menyederhanakan bentuk campuran masing-masing jumlah kain, kita dapat menambahkan ketiga jumlah kain tersebut untuk mengetahui jumlah keseluruhan kain yang dibeli Abi. Jumlah keseluruhan kain yang dibeli Abi adalah 1 1/4 + 3/4 + 4/7 meter = (5/4 + 3/4 + 4/7) meter = (12/7 + 12/7 + 8/7) meter = (32/7) meter.

Dengan demikian, jumlah keseluruhan kain yang dibeli Abi adalah 32/7 meter. Jadi, jawaban dari pertanyaanmu adalah 32/7 meter.

23. Bola boling menggelinding dilantai memiliki energi kinetik 10 joule. Jika massa bola bolinh 5 kg, kecepatan gerak bola tersebut sebesar...


Diket
Energi kinetik (Ek) = 10 joule
massa (m) = 5 kg

ditanya kecepatan (v). maka

Ek=1/2×m×v²
10=1/2×5×v²
10=5/2×v²
10×2/5=v²
20/5=v²
4=v²
√4=v
2=v

jadi kecepatan bola tsb adalah 2 m/s²
