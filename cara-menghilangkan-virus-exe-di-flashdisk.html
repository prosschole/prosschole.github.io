12. Bagaimana cara mengatasi file exe yang tidak ada di visual studio ?


Penjelasan:

:

1. Buka project yang sudah dibuat

2. Pilih menu Project – (klik nama project) Properties

3. Setelah jendela properties dari project Anda terbuka kemudian pilih menu publish dari jendela tersebut

4. tentukan dimana lokasi file hasil publish disimpan pada “Publish Location”

5. Pilih Publish Now (tunggu hingga Visual Studio selesai mempublish project Anda menjadi file setup.exe.)

6. Cara bukanya di (nama project anda)/bin/debug/app.publish

7. Setelah mempunyai file setup dari aplikasi yang di buat, selanjutnya install di komputer yang diinginkan dengan mengklik file setup yang ada.

20. Apa perbedaan virus dan worm? Apa yg menyebabkan seluruh data di suatu flashdisk menjadi ter-hidden, dan bagaimana cara menormalkannya kembali?


Worm merupakan virus komputer yang menginfeksi suatu sistem dengan cara menggandakan dirinya secara otomatis, bahkan dapat menjangkit sebuah system operasi.

Biasanya komputer yang sudah terinfeksi berjalan dengan perintah otomatis yang berformat ( .inf, .tiff, .exe)

untuk cara menghilangkannya mempunyai banyak variasi, jika keseluruhan system terjangkit maka cara efektif adalah menggunakan antivirus avira (standlone edition) pilih free saja, lalu install melalui flashdisk / perangkat penyimpanan lain, dan pastikan scan secara keseluruhan.

untuk alternatif, jika file system yang terinfeksi tidak bisa dijalankan, maka cara satu - satunya adalah menginstall ulang system operasi.
