21. Ketua rt akan memilih 3 orang dari 20 warganya untuk dikirim pelatihan. banyaknya cara ketua rt tersebut untuk memilih adalah​


Jawab:

Dengan teori kombinatorial didapat [tex]nCp = \frac{n!}{(n-p!).p!}[/tex]  

Penjelasan dengan langkah-langkah:

[tex]30C2 = \frac{30!}{(28!)*2!} = \frac{28!.29.30}{28!.2} = \frac{29.30}{2} = 29.15 = 435[/tex] Cara

Jawab:

1140

Penjelasan dengan langkah-langkah:

diketahui :

ada 20 orang ---> akan dipilih 3 orang

ditanya : banyak cara memilih 3 orang tersebut?

jawab :

jika yang ditanya adalah banyak cara memilih, maka disini kita gunakan kombinasi

rumus kombinasi

nCr = [tex]\frac{n!}{(n-r)! .r!}[/tex]

n = 20

r = 3

20C3 = [tex]\frac{20!}{(20-3)!.3!}[/tex] = [tex]\frac{20!}{17!. 3!}[/tex] = 1140

maka banyak cara ketua rt untuk memilih 3 orang tersebut adalah 1140 cara
