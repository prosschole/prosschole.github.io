6. Sintak yang benar untuk menambahkan warna background atau latar belakang dibawah ini adalah... a. <body =’background-:color:red’>b. <background>red</background>c. <body style=’background-color:red’>d. <body style=background-red>​


Sintaks yang benar untuk menambahkan warna background atau latar belakang adalah:

<body style=’background-color:red’>

Jadi pilihan yang tepat adalah C.

Pembahasan

Di dalam HTML sintaks yang benar untuk melakukan perubahan warna background adalah:

<body style=’background-color:red’>

Sintaks tersebut adalah sintaks yang sesuai dengan panduan HTML yaitu:

style="background-color:" ke elemen <body>.

Pilihan lainnya salah karena:

a. <body =’background-:color:red’> tidak sesuai dengan sintaks HTML.

b. <background>red</background> tidak sesuai dengan sintaks HTML.

d. <body style=background-red>​ tidak sesuai dengan sintaks HTML.

Sintaks HTML atau sintaks dalam bahasa pemrograman apa pun adalah bentuk perintah baku yang harus diikuti agar baris sintaks tersebut dapat dikenali dan dieksekusi oleh komputer. Bila salah walau satu karakter saja maka program tidak bisa mendeteksi baris perintah tersebut dan membuat baris kode yang kita buat tidak bisa dijalankan.

Baris kode di atas adalah salah satu contoh baris kode dalam HTML-CSS yang disebut inline CSS. Kita dapat menerapkan pedoman style ke komponen HTML tertentu dengan menggunakan inline CSS. Alih-alih menggunakan CSS eksternal, menyisipkan CSS melibatkan penyertaan CSS dalam file HTML. Meskipun penggunaannya dibatasi, inline CSS berguna untuk menentukan karakteristik unik karena memungkinkan penerapan gaya khusus ke satu elemen HTML.

13. Diketahui vector a = 4i – 2j + 2k dan vector b = 2i – 6j + 4k. Proyeksi vector orthogonal vector a pada vector b adalah .....


vektor a = 4i – 2j + 2k

vektor b = 2i – 6j + 4k

a . b = (4).(2) + (-2).(-6) + (2).(4) = 8 + 12 + 8 = 28

|b|² = 2² + [-6]² + 4² = 4 + 36 + 16 = 56

Proyeksi vector orthogonal vector a pada vector b :

= ( [a . b] / |b|² ) . vektor b

= (28 / 56) . vektor b

= ½ . vektor b

= ½ . (2i – 6j + 4k)

= i - 3j + 2k
