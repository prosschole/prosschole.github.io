Gunakan konsep perulangan (for/while).

Terlebih dahulu kuasai konsep dasar sebelum mempelajari bagaimana membuat program dengan konsep dasar perulangan (for/while). Materi dasar tersebut yaitu :

1) Cara input dan output program (scanf & printf)

2) Mengenal apa itu variabel

3) Mengetahui bagaimana menentukan tipe data yang tepat (contoh tipe data : bil. bulat, bil. real, boolean, char, string)

4) Konsep percabangan (if & else)

Setelah materi di atas telah di kuasai-kuasainya barulah mempelajari konsep perulangan.

Dalam soal tersebut, disuruh untuk menentukan n bilangan prima pertama dan menjumlahkan nya.

Logika bepikir :

Gunakan printah yang dapat membuat suatu variabel (kita namakan saja dengan nama variabel 'i') yang dapat mengembalikan/menampilkan/membuat variabel i bernilai dari 2, 3, 4, 5....... dan seterusnya. Yang dimaksud "dan seterusnya" ialah sampai banyaknya bilangan prima pertama yang di-inputkan (misalnya 3) ditemukan, kemudian berhentikan perulangan

.

Gunakan printah di bawah ini untuk melakukan hal tsb.

for (int i = 2; count_prima < n; i++) {

......

}

Variabel count_prima ialah variabel yang menentukan berapa banyak kah bilangan prima pertama telah ditemukan. Jika count_prima sudah sama dengan n maka perulangan akan berhenti.

Di perulangan tersebut anda dapat membuat variabel i bernilai 2 3 4 5 6 .... dst. Di variabel i tersebutlah kita dapat mengambil bilangan prima contohnya 2 3 5. Untuk dapat mengambil atau menyimpan bilangan i tersebut yang hanya bernilai prima maka perlu dilakukan syarat-syarat kondisi tertentu. Tentu saja gunakan konsep percabangan yaitu if else.

Perhatikan potongan program berikut untuk menyimpan variabel i yang hanya bernilai prima dan kemudian menjumlahkan masing masing bilangan primanya.

int jumlahkan_jika_prima = 0;

if (i ℅ j == 0) {

Bilangan prima ialah bilangan yang habis dibagi oleh 1 dan bilangannya sendiri, contoh 7 hanya dapat dibagi dengan 1 dan 7. maka Kita gunakan kondisi jika 7 habis dibagi habis dengan bilangan 2, 3, 4, 5, 6, maka 7 ialah bilangan bukan prima, tetapi nyatanya tidak, 7 tidak dapat dibagi habis dengan bilangan-bilangan tersebut.

Jika i "saat ini" = 7

kemudian perintah for (int j = 2; j < i; j++) dijalankan, menandakan bahwa nilai j bernilai 2 3 4 5 6 (sampai 6, tidak lebih karena j < i).

SOAL 2

Untuk menentukan suatu bilangan merupakan bilangan prima atau bukan, sepengetahuan saya, terdapat dua metode. Saya namakan kedua metode ini: metode biasa (naif) dan metode teroptimasi.

1. METODE BIASA (NAIF)

Misalkan bilangan yang diperiksa adalah n. Pada metode ini, kita periksa apakah n habis dibagi oleh 2, 3, 4, dst. hingga n–1. Salah satu saja habis membagi n, maka bilangan n bukan prima.

Kita gunakan variabel bantuan bertipe boolean (true atau false), yaitu [tex]\tt apakahPrima[/tex]. Kita juga akan menggunakan variabel untuk looping/iterasi.

Untuk menentukan keterbagian bilangan, kita gunakan operator “mod“. Jika a mod b = 0, maka a habis dibagi b. Pada pseudocode, operator ”mod“ dilambangkan dengan ”%“, seperti pada implementasi bahasa pemrograman pada umumnya.

Algoritma Bahasa Manusia (Bahasa Alami/Natural)Baca masukan bilangan bulat positif \tt n.Jika [tex]\tt n = 1[/tex], maka tampilkan “Bukan prima”, dan proses selesai.Asumsikan n prima dengan inisialisasi [tex]\tt apakahPrima = true[/tex].Dari [tex]\tt i=2[/tex] hingga [tex]\tt i=n-1[/tex], lakukan iterasi:
◇ Jika [tex]\tt n\ mod\ i = 0[/tex],
    maka [tex]\tt apakahPrima = false[/tex], dan hentikan iterasi.
◇ Jika tidak, lanjutkan iterasi.Jika [tex]\tt apakahPrima = true[/tex], maka tampilkan “Prima”.
Jika tidak ([tex]\tt apakahPrima = false[/tex]), maka tampilkan “Bukan prima”.Algoritma dalam bentuk Pseudocode

Algoritma Menentukan Prima atau Bukan Prima
{ Menerima masukan bilangan bulat positif,
menentukan apakah bilangan tersebut prima,
dan menampilkan hasilnya. }
Kamus
   n: integer ≥ 0  {unsigned integer}
   i: integer
   apakahPrima: boolean
Algoritma
    read n
   if n = 1 then
       print “Bukan prima”
       exit   {proses selesai, keluar dari program}
   end if
   apakahPrima ← true
   for i ← 2 to n–1 do
       if (n % i = 0) then
           apakahPrima ← false
           break   {hentikan iterasi}
       end if
   end for
   if apakahPrima then
       print “Prima”
   else
       print “Bukan prima”
   end if

2. METODE TEROPTIMASI

Optimasi algoritma penentuan sebuah bilangan prima atau bukan, dapat dilakukan dengan sebuah metode yang dapat disebut sebagai “optimasi 6k+1”.

Kita langsung jabarkan prosesnya dalam bentuk algoritma bahasa manusia.

Asumsi: Terdefinisi fungsi [tex]\tt floor(x)[/tex], yang menghasilkan bagian bilangan bulat dari sebuah bilangan pecahan (pecahan desimal atau pecahan campuran). Misalnya: [tex]\tt floor(2.6)[/tex] menghasilkan 2.

Algoritma Bahasa Manusia (Bahasa Alami/Natural)

1. Baca masukan bilangan bulat positif [tex]\tt n[/tex].
2. Asumsikan n prima dengan inisialisasi [tex]\tt apakahPrima = true[/tex].
3.1. Jika [tex]\tt n \le 3[/tex], maka [tex]\tt apakahPrima = true[/tex] jika [tex]\tt n = 2[/tex].
3.2. Selain itu (untuk n > 3):
   3.2.1. Jika n mod 2 = 0 atau n mod 3 = 0,
              maka .
   3.2.2. Jika tidak:
       3.2.2.1. Inisialisasi [tex]\tt i=5[/tex].
       3.2.2.2. Selama [tex]\tt i \le floor(\sqrt{n}\,)[/tex]:
            3.2.2.2.1. Jika n mod i = 0, atau n mod (i+2) = 0,
                 maka [tex]\tt apakahPrima = false[/tex], dan hentikan iterasi.
            3.2.2.2.2. Inkrementasi [tex]\tt i[/tex] dengan 6: [tex]\tt i = i+6[/tex].
4.1. Jika [tex]\tt apakahPrima = true[/tex], maka tampilkan “Prima”.
4.2. Jika tidak ([tex]\tt apakahPrima = false[/tex]), maka tampilkan “Bukan prima”.

28. Pseudocode bilangan ganjil 1-20?


Pseudocode bilangan ganjil 1-20?JawabanPendahuluan

Pseudocode adalah sebuah cara penulisan algoritma yang paling dengan dengan bahasa pemrograman. Pseudocode berisi algoritma untuk mengerjakan sesuatu. Algoritma sendiri adalah susunan logis dan sistematis dari langkah-langkah untuk memecahkan atau menyelesaikan suatu permasalahan. Misalnya adalah langkah – langkah untuk memasak mie instan, langkah – langkah yang dapat kita temukan di balik bungkus mie instan tersebut juga dapat kita sebut sebagai sebuah algoritma karena merupakan langkah-langkah untuk menyelesaikan permasalahan yaitu memasak mi instan menjadi matang.

Dalam dunia komputer, algoritma merupakan alur pemrosesan pada sistem komputer yang terbentuk dari kumpulan command atau perintah yang terintegrasi pada setiap aspek komputer (brainware, software, dan hardware).

Dalam dunia pemrograman, algoritma digunakan untuk membuat software dengan berbagai kegunaan yang digunakan pada perangkat komputer. Algoritma dalam pembuatan software diimplementasikan dengan beberapa bahasa pemrograman seperti C/C++, C#, Visual Basic, dan bahasa pemrograman lainnya.Tanpa algoritma yang tepat, baris demi baris kode tidak akan menghasilkan output yang diinginkan. Oleh karena itu algortima sangat penting dalam membuat program berfungsi sebagaimana mestinya.

Semakin kompleks suatu algoritma, semakin banyak pula statement di dalamnya dan tidak selamanya berurutan. Statement-statement ini perlu diseleksi dan disesuaikan dengan permasalahan dan prosedur menyelesaikan suatu masalah. Proses penyeleksian statement inilah yang disebut dengan percabangan.

Pembahasan

Bilangan ganjil adalah bilangan yang tidak habis dibagi 2. Untuk menghitung sisa dari hasil bagi kita perlu menggunakan operator modulo atau modulus yang di simbolkan dengan operator %. Modulo ini akan menghasilkan sisa dari pembagian 2 buah angka. Misalnya 5 dibagi 3 akan menghasilkan 1 dengan sisa 2. Maka 5 modulo 3 akan manghasilkan angka 2. Jika kita manfaatkan untuk mengetahui apakah bilangan itu genap atau ganjil, maka kita gunakan % 2, jika suatu bilangan di modulo dengan 2 hasilnya tidak 0, maka bilangan tersebut pasti bilangan ganjil.

Oleh karena itu kita bisa membuat pseudo code bilangan ganjil 1-20 dengan mengikuti algoritma berikut ini.

MulaiAngka dimulai dari 0Cek angka apakah habis dibagi 2, jika tidak ke langkah 4, jika ya ke langkah 5Angka adalah bilangan ganjil, cetak angka, ke langkah 6Angka adalah bilangan genap, ke langkah 6jika angka kurang dari 20, angka ditambah 1 kembali ke langkah 3, jika tidak ke langkah 7Selesai.

Pseudo code dari algortima tersebut adalah sebagai berikut

Start;

set angka = 0;

while angka < 20

do

{

    if angka mod 2 ! = 0, print angka;

    angka = angka + 1;

}

End;


Setelah ada pseudo code, membuat program pun akan terasa jauh lebih mudah. Kita tinggal mengikuti pseudo code dan merubah sedikit ke bahasa pemrograman.

Contoh program akan dituliskan dalam bahasa C / C++

#include <stdio.h>

#include <conio.h>

int main()

{

    int angka = 0;

    pritnf("Angka ganjil dari 0 sampai 20\n");

    while(angka < 20)

    {

         if(angka % 2 != 0)     pritnf("%i\n", angka);

         angka++;

    }

    return 0;

}

Kesimpulan

Membuat pseudocode maupun program lebih mudah jika kita menuliskan algoritmanya terlebih dahulu. Untuk mencari bilangan ganjil, kita menggunakan operator modulus untuk mengetahui apakah bilangan habis dibagi 0 atau tidak.
