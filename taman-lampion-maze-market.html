        = 2 x 22/7 x 3,5 x 3,5
        = 77 cm²
Kerucut
tinggi (t) = 15,5 -3,5 = 12 cm
sisi mering (s) =√3,5² + 12²               
                     =√12,25 + 144
                     = √156,25

20. contoh dari maze solver robot​


Jawaban:

(First Pass): Robot menemukan jalan keluar dari "labirin sempurna yang tidak diketahui". Tidak peduli di mana Anda meletakkannya di dalam labirin, itu akan menemukan "solusi".

(Second Pass): Setelah robot menemukan solusi labirin yang memungkinkan, robot harus mengoptimalkan solusinya dengan menemukan "jalur terpendek dari awal hingga akhir".

ALGORITMA =

Algoritma pemecahan labirin, yaitu metode otomatis untuk memecahkan labirin. Algoritme mouse, pengikut dinding, Ikrar, dan Trémaux acak dirancang untuk digunakan di dalam labirin oleh pelancong tanpa pengetahuan sebelumnya tentang labirin, sedangkan algoritma pengisian buntu dan jalur terpendek dirancang untuk digunakan oleh seseorang atau program komputer yang dapat melihat seluruh labirin sekaligus.

MAZE ROUTING CODE =

// cur also indicates the coordinates of the current location

int MD_best = MD(src, dst);// It stores the closest MD we ever had to dst

// A productive path is the one that makes our MD to dst smaller

while (cur != dst) {

   if (there exists a productive path) {

       Take the productive path;

   } else {

       MD_best = MD(cur, dst);

       Imagine a line between cur and dst;

       Take the first path in the left/right of the line; // The left/right selection affects the following hand rule

       while (MD(cur, dst) != MD_best || there does not exist a productive path) {

           Follow the right-hand/left-hand rule; // The opposite of the selected side of the line

   }

}

23. 9. Taman berbentuk persegi panjang berukuran panjang 72 m. Lebar taman seperempat panjangnya. Dibagian tepi taman dipasang lampion berjarak 4 m. banyak lampion yang dipasang ada …. pakai caranya


Jawaban:

jawabannya 45

Penjelasan dengan langkah-langkah:

semoga membantu ya :)

Jawaban:

45 buah

Penjelasan dgn langkah-langkah:

Diketahui:

Panjang taman = 72 mLebar taman = seperempat panjangnyaBagian tepi taman dipasang lampion berjarak = 4 m

Ditanya:

Banyak lampion yang dipasang = ...?

Penyelesaian:

Lebar taman seperempat dari panjangnya, maka:

[tex]l = p \times \frac{1}{4} [/tex]

[tex] = 72 \times \frac{1}{4} [/tex]

[tex] = \frac{72}{4} [/tex]

= 72 ÷ 4

= 18 cm

Karena lampion hanya dipasang di tepi taman, maka cari keliling taman tersebut.

K = 2 × (p + l)

= 2 × (72 + 18)

= 2 × 90

= 180 m

Banyak lampion

[tex] = \frac{180 }{4} [/tex]

= 180 ÷ 4

= 45 buah

Kesimpulan:

Jadi, banyak lampion yang di pasang adalah 45 buah.

________________________________

Keterangan:

K = kelilingp = panjangl = lebar

29. sabrina mempunayai sebuah lampion yg berbentuk kubus. volume lampion tersebut 4.913 cm³ panjang rusuk lampion sabrina.......?​


Jawaban:

panjang rusuk = 17cm

Penjelasan dengan langkah-langkah:

Untuk soal ini , pahami dulu bahwa volume kubus itu V = x³ dengan x adalah panjang rusuk nya.

Maka, kerana Volume adalah 4.913, maka ,

V = 4.913 = x³

Untuk mencari x , tinggal di akar pangkat 3 kedua ruas ,

³√(4.913) = ³√x³

Menjadi ,

x = ³√(4.913)

x = 17cm
