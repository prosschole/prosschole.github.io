Bait 1 :

var gajah: integer;

dideklarasikan variabel gajah dengan tipe data integer

Bait 2:

Begin

memulai proses

bait 3:

gajah:=2018;

dideklarasikan variabel gajah memiliki value 2018

bait 4:

if gajah mod 100> 20 then

diberikan kondisi dimana jika hasil bagi value gajah dengan 100 memiliki sisa besar dari 20 maka di lakukan proses

bait 5:

gajah:=gajah+1

proses yg dijalankan jika kondisi pertama bernilai benar maka di proses value gajah ditambah 1

bait 6

else

pendeklarasi untuk proses yang dilakukan jika kondisi pertama tidak terpenuhi

bait 7

gajah:=gajah+14

proses yang dilakukan jika kondisi tidak terpenuhi dimana value gajah ditambah 14

bait 8

writeln(gajah+gajah)

output terakhir dimana nilai gajah sebelumnya ditambah nilai gajah yg sekarang (perulangan)
