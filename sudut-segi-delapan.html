13. banyak titik sudut pada prisma segi delapan berapa buah?


Banyak titik sudut pada prisma segi delapan berapa buah?

Bangun prisma adalah bangun ruang sisi datar yang memiliki alas dan tutup yang sama bentuknya.

Banyak sisi pada prisma segi-n dapat dirumuskan sebagai n + 2Banyak rusuk pada prisma segi-n dapat dirumuskan sebagai 3nBanyak titik sudut pada prisma segi-n dapat dirimuskan sebagai 2n

Pembahasan

Prisma segi-8

Banyak sisi pada prisma segi-8

=  8 + 2

= 10

Banyak rusuk pada prisma segi-8

= 3 x 8

= 24

Banyak titik sudut pada prisma segi-8

= 2 x 8

= 16

17. sudut putar terkecil dari segi tujuh dan segi delapan


Sifat istimewa dari segi-n beraturan adalah:

- memiliki n buah sisi sama panjang

- memiliki n titik sudut, dengan sudut sama besar

- memiliki n buah sumbu simetri

- memiliki n buah simetri putar

- memiliki sudut putar terkecil sebesar 360°/n

Dari penjelasan di atas, maka:

a. Segi tujuh memiliki sudut putar terkecil sebesar 360°/7 = 51,43°.

b. Segi delapan sudut putar terkecil sebesar 360°/8 = 45°.
