2. yang termasuk sebagai perangkat masukan adalah​


Perangkat Yang Digunakan untuk mengatur suatu sistem di dalam komputer

Seperti ; Mouse, Keyboard, Monitor, dll.

. Keyboard

perangkat komputer keyboard

Keyboard (papan ketik) adalah perangkat keras yang penuh dengan tombol-tombol yang mewakili hurup, angka, karakter, dan beberapa fungsi tertentu. Keyboard komputer tentu saja berbeda jauh dengan keyboard (alat musik) yang hanya berisi beberapa tombol putih dan hitam yang mewakili nada tertentu. Terkadang keyboard juga dilengkapi dengan trackpad untuk menggerakkan kursor. Kombinasi beberapa tombol keyboard juga berguna untuk memberi suatu perintah shortcut pada komputer. Contohnya shortcut Ctrl+C yang berfungsi untuk memberi perintah copy (salin) pada komputer.

2. Mouse

perangkat komputer mouse

Mouse (tetikus) adalah perangkat keras yang mudah digenggam, umumnya berisi dua tombol dan sebuah bola atau laser dibawahnya untuk menggerakan kursor (pointer). Fungsi mouse adalah untuk menggerakan kursor, melakukan scroll (tidak semua mouse), klik kiri, dan klik kanan. Gerakan kursor dalam keadaan default selalu mengikuti arah perpindahan mouse.

3. Webcam

perangkat komputer webcam

Webcam adalah perangkat keras berupa kamera kecil yang biasa ditempatkan di atas monitor/LCD. Fungsi webcam adalah untuk melakukan video call (panggilan video) dan bahkan juga dapat digunakan untuk ber-selfie di depan komputer. Hampir setiap laptop keluaran saat ini telah memiliki web cam.

4. Barcode Scanner

perangkat komputer barcode scanner

Barcode scanner adalah alat yang berupa gagangan dan berisi laser berwarna merah untuk memindai barcode. Fungsinya adalah untuk memindai barcode. Barcode adalah kode garis-garis yang biasa terdapat di kemasan sebuah produk swalayan atau gudang perusahaan. Tujuannya adalah untuk mempermudah kasir menghitung total belanjaan dan mempermudah karyawan untuk mengecek isi gudang.

5. Joystick

perangkat komputer joystick


3. sebutkan yang termasuk perangkat masukan dan perangkat keluaran?​


Jawaban:

perangkat masukan (input) yaitu perangkat yang digunakan untuk memasukkan data. Seperti mouse, keyboard, scanner, dan lain-lain.

Perangkat keluaran (output) yaitu perangkat yang berfungsi untuk menampilkan hasil data yang diproses, seperti monitor, printer, speaker, dan lain-lain.

5. Coba kalian terangkan mengapa microphon termasuk ke dalam perangkat keras masukan


Perangkat keras masukan merupakan perangkat keras yang memiliki fungsi memberi input untuk berkomunikasi dengan komputer. Salah satu perangkat keras masukan adalah microphone. Microphone berfungsi untuk memasukkan suara atau audio ke dalam komputer. Audio tersebut nantinya akan dioleh dengan perangkat lunak yang khusus mengolah file audio hingga menjadi sesuatu, bisa sebuah lagu atau musik. Dikarenakan Microphone bisa memasukan atau menginput suatu data yang bisa diolah oleh komputer, sehingga Microphone masuk ke dalam perangkat keras masukan.

18. tuliskan 5 peralatan yg termasuk perangkat keras masukan beserta kegunaannya​


Jawaban:

Contoh perangkat keras (hardware) masukan (input), antara lain:

1. Mouse

2. Keyboard

3. Microphone

4. Scanner

5. Joystick

Penjelasan:

1. Mouse berfungsi sebagai perangkat keras untuk menginput data berbasis GUI (Graphical User Interface) dan ditampilkan dalam bentuk tanda panah di layar monitor.

2. Keyboard berfungsi sebagai perangkat keras untuk menginput data berbasis teks.

3. Microphone berfungsi sebagai perangkat keras untuk menginput data audio (suara) ke dalam komputer.

4. Scanner berfungsi sebagai perangkat keras untuk menginput data berupa pemindai (dapat memindai data tersebut), selanjutnya hasil pemindaian dapat disimpan dalam komputer.

5. Joystick berfungsi sebagai perangkat keras untuk menginput data khusus bila pengguna komputer sedang bermain game tertentu.

Jawaban :

Perangkat keras(hardware) terdiri atas :

1). Input Device

Peralatan input :

~ Keyboard

- Digunakan untuk memasukkan data atau untuk memberikan perintah pada komputer

~ Mouse

- digunakan untuk menggerakkan pointer di layar untuk menjalankan ikon perintah atau program yang tampil pada layar monitor.

~ Scanner

-digunakan untuk mentrasfer atau mengonversi gambar,foto,teks manual,menjadi data digital sehingga dapat dimengerti oleh komputer

~Microphone

- digunakan untuk merekam atau memasukkan suara yang akan disimpan dalam memori komputer

~Webcam

-merupakan kamera real time yang menggambarkan keadaan pada saat itu,sehingga gambarnya bisa di akses atau dilihat melalui koneksi internet dgn menggunakan aplikasi video call.

2).Process Device

peralatannya :

~ Motherboard

-digunakan sebagai tempat untuk memasang atau meletakkan seluruh rangkaian sistem komputer.

~Processor

-digunakan sebagai otak pemrores dan pusat pengendali berbagai perangkat lain sehingga komputer dapat bekerja satu dgn lainnya

~Memory

-Digunakan sebagai tempat menyimpan data sementara pada komputer.

~Harddisk

-digunakan sebagai tempat menyimpan data pada CPU.

2).Output Device

peralatannya :

~Monitor

-digunakan untuk menampilkan data,instruksi,dan informasi dalam bentuk teks dan grafik atau gambar layar monitor.

~Printer

-digunakan untuk mencetak data atau informasi dari komputer dan kertas sebagai medianya

~Speaker

-peralatan output untuk menghasilkan atau mengeluarkan efek suara dari komputer

~Proyektor

-Menampilkan gambar atau visual hasil pemroresan data.
