2. rumus mencari kuat arus pada kumparan sekunder


Transformator atau Trafo adalah alat yang tersusun atas lempeng-lempeng besi yang dililit dua buah lilitan kumparan, yaitu :

- Kumparan Primer atau input

- Kumparan Sekunder atau output

Fungsi Transformator adalah untuk menaikkan atau menurunkan tegangan listrik.

1. Jenis-jenis Transformator :

a. Transformtor step-up

Berfungsi untuk menaikkan tegangan listrik.

Perlu di ingat :

∴ Jumlah lilitan primer lebih sedikit daripada lilitan sekunder.

∴ Tegangan primer lebih kecil daripada tegangn sekunder.

∴ Kuat arus primer lebih besar daripada kuat arus sekunder.

b. Transformator step-down

Berfungsi untuk menurunkan tegangan listrik.

Perlu di ingat :

∴ Jumlah lilitan primer lebih banyak daripada jumlah lilitan sekunder.

∴ Tegangan primer lebih besar daripada tegangan sekunder.

∴ Kuat arus primer lebih kecil daripada kuat arus sekunder.

2. Prinsip kerja transformator

∴ Kumparan primer dihubungkan dengan tegangan input berupa sumber tegangn bolak-balik.

∴ Perubahan jumlah garis-garis gaya magnet ini mengakibatkan terjadinya GGL induksi pada kumparan sekunder.

∴ Besar dan arah arus yang berubah-ubah menyebabkan terjadinya perubahan jumlah garis-gris gaya magnet secara terus menerus pada inti besi.

∴ GGL induksi pada kumparan sekunder digunakan sebagai tegangan output.

3. Transformator ideal

Transformator dikatakan ideal, apabila tidak ada energi yang terbuang menjadi panas saat energi listrik dihantarkan dari kumparan primer ke kumparan sekunder.

Transformator ideal → Daya input (primer) sama dengan daya output (sekunder).

Hubungan antara daya listrik, tegangan listrik, arus listrik, dan jumlah lilitan pada transformator ideal dapat dituliskan dengan persamaan :

Pp = Ps

Ip Vp = Is Vs , dan

[tex]\frac{Vp}{Vs} = \frac{Np}{Ns} = \frac{Is}{Ip}[/tex]

Keterangan :

Pp = daya primer (watt)

Ps = daya sekunder (watt)

Vp = tegangan primer (V)

Vs = tegangan sekunder (V)

Np = lilitan primer

Ns = lilitan sekunder

Ip  = arus primer (A)

Is   = arus sekunder (A)

Mapel : IPA FISIKAKelas  : IX SMPBab     : Induksi ElektromagnetikMateri : Transformator (Trafo)

a. 220 V

b. 20 A

c. 100 %

d. Step up karena Ns > Np

Penjelasan:

a. Np / Ns = Vp / Vs

300 / 1200 = 55 / Vs

Vs = 220 V

b. Vp / Vs = Is / Ip

55 / 220 = Is / 80

Is = 20 A

c. η = Ns Is / (Np Ip) × 100 %

= 1200 × 20 / (300 × 80) × 100 %

= 100 %
