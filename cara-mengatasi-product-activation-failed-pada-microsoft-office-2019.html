12. Tools "illusstrations" pada microsoft office word 2019 terdapat pada toolbar​


Jawaban:

Grup "Ilustrasi" di toolbar Microsoft Office Word 2019 berisi alat untuk menyisipkan dan memformat gambar dan elemen grafis lainnya di dokumen Word Anda. Grup ini biasanya mencakup alat-alat berikut:

1. Gambar: Menyisipkan gambar dari file atau dari sumber online.

2. Bentuk: Menyisipkan berbagai bentuk pra-desain, seperti kotak, lingkaran, dan panah.

3. SmartArt: Menyisipkan grafik SmartArt, yang merupakan diagram atau bagan alur pradesain yang dapat digunakan untuk mengilustrasikan ide atau konsep secara visual.

4. Bagan: Menyisipkan bagan yang dapat digunakan untuk menampilkan data dalam format grafik.

5. Screenshot: Menyisipkan screenshot dari jendela atau elemen lain di layar komputer Anda.

6. Gambar Online: Menyisipkan gambar dari sumber online, seperti Pencarian Gambar Bing atau OneDrive.

Untuk mengakses alat ini, klik grup "Ilustrasi" di toolbar Word. Alternatifnya, Anda dapat menggunakan menu "Sisipkan" dan memilih opsi yang sesuai dari submenu "Ilustrasi".

20. cara mengatasi error 1713. setup cannot install one of the required products for microsoft office


Step 1. Temporarily disable Firewall and Antivirus

Step 2. SFC Scan to fix Error code 1713

Step 3. Registry repair to fix Error code 1713

Solution 4. Remove and reinstallation.

If Error code 1713 is not fixed in registry repair, SFC scan and firewall/antivirus exception, it is required to remove the Office suite completely from the programs and features, and then to reinstall the product for a proper use.

Follow the steps given below to fix Error code 1713 completely.

Step 1. Open control panel

Step 2. Click on uninstall programs and features

Step 3. Search for Microsoft Office in the list.

Step 4. Select Microsoft Office and click on uninstall.

Step 5. Once uninstall is finished, try running Office setup to install Microsoft Office.

25. format file bawaan yang digunakan pada microsoft office word 2019 adalah​


Jawaban:

Format file bawaan yang digunakan pada Microsoft Office Word 2019 adalah .docx (Word Document). File dengan format ini dapat menyimpan teks, gambar, tabel, dan elemen-elemen lain yang terkait dengan dokumen Word. File dengan format ini juga dapat dibuka di versi-versi Word sebelumnya, seperti Word 2010, Word 2007, dan Word 2003, dengan menggunakan kompatibilitas mode.

Penjelasan:

File format bawaan yang digunakan pada Microsoft Office Word 2019 adalah .docx. Ini adalah format file yang didukung oleh versi terbaru dari Microsoft Word dan merupakan pengganti dari format .doc yang lebih lama. Format .docx menyimpan dokumen Word dalam format yang lebih efisien dan ringkas, dan dapat menyimpan sejumlah fitur yang tidak tersedia pada format .doc lama, seperti tabel yang terkait, rujukan yang dinamis, dan konten yang dapat diedit.
