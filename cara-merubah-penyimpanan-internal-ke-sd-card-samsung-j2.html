17. Cara mengatasi multitouch error di samsung j2 2015,,,,yang tidak bisa touch lebih dari 1,,,padahal suport 5 touch


Pada telepon genggam, multitouch error dapat terjadi karena berbagai alasan baik yang bersifat software maupun hardware. Beberapa langkah yang dapat diambil jika terjadi multitouch error adalah sebagai berikut


- Restart handphone. Biasanya dengan meng-restart handphone, maka cache yang tersimpan dapat dihapus secara otomatis.


- Lakukan Factory Reset / Pengaturan Pabrik. Seringkali error terjadi karena ada masalah pada software. Lakukan Factory Reset dengan masuk ke menu Setting, Back Up and Restore, Factory Data. Perlu diingat, sebelum melakukan Factory Reset, harus mem-backup data data yang ada dalam handphone karena Factory Reset akan menghapus semua data didalam handphone.


- Jika Factory Reset telah dilakukan dan masih terdapat error, kemungkinan besar terjadi kerusakan pada bagian hardware ( layar ). Anda dapat mengunjungi service center terdekat untuk mengganti layar handphone anda

23. Kenapa habis saya install ulang dengan kies 3 hp samsung j2 saya jadi penuh begini... mohon bantuannya gan...


Itu kemungkinan terdapat terdapat error writing pada Sistem androidnya. Karena aplikasi Mirroring/ Transfer biasa mengalami kejanggalan seperti yang anda sampaikan tersebut. Faktor-faktor yqng biasanya menyebabkan memori pernuh ketika re-install app adalah:

1. Memori anda sebelumnya memang sudah hampir penuh

2. Terjadi Glitch pada Smartphone anda. Untuk gejala ini, anda harus membackup data anda dan melakukan factory reset.

3. Biasanya jika smartphone RAM kecil akan lebih sering terjadi kekurangan memori internal. Karena disebabkan system akan membagikan data berjalan ke meori internal, supaya memori RAM tidak akan pernuh secara keseluruhan. Cara memgatasinya dengan menghapus Recent App, Force stop aplikasi tidak penting dan jangan menggunakan aplikasi seperti RAM booster, karena ini akan lebih membebankan memorimu.

4. Restart Smartphone

5. Aplikasi tersebut terdapat bug yang mendapati masalah seperti itu

6. Cara paling ampuh terakhir adalah Wipe data/ cache dan ditambahkan format factory. Dan sesudah itu upgrade/ perbarui OS anda (Terdapat pada: About Device > Perbarui perangkat lunak)

7. Karena masalah smartphone anda. Sebaiknya mengupdate perangkat lunak system dulu. Tetapi sebelumnya hapus dulu cache aplikasi tersebut dan uninstall aplikasi tersebut

Sebaiknya jangan menggunakan aplikasi kies untuk mentransfer, anda dapat mentransfer ke pc tanpa kies, hanya melalui kable data saja. Atau menggunakan aplikasiXxender/ ShareIt untuk memindahkan data dari hp ke pc atau sebaliknya
