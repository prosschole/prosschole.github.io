Jawaban dan Pembahasan

Cangkriman yaiku unen-unen sing kudu dibedhek utawa dibatang maksude. Mula saka iku, cangkriman uga diarani bedhekan utawa batangan. Cangkriman iku wujude ana 4, yaiku:

1. cangkriman sing wujude wancahan (cekakan),

Tuladhane :

Burnas Kopen = bubur panas kokopen.Pakboletus = tipak kebo lelene satus.Surles penen, suring muten = susur teles pepenen, susur garing mutenPindhang kileng = sapi nang kandhang, kaki menthelengTuwan sinyo = untune kedawan gusine menyonyo

2. cangkriman sing wujude pepindhan (irib-iriban),

Tuladhane :

Sega sakepel dirubung tinggi = salakGajah nguntal sangkrah = pawonanPitik walik saba meja = sulakEmbokne wuda anake tapihan = rebung/pringPitik walik saba kebon = nanas

3. cangkriman sing wujude plesetan, lan

Tuladhane :

Wong wudunen iku sugih pari = paringisan.Ana gajah numpak becak = Ketok mbujuke.Wong adol krambil dikepruki = sing dikepruki dudu wonge nanging krambileSuta iku wis anguk-anguk kubur = wis arep mati Bosok malah enak = tape

4. cangkriman sing sinawung ing tembang.

Tuladhane :

Bapak pucung renteng-renteng kaya kalung,

Dawa kaya ula,

Pencokanmu wesi miring,

Yen lumaku si pucung ngumbar swara.

Bedhekane: sepur

29. bahasa krama inggil simbah lara untu sewengi ora turu


           Bahasa krama yaiku bahasa sing tataran paling dhuwur sajeroning undha-usuk Basa Jawa. Ing ngisor iki contonane Bahasa Jawa Ngoko, Madya, lan Krama Alus.

Simbah lara untu sewengi ora turu Ngoko   = Simbah lara untu sewengi ora turu. Krama Madya  = Simbah sakit untu sakndalu boten tilem. Krama Inggil (Alus) = Simbah gering waja sakndalu boten sare.

Bapak lungo menyang Jakarta numpak sepur Ngoko   = Bapak lunga menyang Jakarta nunggang sepur. Krama Madya  = Bapak kesah dhateng Jakarta numpak sepur. Krama Inggil (Alus) = Rama tindak dhateng Jakarta nitih sepur.

Bapak tangi turu Ngoko   = Bapak tangi turu. Krama Madya  = Bapak tangi tilem. Krama Inggil (Alus) = Rama wungu sare.
