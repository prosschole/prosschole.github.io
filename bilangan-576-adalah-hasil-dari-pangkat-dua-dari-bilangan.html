9. Hasil kali dua bilangan cacah adalah 576,sedang hasil baginya adalah 9 kedua bilangan tersebut adalah


~Operasi Bilangan

Misal kedua bilangan tersebut a dan b

a × b = 576 → a = 576 / b

a ÷ b = 9

Diperoleh :

           a ÷ b = 9

(576/b) × 1/b = 9

      576 / b² = 9

                b² = 64

                 b = ±8

Maka :

a = 576 / b

a = 576 / 8

a = 72

Kedua bilangan → 72 dan 8 atau -72 dan -8

12. Hasil kali dua bilangan asli adalah 576. Selisih nya 14,berapa jumlah kedua bilangan tersebut/


dari pernyataan soal

dimisalkan 2 bilangan A dan B, maka :

A x B = 576

A - B = 14   ==>  A = 14 + B

maka

A x B = 576

(14 + B) x B = 576

14 B + B^2 = 576

B^2 + 14 B - 576 = 0

(B-18)(B+32) =0

jadi B = 18   atau B = -32

jika B = 18

A x 18 = 576  ==> A = 576 /18 = 32

Jadi A = 32 dan B = 18

Penjelasan dengan langkah-langkah:

kita misalkan dua bilangan cacah itu adalah a dan b .

a × b = 576 .....pers (1)

a/b = 9 .....pers (2)

maka , a = 9 × b =>> 9b

subtitusikan nilai a = 9b ke pers (1)

a × b = 576

9b × b = 576

9b² = 576

b² = 576/9

b² = 64

b = √64

b = 8 ( nilai kedua )

untuk nilai pertama nya kita subtitusikan nilai b = 8 ke pers (2)

a/b = 9

a/8 = 9

a = 9 × 8

a = 72 ( nilai pertama )

8 + 72 = 80 { opsi C }

• Pembahasan :

Pada dasarnya, bilangan berakar selalu memiliki bentuk dimana akar bil pokok sama saja dengan bil pokok pangkat setengah, yang jika bila dirumuskan akan seperti ini :

[tex] \sqrt{a} = {a}^{ \frac{1}{2} } \\ \\ a = bil \: pokok[/tex]

Dan jika ingin menentukan hasil dari bilangan berakar tersebut, jika hasilnya bil pokok itu sendiri. Bil pokok tersebut wajib memenuhi akar bil pokok kuadrat yang jika dirumuskan akan menjadi seperti ini :

[tex] \sqrt{ {a}^{2} } = a[/tex]
