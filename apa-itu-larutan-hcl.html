1. Sebanyak 10mL larutan HCL 2M dicampur dengan 140mL larutan HCL 0,5M.konsentrasi larutan HCL sekarang adalah...M


M akhir = ((V1 x M1) + (V2 x M2)) / V total
= ((10 x 2) + (140 x 0,5)) / 150
= 90 / 150
= 0,6 M

Jadi, konsentrasi larutan hcl sekarang sebesar 0,6 M

2. sebanyak 10ml larutan HCl 2M dicampur dengan 140ml larutan HCl 0,5M. Konsentrasi larutan HCl sekarang adalah


10 ml HCl 2 M + 140 ml HCl 0.5 M
HCl + HCl ------> HCl + H2O
Mula" : 20 mmol 70 mmol - -

___________________________________________ -
Final: 0 mmol 50 mmol 0 mmol 0 mmol

Jadi sisa HCl = 50 : 150 = 0,33
pH = -log 0,33
= - log 33 x 10^-2
= 2 - log 33
= 0.48

5. Larutan hcl mengandung 36% hcl berdasarkan massa. Hitunglah fraksi mol hcl dan h2o dalam larutan ini


HCl = 36 gram, H2O = 100 - 36 = 64 gram.

Fraksi mol HCl = ( mol HCl / (mol HCl + mol H2O) = ((36/36)/((36/36) +(64/18)) = (1 /(1+4)) = 1/5 = 0,2.

Fraksi mol H2O = 1 - 0,2 = 0,8

22. Sebanyak 200mL HCl 0,1M dicampur dengan 300mL larutan HCl 0,2M. Tentukan Molaritas HCl di larutan yang diperoleh adalah...​


0,16M

Penjelasan:

Mencari Molaritas campuran

Molaritas campuran = mol larutan pertama + mol larutan kedua + .... mol larutan ke n / volume larutan 1 + volume larutan 2 + .... volume larutan ke n

M Campuran = M1 × V1 + M2 × V2 / V1 + V2

M Campuran = 0,1M × 200mL + 0,2M × 300mL / 200mL + 300mL

M Campuran = 20mmol + 60mmol / 500mL

M Campuran = 80mmol / 500mL

M = 0,16Molar
