16. Contoh anmasi 2d contoh animasi studi dan animasi 3D ​


Animasi 2D adalah jenis animasi dalam bentuk dua dimensi, artinya animator 2D membuat gambar dan karakter dalam format dua dimensi dan menghidupkannya dengan gerakan. Jenis animasi ini dianggap sebagai bentuk animasi tradisional dengan ciri karakter polos, tidak bervolume, dan hanya bergerak ke atas, bawah, kiri dan kanan.

Contoh animasi 2D ada di serial kartun yang kamu tonton semasa kecil seperti kartun Tom & Jerry, Scooby Doo, atau Spongebob Squarepants. Salah satu animasi 2D hasil karya anak bangsa yang terbaru adalah Si Juki The Movie yang terinspirasi dari komik populer asal Indonesia.

Animasi 3D adalah seni untuk menciptakan gambar bergerak dalam ruang digital 3 dimensi. Melalui manipulasi objek atau model 3D dalam sebuah software untuk mengolah dan membuat animasi, animator mengurutkan gambar yang akan memberikan ilusi gerakan.

Untuk animasi 3D, semua proses dilakukan di komputer. Beberapa contoh animasi 3D yang terkenal, seperti yang telah disebutkan sebelumnya adalah Toy Story, Up, dan Coco. Kalau untuk animasi 3D karya anak bangsa, kamu bisa lihat di serial anak Adit Sopo Jarwo ya.

27. jelaskan perbedaan .fla .swf dan .exe


jelaskan perbedaan .fla .swf dan .exePendahuluan

Macromedia Flash adalah sebuah perangkat lunak editor mutimedia dan animasi yang dibuat untuk para pecinta desain dan animasi supaya bisa berkreasi dengan mudah dalam membuat aplikasi-aplikasi sederhana, animasi-animasi yang interaktif untuk halaman web, film animasi kartun sederhana, dan presentasi. Selain itu, program ini tidak membatasi pengguna jika mampu menggunakan fitur software ini secara optimal dalam penggunaan fasilitas menggambar dan kemampuan menggunakan bahasa pemrograman pada software macromedia Flash (action script) ini kita bisa membuat mini game yang bagus.  

Kelebihan dari macromedia Flash dibandingkan program editor animasi yang lain adalah:

1. mudah dipelajari untuk sekalipun untuk seorang pemula yang masih awam dengan desain dan animasi.

2. dapat dengan mudah dan bebas dalam menuangkan kreatifitasnya berkreasi membuat animasi dengan gerakan yang dapat diatur sesuai dengan flow yang diinginkan.

3. output filenya berukurannya kecil. Karena Flash menggunakan animasi dengan gambar yang berbasis vektor

4. Software Macromedia Flash dapat menghasilkan file bertipe (ektensi). FLA yang bersifat fleksibel, karena dapat dikonversikan menjadi file lain bertipe .swf, .html, .gif, .png, .exe, .mov.

Pembahasan

.fla adalah ekstensi file project dari macromedia flash yang utama.

.swf adalah hasil kompilasi dari file project macromedia flash yang bisa langsung dijalankan animasinya jika sudah terdapat plugin flash player di komputer.

.exe adalah hasil kompilasi dari file project macromedia flash yang berupa executable file yang kompatibel dengan OS windows dan bisa langsung di jalankan tanpa plugin flash sekalipun.

Kesimpulan

perbedaan .fla .swf dan .exe adalah .fla adalah file project, perlu dibuka di program flash dulu untuk membukanya. .swf adalah file animasi yang siap dijalankan tapi membutuhkan plugin flash player, sedangkan. exe adalah file executable windows yang bisa langsung dibuka.
