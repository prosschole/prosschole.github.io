28. pak Harjo berkeinginan Menganti talang rumah. untuk bagian depan rumah,talang yang diperlukan 5 1/4 meter .adapun untuk dapur memerlukan 3 3/8 meter talang. pak Harjo mempunyai persediaan talang 4 1/2 meter berapa meter talang yang harus dibeli pak Harjo agar dapat mengganti seluruh talang rumahnya?​


Penjelasan dengan langkah-langkah:

( 3⅜ + 5 ¼ ) - 4½ = ( 3 + 5 ) + ( 3/8 + 1/4 ) - 4½

= 8 + ( 3/8 + 2/8 ) - 4½

= 8 ⅝ - 4½

= ( 8 - 4 ) + ( ⅝ - ½ )

= 4 + ( 5/8 - 4/8 )

= 4⅛

Jadi Talang yang harus di beli pak harjo adalah = 4⅛
