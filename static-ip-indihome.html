4. Cara mendapatkan ip address static di windows kita harus melakukan setting secara


Jawaban:

Buka Control Panel.Navigasi ke Network and Internet > Network and Sharing Center.Pada panel kiri, klik opsi Change adapter settings.Pada network adapter klik kanan lalu pilih Properties.Klik 2x pada opsi Internet Protocol Version 4 (TCP/IPv4).Klik opsi Use the following IP address dan masukan IP address sesuai kasus yang kamu hadapi (misalnya 192.168.1.7)Subnet mask (biasanya untuk pengguna koneksi lokal adalah 255.255.255.0.Default gateway yang biasanya merupakan IP address yang digunakan pada router. (misalnya 192.168.1.1)Pada menu DNS server, klik opsi Use the following DNS server addresses dan masukan alamat Preferred DNS server, dimana biasanya merupakan IP address yang sama dengan Default gateway.Alternate DNS server dapat kamu isi sebagai opsi lain jika kamu tidak dapat terkoneksi dengan DNS server utama, contohnya bisa kamu masukan DNS google (8.8.8.8). Namun hal ini bersifat opsional (yakni bisa kamu isi, bisa juga tidak).Klik OK untuk menyimpan perubahan.

9. Perintah untuk membuat static routing adalah A. ip route destination mask nexthop B. ip host nama host ip address C. ip route source mask nexthop D. ip route mask nexthop source E. ip route destination nexthop


Perintah untuk membuat static routing adalah ip route destination mask nexthop. Ip route adalah perintah untuk membuat static routing. Destination adalah jaringan tujuan yang ingin ditambahkan ke routing table. Mask adalah subnet mask yang digunakan dalam jaringan. 20. Rancanglah jaringan Lab computer dan layanan hotspot Lab computer dan layanan hotspot! 1. Tentukan peralatan yang dibutuhkan!2. Gambarkan topologi perancangan jaringan!3. Menentukan metode pengkabelan! 4. Menggunakan IP kelas C dan kategori IP Static!5. Disediakan IP public 125.20.40.12​


1. Peralatan yang dibutuhkan untuk merancang jaringan Lab computer dan layanan hotspot adalah:

Router: digunakan sebagai gateway untuk menghubungkan jaringan Lab computer dengan internetSwitch: digunakan untuk menghubungkan komputer-komputer di Lab computerAccess Point: digunakan untuk menyediakan layanan hotspot di Lab computerKabel UTP: digunakan untuk menghubungkan peralatan jaringan


2. Topologi perancangan jaringan yang dapat digunakan adalah star, dimana router berfungsi sebagai pusat jaringan dan dihubungkan dengan switch yang akan menghubungkan komputer-komputer di Lab computer. Selain itu, access point juga dihubungkan ke switch untuk menyediakan layanan hotspot.


3. Metode pengkabelan yang dapat digunakan adalah straight-through, dimana kabel UTP dihubungkan antara port WAN router dengan port LAN switch. Kemudian, port LAN switch dihubungkan dengan port LAN access point dan port LAN komputer-komputer di Lab computer.


4. Untuk menggunakan IP kelas C dan kategori IP Static, dapat dilakukan dengan menetapkan alokasi IP yang sesuai dengan kelas C dan mengkonfigurasi IP statis pada setiap perangkat jaringan. Misalnya, untuk router dapat diberikan alokasi IP 125.20.40.1/24, untuk switch dapat diberikan alokasi IP 125.20.40.2/24, dan untuk access point dapat diberikan alokasi IP 125.20.40.3/24. Kemudian, untuk setiap komputer di Lab computer dapat diberikan alokasi IP di range 125.20.40.4 - 125.20.40.50/24.

5. IP public 125.20.40.12 dapat diberikan kepada router sebagai IP publik yang terhubung ke internet. Kemudian, router akan bertugas untuk meneruskan traffic ke jaringan Lab computer melalui switch dan access point. Dengan demikian, jaringan Lab computer akan terhubung ke internet melalui IP public 125.20.40.12.

25. Jelaskan bagaimana static NAT mentranslasikan ip address host


Jawaban:

- Pertama NAT menerima permintaan dari klien yang berupa paket data yang ditunjukan untuk sebuah server remote di internet.

- NAT kemudian akan mencatat alamat IP Klien , kemudian menyimpannya ke dalam table translasi alamat. Selanjutnya , alamat IP computer klien tersebut akan di ubah oleh NAT menjadi nomor IP NAT , lalu NAT lah yang akan melakukan permintaan kepada server.

- Server selanjutnya akan merespon permintaan tersebut. Melalui sudut pandang server , yang terlihat adalah alamat IP NAT , bukan alamat IP klien yang meminta data yang bersangkutan.

- NAT menerima respon dari server , kemudian melanjutkannya dengan mengirimkan ke alamat IP klien yg bersangkutan.

30. konfigurasi koneksi internet dengan modem ADSL menggunakana. static IP addressb. automatic configuration (DHCP)c. browser Mozillad. galeon​


Konfigurasi koneksi internet dengan model ADSL dapat dilakukan dengan menggunakan Static IP address atau Automatic configuration (DHCP).

Jika menggunakan Static IP address, pengguna harus memasukkan alamat IP yang diberikan oleh penyedia layanan internet (ISP) secara manual ke dalam konfigurasi koneksi internet. Sedangkan jika menggunakan Automatic configuration (DHCP), alamat IP akan didapatkan secara otomatis oleh modem dari server DHCP ISP.

Browser Mozilla dan Galeon adalah aplikasi web browser dan tidak terkait dengan konfigurasi koneksi internet ADSL. Namun, setelah koneksi internet telah dikonfigurasi dengan benar, pengguna dapat menggunakan browser Mozilla atau Galeon untuk mengakses internet.
