1. Apa itu artinya irisan himpunan dan gabungan himpunan???


Irisan

Irisan adalah dua himpunan yang bagian-bagiannya menjadi anggota dari keduanya.

Contohnya

Irisan himpunan A dan B
A B = { x | x A dan B }
Jika A = { 2, 7, 9, 11 }
Jika B = { 1, 5, 9, 10}
Maka A B = 9

Gabungan

Gabungan adalah dua himpunan yang anggotanya hanya bilangan itu saja misalnya anggota bilangan A saja, anggota bilangan B saja dan anggota A, B keduanya.

Contohnya

A B = { x A, atau x B}
Jika A = { 5, 7, 9, 11 )
Jika B = { 6, 7, 8, 9, 10 }
A B = { 5, 6, 7, 8, 9 10, 11 )

irisan himpunan adalah himpunan dari beberapa anggota yang sama dari dua himpunan lainnya
Contoh:A:{1,3,5,7,9} B:{1,2,3,5,7,11}
irisan nya adalah {3,5,7}
gabungan himpunan adalah gabungan dari dua anggota himpunan
Contoh:O:{1,5,10} Y:{1,2,7}
gabungan nya adalah {1,2,5,7,10}

8. Cara mencari gabungan/irisan dari suatu himpunan


Mencari gabungan : anggota himpunan satu dengan lainnya di gabung. Contoh
A = {1,2,4,6}
B = {1,3,7}
Tentukan A U B (A gabungan B)
A U B = {1,2,3,4,6,7}

Mencari irisan : tulis anggota himpunan yang sama. Contoh
A = {1, 3, 5, 7, 9}
B = {3, 6, 9}
Anggota himpunan yang sama adalah 3, dan 9
Maka
A irisan B = {3,9}

15. Merangkum operasi himpunan. ~irisan~gabungan~selisih~komplemen​


MAPEL: MTK

MATERI: HIMPUNAN

TINGKAT: SMP

KELAS: 7

~Irisan (Intersect)

Irisan merupakan anggota yang berada di dalam dua Himpunan dan dua-duanya harus memiliki, contoh:

A = {1, 2, 3}

B = {2, 3, 4}

A ∩ B = {2, 3}

~Gabungan (Union)

Gabungan merupakan anggota yang berada di dalam dua Himpunan tetapi keduanya tidak harus memiliki anggota yang sama, misal:

A = {3, 4, 5}

B = {4, 5, 6}

A ∪ B = {3, 4, 5, 6}

~Selisih

Selisih antara himpunan A dengan himpunan B dimana jika anggota A ada pada B maka selisihnya tidak, contoh:

A = {3, 4, 1, 6}

B = {2, 4, 1}

A - B = {3, 6}

~Kompelen

Mirip dengan selisih tetapi mempunyai himpunan khusus, (biasannya S), misal:

S = {1, 2, 3, 4, 5, 6}

A = {2, 3, 4}

[tex]A^{c}[/tex] = S - A = {1, 5, 6}

17. Jelaskan gabungan,selisih,irisan,komplemen himpunan

25. apakah arti dari gabungan,irisan,diagram ven,dan himpunan
