1. fungsi rumus if pada excel


Fungsi rumus if pada excel adalah menentukan pernyataan atau nilai syarat dan pilihan yang ditentukan lebih dari satu, sebagaimana pada if tunggal.
Fungsi IF pada Microsoft Excel dapat digunakan sebagai formula logika untuk menghitung atau menentukan nilai sebuah pernyataan dimana fungsi IF akan memberikan nilai tertentu jika logika benar atau salah.

2. tuliskan rumus if di excel


Rumus Fungsi IF dalam Micrasoft Excel

Rumus

Contoh
= IF(C6="X","CPU","Monitor")

28. bagaimana caranya menggunakan IF function di microsoft office excel?apa kegunaan directx,direct3d(semua direct) pada program yang mengandung 3D?


Pada Langkah dibawah ini contoh penggunaan rumus if dengan 3 Kriteria seperti
Pada Gambar Dibawah ini :
1. Buatlah Tabel kode barang dan nama barang pada Microsoft excel
2. Pada Kolom Nama Barang Masukan Rumus Dibawah ini:

Contoh 1 Rumus menggunakan 2 IF Dengan 3 Kriteria
=IF(A2="AA",SAMPO,IF(A2="BB","SABUN","GULA"))
Penjelasan 
Jika Cell  A2= kode Barang AA Maka Nama Barang adalah SAMPO Jika Cell A2= BB Maka Nama Barang Adalah SABUN selain kode barang AA dan BB maka Nama Barang Adalah GULA.

Contoh 2 Menggunakan 3 IF Dengan 3 Kriteria
=IF(A2="AA",SAMPO,IF(A2="BB","SAMBUN",IF(A2="CC","GULA","")))
Penjelasan 
Jika Cell  A2= kode Barang AA Maka Nama Barang adalah SAMPO Jika Cell A2= BB Maka Nama Barang Adalah SABUN Jika Cell A2= Kode Barang CC maka Nama Barang Adalah GULA selain kode barang AA ,BB dan CC maka Nama Barang kosong Atau Tidak Ada.

Catatan:
Untuk Penggunaan Rumus IF Dengan Banyak Kriteria Atau Barang anda Dapat Menggunakan IF sebanyak kriteria yang disediakan  sebagai contoh :
Jika Kriteria Yang  Disediakan  Sebanyak 3 maka Kita Dapat Menggunakan IF Sebanyak 2 Atau 3 Seperti Pada Contoh DiatasJika Kriteria Yang Disediakan Sebanyak 4 Maka Kita Menggunakan IF Sebanyak 3 Atau 4Jika Kriteria Yang Disediakan Sebanyak 5 Maka Kita Menggunakan IF Sebanyak 4 Atau 5Dan Seterusnya.Kegunaan direct DirectX mempunyai fungsi untuk memaksimalkan tampilan kualitas suara dan gambar. Tanpa adanya directX, game tidak bisa dijalankan dan audionya pun kurang terasa.
