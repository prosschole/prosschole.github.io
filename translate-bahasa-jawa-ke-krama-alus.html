2. translate ke bahasa jawa (krama alus)


Translate ke bahasa jawa (krama alus) maksudnya mengubah penulisan atau ejaan dari bahasa lain ke dalam bahasa jawa yang termasuk krama alus. Basa jawa iku salah sijining basa daerah asli saka Indonesia. Basa Jawa punika kalebu ing basa Austronesia kang utamane dentuturake dening penduduk suku Jawa.  Basa Jawa uga kalebu basa Ibu ing Indonesia kang nomer siji. Penggunaane basa Jawa dening rakyat rakya saka suku Jawa kang manggon ing daerah Jawa Tengah lan Jawa Timuran.

Pembahasan

Krama alus yaiku salah siji jinising basa jawa, sebab basa jawa dhewe dibagi - bagi klawan piro - piro tingkatan. Aturan - aturan migunaake basa jawa :

Basa jawa iku nduweni akeh pinutur. Tegese Bahasa jawa akeh den tuturake dening rakyat sing manggon ing wilayah Jawa Tengah, Yogyakarta, Jawa Timur lan ing kawasan pantai lor jawa.Basa jawa nduweni macem - macem logat. Basa Jawa nduweni logat sing beda - beda antarane daerah kang siji lan sijini. Koyo dene logat wong sing manggon ing daerah pinggire pantai lor bedo karo rakyat Yogyakarta sing basa jawane tansah alus banget. Dibagi ing ndalem piro - piro tingkatan. Ing rakyat Jawa, ana piro - piro tinggakatan basa (wiwit saka basa kasar sampek alus) sesuai kegunaane. Ana 3 jinise basa, yaiku basa jawa krama inggil, basa jawa krama madya lan basa jawa ngoko.

Translate ing basa jawa krama alus (krama inggil), piro - piro tembung Bahasa Indonesia :

Saya = DalemKamu = PanjenenganKami = Kita (re : kito)Dia = PiyambakipunIni = punika (re : meniko)Itu = NikuApa = MenopoDimana = Wonten PundhiYang Mana = IngkangpundhiSiapa = SintenMengapa = KadhosmenopoMahal = AwisPanas = BentherOrang = Piyantun

Tuladha ukara :

Ayah baru saja mandi = Bapak nembe siram

Ibu sedang tidur = Ibu sare

Kakek sakit panas = Eyang kakung gerah benther

Nenek sudah meninggal = Eyang putri sampun seda

23. Apa perbedaan jenengan,sampeyan,kowe,kulo dan apa maksud dari Bahasa Jawa Ngoko Lugu, Bahasa Jawa Ngoko Alus, Bahasa Jawa Krama Lugu, dan Bahasa Jawa Krama Alus.


★ Perbedaan antara :

➡️jenengan = Krama Alus

➡️sampeyan = Krama lugu

➡️Kowe = Ngoko lugu

➡️Kula = Ngoko Alus

➡️Ngoko lugu yaiku basa sing tembung-tembunge nggawe basa Ngoko lugu kabeh.

➡️Ngoko alus yaiku basa sing tembung-tembunge wis kecampuran antara basa Ngoko Karo basa Krama.

➡️Krama lugu yaiku basa sing tembung-tembunge nggawe basa Krama lugu kabeh.

➡️Krama alus yaiku basa sing tembung-tembunge nggawe basa Krama Alus kabeh.

Pembahasan :

➡️ Unggah ungguh basa kaperang dadi 2 :

Basa NgokoBasa Krama

➡️Basa Ngoko ana 2 :

Basa Ngoko lugu.Basa Ngoko Alus.

★ Basa Ngoko lugu tegese basa sing nggunakake Tembung basa Ngoko lugu kabeh.

➡️ tuladha : Budi tuku jajan.

★ Basa Ngoko Alus tegese basa sing tembung-tembunge wis kecampur basa Ngoko lan Inggil.

➡️ tuladha : Arep tindak menyang Endi?

➡️Basa Krama uga ana 2 :

Basa Krama lugu.Basa Krama Alus.

★ Basa Krama lugu tegese basa sing tembung-tembunge nggawe basa Krama lugu kabeh, ora kecampur Ngoko lan Inggil.

➡️tuladha : Ibu badhe kesah ten malang.

★ Basa Krama Alus tegese basa sing tembung-tembunge nggawe basa Krama Alus kabeh, tanpa dicampur Ngoko utawa Krama lugu.

➡️tuladha : bapak tindak kantor nitih sepedhah.
