7. fungsi file config di codeigniter


Fungsi file config di codeigniterJawaban Pendahuluan

Fungsi file config pada codeigniter adalah sebagai file yang berfungsi untuk melakukan berbagai macam pengaturan pada framework codeiginiter. Pada codeigniter, file config merupakan sebuah folder.

Pembahasan

Setelah membahas mengenai file config pada codeigniter, mari kita bahas juga secara singkat mengenai apa itu codeigniter. Codeigniter merupakan salah satu framework untuk membuat website menggunakan bahasa php. Pada Codeigniter terdapat beberapa folder yang didalamnya berisi berbagai macam file dengan fungsinya yang berbeda-beda. Salah satunya adalah file config yang telah kita sebutkan dan jelaskan tadi.

Pada folder config sendiri terdiri dari berbagai macam file, antara lain adalah

1. autoload.php

2. database.php

3. routes.php

4. config.php

Kesimpulan

Demikian pemabahasan mengenai apa itu codeigniter dan pembahasan mengenai file config pada codeigniter. Dengan menggunakan framework seperti codeigniter maka proses pembuatan website akan semakin mudah dilakukan. 10. Apa hubungan e-learning dengan learning Manajemen system dan learning content manajemen system


Beberapa sistem memiliki alat untuk mengirimkan dan mengelola instruktur sinkron dan asinkron ‘’online’’ pelatihan dan instruktur berdasarkan objek belajar metodologi. Sistem ini disebut ‘’Learning Content Management Systems’’ atau LCMS. Meskipun perbedaan ini, LMS sering digunakan untuk merujuk antara LMS dan LCMS, meskipun LCMS merupakan pengembangan lebih lanjut dari LMS. Karena kesesuaian, akronim CLCMS ‘’(Computer Learning Content Management System)’’ kini banyak digunakan untuk menciptakan cara fonetik seragam referensi perangkat lunak sistem pembelajaran berbasis pada pembelajaran teknologi metodologi maju.


11. apa itu e-learning? bagaimana pemanfaatannya? apa kelebihan dan kekurangan e-learning? dan bagaimana pendapat anda jika e-learning diterapkan di sekolah umum seperti di SMA???


Mapel : Teknologi Informasi dan Komunikasi (Kelas VII SMP)
Materi : Internet
Indikator : Dampak positif - negatif penggunaan Internet

Jawaban :

E - Learning (Electronic Learning/Pembelajaran Elektronik) merupakan aplikasi dari pemanfaatan internet yang bertujuan untuk mempelajari suatu informasi dengan menggunakan jaringan internet.

Untuk sekarang ini, E - Learning sangat bermanfaat untuk pelajar. Apa alasannya ? Ya, banyak pelajar yang menganggap E - Learning sebagai "pelengkap" suatu informasi yang diterima di bangku sekolah. Selain itu juga, E - Learning sangat mudah untuk digunakan, pengguna hanya membutuhkan perangkat dan internet untuk mengaksesnya. Namun, untuk mengakes E - Learning harus diperhatikan pula tatacaranya mengingat internet sudah semakin luas.

Dari penjelasan diatas, dapat simpulkan :
+ E - Learning dapat digunakan oleh siapa saja, mulai dari pelajar hingga guru sekalipun.
+ E - Learning menggunakan perangkat yang canggih.
+ E - Learning sebagai pelengkap suatu informasi yang didapat dari bangku sekolah. Mengingat informasi yang ada di internet sangat luas sekali.
+ Dengan E - Learning, kita dapat belajar kapan saja dan dimana saja tanpa mengenal waktu.

- Karena E - Learning menggunakan teknologi internet, maka pengguna harus berhati - hati dengan konten negatif
- Karena E - Learning bersifat canggih, maka tidak menutup kemungkinan untuk mendapat layanan tersebut harus merogoh kocek dalam - dalam (mahal)
- Untuk pengguna, jika terlalu sering menggunakan E - Learning, bisa saja kesosialan pengguna terganggu, misalkan : Tatap muka antara guru dan siswa berkurang, cenderung menyendiri, dsb

Jadi, bagaimana penerapan E - Learning di sekolah ?
Menurutku, langkah tersebut sudah sangat bagus, didukung pula masyarakat yang sudah melek teknologi dan mengikuti arus teknologi. Siswa semakin aktif pula dalam mencari suatu informasi. Namun, peran sekolah, guru, dan orang tua sangatlah penting untuk kasus ini. Jangan sampai mereka terlalu bergantung kepada alat teknologi. Oleh karena itu, sekolah, guru, dan orang tua harus memberikan bimbingan atau pendidikan dini mengenai penggunaan internet kepada muridnya. Jika dibimbing, maka kedepannya siswa dapat memahami apa dampak positifnya jika menggunakan E - Learning, sekaligus menghindari mana yang termasuk hal - hal yang bersifat negatif.

18. Bagaimana cara kerja PHP pada CodeIgniter ?


Jawaban:

CodeIgniter adalah Framework yang dikembangkan pada Bahasa Pemrograman PHP. Code Igniter bersifat Open-Source yang banyak digunakan oleh para developer dalam mengembangkan website yang dinamis.

Cara Kerja CodeIgniter

Code igniter merupakan framework MVC. MVC adalah Model-view-controller. Ketika kamu anggaplah mencari suatu informasi, controller akan merespon permintaan kamu, lalu menyajikan informasi yang kamu cari jika memang tersedia.

Contoh kecilnya yakni, jika kamu ingin mencari id=3, maka controller akan merespon permintaan kamu, lalu meminta Model untuk mengambil data dari id=3. Model nantinya akan memberikan data tersebut kembali ke controller. Controller akan menyajikan data tersebut kepada View dengan format yang mudah dikenali (Human-readable format). Dan hasilnya akan muncul pada browser kamu.

27. tuliskan tahapan configurasi pada codeigniter


Kalau aku pake XAMPP

1. Download Codeigniter Framework di situs resminya
2. Ekstrak kemudian taruh di folder htdocs dengan nama project. Contoh : htdocs/aplikasiku/(taruh disini)
3. Aktifkan Apache dan Mysql pada XAMPP
4. Ketikkan localhost/project yang dibuat pada web browser
5. Ketika muncul tulisan "welcome to codeigniter !" framework siap digunakan

Untuk konfigurasi, masuk ke folder codeigniter dan double-click pada folder config. Di folder config ada beberapa file php yang bisa diubah oleh beberapa pengembang.

1. Database.php
file ini berisikan untuk membuat koneksi dan memilih database

2. Config.php
mengatur basis file yang nantinya akan dibuka

3. Autoload.php
Karena CI bersifat MVC, autoload berfungsi untuk memanggil library, helper, model,dll

4.  Routes.php
Memanggil default controller yang tersedia/akan dibuat
