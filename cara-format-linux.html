5. Bagaimana cara menginstal sistem operasi linux


Bagaimana cara menginstall sistem operasi Linux. Linux merupakan salah satu sistem operasi yang populer sekarang. Macam-macam linux disebut dengan distro. Pilihlah distro yang sudak cukup terkenal dan terdapat dukungan dari pembuat distro atau dari forum-forum distro tersebut.

Cara menginstaal linux tidak jauh berbeda dengan menginstall sistem operasi Windows. Langkah pertama memastikan partisi hardisk. Namun terdapat perbedaan pada tipe partisi, jika windows menggunakan tipe NTFS, pada linux menggunakan ext4.

Pada saat membuat partisi secara manual pada linux bisa menggunakan format sebagai berikut

1. Pilih / dibaca root untuk menginstall sistem linux, alokasikan sesuai kebutuhan
2. Pilih /home untuk letak partisi menyimpan data-data.
3. Swap ruang swap berfungsi sebagai backup dari RAM kita, namun pada perangkat komputer laptop sekarang RAM sudah memiliki kapasitas yang besar, jadi alokasikan swap sebesar 1-2 GB saja.

Selain menggunakan partisi manual, Linux juga menyediakan partisi secara otomatis dan semua pembagian hardisk akan dilakukan secara otomatis namun data yang ada pada hardisk sebelumnya juga akan terkena partisi dan terhapus.

4. untuk mengatur hak akses file di GNU/Linux dapat menggunakan perintah chmod (change mode) -> "sudo<spasi>chmod<spasi>xxx<spasi>nama file/folder"

dimana xxx diganti dengan angka :

1 -> execute

2 -> write

4 -> read

x pertama untuk owner, x kedua untuk grup, x ketiga untuk other

contoh saya akan mengubah ubuntu.txt menjadi read-only, maka saya ketikan "sudo chmod 444 ubuntu.txt"

5. sama dengan nomor 4

6. "sudo,<spasi>groupadd<spasi>namaGrup"

contoh : saya ingin membuat gurp dengan nama mamalia, maka saya akan mengetikan "sudo groupadd mamalia"

7. "sudo<spasi>chwon<spasi>namaUser:grup<spasi>namaFile"

untuk mengubah kepemilikan dapat menggunakan perintah chown (change owner), misalya ada file ubuntu.txt dengan kepemilikan root:root (untuk mengecek kepemilikan dapat menggunakan perintah "ls -l") akan diubah ke user budi, maka ketikan  "sudo chown budi ubuntu.txt". kemudian untuk mengecek apakah berhasil ketikan perintah "ls -l"

8. "sudo<spasi>chwon<spasi>namaUser:grup<spasi>namaFile"

sama dengan nomor 7,  misal ubuntu.txt kepemilikannya root:root (user root groupnya juga root), diubah gurp ke mamalia, ketikan perintah "sudo chown  root:mamalia ubuntu.txt"

13. format partisi harddisk yang sesuai dengan sistem operasi linux


Partisi Primary : merupakan jenis partisi utama diharddisk untuk system operasi umumnya. Partisi primary hanya bisa dibuat maksimal 4 partisi, hal ini sangat berbeda dengan sistem DOS yang hanya mengijinkan satu jenis partisi primary untuk system. Jenis partisi ini menempati nomor partisi 1, 2, 3 dan 4. Misalnya harddisk sda bisa dibuat sda1, sda2, sda3 dan sda4.

Partisi Extended : Partisi Extended berfungsi untuk mengatasi keterbatasan pembagian partisi. Partisi Extended tidak menangani pengolahan data secara langsung. Untuk dapat menggunakannya, kita harus menciptakan Partisi Logical terlebih dahulu.

Partisi Logical : Partisi yang akan selalu dibuat didalam partisi extended. Nomor partisi akan selalu dimulai  5 dan seterusnya. Jika ada 3 jenis partisi logical maka masing-masing akan menempati sda5, sda6, dan sda7.

27. bagaimana cara mengetahui spesifikasi komputer di linux ?


1.Informasi spesifikasi prosessor (CPU)
Untuk mengetahui informasi detil prosessor yang ada di komputer Anda, ketikan perintah berikut: cat /proc/cpuinfo
Untuk mengetahui salah satu item data yang Anda perlukan adalah menambahkan perintah | grep 'item data'. Contohnya jika Anda hanya ingin mengetahui model prosessornya saja, maka gunakan perintah berikut: cat /proc/cpuinfo | grep 'model name'
2.Informasi spesifikasi memori (RAM)
Perintah berikut digunakan untuk mengetahui spesifikasi detil memori (RAM) yang ada di komputer Anda: cat /proc/meminfo
Untuk mengetahui kapasitas total memori terpasang, maka gunakan perintah berikut: cat /proc/meminfo | grep MemTotal
3.Informasi spesifikasi chipset kartu grafis (VGA)
Ketikan perintah berikut untuk mengetahui jenis chipset kartu VGA yang Anda pakai: lspci -nnk | grep VGA -A1

4.Informasi kapasitas total harddisk
Ketikan perintah berikut untuk mengetahui kapasitas total harddisk komputer Anda: sudo fdisk -l | grep Disk
5.Perintah Informasi semua hardware
Perintah alternatif lainnya yang akan kita pakai adalah lshw. Perintah ini akan memberikan informasi keluaran spesifikasi hardware komputer Anda secara keseluruhan. Untuk menggunakan perintah ini, buka terminal (CTRL+ALT+T untuk Ubuntu), lalu ketikan perintah berikut: sudo lshw -short
Perintah lshw akan memberikan informasi keluran yang sangat banyak dan detil. Jika ingin menyingkat informasi keluaran tersebut, pakailah opsi -short. Apabila perintah lshw tidak dapat dijalankan karena sistem operasi Linux Anda belum terinstal aplikasi tersebut, maka gunakan perintah berikut untuk menginstalnya: sudo apt-get install lshw
6. Informasi spesifikasi hardware melalui tampilan grafis (GUI)
Sebagai tambahan, jika Anda menginginkan informasi tentang spesifikasi hardware untuk ditampilkan secara grafis (GUI), maka Anda bisa menggunakan aplikasi hardinfo. Aplikasi ini akan memudahkan Anda dalam melihat spesifikasi hardware yang telah dikelompokan menurut jenis hardwarenya. Untuk menginstal aplikasi hardinfo, gunakan perintah berikut: sudo apt-get install hardinfo
Setelah itu ketikan perintah berikut untuk menjalankannya: hardinfo
