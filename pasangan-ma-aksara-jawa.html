Aksara Jawa saking "Tembung Saroja" :

꧋ꦠꦼ ꦩ꧀ꦧꦸꦁ ꦱ ꦫꦺꦴ ꦗ꧉

PEMBAHASAN"Aksara Jawa"

Aksara Jawa yaiku salah sawijining tulisan tradisional kang turun temurun saka nenek moyang kangge nulis bahasa jawa. Aksara jawa katulis saka kiwa marang tengen, lan penulisane ugi mboten wonten spasi. Sajroning Aksara Jawa uga wonten sandhangan, pasangan, aksara nglegena, aksara rekan, aksara angka, aksara murda lan sapiturute.

• Aksara Nglegena

Ha [ ꦲ ], Na [ ꦤ ], Ca [ ꦕ ], Ra [ ꦫ ], Ka [ ꦏ ],

Da [ ꦢ ], Ta [ ꦠ ], Sa [ ꦱ ], Wa [ ꦮ ], La [ ꦭ ],

Pa [ ꦥ ], Dha [ ꦝ ], Ja [ ꦗ ], Ya [ ꦪ ], Nya [ ꦚ ],

Ma [ ꦩ ], Ga [ ꦒ ], Ba [ ꦧ ], Tha [ ꦛ ], Nga [ ꦔ ].

• Aksara Murda

Na [ ꦟ ], Ka [ ꦑ ], Ta [ ꦠ ], Sa [ ꦯ ],

Pa [ ꦦ ], Nya [ ꦘ ], Ga [ ꦓ ], Ba [ ꦨ ].

• Aksara Rekan

Kha [ ꦏ꦳ ], Fa [ ꦥ꦳ ], Dza [ ꦢ꦳ ], Gha [ꦒ꦳ ], Za [ ꦗ꦳ ].

• Aksara Angka

1 [ ꧑ ], 2 [ ꧒ ], 3 [ ꧓ ], 4 [ ꧔ ], 5 [ ꧕ ],

6 [ ꧖ ], 7 [ ꧗ ], 8 [ ꧘ ], 9 [ ꧙ ], 0 [ ꧐ ].

• Aksara Swara

a [ ꦄ ], i [ ꦆ ], u [ ꦈ ], e [ ꦌ ], o [ ꦎ ].

Aksara Jawa saking "Tembung Saroja" kasusun saka :

Aksara Nglegena : Ta [ ꦠ ], Ma [ ꦩ ], Sa [ ꦱ ], Ra [ ꦫ ], Ja [ ꦗ ].

Aksara Murda : -

Pasangan : Ba [ ꧀ꦧ ]

Sandhangan :

Suku [ ꦸ ] = tanda vokal u.Taling tarung [ ꦺꦴ ] = pratandha vokal o.Pada lungsi [ ꧉ ] = titik.Adeg-adeg [ ꧋ ] = pratandha wiwitane ukara.Cecak [ ꦁ ] = pratandha konsonan ng.Pepet [ ꦼ ] = pratandha vokal e.

Mila yen katulis dados :

꧋ꦠꦼ ꦩ꧀ꦧꦸꦁ ꦱ ꦫꦺꦴ ꦗ꧉

1) Inggilipun Aksara Ta [ ꦠ ] dipunsukani sandhangan pepet [ ꦼ ] , mila ungelipun dados Te [ ꦠꦼ ].

2) Ngandhapipun aksara Ma [ ꦩ ], dipunsukani pasangan Aksara Ba ingkang sampun disigeg suku ꧀ꦧꦸ . Lajeng, sandhangan ceceg [ ꦁ ] dipunsukani wonten inggilipun aksara Ma. Mila menika ungelipun dados "mbung."

3) Aksara Sa [ ꦱ ] ajeg, mboten dipunsukani menapa-menapa. Mila ungelipun tetep "Sa".

4) Ing satengahe aksara Ra [ ꦫ ] dipunsukani taling tarung [ ꦺꦴ ]. Mila ungelipun dados Ro [ ꦫꦺꦴ ].

5) Aksara Ja [ ꦗ ] ajeg, mboten dipunsukani menapa-menapa. Mila ungelipun tetep "Ja".

Pasangan aksara Jawa sing panulise jejer karo aksara sing dipasangi yaiku …

A. na, ga, pa

B. ha, sa, pa ⚫

C. sa, ra, nga

D. nya, tha, ma

__________________Pembahasan

Pasangan aksara Jawa iku cacahe padha karo aksarane, yaiku 20 :

ha, na, ca, ra, ka

꧀ꦲ ꧈ ꧀ꦤ꧈ ꧀ꦕ꧈ ꧀ꦫ ꧈ ꧀ꦏ

da, ta, sa, wa, la

꧀ꦢ ꧈ ꧀ꦠ ꧈ ꧀ꦱ ꧈ ꧀ꦮ ꧈ ꧀ꦭ

pa, dha, ja, ya, nya

꧀ꦥ ꧈ ꧀ꦝ ꧈ ꧀ꦗ ꧈ ꧀ꦪ ꧈ ꧀ꦚ

ma, ga, ba, tha, nga

꧀ꦩ ꧈ ꧀ꦒ ꧈ ꧀ꦧ ꧈ ꧀ꦛ ꧈ ꧀ꦔ

wis kena dideleng tulisan ing dhuwur, pasangan pasangan aksara kang mapane jejer karo aksarane :

- ha = ꦲ ➡️ ꧀ꦲ- sa = ꦱ ➡️꧀ꦱ- pa = ꦥ ➡️ ꧀ꦥ

★ catetan

aksara lan huruf sing diwenehi garis ngisor iku pasangan aksara kang mapane jejer karo aksarane.

Jawaban:

Pasangan aksara Jawa sing panulise jejer karo aksara sing dipasangi yaiku … *

A. na, ga, pa

B. ha, sa, pa ✔

C. sa, ra, nga

D. nya, tha, ma

_________________Penjelasan:

Pasangan aksara jawa sing panulisane ana ing jejer karo aksara sing dipasangai yaiku; ha, sa, pa

ha ꧀ꦲsa ꧀ꦱpa ꧀ꦥ

pasangan sing kacitak kandel, iku pasangan sing panulisane ana ing jejer aksara sing dipasangi.
