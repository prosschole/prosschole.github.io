1. Cara agar bisa membuat daftar halaman angka romawi di word ?


Jawaban:

Misalkan di Ms. Word2013, caranya:

langkah 1 : klik ribbon Insert

langkah 2 : klik Page Number

langkah 3 : klik Format Page Numbers

langkah 4 : klik pilihan Number Formats untuk memilih romawi

langkah 5 : klik tombol OK

Penjelasan tambahan:

Amatilah gambar lampiran:

langkah 1 : ditunjukkan tanda panah merah

langkah 2 : ditunjukkan tanda panah biru

langkah 3 : ditunjukkan tanda panah hijau

langkah 4 : ditunjukkan tanda panah coklat

langkah 5 : ditunjukkan tanda panah abu-abu

Perlu diketahui bahwa setelah langkah 3 dilakukan, akan muncul kotak dialog Page Number Format yg ditunjukkan tanda panah hitam


2. Bagaimana Cara membedakan nomor halaman di word romawi dan angka


Jawaban:

Untuk pengguna windows

1. Pilih Insert > Page Number, lalu tentukan di bagian mana kamu mau menyisipkan halaman apakah di bagian atas, bawah, dan yang lain

2. Klik Page Number > Format Page Number, selanjutnya tinggal pilih mau format angka atau romawi

3. Misalnya kamu mau memberi halaman tapi bukan di halaman pertama melainkan halaman ketiga paper, centang kotak Different First Page

4. Klik Page Number > Format Page Number, pilih 'Start at' di Page Numbering. Kamu tinggal ketik angka 3

5. Proses selesai. Kamu bisa klik Close Header and Footer atau tekan tombol Esc.

- Membuat Halaman Romawi dan Angka di Mac

1. Pilih Insert > Page Number > Page Number, lalu tentukan posisi untuk meletakkan nomor halaman

2. Selanjutnya klik Format > Number Format, tinggal pilih apakah kamu mau halamannya berupa angka atau huruf romawi

3. Kalau ingin membuat halaman tapi bukan di halaman pertama, klik untuk hilangkan tanda ceklis di box Show number on first page

4. Lalu pilih Format, pilih 'Start at' dan masukkan di halaman berapa kamu ingin mulai ada nomor halaman

5. Lalu klik OK dan halaman akan muncul di lembar paper kamu

Nah, itu dia langkah-langkah untuk membuat halaman di Microsoft Word, Urbanreaders. Selamat mencoba!

7. Tuliskanlah bagaimana cara membuat tabel pada microsof word 2013


Klik “Insert” pada menu bar.
Kamu bisa langsung klik menu “Table”dan menentukan baris dan kolom yang diperlukan. Pada menu tersebut terdapat 20 kotak-kotak kecil yang berguna untuk menentukan baris dan kolom dengan cara menggeser-geser kursor.

Klik “Design” pada menu bar untuk mengubah desain tabelmu.
Pilih desain tabel sesuai keinginanmu dibagian “Table Style”. Tersedia beberapa varian desain yang dapat kamu pilih dengan cara klik tombol panah yang ada pada menu tersebut.


Klik template desain tabel yang kamu inginkan.

Kini tampilan tabelmu tidak lagi monoton dan lebih menarik untuk dilihat.

13. Bagaimana cara mengubah format nomor pada halaman,maksudnya untuk nomor halaman yang berbeda dengan lembar pertama.Maksudnya supaya halaman daftar isi satu romawi dan halaman bab angka 1 biasa?,saya sudah melihat tutorial untuk mengubah format angka dari romawi ke yg 1,2,3 yg biasa namun saya belum berhasil melakukan sesuai dengan tutorial


Dibikin 3 file aja.
1. File judul (gausah dikasih halaman)

2. File daftar isi dan kata pengantar,dll (mulai dari romawi i,ii,iii,iv,dll)

3 . File isi (mulai dari angka 1 dan seterusnya).

Misalkan pada Ms. Word 2007:
- buka halaman yang ingin diformat halamannya
- posisikan kursor pada halaman yang akan menjadi halaman 1 (bukan romawi)
- tampilkan kotak dialog Page Setup:
- klik tab. Paper dan klik Apply to dengan memilih This Point Forward
- klik tombol OK
- Hasilnya adalah halaman yang kursornya aktif tadi menjadi halaman romawi i
- blok nomor halaman romawi i yang akan diubah menjadi halaman 1 (bukan romawi)
- klik tab. design dan klik Page Number lalu pilih Format Page Numbers...
- klik format angka yang bukan romawi (1,2,3) dan klik tombol OK
- selanjut-selanjutnya apabila akan menambah halaman, maka nomor halaman akan otomatis menjadi 1,2,3 (bukan romawi)

# konsekuensi dari cara di atas adalah: posisi nomor halaman selalu di bawah (contoh yang saya gunakan ialah nomor halaman di posisi tengah bawah)

Penjelasan:

Jawaban:

   Untuk superscript gunakan kombinasi keyboard 'Ctrl'+'Shift'+'+' (tekan secara bersamaan tobol Ctrl, Shift dan + atau plus). Jika pada jenis keyboard numeric cukup tombol 'Ctrl'+'+' . Dan untuk subscript gunakan kombinasi tombol keyboard 'Ctrl'+'=' (tekan secara bersamaan tombol Ctrl dan = atau samadengan). Keyboard shorcut ini juga berlaku untuk MS Word 2003 dan MS Word 2007, bahkan versi yang lebih baru. Bagi operator computer yang sering melakukan pengetikan penggunaan shorcut akan lebih efisien dalam bekerja.
