9. sebutkan langkah langkah membuka program aplikasi Microsoft excel


Memulai Microsoft Excel

Untuk memulai menggunakan aplikasi Microsoft Excel, ada beberapa cara yang bisa Anda gunakan diantaranya adalah sebagai berikut : 

Cara Pertama

Nyalakan Komputer terlebih dahuluKlik tombol Start pada Taskbar KomputerPilih menu All Program, Pilih Microsoft OfficeKemudian klik Microsoft Excel 2007

Cara Kedua

Nyalakan Komputer terlebih dahuluDouble Klik icon Microsoft Excel 2007 yang ada pada Desktop Komputer Anda.

Cara Ketiga

Nyalakan Komputer terlebih dahulu Tekan kombinasi tombol Windows (bendera di pojok kiri bawah) dan tombol R secara bersamaan (Wndows+R), maka akan muncul kotak dialog kecil di pojok kiri bawah. Ketikkan Excel kemudian tekan tombol Enter.

20. Sebutkan beberapa software pengolah kata Sebutkan langkah-langkah membuka aplikasi Microsoft Word Sebutkan elemen-elemen dasar pengelola kata di Microsoft Word Tuliskan cara penampilan penggaris ruler pada Word​


Jawaban:

element dasar MS. Office word 2010 terdiri dari:

1. title bar = judul untuk menampilkan nama file yang sedang dibuka contohnya : document1-Microsoft Word

2. menu bar = deretan menu yang berisi perintah-perintah sesuai nama setiap menu edit view dan sebagainya

3. toolbar standard = deretan menu standard yang diwakili dengan gambar icon sebagai tombol percepatan dan alternatif untuk menjalankan perintah dalam menu bar

4. toolbar fomating = berisi sejumlah perintah untuk menformat naskah yang sudah diketikkan

5. ruler = berisi angka layaknya penggaris

6. kursor = adalah tanda yang berkedip dilembaran pengetikan

7. lembar kerja = tempat kita mengetikan teks

8. status bar = baris yang berisi pentunjuk mengenai posisi kursor pada halaman baris keberapa, jumlah halaman dan lain lain

9. sizing button = mengatur ukuran jendela kerja dan menutup program aplikasi yang sedang terbuka

10. scroll bar = menggeser layar ke kiri atau kekanan gunakan horizontal scroll bar sedangkan untuk menggeser layar keatas bawah menggunakan vertical scroll bar

Namun, pastikan bahwa sudah terdapat ikon Microsoft Word dalam bentuk jalan pintas (shortcut).Jika pada desktop pengguna tidak ada shortcut Microsoft Word, maka cara mengaktifkannya adalah dengan menekan tombol start (logo Windows) yang ada di pojok sebelah kiri bawah pada panel utama (taskbar) menggunakan mouse . Kemudian pengguna dapat men-scroll mouse ke bawah untuk mencari program Microsoft Word, setelah itu tekan sekali ikon beserta tulisan Microsoft Word menggunakan tombol mouse sebelah kiri.Hampir sama dengan cara yang kedua, cara ini menggunakan menu start . Alih-alih menggunakan mouse, cara ini dapat menggunakan peranti keras bernama keyboard. Pengguna dapat menekan logo Windows pada keyboard, lalu akan muncul panel yang sama jika menggunakan mouse. Ketik kata kunci yaitu Word maka panel akan menunjukkan program yang dicari, kemudian tekan tombol enter pada keyboard maka program akan berjalan.Dalam penggunaan kedepannya jika ingin akses yang mudah, pengguna juga dapat membuat shortcut pada taskbar. Caranya adalah dengan terlebih dahulu membuka program Microsoft Word menggunakan salah satu dari 3 cara di atas. Setelah program sudah berjalan, pengguna tinggal menekan tombol sebelah kanan mouse pada ikon Microsoft Word yang aktif di taskbar, kemudian akan muncul panel kecil berisikan beberapa opsi dan pilihlah opsi pin to taskbar (Windows 8-later).

Pembahasan

Microsoft Word adalah sebuah perangkat lunak/program yang dikembangkan oleh perusahaan teknologi Microsoft. Program ini berfungsi untuk memproses kata dan membuat dokumen secara digital. Microsoft pertama kali meluncurkan program ini pada tahun 1983 dengan nama Multi-Word Tool. Kemudian, pengembang melakukan berbagai macam pembaruan pada program ini. Sehingga terdapat banyak versi yang ada di pasaran. Namun untuk saat ini, versi terbarunya dirilis tahun 2019 bernama Word yang menjadi satu dengan program yang lain di Microsoft Office 2019. Selain itu, Microsoft juga merilis program ini agar dapat digunakan dibeberapa Sistem Operasi, yaitu Windows, macOS, Android, dan iOs.
