7. jelaskan fungsi dari wordart?


Fungsi WordArt yaitu untuk menyisipkan teks atau membua tulisan dengan desain yang bervariasi dengan gaya artistik.

Pembahasan

Langkah menggunakan WordArt :

Klik Tab Insert pada Microsoft Word Klik WordAt makan akan tampil beberapa pilihan menu Pilih salah satu, selanjutnya ketikan apa yang ingindiketikan pada kolom " Your Text Here" Klik Ok,

Fungsi fasilitas pada wordart styles :

WordArt styles, digunakan untuk mengubah gaya teks. Shape Fill, mengubah tema warna teks, secara default warna teks wordart style 1 adalah putih dengan garis pinggir abu-abu.  Shape autline, digunakan untuk mengubah warna garis pinggir teks. Change shape, digunakan untuk mengubah bentuk teks.

Group text terdapat 5 menu :

Edit text, berfungsi untuk mengedit teks yang  dibuat sebelumnya. Spacing, digunakan untuk mengatur jarak antara teks satu dengan teks lainya. Even height, digunakan untuk menyamaratakan tinggi dari teks satu dengan teks lainya. Vertical text, digunakan untuk mengubah posis teks yang tadinya horizontal ke posisi vertikal atau sebaliknya. Alignment, digunakan untuk menentukan rata untuk tepi teks. Dengan menggunakan fasilitas alignment dapat menentukan peletakan teks, misal teks  rata kiri, kanan bisa juga rata kiri kanan.

Microsoft Word

Microsoft Word merupakan salah satu software pengolah huruf/teks besutan Microsoft. Di dalamnya terdapat lembar kerja yang dilengkapi denga berbagai tools yang menunjang kebutuhan penggunanya baik untuk dokumen resmi, undangan, makalah dan jenis lainnya.

Microsoft Word sendiri sudah terdapat beberapa versi diantaranya 1997, 2003, 2007, 2010, 2013 dan 2017. Yang membedakan pada setiap versinya hanyalah interface dan kelengkapan isi tools walau susunan menu juga tidak jauh berbeda.

Menu yang terdapat di Microsoft Word diantaranya

Home, di menu ini terdapat tools umum seperti cut, copy, paste, font, font size, paragraph dan pengaturan umum lainnya

Insert, dimenu ini terdapat tools untuk memasukkan tabel, gambar,  grafik dan keperluan lain yang bisa dimasukkan ke dalam dokumen melalui menu ini

Page Layout, menu ini berisi tools unutk mengatur ukuran halaman kerja mulai dari ukuran kertas, margin, hingga background page.

References, menu ini berisi tools untuk catatan kaki, isi konten, caption hingga indeks dokumen

Mailings, menu ini berisi untuk mengumpulkan data - data

Review, menu ini digunakan untuk mempermudah pengguna dalam melakukan cek spelling, memproteksi dokumen, memberi komentar hingga membandingkan dengan sumber lain

View, meu ini keberadaanyya tidak selalu dan hanya muncul jika pengguna sedang mengedit gambar, grafik, ilustrasi, tabel, maupun shape.
