Jawaban:

Kuartil terdiri dari tiga macam yaitu :

> Kuartil tengah/median ( Q2 )

> kuartil atas ( Q3 )

> kuartil bawah ( Q1 )

Dalam mencari nilai kuartil, dibedakan menjadi dua macam yaitu:

1. kuartil data genap

2. kuartil data ganjil.

Cara mencari kuartil data ganjil yaitu :

1. Urutkan data terlebih dahulu

2. Kuartil tengah/median ( Q2 ) => X_{\frac{1}{2}( n + 1 )}X

Sedangkan untuk mencari kuartil data genap yaitu : ( perhatikan gambar terlampir )

1. Urutkan data terlebih dahulu

2. Tentukan nilai tengah dari data tersebut ( Median )

3. Untuk mencari kuartil bawah maka bagilah data di sebelah kiri median menjadi dua bagian sama besar.

4. Untuk mencari kuartil atas maka bagilah data di sebelah kanan median menjadi dua bagian sama besar.

Jangkauan Interkuartil ( H )

Jangkauan interkuartil atau Hamparan ( H ) adalah selisih antara kuartil ketiga ( Q3 ) dengan kuartil pertama ( Q1 ). Rumus jangkauan interkuartil = Q3 - Q1

Simpangan Kuartil ( Qd )

Simpangan Kuartil ( Qd ) yaitu setengah dari selisih kuartil ketiga dengan kuartil pertama. Maka Qd yaitu = \frac {1} {2}

STATISTIKA

mungkin yang dimaksud Jangkauan interkuartil

Urutkan data :

7, 7, 8, 8, 9, 9, 9, 10, 10, 10 misal [tex] x_n \: \text{data ke n} [/tex]

Mak urutan data tadi diurut dari [tex] x_1 \: \text{sampai} \: x_10 [/tex]

Jumlah data genap, diperoleh :

[tex] Q_1 = x_{ \frac{n + 2}{4} } = x_{ \frac{10 + 2}{4}} = x_3 [/tex]

[tex] Q_1 = 8 [/tex]

[tex] Q_3 = x_{\frac{3(n + 2)}{4} } = x_{ \frac{3(10 + 2)}{4}} = x_8 [/tex]

[tex] Q_3 = 10 [/tex]

Jangkauan interkuartil = Q3 -Q1

= 10 -8

jangkauan interkuartil = 2
