2. Volume suatu tabung 120cm² jika tingginya di jadikan setengahnya maka volume tabung sekarang adalah


Bab Bangun Ruang
Matematika SMP Kelas VIII

tabung

t2 = 1/2 x t1
r2 = r1

volume sekarang = volume awal x (r2)² x t2
                            = 120 x 1² x 1/2
                            = 60 cm³BRSL
• tabung

V₁ = 120 cm³
t₂ = ½ t₁, pada jari-jari tetap
V₂ = __?

volume tabung
V = π r² t

perbandingan volume
V₂ / V₁ = (π r² t₂) / (π r² t₁)
V₂ / V₁ = t₂ / t₁
V₂ / V₁ = t₂ / t₁
V₂ / 120 = (½ t₁) / t₁
V₂ / 120 = ½
V₂ = ½ • 120
V₂ = 60 cm³ ✓
