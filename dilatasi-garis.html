15. Tentukan bayangan dari garis 2y=4x+5 pada transformasi transformasi d.pada dilatasi[0,–3] e.pada dilatasi [s(0,3),–2]


d. 2y = 4x - 15

e. y = 2x - 4.

Pembahasan

Dilatasi adalah transformasi yang memperbesar atau memperkecil ukuran suatu obyek. Hasil dilatasi bergantung kepada titik pusat dilatasi (P) dan faktor skala (k).

Jika titik A (x,y) didilatasikan terhadap titik pusat O(0,0) dengan faktor skala k, maka :

Transformasi dilatasinya :

         [O, k]

A(x, y) ----------→ A'(kx, ky)

x'  = kx

y'  = ky

Apabila titik A (x,y) didilatasikan terhadap titik pusat P(a,b) dengan faktor skala k, maka :

Transformasi dilatasinya

            [P, k]

A(x, y) ----------→ A'(a + k(x - a), b + k(y - b))

x'  = a + k(x - a)

y'  = b + k(y - b)

================================================

Diketahui :

Garis 2y = 4x + 5

Ditanya :

Bayangan garis akibat transformasi :

d. Dilatasi [O, -3]

e. Dilatasi [S(0, 3), -2]

Jawab :

d. Garis 2y = 4x + 5 dengan dilatasi [O, -3]

k = -3

x'  = kx

x' = -3x

x = [tex]-\frac{1}{3}[/tex]x'

y'  = ky

y' = -3y

y = [tex]-\frac{1}{3}[/tex]y'

2y = 4x + 5

2 ([tex]-\frac{1}{3}[/tex]y') = 4 ( [tex]-\frac{1}{3}[/tex]x') + 5

Kalikan ruas kiri dan kanan dengan -3

2y' = 4x' + 5 (-3)

2y' = 4x' - 15

Jadi bayangan garis garis 2y = 4x + 5 oleh dilatasi [O, 3] adalah 2y = 4x - 15.

e. Garis 2y = 4x + 5 dengan dilatasi [S(0, 3), -2]

a = 0, b = 3, dan k = -2

x'  = a + k(x - a)

x' = 0 + (-2) (x - 0)

x' = -2(x)

x' = -2x

x = [tex]-\frac{1}{2}[/tex]x'

y'  = b + k(y - b)

y' = 3 + (-2) (y - 3)

y' = 3 - 2y + 6

y' = 9 - 2y

-2y = y' - 9

y = [tex]-\frac{1}{2}[/tex] (y' - 9)

2y = 4x + 5

2 ([tex]-\frac{1}{2}[/tex]) (y' - 9) = 4 ( [tex]-\frac{1}{2}[/tex]x') + 5

(-1) (y' - 9) = -2x' + 5

-y' + 9 = -2x' + 5

-y' = - 2x' + 5 - 9

-y' = - 2x' + 4

y' = 2x' - 4

Jadi bayangan garis garis 2y = 4x + 5 oleh dilatasi  [S(0, 3), -2] adalah y = 2x - 4.

25. sebuah garis g 3x-6y-1=0 dilatasi dengan skala=3 dan pusat dilatasi pada titik P(1,-2) maka bayangannya adalah


garis g : 3x - 6y - 1 = 0

(x, y) D[(1, -2), 3] (x', y')

x' = (x - p) k + p
x' = (x - 1) 3 + 1
x' = 3x - 3 + 1
x' = 3x - 2
x = (x' + 2) / 3

y' = (y - q) k + q
y' = (y + 2) 3 + (-2)
y' = 2y + 6 - 2
y' = 2y + 4
y = (y' - 4) / 2

akan diperoleh titik bayanganya
x' = 3x - 2
y' = 2y + 4

matriksnya

[tex] \binom{ {x}^{l} }{ {y}^{l} } = k \binom{x - p}{y - q} + \binom{p}{q} [/tex]

[tex] \binom{ {x}^{l} }{ {y}^{l} } = \binom{3 \: \: \: \: \: 0}{0 \: \: \: \: \: \: 3} \binom{x - 1}{y + 2} + \binom{1}{ - 2} \\ \binom{ {x}^{l} }{ {y}^{l} } = \binom{3x - 3}{3y + 6} + \binom{1}{ - 2} \\ \binom{ {x}^{l} }{ {y}^{l} } = \binom{3x - 2}{3y + 4} \\ x = \frac{ {x}^{l} + 2}{3} \: dan \: y = \frac{ {y}^{l} - 4 }{3} \\ [/tex]
