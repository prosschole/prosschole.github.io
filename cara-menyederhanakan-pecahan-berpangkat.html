8. Cara menyederhanakan pecahan berpangkat aljabar


Jawaban:

Suatu pecahan bentuk aljabar dikatakan paling sederhana apabila pembilang dan penyebutnya tidak mempunyai faktor persekutuan kecuali 1, dan penyebutnya tidak sama dengan nol. Untuk menyederhanakan pecahan bentuk aljabar dapat dilakukan dengan cara membagi pembilang dan penyebut pecahan tersebut dengan FPB kedunya.

Penjelasan dengan langkah-langkah:

Contoh :

Sederhanakan pecahan bentuk aljabar berikut, jika x, y ≠ 0

[tex] \frac{3x}{6x {}^{2} y} [/tex]

Penyelesaian:

Bentuk perkalian faktor prima dari 3x dan 6x2y adalah sebagai berikut.

3x = 3 × x

6x²y = 2 × 3 × x² × y

Maka FPB kedua bentuk aljabar tersebut adalah sebagai berikut.

FPB = 3 × x = 3x

Untuk menyederhanakan bentuk pecahan aljabar di atas, kita bagi pembilang dan penyebut dengan 3x sehingga hasilnya adalah sebagai berikut.

[tex] \frac{3x}{6x {}^{2} y} = \frac{3x \div 3x}{6x {}^{2} y \div 3x} \\ = \frac{1}{2xy} [/tex]

18. cara menyederhanakan bilangan pecahan berpangkat


1. Untuk bilangan pecahan berpangkat positif --> dipangkatkan dulu pembilang dengan penyebutnya --> sederhanakan pecahan jika masih bisa disederhanain
Contoh :
[tex](\frac{5}{3})^{3} = \frac{125}{27} = 4 \frac{17}{27} [/tex]

2. Untuk bilangan pecahan berpangkat negatif --> dibalikkan pembilang dengan penyebutnya --> pangkatnya dijadikan positif --> dipangkatkan pembilang dengan penyebutnya --> sederhanakan pecahan jika masih bisa disederhanain
Contoh :
[tex](\frac{3}{5})^{-3} = (\frac{5}{3})^{3} = \frac{125}{27} = 4 \frac{17}{27} [/tex]

3. Untuk perkalian dua bilangan pecahan dengan bilangan pokok yang sama hasilnya sama dengan bilangan pokok dipangkatkan dengan jumlah kedua pangkat pada pecahan tersebut. 
Contoh :
[tex](\frac{1}{2})^{2} x (\frac{1}{2})^{3} = (\frac{1}{2})^{2+3} = (\frac{1}{2})^{5} = \frac{1}{32}[/tex]

4.  Untuk pembagian dua bilangan pecahan dengan bilangan pokok yang sama hasilnya sama dengan bilangan pokok dipangkatkan dengan selisih kedua pangkat pada pecahan tersebut.
Contoh :
[tex](\frac{1}{2})^{3} : (\frac{1}{2})^{2} = (\frac{1}{2})^{3-2} = (\frac{1}{2})^{1} = \frac{1}{2}[/tex]

5. Untuk Pecahan berpangkat kemudian dipangkatkan hasilnya sama dengan pecahan dipangkatkan dengan hasil perkalian kedua pangkat tersebut.
Contoh :
[tex]((\frac{1}{2})^{2})^{3} = (\frac{1}{2})^{2x3} = (\frac{1}{2})^{6} = \frac{1}{64}[/tex]
