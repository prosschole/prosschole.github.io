23. Grafik fungsi y=sin 2x adalah​


Jawaban : ( C )

Tabel nilai x => y = sin 2x

0° => 0

(sin [2 × 0°] = sin 0° = 0)

45° => 1

(sin [2 × 45°] = sin 90° = 1)

90° => 0

(sin [2 × 90°] = sin 180° = 0)

135° => –1

(sin [2 × 135°] = sin 270° = –1)

180° => 0

(sin [2 × 180°] = sin 360° = 0)

28. Jelaskan perubahan grafik y= sin X+3 terhadap y= sin X disertai sketsa grafik nya


Jawaban:

Perubahan grafik trigonometri ini memiliki beberapa aturan penting yaitu:

• Untuk fungsi y = k.sin a (x ± α) mempunyai nilai maksimum = k (1) = k, nilai minimum = k (-1) = -k, Periode = 360⁰/a.

• Untuk fungsi y = k.cos a (x ± α) mempunyai nilai maksimum = k (1) = k, nilai minimum = k (-1) = -k, Periode = 360⁰/a.

• Untuk fungsi y = k.tan a (x ± α) mempunyai nilai maksimum = k (∞) = ∞, nilai minimum = k (-∞) = –∞, Periode = 180⁰/a.
