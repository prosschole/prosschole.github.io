15. akar pangkat 2 dan akar pangkat 3 dari 81 menggunakan faktorisasi prima metode tabel​


Jawaban:

[tex]81 = 3 \times 3 \times 3 \times 3 \\ 81 = {3}^{4} [/tex]

Maka,

Akar pangkat dua dari 81 adalah

[tex] \sqrt{81} = {3}^{2} \\ = 9[/tex]

Sedangkan akar pangkat tiganya adalah

[tex] \sqrt[3]{81} = {3}^{3} \times 3 \\ = 3 \sqrt[3]{3} [/tex]

23. ^' Tabel akar pangkat 3 1-100


Kelas : VII (1 SMP)
Materi : Bilangan Bulat
Kata Kunci : bilangan, akar, pangkat, tiga

Pembahasan :
Akar pangkat tiga merupakan operasi kebalikan (invers) dari operasi pangkat tiga.
 
Untuk menghitung akar pangkat tiga dari bilangan a, dapat ditentukan dengan menggunakan sifat berikut.
∛a = p ⇔ p³ = a,
dengan p bilangan bulat.
(∛a dibaca akar pangkat tiga dari a).

Mari kita lihat soal tersebut.
Tentukan tabel akar pangkat tiga dari 1 - 100!

Jawab :
Berikut tabel akar pangkat tiga dari 1 - 100.

Akar pangkat tiga    hasil

∛1                              ∛1 = ∛1³ = 1;
∛8                             ∛8 = ∛2³ = 2;
∛27                           ∛27 = ∛3³ = 3;
∛64                           ∛64 = ∛4³ = 4.

Tambahan : tabel akar pangkat tiga dari 100 - 1000.
Akar pangkat tiga    hasil
∛125                          ∛125 = ∛5³ = 5;
∛216                          ∛216 = ∛6³ = 6;
∛343                         ∛343 = ∛7³ = 7;
∛512                          ∛512 = ∛8³ = 8;
∛729                         ∛729 = ∛9³ = 9;
∛1.000                       ∛1.000 = ∛10³ = 10.

24. Tabel fungsi y=3 pangkat x + 1


melalui titik A(a,-2)

y = 3^x - 3 ^(x+1)

-2 = 3^a - 3^(a + 1)

-2 = 3^a - (3 . 3^a)

-2 = 3^a (1 - 3)

-2 = 3^a (-2)

1 = 3^a

3⁰ = 3^a

a = 0


melalui B(b,-18)

y = 3^x - 3 ^(x+1)

-18 = 3^b - 3^(b + 1)

-18 = 3^b - (3 . 3^b)

-18 = 3^b (1 - 3)

-18 = 3^b (-2)

9 = 3^b

3² = 3^b

b = 2


a + b = 0 + 2 = 2
