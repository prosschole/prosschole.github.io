19. Janok slowin Pagongan pada peta 1,5cm peta ito 1.800.000Jarak sebenarnya slawi-Pagongan adalah​


Diketahui :
Jp = 1,50 cm
Skala = 1 : 1.800.000

Ditanya :
Js?

Jawab :
Js = Jp : Skala
Js = 1,50 : [tex] \frac{1}{800.000} [/tex]
Js = 1,50 × 1.800.000
Js = 2.700.000 cm
Js = 27 kmDiket :

jarak pada peta

==> 1,5 cm

skala

==> 1 : 800.000

Ditanya :

jarak sebenarnya....???

Penyelesaian :

[tex]skala = \frac{jp}{js} \\ \frac{1}{800.000} = \frac{1.5}{js} \\ js = 800.000 \times 1.5 \\ js = 1.200.000 \\ js = 12 \: km[/tex]

Jawaban :

12 km
