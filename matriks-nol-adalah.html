Matriks Singular

13. apa alasan matriks nol pasti merupakan matrik persegi?


Berdasarkan elemen-elemen penyusunnya matriks dapat di bagi menjadi beberapa jenis yaitu salah satunya Matriks nol dimana ia adalah suatu matriks yang semua elemennya adalah 0 (nol).

Berdasarkan ordo Matriks, matriks dibedakan menjadi salah satunya matriks persegi, matriks persegi panjang,.

Matriks persegi adalah matriks yang banyak barisnya sama dengan banyak kolomnya, matriks ini memiliki ordo n × n.

Sedangkan matriks persegi panjang adalah matriks yang banyak barisnya sama dengan banyak kolomnya, matriks ini memiliki ordo m × n.

Namun matriks nol kemungkinan besar termasuk matriks persegi atau matriks persegi panjang, sehingga kemungkinan yang setiap unsurnya 0 berordo m x n dan berordo n x n.

Tentukanlah nilai kebenaran setiap pernyataan di bawah ini! a) Jika elemen kolom ke-1 matriks A semuanya nol, maka elemen kolom ke-1 matriks AB juga semuanya nol. b) Jika elemen pada baris ke-1 pada matriks A semuanya nol, maka elemen baris ke-1 matriks AB juga semuanya nol.


Kita ambil satu sampel. Misalkan matriks A dan B berordo 3 x 3 :

a.
[tex] A = \left[\begin{array}{ccc}0&a&b\\0&c&d\\0&e&f\end{array}\right] [/tex]
   [tex] B = \left[\begin{array}{ccc}g&h&i\\j&k&l\\m&n&o\end{array}\right] [/tex]
[tex]AB = \left[\begin{array}{ccc}0&a&b\\0&c&d\\0&e&f\end{array}\right] \left[\begin{array}{ccc}g&h&i\\j&k&l\\m&n&o\end{array}\right] [/tex]   
Kita lihat bahwa elemen kolom 1 matriks AB :
Kolom 1 matriks AB = {(aj + bm), (cj + dm), (ej + fm)}.
Maka pernyataan elemen kolom 1 matriks AB semuanya nol bernilai SALAH.

b.
[tex] A = \left[\begin{array}{ccc}0&0&0\\a&b&c\\d&e&f\end{array}\right] [/tex]
   [tex] B = \left[\begin{array}{ccc}g&h&i\\j&k&l\\m&n&o\end{array}\right] [/tex]
[tex]AB = \left[\begin{array}{ccc}0&0&0\\a&b&c\\d&e&f\end{array}\right] \left[\begin{array}{ccc}g&h&i\\j&k&l\\m&n&o\end{array}\right] [/tex]   
Kita lihat bahwa elemen baris 1 matriks AB :
Kolom 1 AB = {0, 0, 0}.
Maka pernyataan elemen baris 1  matriks AB semuanya nol bernilai BENAR.



30. Tentukan nilai x jika determinan setiap matriks berikut adalah nol


[tex]\text{determinan dari matriks}\begin{array}{rcl}\left(\begin{array}{cc}a&b&c&d\end{array}\right)\end{array}\text{dapat dicari dengan persamaan }\\\text{determinan matriks } =ad-bc[/tex]

kalau determinan matriksnya = 0, berarti

ad-bc = 0

a)

a = -3

b = x

c = x+5

d = 2

            ad-bc = 0

(-3)(2)-(x)(x+5) = 0

        -6-x²-5x = 0

        x²+5x+6 = 0

  x²+2x+3x+6 = 0

x(x+2)+3(x+2) = 0

      (x+2)(x+3) = 0

x = -2 atau x = -3

b)

               ad-bc = 0

(x+2)(2)-(-5)(2x) = 0

        2x+4+10x = 0

                  12x = -4

                      x = -1/3

c)

x(4x)-2x(4) = 0

     4x²-8x = 0

       x²-2x = 0

       x(x-2) = 0

x = 0 atau x = 2

d)

2(-5)-(-x)(x+3) = 0

     -10+x²+3x = 0

      x²+3x-10 = 0

 x²+5x-2x-10 = 0

x(x+5)-2(x+5) = 0

     (x-2)(x+5) = 0

x = 2 atau x = -5

e)

(x-1)(x+2)-2x(x+2) = 0

        (x+2)(x-1-2x) = 0

         (x+2)(-(x+1)) = 0

           -(x+2)(x+1) = 0

            (x+2)(x+1) = 0

x = -2 atau x = -1

f)

  2x(x-1)-(-1)(x-4) = 0

              2x²-x-4 = 0

                 2x²-x = 4

               x²-1/2x = 2

x²-1/2x+((-1/2)²/4) = 2+((-1/2)²/4)

         (x+(-1/2)/2)² = 33/16
