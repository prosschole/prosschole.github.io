Wangsulan

➡️ Basa kramane yaiku Prabu Yudhistira watakipun jujur lan srantos.

Penjelasan

Basa Krama

Digunakake kanggo wong nom menawa matur marang wong tuwa, anak buah marang pimpinan, lan wong sing durung kenal.

☞ {Krama alus}

→ basa kang wujude kabeh tembunge migunakake basa krama lan krama inggil.

✧ Titikan/Ciri-ciri Krama Alus :

- Tembung-tembunge krama kecampur Krama Inggil.

- Ater-ater lan panambang dikramakake.

- Tembung aku owah dadi kula.

- Tembung kowe owah dadi panjenengan.

- Kang dikramakake inggil perangan awak, sandhangan lan kriyane wong sing diajak guneman lan wong sing di gunem.

☆ Tuladha :

1. Dalemipun panjenengan wonten pundi?

2. Bu, panjenengan dipundhawuhi eyang sowan mrika sakmenika.

3. Mangga pinarak, panjenengan sampun dipuntengga bapak.

4. Menawi sampun dados tugasipun saged dipunkempalaken.

5. Bapak lan ibu sampun tindak dhateng Jogja kala wingi.
