13. Tentukan diketahui rumus kimia aluminium sulfat AL2(SO4)3 .berapakah mol masing masing unsur (AL,S dan O) terdapat dalam 1 mol aluminium sulfat ?


Jawaban:

Terlampir di penjelasan

Penjelasan:

Mr Al2(SO4)3 = ( 2 x Ar Al ) + ( 3 x Ar S ) + ( 12 x Ar O )

=> ( 2 x 27 ) + ( 3 x 32 ) + ( 12 x 16 )

=> 342

mol unsur = ( jumlah atom x Ar unsur ) / Mr senyawa x mol senyawa

mol Al = ( 2 x 27 ) / 342 x 1 mol

=> 0,158 mol

mol S = ( 3 x 32 ) / 342 x 1 mol

=> 0,281 mol

mol O = ( 12 x 16 ) / 342 x 1 mol

=> 0,561 mol
