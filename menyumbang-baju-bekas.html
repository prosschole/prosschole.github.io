= 24%

Baju balita sebanyak = 0,28 bagian

Baju anak-anak sebanyak = 6/25 bagian

Sisanya adalah baju untuk remaja.

Ditanyakan :

Baju remaja adalah ..... bagian

Penyelesaian :

Rubah semua data dalam bentuk pecahan :

Baju bayi = 35% = [tex]\frac{35}{100}[/tex]

Baju balita = 0,28 bagian = [tex]\frac{28}{100}[/tex]

Baju anak-anak = [tex]\frac{6}{25}[/tex]

Sehingga bagian baju remaja adalah :

= 1 - ([tex]\frac{35}{100}[/tex] + [tex]\frac{28}{100}[/tex] + [tex]\frac{6}{25}[/tex])

Samakan semua penyebut :

[tex]=\frac{100}{100}-(\frac{35}{100}+\frac{28}{100}+\frac{24}{100})[/tex]

[tex]=\frac{100}{100}-\frac{87}{100}[/tex]

[tex]=\frac{13}{100}[/tex]Dalam bentu desimal = 0,13Dalam bentu prosentase = 13%
