17. Ada yg bisa Diketahui nilai logaritma dasar : Log 2 = 0,3 Log 3 = 0,48 Tanpa kalkulator hitunglah ¹³log 18 dengan menggunakan nilai tsb !


Untuk bilangan prima, kita bisa menentukan nilainya dgn interpolasi linear,
Mengingat log 13 antara log 12 dan log 14
x1 = 12
y1 = log 12 = 2 log 2 + log 3 = 2(0,3)+0,48 = 0,6 + 0,48 = 1,08
x2 = 14
y2 = log 14 = log 2 + log 7
Karena log 7 antara log 6 dan log 8, maka
Log 7 = (0,9-0,78)(7-6)/(8-6) + 0,78
= 0,12/2 + 0,78 = 0,06 + 0,78 = 0,84
y2 = 0,3 + 0,84 = 1,14
Log 13 = (1,14-1,08)(13-12)/(14-12) + 1,08 = 0,06/2 + 1,08 = 0,03 + 1,08 = 1,11

26. Bagaimna mencari log Tanpa kalkulator


Cara menghitung logaritma dg menggunakan sifat-sifat logaritma.

Pembahasan

Logaritma merupakan kebalikan dari perpangkatan

ᵃlog b = n dalam bentuk perpangkatan → aⁿ = b

Cara menghitung logaritma dg menggunakan sifat-sifat logaritma :

Sifat-sifat logaritma

ᵃlog (b.c) = ᵃlog b + ᵃlog c.ᵃlog b x ᵇlog c = ᵃlog c.ᵃlog (ᵇ/ₓ) = ᵃlog b - ᵃlog x.ᵃlog b = 1/ᵇlog a.ᵃlog (b/c) = - ᵃlog (c/b).ᵃlog bⁿ = n. ᵃlog b.ᵃ²log b = ¹/₂. ᵃlog b.ᵃlog aⁿ = n.ᵃlog a = 1.
