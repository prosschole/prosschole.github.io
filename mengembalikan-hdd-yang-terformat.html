5. Bagaimana cara mengembalikan data yang diformat saat instal ulang?


Penjelasan:

Langkah 1

Silahkan buka dan jalankan EaseUS data recovery wizard yang sudah anda install.

Langkah 2

Pada jendela pertama yang muncul anda akan diminta memilih type file yang hendak dikembalikan. Type file tersebut dapat berupa Graphics, Audio, Document, Video, Email, dll. Pada bagian ini anda dapat memilih All File Types atau memilih beberapa type file saja. Setelah itu klik Next pada pojok kanan bawah.

Langkah 3

Pada jendela selanjutnya anda harus memilih lokasi atau drive partisi yang terhapus. Jika anda sudah menimpanya dengan partisi baru silahkan pilih partisi tersebut, jika belum (unalocated space) pilih driver tersebut. Jika sudah menentukan drive partisi silahkan klik Scan pada pojok kanan bawah.

Langkah 4

Kemudian tunggu sampai proses Scanning selesai.

Langkah 5

Setelah proses Scanning selesai, maka akan muncul daftar file yang merupakan isi dari partisi anda yang tehapus. Silahkan ceklis file atau folder mana saja yang hendak dikembalikan kemudian klik Recover pada pojok kanan bawah. Tetapi jika file anda tidak ditemukan silahkan lakukan Deep Scanning dengan menekan tombol Deep Scanning. Kemudian anda dapat melakukan recover setelah proses Deep Scanning selesai.

Langkah 6

Setelah anda klik tombol Recover sebelumnya, maka akan muncul jendela baru yang mengharuskan anda memilih sebuah direktori. Direktori ini merupakan tempat dimana file atau folder yang sudah direcover atau dikembalikan nantinya akan disimpan. Jadi silahkan anda memilih tempat dimana anda akan menyimpan file atau folder yang sudah dikembalikan.

Langkah 7

Setelah itu silahkan tunggu sampai proses Recover atau pengembalian data selesai dan anda akan menemukan file isi partisi anda yang sudah dikembalikan pada direktori yang anda tentukan sebelumnya.

7. mengapa HDD perlu di partisu


Hasil dari partisi HDD umumnya berupa drive. Misalkan ada 2 partisi, maka ada drive C dan drive D. Misalkan ada 3 partisi, maka ada drive C, drive D dan juga drive E.

HDD dipartisi umumnya untuk mengalokasikan data-data di dalamnya sesuai fungsinya. Misalnya di drive C berisi data-data sistem. drive D berisi data-data dokumen umum. drive E berisi data-data cadangan (backup).

27. Apa Itu RAM dan HDD?


Jawaban:

Ram adalah Random Access Memory, yang berfungsi untuk menyimpan memory singkat untuk kinerja dalam suatu cpu. Ram biasa berbentuk kepingan board yang bisa dipasangkan pada mother board.

Hardisk adalah komponen untuk penyimpanan data pengguna OS dan banyak lagi. hardisk berbentuk Bar yg di dalamnya terdapat piringan, namun skrng sdh banyak hrdisk yg ber tipe SSD.

Penjelasan:

RAM, random access memory. Berfungsi sebagai cache program, semua prpgram termasuk sistem operasi. Jadi, saat mnejalankan program (misalkan, Microsoft Word), data inti program dimasukkan terlebih dahulu ke RAM dan kemudian dijalankan prosesor. Saat program ditutup, kadang ada beberapa inti porgram yang tetap ada di dalam RAM sehingga saat anda jalankan program yamg sama, loading semakin cepat. Semakin besar RAM, semakin lancar program dijalankan.

Hard disk, HDD atau SSD. Berfungsi untuk menyimpan data pengguna, seperti file pekerjaan, foto dan musik. Semakin besar HDD, semakin banyak file yamg bisa anda simpan.


28. . Apakah perbedaan antara HDD dengan FDD


Kalau HDD adalah Cakram keras (dalam bahasa inggris sering disebut  harddisk atau harddisk drivedisingkat HDD atau hard drive disingkat HD) adalah sebuah komponen perangkat keras yang menyimpan data sekunder dan berisi piringan magnetis.

Kalau FDD adalah Penggerak kilat USB (dalam bahasa inggris sering juga disebut USB flash drive) adalah alat penyimpan memori kilat tipe NAND yang memiliki alat penghubung USB yang terintegrasi.
