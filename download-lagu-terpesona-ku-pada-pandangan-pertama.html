Pandangan -> n = 9, k = 3, 3

P = 9!/(3!×3!) = 9×8×7×6×5×4×3×2 / 3×2×3×2 = 9×8×7×5×4 =

10.080 susunan kata

↓

Mata -> n = 4, k = 2

P = 4!/2! = 4×3×2/2 = 4×3 =

12 susunan kata

↓

Yang -> n = 4

P = 4! = 4×3×2 = 12×2 =

24 susunan kata

↓

Mempesona -> n = 9, k = 2, 2

P = 9!/(2!×2!) = 9×8×7×6×5×4×3×2 / 2×2 = 9×8×7×6×5×3×2 =

90.720 susunan kata
