5. Tolong buatin progran c++ menampilkan bilangan genap dan bilangan ganjil 0 sampai 100 Dan total hasil dari bilangan genap berapa dan bilangan ganjil berapa Kirimin kode nya juga


Metode ini bisa dicapai dengan istilah LOOPING dengan FOR:
Dengan statement syntax seperti berikut:
for(init; condition; next){
//statement1
//statement2
//statement xx
}
init: dimana ini dilakukan pertama kali saat masuk ke for loop, ini adalah tempat dimana deklarasi variable ini akan digunakan sebagai penanda dalam loop.

condition: setelah semua statement dijalankan di dalam badan loop, program akan mengecek apakah condition itu masih berlaku(true) atau tidak. Jika ya, maka badan loop akan dijalankan kembali, jika tidak, maka program akan keluar dari for loop dan berpindah ke statement selanjutnya.

next: Setelah semua statement dijalankan, program akan melakukan apa yang ada di statement next tersebut, biasanya antara menambahkan atau mengurangi.

Contohnya pada di gambar pertama(baris 9 dan 18):
for(int i = 0; i < 100; i++){
//statements...
}
program akan berjalan dalam loop mendeklarasikan variable i dengan nilai 0; kondisi i < 100; aksi selanjutnya i bertambah 1.
Setelah semua statement dijalankan maka i akan bertambah 1 dan mengecek lagi apakah i masih < 100, jika IYA maka akan dijalankan lagi.

Bagian selanjutnya(baris 10 dan baris 19) di gambar pertama:
if(i % 2 == 0){/*statements*/} atau if(i % 2 == 1){/*statements*/}
operator % ini dinamakan modulus, yaitu sisa pembagiannya. Misalnya 4 % 2 akan menghasilkan 0, atau 5 % 2 akan menghasilkan 1. operator modulus ini berguna jika kita ingin memastikan kelipatan sebuah angka.

buatlah sebuah prosedur dengan nama tampilPrima, codingnya seperti di bawah ini . 
public void tampilPrima(int batas){ Prima prima = new Prima(); int counter = 1; System.out.print("Angka prima sampai angka " + batas + " : 2"); for (int i = 3; i <= batas; i++) { if(prima.cekPrima(i)){ System.out.print(" , " + i); counter++; } } System.out.println("\nTotal angka prima adalah "+ counter); } Fungsi ini bertujuan untuk menampilkan angka prima dari angka prima pertama sampai dengan angka yang menjadi batas.Angka yang menjadi batas kita dapatkan dari parameter.Kemudian saya juga ingin mengetahui berapa banyak angka prima yang muncul untuk itu saya membuat variabel counter . 
Kemudian gunakan looping for untuk medapatkan angka yang akan dicek apakah prima atau tidak. lalu gunakan fungsi pada pembahasan sebelumnya untuk mengeceknya dan jangan lupa fungsi ini di gunakan sebagai kondisi dalam percabangan if. jika if bernilai TRUE maka tampilkan angkanya dan juga incrementkan counter. Isi dari counter akan tampil di akhir looping for .
Selain itu saya juga ingin membuat fungsi yang tujuannya sama seperti di atas, cuma bedanya kalo di atas parameternya di gunakan untuk batas angka sedangkan di sini di gunakan untuk banyaknya angka prima yang ingin di tampilkan .beri nama fungsi ini dengan nama tampilkanPrimaSebanyak() , codingnya seperti di bawah . 
public void tampilPrimaSebanyak(int batas){ Prima prima = new Prima(); int counter = 0; int angka = 3; System.out.print("Angka prima sebanyak " + batas + " : 2"); while(true){ if(prima.cekPrima(angka)){ counter++; System.out.print(" , " + angka); } if(counter == batas){ break; } angka++; } } Berbeda dengan fungsi tampilPrima , disini saya memakai looping while . Saya belum tau sampai berapa looping yang di perlukan untuk menampilkan angka prima sebanyak misalnya 30 , while di sini looping yang paling cocok untuk di gunakan . While sengaja kita buat kondisinya selalu TRUE , cara agar loopingnya berhenti adalah dengan menggunakan code break yang kita tempatkan dalam if . 
Lalu ubah sedikit di mainya seperti code di bawah ini . 
public static void main(String[] args) { Prima prima = new Prima(); prima.bacaPrima(7); System.out.println(""); prima.tampilPrima(30); prima.tampilPrimaSebanyak(12); }

}

}

(gunakan perulangan for) [Bobot: 25] Contoh output program Input batasan angka : 20 ==================================== Angka-angka yang merupakan kelipatan 3 atau 5 kurang dari 20 adalah: 3 5 6 9 10 12 15 18 Jumlah angka genap : 4 angka Jumlah angka ganjil : 4 angka

Java Perulangan For

import java.util.Scanner;

// Wrapper class untuk program Kelipatan3Atau5

public class Kelipatan3Atau5 {

  // Membaca dan mengembalikan nilai angka masukan

  private static int bacaInputBilangan(String teks) {

      int tmp = 0;

      Scanner in = new Scanner(System.in);

      try {

          System.out.print(teks);

          tmp = in.nextInt();

      } catch (Exception e) {

          System.out.println("Terjadi kesalahan.");

      } finally {

          in.close();

      }

      return tmp;

  }

  private static boolean apakahKelipatan(int bilangan, int faktor) {

      return bilangan % faktor == 0;

  }

  private static boolean apakahGenap(int bilangan) {

      return bilangan % 2 == 0;

  }

  // Program Utama

  public static void main(String[] args) {

      int batas = 0, nGenap = 0, nGanjil = 0;

      // Minta input batas bilangan

      batas = bacaInputBilangan("Input batasan angka: ");

      System.out.println("================================");

      // Batas harus lebih dari atau sama dengan 3

      // agar minimal ada 1 output

      if (batas < 3) {

          System.out.println("Batasan angka tidak boleh kurang dari 3.");

      }

      else { // batasan angka valid

          System.out.print("Angka-angka yang merupakan kelipatan 3 atau 5 dan kurang dari "

              + String.valueOf(batas)

              + " adalah: ");

          // Iterasi dari 1 hingga batas-1

          for (int i=1; i < batas; i++) {

              if (apakahKelipatan(i, 3) || apakahKelipatan(i, 5)) {

                  // cetak angka

                  System.out.print(String.valueOf(i) + " ");

                  // counter genap dan ganjil

                  if (apakahGenap(i)) nGenap++;

                  else nGanjil++;

              }

          }

          System.out.println();

          // Cetak jumlah angka genap dan ganjil

          System.out.println("Jumlah angka genap : " + String.valueOf(nGenap) + " angka.");

          System.out.println("Jumlah angka ganjil: " + String.valueOf(nGanjil) + " angka.");

      }

  }

}
