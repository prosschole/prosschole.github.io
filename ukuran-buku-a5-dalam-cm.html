21. 1. Sebuah toko alat lukis menyediakan buku gambar sket (sketchbook) ukuran A5, A4, dan A3. Harga 2 buku gambar A5 dan 3 buku gambar A4 adalah Rp74.000,00. Harga 1 buku gambar A5 dan 2 buku gambar A3 adalah Rp76.000,00. Harga 1 buku gambar A4 lima belas ribu rupiah lebih murah daripada harga 1 buku gambar A3. Tentukan: a. harga 1 buku gambar A3; b. harga 4 buku gambar A5. pls sama caranya yaa​


a. Harga 1 buku gambar A3 adalah Rp 33.000,00.

b. Harga 4 buku gambar A5 adalah Rp 40.000,00.

Penjelasan dengan langkah-langkah:

Diketahui:

Harga 2 buku gambar A5 dan 3 buku gambar A4 adalah Rp74.000,00.

Harga 1 buku gambar A5 dan 2 buku gambar A3 adalah Rp76.000,00.

Harga 1 buku gambar A4 lima belas ribu rupiah lebih murah daripada harga 1 buku gambar A3.

Ditanya:

Tentukan:

a. harga 1 buku gambar A3;

b. harga 4 buku gambar A5.

Jawab:

Misalkan:

buku gambar A5 = a

buku gambar A4 = b

buku gambar A3 = c

2a + 3b = 74.000         (persamaan 1)

a + 2c = 76.000           (persamaan 2)

b = c - 15.000              (persamaan 3)

Substitusikan nilai b pada persamaan 3 ke persamaan 1.

2a + 3b = 74.000

2a + 3(c - 15.000) = 74.000

2a + 3c - 45.000 = 74.000

2a + 3c = 74.000 + 45.000

2a + 3c = 119.000                       (persamaan 4)

Eliminasi persamaan 2 dan 4.

a + 2c = 76.000             (dikali 2)

2a + 3c = 119.000           (dikali 1)        

2a + 4c = 152.000

2a + 3c = 119.000       -

c = 33.000

Substitusikan nilai c pada persamaan 2.

a + 2c = 76.000

a + 2(33.000) = 76.000

a + 66.000 = 76.000

a = 76.000 - 66.000

a = 10.000

Substitusikan nilai c pada persamaan 3.

b = c - 15.000

b = 33.000 - 15.000

b = 18.000

Jadi, harga 1 buku gambar A3 adalah Rp 33.000,00.

Harga 4 buku gambar A5

4a = 4 x 10.000

= 40.000

Jadi, harga 4 buku gambar A5 adalah Rp 40.000,00
