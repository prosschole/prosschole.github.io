13. Start yg digunakan dalam perlombaan lari jarak jauh adalah.... A. Start melayang b. Start berdiri. C. Start kombinasi


          Start yang digunakan pada lari jarak jauh adalah Start Berdiri, Start yang digunakan pada lari jarak pendek adalah Start Jongkok, dan Start yang digunakan pada lari estafet adalah Start Jongkok dan Start Melayang.

Pembahasan

           Penjelasan macam-macam start adalah sebagai berikut :  

START JONGKOK

           Start Jongkok (crouching start) adalah start yang digunakan pada cabang olahraga atletik lari jarak pendek. Start Jongkok digunakan pada panjang lintasan 100m, 200m, dan 400m.

________________

START BERDIRI

           Start Berdiri (Standing Start) adalah start yang digunakan pada cabang olahraga atletik lari jarak menengah dan lari jarak jauh. Start berdiri digunakan pada panjang lintasan 800m, 1500m, 5000m, dan 10.000m.

________________

START MELAYANG

           Start Melayang (Flying Start) adalah start yang digunakan dalam olahraga cabang atletik lari estafet. Start melayang digunakan oleh pemain ke-2, ke-3, dan ke-4 dalam olahraga estafet

14. Start yang digunakan dalam perlombaan jalan cepat adalah... a. start melayang b. start jongkok c. start berdiri d. start berlari


Start yang digunakan dalam perlombaan jalan cepat adalah (C) start berdiri.

Pembahasan

Jalan cepat adalah gerakan berjalan dengan cara melangkah tanpa adanya putus kontak dengan tanah. Gerakan melangkah pada jalan cepat deilkaukan dengan cara ketika kaki depan melangkah, kaki depan harus menyentuh tanah sebelum kaki kiri melangkah meninggalkan tanah. Dalam jalan cepat, terdapat beberapa teknik dasar yang harus dikuasai oleh setiap pemain. Teknik dasar dalam jalan cepat adalah.

1. Teknik start.

Teknik start pada jalan cepat dilakukan dengan teknik start berdiri. Start berdiri dilakukan dengan posisi badan berdiri tegap dengan salah sati kaki di depan dan kaki lainnya di belakag. Posisi badan sedikit condong kedepan dan posisi kedua tangan di tekuk di samping badan dan dalam keadaan relaks.

2. Teknik berjalan.

Teknik berjalan pada jalan cepat dilakukan dengan kedua kaki tidak boleh putus kontak dengan tanah. Ketika kaki kanan melangkah kedepan dan mendarat ke tanah, sesegera mungkin angkat kaki kiri untuk melangkah ke depan. Gerakan ayunan lengan dilakukan dengan mengayun setinggi dada dan pinggul tetap rendah selama berjalan.

3. Teknik memasuki garis finish.

Dalam jalan cepat, tidak ada teknik khusus ketika memasuki garis finish. Peserta harus tetap berjalan biasa hingga melewati garis finish.

21. Bagas sedang mengikuti perlombaan lari antar sekolah. Dalam perlombaan lari tersebut, terdapat beberapa jenis start yang digunakan. Salah satunya adalah start jongkok. Bagaimana tahapan dalam melakukan start jongkok​


Start Pendek Posisikan kedua kaki terbuka selebar bahu.

Kaki depan menginjak papan start depan.

Kaki belakang menginjak papan start belakang.

Lutut kaki belakang berada di posisi sejajar dengan jari-jari kaki depan.

Atau posisikan lutut kaki depan kurang lebih satu jengkal dari telapak kaki depan.

Posisikan letak tangan membuka sejauh dengan lebar garis bahu.

Posisikan ibu jari di depan tangan agar mendapatkan manfaat peregangan otot.

Kemudian posisikan kepala menunduk ke bawah dengan mata lurus melihat ke depan, kemudian lakukan tolakan lari jarak pendek.

Start Medium Posisikan awalan dengan berjongkok dan tempatkan tangan secara terbuka selebar bahu dan sejajar dengan bahu.

Lakukan posisi lutut kaki belakang sejajar dengan tungkai kaki depan.

Kemudian posisikan lutut kaki ke belakang secara sejajar ke kaki depan. Mundurkan sedikit ke belakang sampai tangan kira-kira 20-30 cm.

Posisikan kepala sedikit ke bawah dengan pandangan ke depan. Lakukanlah lari jika aba-aba sudah terdengar.

Start Jauh (Long Start) Posisikan letak tangan secara sejajar dengan garis di lapangan. Posisikan juga bahu secara sejajar dengan garis di lapangan.

Posisikan lutut condong ke belakang tubuh. Perhatikan bahwa posisi kaki depan mesti ditungkaikan.

Posisikan pula lutut condong ke belakang. Hal ini dilakukan agar posisi lutut agak jauh dari tangan.

29. start yang digunakan dalam perlombaan jalan cepat adalah... A. start melayang b. start jongkok c. start berdiri d. start berlari


Start yang digunakan dalam perlombaan jalan cepat adalah  Jawaban  Pendahuluan  

Jalan cepat biasanya ada di dalam sebuah perlombaan. Jalan cepat berbeda dengan sebuah kegiatan lari walaupun sama - sama melangkah maju dan sama dalam pelaksanaannya yang di awali dengan start dan di akhiri dengan melewati garis finish. Jalan cepat sendiri merupakan sebuah kegiatan berjalan/melangkah maju yang di lakukan dengan mempercepat langkah kakinya dengan irama tertentu sehingga tetap terpelihara dan tidak terputus.  

Pembahasan  

Berikut prinsip dasar pada jalan cepat, antara lain sebagai berikut :  

Start, pada perlombaan jalan cepat di lakukan dengan cara start berdiri. Tidak ada teknik khusus didalam perlombaan ini, berbeda dengan perlombaan lari berpengaruh pada hasil perlombaan maka diperlukannya teknik khusus yang harus dipelajari atau dilatih.Jalan cepat, teknik yang sama dengan perlombaan lari yakni dengan melangkahkan kaki belakang kedepan bersamaan dengan lengan yang diayunkan.  Finish, tidak adanya teknik khusus dalam mencapai finish, jalan cepat hanya dengan berjalan terus dengan menjaga iramanya hingga melewati garis finish.  Kesimpulan  

Jadi, dapat disimpulkan bahwa start yang digunakan dalam perlombaan jalan cepat adalah start berdiri. Sesuai pertanyaan diatas jawaban yang tepat adalah C. Start Berdiri.
