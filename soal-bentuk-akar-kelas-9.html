Perangkatan

= 9² x 9³

= 9^2 + 3

= 9⁵

Sifat nya a^m x a ^n = a^m + n

CMIIW

Bentuk sederhana = 9⁵

Hasil = 59.049

_________________________________

Pendahuluan

Bilangan Berpangkat ( Eksponent ) adalah bilangan yang dikalikan berulang sebanyak pangkatnya. Bilangan Berpangkat terdiri dari :

Bilangan Pangkat PositifBilangan Pangkat NegatifBilangan Pangkat Nol

Sifat - sifat Bilangan Berpangkat :

[tex] \sf \bf \: {a}^{0} = \: 1[/tex][tex] \sf \bf \: {a}^{b} \times {a}^{c} = \: {a}^{b + c} [/tex][tex] \sf \bf \: {a}^{b} \div {a}^{c} = \: {a}^{b - c} [/tex][tex] \sf \bf \: ( {a}^{b} )^{c} = \: {a}^{bc} [/tex][tex] \sf \bf \: ({ab})^{c} = {a}^{c} {b}^{c} [/tex]Dll

_________________________________

Diketahui

9² × 9³

Ditanya

Bentuk sederhana & Hasil !

JawabBentuk sederhana = 9⁵Hasil = 59.049

Penyelesaian

Gunakan sifat :

[tex] \sf \: {a}^{b} \times {a}^{c} = \: {a}^{b + c} [/tex]

Sehingga :

[tex] \sf \: = \: {9}^{2} \times {9}^{3} [/tex]

[tex] \sf \: = \: {9}^{2 + 3} [/tex]

[tex] \boxed{ \sf \bf \huge \: = \: {9}^{5}} \sf \: - - > \: bentuk \: sederhana[/tex]

[tex] \sf \: = \: 9 \times 9 \times 9 \times 9 \times 9[/tex]

[tex] \boxed{ \sf \bf \huge \: = \: 59.049} \sf \: - - > \: hasil[/tex]

Bentuk sederhana = 9⁵

Hasil = 59.049

Kesimpulan

Bentuk sederhananya adalah 9⁵

Hasilnya adalah 59.049

Sifat - sifat Bilangan berpangkat :  

1. [tex]a^{m}\times{a^{n}=a^{m+n}[/tex]  

2. [tex]a^{m}:a^{n}=a^{m-n}[/tex]  

3. [tex](a^{m)^{n}}=a^{m\times{n}[/tex]  

4. [tex]a^{m}\times{b^{m}=(ab)^{m}[/tex]  

5. [tex]a^{0}=1[/tex]  

6. [tex]\frac{1}{a^{n}}=a^{-n}[/tex]  

Dengan m, n bilangan bulat  

Bentuk Pangkat dengan bilangan berupa pecahan :

1. [tex](\frac{a}{b} )^{n}=\frac{a^{n}}{b^{n}}[/tex]

2. [tex](\frac{a}{b})^{-n}=(\frac{b}{a})^n=\frac{b^{n}}{a^{n}}[/tex]

3. [tex](\frac{a}{b})^0=1[/tex]

untuk a ≠ 0 , b ≠ 0

3⁰= 1

Jadi : 3 + 1 = 4

b. -2⁻⁶ = [tex]-\frac{1}{2^{6}}[/tex]

[tex](\frac{a}{b})^{-n}=(\frac{b}{a})^n=\frac{b^{n}}{a^{n}}[/tex] :

-2⁶ = [tex]-\frac{1}{2^{6}}[/tex]

= [tex]-\frac{1}{64}[/tex]

c. -3³ x (-3⁰) = -27

-3³ = -27

-3⁰ = 1

-27 x 1 = -27

[tex](\frac{1}{6})^{-3}=6^{3}[/tex] = 216

e. [tex](-\frac{2}{3})^{-2}=\frac{9}{4}[/tex]

No 3

  [tex]\frac{2^7}{2^6}=2^1=2[/tex]

 [tex](-\frac{1}{4} )^{-4}=-4^4[/tex]

 [tex](-\frac{1}{4})^0=1[/tex]

 [tex](-\frac{1}{4})^4=-\frac{1}{4^4}[/tex]

[tex]-4^4\times{1}\times{-\frac{1}{4^4}}=1[/tex]

c. [tex]\frac{1}{3^{5}}\times{\frac{1}{3^{-7}}}=3^{2}[/tex]

[tex]\frac{1}{3^5}\times{3^7}=\frac{3^7}{3^5}=3^{7-5}=3^2=9[/tex]
