1. Apa rumus least square?


Metode Least Square : Metode yang digunakan untuk analisis time series adalah Metode Garis Linier Secara Bebas (Free Hand Method), Metode Setengah Rata-Rata (Semi Average Method), Metode Rata-Rata Bergerak (Moving Average Method) dan Metode Kuadrat Terkecil (Least Square Method). Dalam hal ini akan lebih dikhususkan untuk membahas analisis time series dengan metode kuadrat terkecil yang dibagi dalam dua kasus, yaitu kasus data genap dan kasus data ganjil. Secara umum persamaan garis linier dari analisis time series adalah : Y = a + b X.

Keterangan :

Y adalah variabel yang dicari trendnya dan X adalah variabel waktu (tahun).

Sedangkan untuk mencari nilai konstanta (a) dan parameter (b) adalah :

 a =  ΣY / N   Dan b =(ΣYx)/ΣX2

Metode Least Square (kuadrat terkecil) Metode ini paling sering digunakan untuk meramalkan y,karena perhitungannya lebih teliti. Rumus Mencari persamaan garis trendY’ = α+bx,  α = (∑У)/n  b =(∑Уx)/ ∑x^2 Untuk melakukan perhitungan diperlukan nilai variabel waktu (x), jumlah nilai variable waktu adalah nol atau ∑x=0.

1     Untuk n ganjil maka n= 2k+1 X k+1=0

  Jarak antara 2 waktu diberi nilai satu satuan

  Diatas 0 diberi tanda negatif ( - )

  Dibawahnya diberi tanda positif ( + )

2     Untuk n genap maka n =2k X1/2 [k+(k+1)]=0

  Jarak antara 2 waktu diberi nilai dua satuan

  Diatas 0 diberi tanda negatif ( - )

  Dibawahnya diberi tanda positif ( + )

Rumus persamaan trend linearnya dengan menggunakan metode least square! b. Prediksikan berapa jumlah wisatawan pada tahun 2030!tolong dong Sekarang soale diserahkan​


a. Rumus persamaan trend linier menggunakan metode least square adalah:

Y = a + bx

dengan:

b = (nΣXY - ΣXΣY) / (nΣX^2 - (ΣX)^2)

a = (ΣY - bΣX) / n

n = jumlah data = 10

ΣX = 20 + 31 + 40 + 20 + 35 + 40 + 45 + 50 + 45 + 52 = 378

ΣY = 20 + 31 + 40 + 20 + 35 + 40 + 45 + 50 + 45 + 52 = 378

ΣXY = (20x2011) + (31x2012) + (40x2013) + (20x2014) + (35x2015) + (40x2016) + (45x2017) + (50x2018) + (45x2019) + (52x2020) = 383,382

ΣX^2 = 2011^2 + 2012^2 + 2013^2 + 2014^2 + 2015^2 + 2016^2 + 2017^2 + 2018^2 + 2019^2 + 2020^2 = 40,361,825

Mengganti nilai ke rumus b dan a, maka diperoleh:

b = (10 x 383,382 - 378 x 378) / (10 x 40,361,825 - 378^2) = 1.505

a = (378 - 1.505 x 378) / 10 = -744.45

Sehingga, persamaan trend linier adalah:

Y = -744.45 + 1.505X

b. Untuk memprediksi jumlah wisatawan pada tahun 2030, nilai X nya adalah 2030. Maka,

Y = -744.45 + 1.505(2030) = 2539.95 ribu wisatawan

Jadi, jumlah wisatawan pada tahun 2030 diprediksi sebanyak 2,539,950 orang.

14. Mengapa biaya transportasi yang dihasilkan dengan metode least cost lebih kecil dari pada biaya transportasi yang dihasilkan dengan metode NWCR?


Metode Least Cost dan Metode Northwest Corner Rule (NWCR) adalah dua metode yang umum digunakan dalam perencanaan transportasi untuk mengoptimalkan biaya transportasi.

Metode Least Cost adalah metode yang mempertimbangkan biaya transportasi terendah untuk mengalokasikan persediaan dari sumber ke tujuan. Metode ini memperhitungkan biaya transportasi per unit dari setiap sumber ke setiap tujuan dan mencari kombinasi yang paling ekonomis.

Sementara itu, Metode NWCR adalah metode yang mengalokasikan persediaan dari sumber ke tujuan secara berurutan, dimulai dari sudut barat laut (northwest corner) dari tabel alokasi persediaan dan kemudian mengurangi persediaan dari sumber tersebut dan memindahkannya ke tujuan hingga persediaan habis.

Perbedaan dalam hasil biaya transportasi yang dihasilkan oleh kedua metode ini terletak pada fakta bahwa Metode Least Cost mempertimbangkan biaya transportasi per unit, sementara Metode NWCR tidak mempertimbangkan faktor biaya transportasi per unit dalam urutan alokasi.

Oleh karena itu, Metode Least Cost dapat menghasilkan biaya transportasi yang lebih rendah dibandingkan Metode NWCR karena ia mempertimbangkan biaya per unit dan mencari solusi yang paling ekonomis. Sedangkan Metode NWCR mengabaikan biaya per unit dalam urutan alokasi dan hanya mengalokasikan persediaan berdasarkan urutan geografis.

b. Buat persamaan garis trend linear dengan menggunakan metode least square. Metode least square dapat digunakan untuk menemukan garis lurus yang terbaik untuk data penjualan. Garis lurus ini akan memberikan persamaan matematis yang menggambarkan tren data penjualan selama 11 tahun.

Persamaan garis trend linear: y = mx + c

Dimana:

y = penjualan

x = tahun

m = gradien (slope)

c = intersep (intercept)

Rumus untuk menghitung gradien dan intersep adalah sebagai berikut:

m = (nΣxy - ΣxΣy) / (nΣx^2 - (Σx)^2)

c = (Σy - mΣx) / n

Dimana:

n = jumlah data (11 tahun)

Σ = jumlah total

x = tahun

y = penjualan

Σy = 1650

Σxy = 36300

Σx^2 = 506

n = 11

m = (11(36300) - (66)(1650)) / (11(506) - (66)^2) = 10

c = (1650 - (10)(66)) / 11 = 10
