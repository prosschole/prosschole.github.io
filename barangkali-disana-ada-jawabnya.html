b. asamnya HNO₃, basa konjugasinya NO₃⁻

2. pakenya rumus buffer asam yah.
pH [tex]=-log 10^{-5} \frac{[0,1]}{[0,2]} \\ =-log5. 10^{-5} \\ =5-log5[/tex]

pH NaOH = 12, maka pOH-nya 2. M NaOH = 10⁻². persamaan di bawah ini, yg kanan asam, yg kiri basa.
V × M = V × M
V × 0,05 = 60 × 0,01
V = 12 mL

15. Barangkali ada yang bisa bantu jawab nomor.11


Kecepatan Max = 75 Km/Jam , Kecepatan Min = 10 Km/Jam , Kecepatan Motor = x Km/Jam

Fungsi Komsumsi BBM :

F(x) = [tex]\frac{1}{200} (\frac{1600}{x}+x)[/tex]

     = [tex]\frac{1}{200} (\frac{1600}{x}+\frac{x^{2}}{x})[/tex]

    = [tex]\frac{8}{x}+\frac{x^{2}}{200x}[/tex]

    =[tex]\frac{1}{x}(8+\frac{x^{2}}{200})[/tex]

Fungsi Biaya :

C(x) = [tex]\frac{9600}{x}+\frac{60x^{2}}{x}[/tex]

Biaya nya akan paling hemat jika Kecepatan nya minimum , berarti x = 10 , dan Konsumsi minimal nya adalah :

F(10) = [tex]\frac{1}{10}(8+\frac{10^{2}}{200})[/tex]

       = [tex]\frac{1}{10}(8+\frac{100}{200})[/tex]

       = [tex]\frac{1}{10}(8+\frac{1}{2})[/tex]

       = [tex]\frac{1}{10}(\frac{17}{2})[/tex]

F(10)  = [tex](\frac{17}{20})[/tex] = 0,85 Liter/Km

Jadi Motor itu akan menghabiskan 0,85 Liter/Km BBM kalau motornya bergerak dengan kecepatan 10 Km/Jam

24. Barangkali ada yg bisa bantu jawab


Penjelasan dengan langkah-langkah:

c)

[tex]f'(t) = \frac{1}{ \sqrt{t + 1} } [/tex]

[tex]f(t) =\int f'(t)[/tex]

[tex] = \int \frac{1}{ \sqrt{t + 1} } dt \\ =\int (t + 1) ^{ - \frac{1}{2} } dt [/tex]

misal

u = t+1

du = dt

[tex] = \int {u}^{ - \frac{1}{2} } du[/tex]

[tex] = \frac{ {u}^{( - \frac{1}{2} + \frac{2}{2} )} }{ - \frac{1}{2} + \frac{2}{2} } + c \\ = \frac{ {u}^{ \frac{1}{2} } }{ \frac{1}{2} } + c \\ = 2 \sqrt{u} + c \\ = 2 \sqrt{t +1 } + c[/tex]

f(3) = 18

[tex] 2\sqrt{3+1}+c = 18\\4+c = 18\\c = 18-4 = 14[/tex]

maka

[tex]f(t) = 2\sqrt{t+1}+14[/tex]

d)

[tex]f'(t) = \sqrt{2t - 1} [/tex]

[tex]f(t) =\int f'(t)[/tex]

[tex] = \int \sqrt{2t - 1} dt \\ = \int(2t - 1) ^{ \frac{1}{2} } dt[/tex]

misal

u = 2t-1

du = 2 dt

[tex]dt = \frac{du}{2} [/tex]

[tex] = \int {u}^{ \frac{1}{2} } \frac{du}{2} [/tex]

[tex] = \frac{ {u}^{ (\frac{1}{2} + \frac{2}{2} )} }{ \frac{1}{2} + \frac{2}{2} } + c \\ = \frac{ {u}^{ \frac{3}{2} } }{ \frac{3}{2} } + c \\ = \frac{2}{3} u \sqrt{u} + c \\ = \frac{2}{3} (2t - 1) \sqrt{2t - 1} + c[/tex]

[tex] f(\frac{1}{2})=-1[/tex]

[tex]\frac{2}{3}(2(\frac{1}{2})-1)\sqrt{2(\frac{1}{2})-1}+c= -1\\0+c = -1\\c = -1[/tex]

maka

[tex]f(t) = \frac{2}{3}(2t-1)\sqrt{2t-1}-1[/tex]
