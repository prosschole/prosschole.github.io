1. Penyelesaian pertidaksamaan |x-5| ≤ |x+7| adalah (pertidaksamaan harga mutlak)


|x - 5| <= |x + 7|
√(x - 5)² <= √(x + 7)²
(x - 5)² - (x + 7)² <= 0
(x - 5 + x + 7) . (x - 5 - x - 7) <= 0
(2x + 2) . (-12) <= 0
-12 <= 0

2x + 2 >= 0
2x >= -2
x >= -2/2
x >= -1

Maka, x >= -1

Mutlak adalah sepenuhnya atau seutuhnya dan tidak tergantung pada kondisi apapun sebagai syarat.

Nilai mutlak bilangan positif atau nol adalah bilangan itu sendiri dan nilai mutlak bilangan negatif adalah lawan dari bilangan tersebut.

Berikut ini adalah sifat-sifat nilai mutlak:

[tex]1) \: |ab| = |a| \: |b| [/tex]

[tex]2) \: | \frac{a}{b} | = | \frac{a}{b} | [/tex]

[tex]3) \: |a + b| \leqslant |a| + |b| [/tex]

[tex]4) \: |a - b| \geqslant | |a| - |b | | [/tex]

[tex]5) \: |x| ^{2} = {x}^{2} [/tex]

Berikut adalah langkah-langkah pengerjaan dari soal diatas:

(i) Pisahkan ke dalam beberapa kasus yang memungkinkan.

(ii) Selesaikan Pertidaksamaan tersebut

(iiii) Temukan irisan

14. Tentukan nilai penyelesaian pertidaksamaan harga mutlak [6-3x] < 12!


Jawab:

Penjelasan dengan langkah-langkah:

|6 - 3x| < 12 (kuadratkan kedua ruas)

(6 - 3x)² < 12²

36 - 36x + 9x² < 144

9x² - 36x + 36 - 144 < 0

9x² - 36x - 108 < 0 (Kedua ruas dibagi 9 agar menjadi lebih sederhana)

x² - 4x - 12 < 0

(x - 6) (x + 2) < 0

x - 6 = 0

x = 6

x + 2 = 0

x = -2

+ - +

-2 6

Untuk x < -2, x = -3 disubstitusikan ke pertidaksamaan x² - 4x - 12 < 0, sehingga:

(-3)² - 4(-3) - 12 < 0

9 + 12 - 12 < 0

9 < 0 (Tidak memenuhi syarat)

Untuk -2 < x < 6, x = 0 disubstitusikan ke pertidaksamaan x² - 4x - 12 < 0, sehingga:

(0)² - 4(0) - 12 < 0

0 + 0 - 12 < 0

0 < 0 (memenuhi syarat)

Untuk x > 6, x = 7 disubstitusikan ke pertidaksamaan x² - 4x - 12 < 0, sehingga:

(7)² - 4(7) - 12 < 0

49 - 28 - 12 < 0

9 < 0 (Tidak memenuhi syarat)

Jadi HP = {x | -2 < x < 6 ; x € R}.

22. Penyelesaian dari pertidaksamaan harga mutlak | 2x + 1 | < 5 adalah...


|2x + 1| → 2x + 1, untuk 2x + 1 ≥ 0 ⇔ x ≥ -1/2
             → -(2x + 1), untuk 2x + 1 < 0 ⇔ x < -1/2
Untuk batasan x ≥ -1/2  ....(1)
|2x + 1| < 5
2x + 1 < 5
2x < 4
x < 2 ...(2)
Dari (1) dan (2) diperoleh -1/2 ≤ x < 2

Untuk batasan x < -1/2 ...(3)
-(2x + 1) < 5
-2x - 1 < 5
-2x < 6
-x < 3
x > -3  ...(4)
Dari (3) dan (4) diperoleh -3 < x < -1/2

Dari penyelesaian kedua batasan
→ -1/2 ≤ x < 2
→ -3 < x < -1/2

Maka diperoleh penyelesaian
HP =  {x | -3 < x < 2, x ∈ R}
