10. mengapa HDD perlu di partisu


Hasil dari partisi HDD umumnya berupa drive. Misalkan ada 2 partisi, maka ada drive C dan drive D. Misalkan ada 3 partisi, maka ada drive C, drive D dan juga drive E.

HDD dipartisi umumnya untuk mengalokasikan data-data di dalamnya sesuai fungsinya. Misalnya di drive C berisi data-data sistem. drive D berisi data-data dokumen umum. drive E berisi data-data cadangan (backup).

20. Apa perbedaan antara hdd dan ssd


HDD (Hard Disk Drive) merupakan perangkat penyimpanan paling umum, generasi lama serta menggunakan teknologi yang tidak berubah sejak puluhan tahun lalu. HDD berbasis platter (piringan) berputar untuk menyimpan data, karena itu cukup beresiko saat bekerja karena komponen-komponennya bergerak.SSD (Solid State Drive) merupakan merupakan perangkat penyimpanan generasi baru, berbasis teknologi flash memory untuk menyimpan data. Karena menggunakan Flash, SSD tidak memiliki komponen bergerak. SSD sudah banyak digunakan oleh perangkat Notebook High-End dan Server.Ukuran HDD dan SSD berbeda. SSD lebih tipis dari HDD bahkan lebih tipis dari SSHDSSD biasanya menyediakan kinerja yang sangat maksimal untuk booting serta memiliki kinerja yang sangat tinggi untuk baca ataupun tulis yang sangat mendukung komputasi yang memerlukan Harga SSD lebih mahal dari HDD karena dapat memuat kapasitas sampai 30 TBPengolahan data pada HDD dilakukan dengan sinergi antara aktivitas mekanik dan elektrik. Sedangkan pada SSD karena tidak ada perangkat yang bergerak maka proses pengolahan data dilakukan secara elektrik seperti pada RAM juga flashdisk.Pada HDD, harus rajin dilakukan defrag agar kinerja HDD tidak menurun sedangkan pada SSD tidak perlu defrag karena SSD menggunakan Chip Flash untuk menyimpan data.SSD lebih tahan banting dibanding HDD. Jika terjatuh HDD bisa rusak total dan kehilangan data di dalamnya sedangkan SSD tidak

Jawaban:

Ram adalah Random Access Memory, yang berfungsi untuk menyimpan memory singkat untuk kinerja dalam suatu cpu. Ram biasa berbentuk kepingan board yang bisa dipasangkan pada mother board.

Hardisk adalah komponen untuk penyimpanan data pengguna OS dan banyak lagi. hardisk berbentuk Bar yg di dalamnya terdapat piringan, namun skrng sdh banyak hrdisk yg ber tipe SSD.

Penjelasan:

RAM, random access memory. Berfungsi sebagai cache program, semua prpgram termasuk sistem operasi. Jadi, saat mnejalankan program (misalkan, Microsoft Word), data inti program dimasukkan terlebih dahulu ke RAM dan kemudian dijalankan prosesor. Saat program ditutup, kadang ada beberapa inti porgram yang tetap ada di dalam RAM sehingga saat anda jalankan program yamg sama, loading semakin cepat. Semakin besar RAM, semakin lancar program dijalankan.

Hard disk, HDD atau SSD. Berfungsi untuk menyimpan data pengguna, seperti file pekerjaan, foto dan musik. Semakin besar HDD, semakin banyak file yamg bisa anda simpan.
