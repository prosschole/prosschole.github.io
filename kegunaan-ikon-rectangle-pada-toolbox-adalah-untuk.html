6. ikon rectangle terletak di group apa​


Jawaban:

Lokasi Ikon Rectangle.

Ikon renctangle berada pada menu insert, dan berada di sub menu illustration. Untuk membuka dan melihat dan menggunakan ikon rectangle berada pada toolbar Shapes.

Yang dimana pada toolbar shapes terdapat berbagai macam ikon seperti:

1. AutoShapes : berguna untuk menyisip gambar autoshapes

2. Line : berguna untuk menyisip menggambar garis

3. Arrow : berguna untuk menyisip menggambar anak panah

4. Rectangle : berguna untuk menyisip menggambar kotak

5. Oval : berguna untuk menyisip menggambar lingkaran

6. Text Box : berguna untuk menyisip menyisip kotak teks ( Mengetik Teks dalam kotak)

7. Insert Word Art : berguna untuk menyisip menyisip Word Art ( Tulisan Indah )

8. Insert Diagram : berguna untuk menyisip menyisipkan diagram

9. Clip Art : berguna untuk menyisipkan gambar yang disediakan oleh Microsoft Word

10. Insert Picture : berguna untuk menyisip menyisip gambar

11. Fill Color : berguna untuk merubah warna isi objek gambar

12. Line Color : berguna untuk merubah warna garis/bingkai objek gambar

13. Font Color : berguna untuk merubah warna huruf

14. Line Style : berguna untuk merubah jenis dan ketebalan garis gambar

15. Dash Style : berguna untuk merubah garis dalam bentuk titik-titik / garis 

putus-putus

16. Arrow Style : berguna untuk merubah bentuk pangkal ujung garis

17. Shadow Style : berguna untuk membuat bayangan objek gambar

18. 3-D Style : membuat gaya 3D.

Jawaban:

5 toolbox yang digunakan untuk menyeleksi gambar

Berikut adalah 5 macam bentuk toolbox yang akan berfungsi untuk melakukan penyeleksian daripada sebuah gambar

1.Rectangular Marque Tool: berfungsi untuk melakukan penyeleksian daripada area dengan cara menggunakan bentuk persegi.

2.Eliptical Marque Tool: berfungsi untuk melakukan penyeleksian daripada sebuah area dengan cara menggunakan sebbuah bentuk lingkaran.

3.Single Row Marque Tool: berfungsi untuk melakukan penyeleksian daripada sebuah gambar vertikal dengan cara menggunakan sebuah tinggi area seleksi 1 pixel.

4.Single Column Marque Tool: berfungsi untuk melakukan penyeleksian gambaran secara sebuah horizontal dengan ketinggian area seleksi 1 pixel.

5.Lasso Tool: berfungsi untuk melakukan penyeleksian daripada gambar yang dimana akan sesuai dengan sebuah gerakan mouse
