6. pola ke - n dari baris bilangan segitiga pascal adalah?pola ke - 100 dari baris bilangan pascal adalah?


misalnya pola ke-n dengan nomer suku ke - p

maka Un = [tex] \left[\begin{array}{ccc}n\\p-1\end{array}\right] = \frac{n!}{(n-(p-1))!(p-1)!} [/tex]

contoh =
jika n=5

U2=[tex] \frac{5!}{(5-1)!(2-1)!}= \frac{5}{1}=5 [/tex]

U3= [tex] \frac{5!}{(5-2)!(3-1)!}= \frac{5.4}{2!1!} =10[/tex]

U4= [tex] \frac{5!}{(5-2)!(3-1)!}= \frac{5.4}{2!1!} =10[/tex]

U5=[tex] \frac{5!}{(5-1)!(2-1)!}= \frac{5}{1}=5 [/tex]

jadi susunan  1 5 10 5 1

24. Cara menentukan jumlah bilangan pada baris ke-n pada pola bilangan Pascal


Pada pola bilangan pascal, jumlah bilangan pada:

Baris 1 = 1 = 2^0
Baris 2 = 2= 2¹
Baris 3 = 4 = 2²
Baris 4 = 8 = 2³

Jadi cara menentukan jumlah bilangan pada baris ke -n adalah
Baris ke-n = 2^(n-1)

Contoh : 2^(12-1) = 2^11 =>2048
Jadi, jumlah bilangan pada baris ke-12 adalah 2048.

26. rumus hukum pascal?A1=5cm2 F1=100 N F2=50 N A2=....cm2?


>>Tekanan<<

[tex]\boxed{\boxed{P=\frac{F}{A} }}[/tex]

dimana:

P = Tekanan (N/m² atau Pa)

F = Gaya (N)

A = Luas Penampang (m²)

Diketahui:

A₁ = 5 cm²

F₁ = 100 N

F₂ = 50 N

A₂ = ?

Jawab:

[tex]\frac{F_1}{A_1} =\frac{F_2}{A_2} \\\frac{100\ \cancel{N}}{5\ cm^2} =\frac{50\ \cancel{N}}{A_2} \\A_2=\frac{5\ cm^2}{100}\times 50\\ \boxed{A_2=\bold{2,5\ cm^2}}[/tex]
