2. WEB CROP ASSIGNMENT Foto 2 (Herojams Strawberry) Bagaimana caranya menyelesaikan langkah-langkah pengeditan di Adobe Photoshop CS6 ?


Jawaban:

1. Buka Adobe Photoshop CS6, lalu buka file gambar yang ingin diedit.

2. Lakukan pengeditan dasar seperti cropping (memotong), resizing (mengubah ukuran), atau adjusting color (menyesuaikan warna) jika diperlukan.

3. Gunakan alat-alat yang tersedia di toolbar untuk melakukan pengeditan lebih lanjut. Beberapa alat yang umum digunakan adalah alat selection, brush, eraser, clone stamp, dan gradient tool.

4. Setelah selesai mengedit, pastikan untuk menyimpan file dengan format yang tepat seperti JPEG, PNG, atau PSD. Pilih menu "File" > "Save As" atau "Export" untuk menyimpan file.

5. Jika perlu, tambahkan layer baru untuk melakukan pengeditan tambahan. Pilih menu "Layer" > "New" > "Layer" atau tekan tombol "Ctrl + Shift + N" untuk membuat layer baru.

6. Gunakan efek atau filter untuk memberikan efek pada gambar, seperti blur, sharpen, atau hue/saturation. Pilih menu "Filter" untuk memilih efek atau filter yang diinginkan.

7. Periksa kembali hasil pengeditan sebelum menyimpan atau mengekspor file. Pastikan tidak ada kesalahan atau cacat pada hasil akhir.

8. Simpan file dalam format yang tepat dan sesuai dengan kebutuhan. Jika membutuhkan, buat backup file untuk menghindari kehilangan data.

4. Tuliskan langkah-langkah membuat foto ukuran 3 x 4 menggunakan aplikasi Adobe Photoshop!​


Jawaban:

Langkah - langkah membuat Foto 3 x 4 pada Photoshop:

1. Buat Layer baru. (File > New)

2. Atur Layer baru dengan ukuran 3 x 4 cm dengan resolusi 300. (rata rata untuk hemat memori memakai 72)

3. Tekan OK untuk Konfirmasi.

4. Buka File Foto yang akan diatur ukurannya. (File > Open)

5. Tarik gambar yang telah dibuka menuju ke Layer Kosong yang dibuat tadi.

6. Lalu Kecilkan sesuai ukuran yang diinginkan.

7. Save.

16. Bagaimana caranya langkah-langkah membuat "reiteration" foto efek kembar, efek gambar awan, dan efek gambar burung di Adobe Photoshop CS6 ?​


Jawaban:

Berikut ini adalah langkah-langkah untuk membuat efek "reiteration" foto kembar, efek gambar awan, dan efek gambar burung di Adobe Photoshop CS6:

Reiteration Foto Kembar

a. Buka foto yang ingin dijadikan efek kembar di Adobe Photoshop CS6.

b. Duplikat lapisan foto dengan menekan Ctrl + J (Windows) atau Command + J (Mac).

c. Pilih lapisan duplikat dan pergi ke menu "Edit" > "Transform" > "Flip Horizontal".

d. Pindahkan lapisan duplikat ke posisi yang diinginkan dengan menggunakan alat "Move".

e. Ubah opasitas lapisan duplikat menjadi sekitar 50% agar lebih terlihat seperti efek kembar.

f. Jika perlu, atur kecerahan dan kontras pada kedua lapisan agar lebih cocok.

Efek Gambar Awan

a. Buka foto yang ingin dijadikan efek gambar awan di Adobe Photoshop CS6.

b. Pilih menu "Filter" > "Filter Gallery".

c. Pilih efek "Artistic" > "Poster Edges".

d. Sesuaikan nilai "Edge Thickness", "Edge Intensity", dan "Posterization" hingga mendapatkan efek awan yang diinginkan.

e. Jika perlu, atur opasitas lapisan agar lebih cocok dengan background atau elemen lainnya.

Efek Gambar Burung

a. Buka foto yang ingin dijadikan efek gambar burung di Adobe Photoshop CS6.

b. Pilih menu "Filter" > "Filter Gallery".

c. Pilih efek "Artistic" > "Cutout".

d. Sesuaikan nilai "Number of Levels", "Edge Simplicity", dan "Edge Fidelity" hingga mendapatkan efek potongan burung yang diinginkan.

e. Jika perlu, atur opasitas lapisan agar lebih cocok dengan background atau elemen lainnya.

Demikianlah langkah-langkah untuk membuat efek "reiteration" foto kembar, efek gambar awan, dan efek gambar burung di Adobe Photoshop CS6. Anda dapat memodifikasi nilai-nilai tertentu pada setiap filter atau menambahkan sentuhan pribadi untuk membuat efek yang lebih kreatif dan sesuai dengan selera.

Cara Install dan Aktivasi Adobe Photoshop CS5

1. Setelah anda selesai mendownload file Adobe Photoshop CS5, selanjutnya anda perlu mengekstraknya. Dengan cara klik kanan > pilih Extract to.

2. Kemudian klik 2X file Set-up.

3. Pilih Accept.

4. Selanjutnya anda akan disuruh memasukkan serial number. Kita bisa skip tahap ini dengan cara memilih Install this product as a trial kemudian pada kolom bahasa pilih English (International). Kemudian pilih Next.

5. Jika laptop anda menggunakan Windows 32 Bit, maka disarankan memilih Adobe Photoshop CS5 (tanpa embel-embel 64 Bit). Jika anda tidak tahu Windows anda 32 atau 64-bit. Selanjutnya pilih Install.

6. Jika muncul notifikasi seperti ini, tinggal pilih Yes.

7. Tunggu proses instalasi Adobe Photoshop CS5 hingga selesai.

8. Adobe Photoshop CS5 sudah berhasil diinstall, selanjutnya pilih Done.

Tahapan aktivasi yang perlu kita lakukan

Download file dibawah ini untuk keperluan aktivasi Adobe Photoshop CS5. Selanjutnya ekstrak dengan cara klik kanan > pilih Extract to.

Selanjutnya buka folder Aktivasi.

Pilih 32 atau 64 bit

Kemudian copy file yang bernama amtlib.dll.

Buka Local Disk (C:) kemudian masuk ke folder Program Files.

Masuk ke folder Adobe > Adobe Photoshop CS5 (tergantung bit)

Pastekan file tadi kesini. Jika muncul notifikasi seperti ini, tinggal pilih Replace the file in the destination.

Jika muncul warning, pilih saja Continoe.

Sekarang anda sudah bisa menggunakan Adobe Photoshop CS5 secara maksimal tanpa perlu takut muncul popup yang menyuruh anda untuk melakukan aktivasi.

29. Bagaimanakah prosedur atau langkah-langkah mengakhiri Photoshop?​


Jawaban:

1.Klik menu File -> Exit (Ctrl+Q).

2.Gunakan tombol Close pada sudut kanan atas dari Title bar.

3.Gunakan tombol Alt+F4.

Jika pada saat anda menutup dengan salah satu langkah tersebut masih ada file yang terbuka dan belum disimpan, maka Photoshop akan menampilkan kotak dialog.

Pembahas:

Kalau mau disimpan klik Yes, klik No untuk tidak menyimpan dan klik Cancel untuk membatalkan keluar dari Photoshop.

Apabila anda memilih Yes, maka Photoshop akan menampilkan kotak dialog Save As untuk proses penyimpanan. Lakukan proses penyimpanan sesuai dengan keperluan.
