8. sebuah kereta api melaju dengan kecepatan 20 m/s meninggalkan stasiun. pluit kereta api berfrekuensi 720 Hz. jika kecepatan bunyi udara adalah 340 m/s, frekuensi bunyi yang didengar oleh orang yang berada di stasiun sebesar ?


Saya akan mencoba menjawab dengan dua jawaban:

Jawaban pendek:

Sebuah kereta api melaju dengan kecepatan 20 m/s meninggalkan stasiun. Pluit kereta api berfrekuensi 720 Hz.

Jika kecepatan bunyi udara adalah 340 m/s, frekuensi bunyi yang didengar oleh orang yang berada di stasiun sebesar 677.6 Hz.

Jawaban panjang:

Berdasarkan efek Doppler, suara yang mendekati pendengar menjadi terdengar lebih keras atau lebih tinggi frekuensinya, sedangkan suara yang menjauhi pendengar terdengar lebih pelan.

Karena kereta api menjaui stasiun maka suara pluit kereta api akan terdengar lebih pelan.

Rumus efek Doppler disini adalah:

f terdengar = f pluit(1 + v kereta/v suara)

Karena pluit kereta berfrekuensi 720 Hz, kereta api menjauhi stasiun dengan kecepatan 20 m/s dan kecepatan suara adalah 340 m/s maka:

f terdengar = 720 (1 - 20/340)
= 720 (1 - 1/17)
= 720 × 16/17
= 677.6 Hz


Jadi oleh orang di stasiun suara tersebut akan terdengar dengan frekuensi 677.6 Hz
