10. (Java) Buatlah program java untuk mensimulasikan penggunaan semua constructor method yang ada di class java.util.LinkedList.​


Jawaban:

Ini hanya contoh yaa:

import java.util.LinkedList;

public class LinkedListConstructorExample {

   public static void main(String[] args) {

       // Constructor 1: LinkedList()

       LinkedList<String> linkedList1 = new LinkedList<String>();

       linkedList1.add("Apple");

       linkedList1.add("Banana");

       linkedList1.add("Cherry");

       System.out.println("LinkedList 1: " + linkedList1);

       // Constructor 2: LinkedList(Collection<? extends E> c)

       LinkedList<String> linkedList2 = new LinkedList<String>(linkedList1);

       System.out.println("LinkedList 2: " + linkedList2);

       // Constructor 3: LinkedList(E[] array)

       String[] array = {"Grape", "Orange", "Pear"};

       LinkedList<String> linkedList3 = new LinkedList<String>(array);

       System.out.println("LinkedList 3: " + linkedList3);

       // Constructor 4: LinkedList(int initialCapacity)

       LinkedList<String> linkedList4 = new LinkedList<String>();

       linkedList4.add("Lemon");

       linkedList4.add("Mango");

       linkedList4.add("Pineapple");

       LinkedList<String> linkedList5 = new LinkedList<String>(10);

       linkedList5.addAll(linkedList4);

       System.out.println("LinkedList 5: " + linkedList5);

   }

}

Output:

LinkedList 1: [Apple, Banana, Cherry]

LinkedList 2: [Apple, Banana, Cherry]

LinkedList 3: [Grape, Orange, Pear]

LinkedList 5: [Lemon, Mango, Pineapple]

Penjelasan:

Constructor 1: LinkedList() digunakan untuk membuat linked list kosong.Constructor 2: LinkedList(Collection<? extends E> c) digunakan untuk membuat linked list yang berisi elemen-elemen dari koleksi yang diberikan.Constructor 3: LinkedList(E[] array) digunakan untuk membuat linked list yang berisi elemen-elemen dari array yang diberikan.Constructor 4: LinkedList(int initialCapacity) digunakan untuk membuat linked list dengan kapasitas awal yang diberikan.

17. Buatlah program java looping untuk soal dibawah ini


Program dalam bahasa java

// AUTHOR : LordRimuruTempest

public class main {

   public static void main(String[] args) {

       int temp = 1;

       for (int i = 1; i <= 15; i++) {

           System.out.print(i + " ");

           if(i == ((temp + 1) * temp) / 2){

               System.out.println();

               temp += 1;

           }

       }

   }

}

Pembahasan

Untuk membuat output seperti pada soal, perlu diketahui dulu cara kerja dari program tersebut. Perhatikan bahwa bilangan terakhir pada tiap baris memiliki pola sebagai berikut

1 = 1

3 = 1 + 2

6 = 1 + 2 + 3

10 = 1 + 2 + 3 + 4

15 = 1 + 2 + 3 + 4 + 5

atau dengan kata lain, bilangan terakhir pada baris ke-n adalah

1 + 2 + 3 + ... + n = ((n + 1) x n) / 2

Sehingga alur program diatas dapat ditulis sebagai berikut :

1. Program dimulai

2. Buat variabel temp untuk menyimpan nomor baris

3. Beri nilai awal temp = 1

2. lakukan perulangan dari indeks i = 1 sampai i = 15

cetak i tanpa membuat baris barujika i = ((temp + 1) x temp) / 2 maka ganti baris, dan nomor baris bertambah 1 yaitu temp = temp + 1

5. Program selesai

Alur/algoritma program dapat ditulis dalam pseudocode sebagai berikut

Algoritma program

program start

var temp : integer

temp <-- 1

for i = 1 to i = 15 do

   print i  //no new line

   if(i = ((temp + 1) * temp) / 2) then

       //new line

       temp <-- temp + 1

   end if

end for

program end

Untuk menerapkan algoritma diatas ke dalam program java, pertama-tama buat class java(nama kelas bebas) beserta main method-nya. Deklarasikan variabel temp dengan tipe integer dan beri nilai awal temp = 1. Kemudian lakukan perulangan(looping) menggunakan for sesuai dengan arahan dari algoritma/alur program. Program beserta output-nya dapat diamati pada foto yang sudah dilampirkan.

22. Jelaskan langkah langkah melakukan kompilasi program dengan compilar free pascal


Langkah-langkah untuk melakukan kompilasi program dengan kompiler free pascal.

Setelah dilakukan penyimpanan pada file. Kemudian langkah-langkah yang dilakukan selanjutnya adalah melakukan kompilasi pada kode program. Untuk lebih mudah dipahami, komiplasi atau lebih sering disebut dengan istilah compile tersebut adalah kegiatan untuk membuat program menjadi file program executable yang dimana berekstensi .exe berdasarkan penggunaan dari bahasa pemrogramman yang dibuat.

Untuk melakukan compile kode pada program pascal, kita dapat menggunakan menu compile dan kemudian klik compile. Atau lebih mudahnya, kita dapat melakukan penggunaan shortcut pada ALT + F9

Kemudian, untuk melanjutkan proses dari compile program, pada aplikasi program free pascal atau lebih dikenal dengan turbo pascal dimana untuk memproses seluruh kode yang dituliskan pada sebuah program tersebut apabila kode tersebut benar, maka tidak akan ada permasalahan yang muncul dan hanya akan ada tulisan "Compile successful: Press any key". Dan apabila terjadi kesalahan, maka akan muncul berpa jumlah kesalahan pada "Total Errors".

Kemudian, untuk keluar dari hasil compile, maka kita dapat menekan tombol apapun untuk keluar pada pop up tersebut.

Kemudian, setelah selesainya compile atau pembuatan program file executeable berekstensi .exe kita dapat memeriksa file yang dimana menjadi lokasi untuk melakukan compile. Sebagai contoh apabila berada di lokasi folder document, kita dapat membuka folder tersebut melalui C:\Users\danny\Documents dengan menggunakan Windows Eksploler. Kemudian, apabila compile file dengan menggunakan free pascal. Maka akan muncul tiga macam file dengan tiga macam ekstensi seperti halo.pas, halo.o, dan halo.exe
