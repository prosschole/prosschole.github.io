13. sebutkan Cara menghitung sampel dengan rumus slovin


Berdasarkan notasi rumus besar sampel penelitian minimal oleh Slovin, maka apabila kamu punya 1.000 orang dalam sebuah populasi, kamu bisa tentukan minimal sampel yang akan diteliti. Margin of error yang ditetapkan adalah 5% atau 0,05.

Perhitungannya adalah:

n = N / (1 + (N x e²))

Sehingga: n = 1000 / (1 + (1000 x 0,05²))n = 1000 / (1 + (1000 x 0,0025))

n = 1000 / (1 + 2,5)

n = 1000 / 3,5

n = 285,7143

Apabila dibulatkan maka besar sampel minimal dari 1000 populasi pada margin of error 5% adalah sebesar 286.

Sampel Dadu = 1, 2, 3, 4, 5, 6
Pada kejadian pelemparan uang logam dan dadu secara bersamaan, maka titik sampelnya dapat dibuat melalui tabel dengan kolom uang logam A dan G dan baris dadu 1, 2, 3, 4, 5, 6 sehingga didapat semua titik sampel sebagai berikut:
{(A, 1) (A,2) (A,3) (A,4) (A,5) (A,6) (G,1) (G,2) (G,3) (G,4) (G,5) (G,6)}

29. cara menentukan ruang sampel dan titik sampel beserta contoh


Untuk menentukan ruang sampel dengan cara mendaftar dapat diambil contoh pada pelemparan sebuah uang koin. Pada pelemparan uang koin kemungkinan muncul sisi angka (A) atau sisi gambar (G). Bagaimana jika melempar tiga uang koin sekaligus?


Pada pelemparan tiga uang koin sekaligus, misalkan muncul sisi angka (A) pada mata uang pertama, muncul sisi gambar (G) pada mata uang kedua, dan muncul sisi angka (A) pada mata uang ketiga. Kejadian ini dapat ditulis AGA. Kejadian lain yang mungkin dari pelemparan tiga uang koin sekaligus adalah AAA, AAG, GAA, AGG, GAG, GGA, atau GGG. Jika ruang sampelnya ditulis dengan cara mendaftar, maka diperoleh S = {AAA, AAG, AGA, GAA, AGG, GAG, GGA, GGG} sehingga diperoleh banyaknya ruang sampel adalah n(S) = 8.
