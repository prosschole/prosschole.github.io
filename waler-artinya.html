3. Punten waler, hatur nuhun​


Jawaban:

》Hukum nun mati( نْ ) atau tanwin ( _ً_/--ٍ--/_ٌ_ ) . adalah empat hukum yang muncul tatkala nun bersukun atau tanwin bertemu dengan 28 huruf hijiyah . Empat hukum tersebut adalah Idzhar Halqi , Idgham,Iqlab , dan ikhfa .

● Huruf idzhar ada 6 yaitu : hamzah ( ء ) ,Ha ( ھ ) , 'ain ( ع ) , ha ( ح ) , ghain ( غ ) , dan kho ( خ ) .

●Huruf idgham ada 6 yaitu : ya ( ي ) , ra ( ر ) , mim ( م ) , lam ( ل ) , wau ( و ) , dan nun ( ن ) . yang terkumpul dalam lafadz يرملون

■catatan : idgham dibagi 2 yaitu idgaham bigunnah dan idgham bilagunnah/ bighairil gunnah. •Huruf idgham bigunnah ada empat yaitu : ya ( ي ) , nun ( ن ) , mim ( م ) , dan wau ( و ) . yang terkumpul ينمو .. • Huruf idgham bilagunnah / bighairil gunnah ada 2 yaitu : lam ( ل ) dan ro ( ر ) .

●Huruf Iqlab ada 1 yaitu : ba ( ب ) .

●Huruf Ikhfa ada 15 yaitu : shod ( ص ) ,dzal ( ذ ) , tsa ( ث ) , kaf ( ك ) ,Jim ( ج ) , sya ( ش ) , qof ( ق ) , sin ( س ) , dal ( د ) , tho ( ط ) , zai ( ز ) ,fa ( ف ) , ta ( ت ), dlad ( ض ) dan zha ( ظ ).

CONTOH BACAAN NUN MATI DAN TANWIN : Idzhar , Idgam ( bigunnah dan bilagunnah ) , iklab , dan ikhfa .

● Contoh Idzhar :

مَنْ اٰمَنَ

حَاسِدٍ اِذَا

●Contoh Idgham bigunnah :

مَنْ يَّعْمَلْ

لِقَوْمٍ يَّعْلَمُوْنَ

●Contoh idgham bilagunnah / bighairil gunnah :

مِنْ لَّدُنْكَ

خَيْرٌلَّكَ

contoh iklab :

مِنْ بَعْدِ

سَيِّئَةٌ بِمَا

●Contoh ikhfa :

مَنْ كَانَ

اَجْرًا كَبِيْرًا

15. Pranyatan ing ngisor iki kang bener kanggo nerangake pangrakiting tembung wewaler yaiku.. A. Wewaler mujudake tembung lingga B. Wewaler saka tembung wawar antuk seselan -el- C. Wewaler saka tembung waler entuk ater-ater wa D. Wewaler mujudake dwipurwa saka tembung waler E. Wangsulan A, B, C, lan D salah


Wangsulan

➡️ D. Wewaler mujudake dwipurwa saka tembung waler.

Penjelasan

Wewaler kuwi asale saka tembung lingga 'waler', banjur diwujudake tembung dwipurwa dadi tembung 'wewaler'

Waler tegese pepacuh, pepali, larangan.

☞ {Tembung Dwipurwa}

Dwi tegese loro, purwa tegese wiwitan. Dadi tembung dwipurwa yaiku tembung sing dirangkep purwaning tembung utawa wanda sing dadi wiwitane tembung.

—☆ Tuladha :

mangan = ma+mangan = mamangan = memangan tulung = tu + tulung = tutulung = tetulung lara = la + lara = lalara = lelara resik = re + resik = reresikwaler = wa + waler = wawaler = wewaler
