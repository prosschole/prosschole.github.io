Penjelasan:

Active Voice (kata kerja aktif) berarti menerangkan sebuah pekerjaan. Contoh menyapu, membaca, dll. Dalam bahasa Inggris active voice dapat diketahui dengan adanya V1/V2/Ving/(have/has/had+V3)

Contoh:

Marry eats fruits today. (S + V1)Marrry ate fruit yesterday. (S + V2)Marry is eating banana now. (S + Ving)Marry has just eatten banana. (S + has + V3)

Passive voice (kata kerja pasif) berarti suatu objek (O) diterangkan oleh suatu pekerjaan. Contoh buku dibaca oleh Rudi.

Dalam bahasa Inggris passive voice dapat diketahui dengan rumus:

S + to be + V3 --- to be (is/am/are/was/were)

S + to be + being + V3

S + has/have/had + been + V3

Contoh :

This book is read by Tia. (S + to be + V3)This book is being read by Tia. (S + to be + being + V3)This book has been read ny Tia. (S + has + been + V3)
