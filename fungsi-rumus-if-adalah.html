1. Fungsi rumus IF adalah..


Mempermudah kita untuk mencara suatu bilanganRumus IF dalam microsoft excel digunakan untuk menentukan kalau kamu memenuhi syarat tersebut, maka kamu mendapatkan hadiah. Kalau kamu tidak memenuhi syarat tersebut, maka kamu tidak mempunyai apa-apa.

Contoh: di dalam supermarket, syarat untuk mendapatkan sebuah ponsel Blackberry X, kamu harus berbelanja mencapai Rp.200.000 . Jika tidak, maka kamu hanya mendapatkan pensil 2B.

Dengan contoh tersebut, dapat disimpulkan dengan rumus:
=IF(X6>200.000;"Ponsel Blackberry X";"Pensil 2B") tekan enter.

Dengan X6 = Total pembayaran belanjamu di supermarket.

12. Tuliskan fungsi if dan cara penulisan rumusnya​


Jawaban:

ungsi IF memungkinkan Anda untuk membuat perbandingan logis antara nilai dan apa yang diharapkan dengan menguji kondisi dan mengembalikan hasil jika True atau False.

=IF(Sesuatu adalah Benar, lakukanlah sesuatu, jika tidak lakukanlah hal lain)

Oleh karena itu, pernyataan IF dapat memiliki dua hasil. Hasil pertama jika perbandingan Anda Benar dan hasil kedua jika perbandingan Salah.

Pernyataan IF sangat kuat, dan membentuk dasar dari banyak model lembar bentang, namun pernyataan ini juga merupakan penyebab utama dari banyak masalah lembar bentang. Secara ideal, pernyataan IF harus berlaku terhadap kondisi minimal, seperti Pria/Wanita, Ya/Tidak/Mungkin, misalnya, namun terkadang Anda mungkin perlu mengevaluasi skenario lebih kompleks yang membutuhkan penumpukan* lebih dari 3 fungsi IF secara bersamaan.

* “Penumpukan” mengacu kepada praktik penggabungan beberapa fungsi secara bersamaan dalam satu rumus.

15. Tuliskan rumus umum dari fungsi logika IF


IF(logical_test, value_if_true, [value_if_false])

Misalnya:

=IF(A2>B2,"Melebihi Budget","OK")

=IF(A2=B2,B4-A4,"")

Deskripsi

logical_test    (diperlukan)

Kondisi yang ingin Anda uji.

value_if_true    (diperlukan)

Nilai yang ingin Anda kembalikan jika hasillogical_test adalah TRUE.

value_if_false    (opsional)

Nilai yang ingin Anda kembalikan jika hasillogical_test adalah False.

Contoh IF sederhana

=IF(C2=”Ya”,1,2)

Dalam contoh di atas, sel D2 mengatakan:IF(C2 = Ya, jadi kembalikan 1, jika tidak kembalikan 2)

=IF(C2=1,”Ya”,”Tidak”)

21. jelaskan fungsi dari rumus IF,HLookup,dan VLookup?​


Fungsi IF ialah salah 1 fungsi di Ms. Excel yang digunakan untuk menghasilkan nilai berdasarkan uji logika (benar/salah) tertentu.

Fungsi HLOOKUP ialah salah 1 fungsi di Ms. Excel untuk mencari sebuah nilai pada kolom(column) pertama sebuah data tabel sehingga didapat sebuah nilai tertentu pada baris lain yang berada pada kolom (column) yang sama dengan hasil pencarian. Data pada fungsi ini disusun secara horizontal.

Fungsi VLOOKUP adalah salah 1 fungsi pada Ms. Excel untuk mencari sebuah nilai pada baris (row) pertama sebuah data tabel sehingga didapat sebuah nilai data tertentu pada kolom lain yang berada pada baris (row) yang sama dengan hasil pencarian. Data pada fungsi ini disusun secara vertikal.
