17. Setarakan reaksi fe2o3+co=fe+co2


Jawaban:

Persamaan reaksi setara:

[tex] Fe_2O_3 \ + \ 3CO \rightarrow 2Fe \ + \ 3CO_2 [/tex]

Penjelasan:

[tex] aFe_2O_3 \ + \ bCO \rightarrow cFe \ + \ dCO_2 [/tex]

Hitung jumlah masing-masing atom di ruas kiri dan kanan

Fe: 2a = c

O: 3a+b = 2d

C: b = d

misalkan nilai a = 1

Fe:

2a = c

2(1) = c

2 = c

Nilai c = 2

O:

3a+b = 2d

3(1)+b = 2d

3 = b

Nilai b = 3

C:

b = d

3 = d

nilai d = 3

Jadi nilai koefisien reaksi setaranya adalah a=1, b=3, c=2, d=3

Persamaan reaksi setara:

[tex] Fe_2O_3 \ + \ 3CO \rightarrow 2Fe \ + \ 3CO_2 [/tex]
