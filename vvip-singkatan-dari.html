Diketahui:

600 tiket terjual, dengan keterangan:

Harga Festival = 180.000 Harga VIP = 200.000 Harga VVIP = 350.000 Jumlah tiket VVIP terjual = 2x jumlah tiket VIP terjualHasil penjualan seluruh tiket = 120.600.000

Ditanya:

a. Model matematika dan persamaan matriks

b. Banyak tiket terjual

Jawaban:

A. Soal A

X + Y + Z = 600 18X + 20Y + 35Z = 12.060 2Y ‚Äì Z = 0

[tex]\left[\begin{array}{ccc}1&1&1\\18&20&35\\0&2&-1\end{array}\right][/tex] ¬†[tex]\left[\begin{array}{ccc}x\\y\\z\end{array}\right][/tex] = [tex]\left[\begin{array}{ccc}600\\12060\\0\end{array}\right][/tex]

B. Soal B

Tiket Festival terjual = 495 tiket Tiket VIP terjual = 35 tiket Tiket VVIP terjual = 70 tiket

Penjelasan

Misalkan jumlah tiket festival adalah x, jumlah tiket VIP adalah y dan jumlah tiket VVIP adalah z. Sehingga diketahui:

ax + by = c ¬†

Menjadi

[tex]\left[\begin{array}{ccc}1&1&1\\18&20&35\\0&2&-1\end{array}\right][/tex] ¬†[tex]\left[\begin{array}{ccc}x\\y\\z\end{array}\right][/tex] = [tex]\left[\begin{array}{ccc}600\\12060\\0\end{array}\right][/tex]

Kemudian untuk menyelesaikan persamaan matriks di atas, pertama-tama kita harus menentukan determinan matriks D, Dx, Dy, dan Dz.

Matriks D merupakan matriks 3 x 3 yang terdiri atas koefisien-koefisien semua variabel persamaan. ¬†

Matriks Dx merupakan matriks 3 x 3 yang kolom pertamanya merupakan konstanta persamaan, kemudian kolom kedua terdiri atas koefisien y, dan kolom ketiga terdiri koefisien z.

Matriks Dy merupakan matriks 3 x 3 yang kolom pertamnya terdiri atas koefisien x, kolom kedua terdiri atas konstanta persamaan, dan kolom ketiga terdiri atas koefisien z. ¬†

Matriks Dz merupakan matriks 3 x 3 yang kolom pertamanya terdiri atas koefisien x, kolom kedua terdiri atas koefisien y, dan kolom ketiga terdiri atas konstanta persamaan. ¬†

Sehingga didapat:

D = [tex]\left[\begin{array}{ccc}1&1&1\\18&20&35\\0&2&-1\end{array}\right][/tex]

Dx = [tex]\left[\begin{array}{ccc}600&1&1\\12060&20&35\\0&2&-1\end{array}\right][/tex]

Dy =[tex]\left[\begin{array}{ccc}1&600&1\\18&12060&35\\0&0&-1\end{array}\right][/tex]

Dz = [tex]\left[\begin{array}{ccc}1&1&600\\18&20&12060\\0&2&0\end{array}\right][/tex]

Kemudian carilah nilai determinan seluruh matriks D, kita dapat menggunakan metode Sarrus di bawah (terlampir di foto). Sehingga didapatkan nilai:

D = -36 Dx = -17820 Dy = -1260 Dz = -2520

Nilai x, y dan z ditentukan dengan rumus:

X = Dx/D = -17820/-36 = 495 Y = Dy/D = -1260/-36 = 35 Z = Dz/D = -2520/-36 = 70

Sehingga didapatkan nilai X = 495, Y = 35 dan Z = 70 dan dapat disimpulkan banyaknya tiket terjual ialah:

Tiket Festival = 495 tiket Tiket VIP = 35 tiket Tiket VVIP = 70 tiket
