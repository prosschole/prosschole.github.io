16. apa artinya sweept,sweep,sweeps,sweeping?​


Jawaban:

sweept = menyapu

sweep = menyapu

sweeps = menyapu

sweeping = luas

Penjelasan:

sama artinya tapi yang membedakan kondisi dimana kita sedang mengalaminya.

Jawaban:

Swept = menyapu ( sudah dilakukan )

Sweep= menyapu ( untuk I, you, they, we )

Sweeps= menyapu ( untuk she, he, it )

Sweeping= menyapu ( sedang menyapu )
