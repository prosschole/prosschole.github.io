Penjelasan:

04. f(x) = 8 – │2x + 4│

Untuk 2x + 4 ≥ 0 ⇒ x ≥ -2

f(x) = 8 - (2x + 4) ⇒ f(x) = -2x + 4

Untuk 2x + 4 < 0 ⇒ x < -2

f(x) = 8 - [-(2x + 4)] ⇒ f(x) = 2x + 12

[tex]\boxed{ \ f(x) = \left \{ {{-2x+4, \ x \geq 2} \atop {2x + 12, \ x < -2}} \right \ }[/tex]

05. f(x) = 5x – │2x - 8│

Untuk 2x - 8 ≥ 0 ⇒ x ≥ 4

f(x) = 5x - (2x - 8) ⇒ f(x) = 3x + 8

Untuk 2x - 8 < 0 ⇒ x < 4

f(x) = 5x - [-(2x - 8)] ⇒ f(x) = 7x - 8

[tex]\boxed{ \ f(x) = \left \{ {{3x + 8, \ x \geq 4} \atop {7x - 8, \ x < 4}} \right \ }[/tex]

06. f(x) = │2x - 6│ + │x + 2│

Untuk 2x - 6 ≥ 0 ⇒ x ≥ 3

f(x) = (2x - 6) + (x + 2) ⇒ f(x) = 3x - 4

Untuk x + 2 < 0 ⇒ x < -2

f(x) = [-(2x - 6)] + [-(x + 2)] ⇒ f(x) = -3x + 4

Dari 2x - 6 < 0 menjadi x < 3, dan dari x + 2 ≥ 0 menjadi x ≥ -2, diambil irisan keduanya sehingga untuk -2 ≤ x < 3 adalah sebagai berikut.

f(x) = -(2x - 6) + (x + 2) ⇒ f(x) = -x + 8

[tex]\boxed{ \ f(x) = \left\{\begin{array}{ccc}-3x + 4, \ x < -2 \\-x + 8, \ -2\leq x < 3 \\3x - 4, \ x \geq 3 \end{array}\right \ }[/tex]

07. f(x) = │x + 3│ + │2x - 4│

Untuk 2x - 4 ≥ 0 ⇒ x ≥ 2

f(x) = (x + 3) + (2x - 4) ⇒ f(x) = 3x - 1

Untuk x + 3 < 0 ⇒ x < -3

f(x) = [-(x + 3)] + [-(2x - 4)] ⇒ f(x) = -3x + 1

Dari 2x - 4 < 0 menjadi x < 2, dan dari x + 3 ≥ 0 menjadi x ≥ -3, diambil irisan keduanya sehingga untuk -3 ≤ x < 2 adalah sebagai berikut.

f(x) = (x + 3) + [-(2x - 4)] ⇒ f(x) = -x + 7
