1. Microsoft Word

Ms.word merupakan aplikasi pengolah kata, dalam artian semua pekerjaan yang berhubungan dengan pengolahan kata dapat dilakuka dengan menggunakan Ms.Word. Contoh nya, membuat surat, membuat tabel, input gambar, membuat desain sederhana, membuat dokumen dan berbagai macam pengolahan data lainnya.

2.Microsoft Power Point

Ms.Power point merupakan aplikasi dari Microsoft Office yang berfungsi sebagai media presentasi dengan menggunakan slide. Dengan menggunakan aplikasi ini pengguna dimudahkan untuk melakukan presentasi, selain itu dalma ms.power point juga terdapat fitur-fitur yang dapat digunakna untuk membuat slide presentasi menjadi lebih menarik.

3. Microsoft Publisher

Ini merupakan paket aplikasi perkantoran dari Microsoft yang berguna untuk membuat surat kabar, Desktop Publishing, kartu ucapan, dan masih banyak lagi. Awalnya Microsoft Publisher digunakan untuk Ms Office 2003 dan kemudian Ms Office 2007. Yang terbaru Publisher hadir di Ms Office 2010 pada edisi Profesional

4. Microsoft Access

Microsoft Office Acces merupakan aplikasi basis data atau database yang dikeluarkan oleh Microsoft salam satu paket ms.Office. Dengan menggunakan Ms.Acces, kita dapat menangani proses manipulasi data dan pembuatan sistem.

5. Microsoft Outlook

Ms.Outlook adalah aplikasi pengelola informais pribadi dari Ms.Office yang digunakn untuk mengirim dan membaca surat elektronik (email), selain itu Ms.Outlook juga memiliki fungsi seperti membuat catatan, jurnal, jadwal kerja dan kalender.

6. Microsoft Info Path

Ms.InfoPath merupakan aplikasi ms.Office yang berfungsi untuk pengembangan formulir data berbasis XML, seperti merancang, mendistribusikan, mengisi dan mengirimkan formulir elektronik yang berisi data terstruktur. Ms.Infopath dikeluarkan pada tahun 2003 dengan menyediakna beberapa fitur menarik.

7. Microsoft Lync

Ini adalah aplikasi Microsoft yang sebelumnya dinamai dengan Office Communicator 2007 R2. Aplikasi ini dikeluarkan pada tahun 2011 dan saat ini sudah hadir dalam paket Ms Office 2013. Sebenarnya Lync mempunyai banyak fungsi. Tetapi bagi kebanyakan pengguna, Lync merupakan instant messenger seperti MSN Messenger dan Google Talk, Skype, dan Yahoo Messenger.

8. Microsoft One Note

Microsof Office One Note adalah aplikasi yang berfungsi sebagai kertas catatan. Karena fungsi Ms.One note seperti sebuah kertas maka kita dapat secara bebas menulis jadwal kegiatan, pengingat atau sekedar mencorat-coret lembar kerja. Berbeda dari Microsoft Word, di OneNote kamu bisa menulis dan meletakkan apapun di tempat manapun yang kamu suka.

9. Microsoft Visio

Ms.Visio merupakan aplikasi yang digunakan untuk membuat diagram Flowchart dan berbagai macam diagram, brainstrorm dan bebebrapa skema jaringan. Aplikasi ini juga menggunakan fitur grafik vektor sehingga pengguna lebih dimudahkan untuk menggunakannya.

10. Microsoft Front Page

Ms.Front Page adalah sebuah aplikasi untuk administrasi website dengan editor HTML yang menggunakan basis WYG. Saat ini ms.Front Page telah digantikan dengan SharePonit Designer yang diluncurkan pada tahun 2006.
