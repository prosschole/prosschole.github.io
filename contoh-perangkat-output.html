5. apakah yang di sebut perangkat input atau output berikan contoh


Jawaban:

Perangkat input adalah perangkat ‘masukkan’ untuk unit komputer yang cara kerjanya memberikan perintah secara langsung dengan menghasilkan informasi berbentuk digital kepada para pengguna. Ada berbagai macam perangkat keras berjenis input yang diciptakan untuk komputer, berikut adalah contohnya:

1. Keyboard

2. Mouse

3. Joystick

4. Scanner

Informasi yang ditampilkan oleh komputer sudah diterjemahkan oleh teknologi komputer itu sendiri sehingga dapat dipahami penggunanya dengan mudah. Berikut ini ada beberapa contoh perangkat output komputer yang sering digunakan.

1. Monitor

2. Projector

9. Jelaskan yang disebut dengan perangkat Input / Output! Berikan contohnya


Perangkat input ialah sebuah proses mengirimkan suatu informasi ke sistem komputer untuk diproses. Adapun contohnya yaitu keyboard, mouse, touchpad, touchscreen, microphone, scanner, web cam.Perangkat output ialah sebuah proses untuk menghasilkan atau menampilkan hasil dari proses tersebut. Adapun contohnya yaitu monitor, speaker, printer, proyektor.

Pembahasan:

Perangkat input adalah sebuah perangkat ‘masukkan’ untuk unit komputer yang dimana cara kerjanya dapat memberikan perintah secara langsung dengan menghasilkan suatu informasi yang berbentuk digital kepada para penggunanya.

Berikut ini merupakan alat input beserta fungsinya, yakni:

Keyboard memiliki fungsi yakni untuk memberi perintah kepada komputer, untuk mengetik sebuah dokumen, atau menjalankan proses input lainnya.Mouse memiliki fungsi yakni untuk menggerakkan kursor, untuk membuka dan menutup program, dan lain sebagainya.Mikrofon memiliki fungsi yakni untuk berbicara dengan lawan bicara yang terhubung dengan internet.Scanner memiliki fungsi yakni untuk mengubah materi cetak seperti misalnya teks atau gambar ke dalam bentuk yang dapat dibaca atau dilihat oleh komputer.Web cam memiliki fungsi yakni untuk mengaktifkan gambar atau video dan orang lain dapat melihat gambar atau video tersebut.

Perangkat output adalah sebuah perangkat komputer yang digunakan untuk menampilkan atau menyampaikan suatu informasi kepada penggunanya. Adapun suatu informasi yang ditampilkan oleh komputer sudah diterjemahkan oleh teknologi komputer itu sendiri sehingga akan dipahami oleh penggunanya dengan mudah.

Berikut ini merupakan alat output beserta fungsinya, yakni:

Monitor memiliki fungsi yakni untuk menampilkan visual data, gambar atau program yang akan/sedang dijalankan.Speaker memiliki fungsi yakni untuk mendengarkan musik, suara maupun bentuk audio lainnya.Printer memiliki fungsi yakni untuk memproduksi data ke dalam bentuk fisik.Plotter memiliki fungsi yakni untuk mencetak hasil desain grafis dalam ukuran yang besar. Plotter hampir sama seperti printer namun plotter cenderung fokus untuk mencetak hasil grafis dengan ukuran besar.Proyektor memiliki fungsi yakni untuk menampilkan data ke dalam bentuk tampilan yang lebih besar.
