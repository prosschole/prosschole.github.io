2. ibel ingin membuat kebaya dan baju pesta ibel membutuhkan 2 1/9 meter kain brokat untuk membuat kebaya baju pesta ibel memebutuhkan 2 2/5 meter kain satin kebaya ataukah baju pesta yang memerlukan kain lebih banyak? coba kamu hitung


Untuk membuat kebaya diperlukan kain

= 2 1/9 meter

= (2 x 9 + 1)/9 meter

= 19/9 meter

= (19 x 5)/(9 x 5) meter

= 95/45 meter


Untuk membuat baju pesta diperlukan kain

= 2 2/5 meter

= (2 x 5 + 2)/5 meter

= 12/5 meter

= (12 x 9)/(5 x 9) meter

= 72/45 meter


95/45 meter > 72/45 meter


Maka, yang memerlukan kain lebih banyak adalah kebaya

14. 2. Untuk membuat 2 lembar kebaya yang modelnya berbeda diperlukan kain dasar 5 meter. Model pertama memerlukan kain 70 cm lebih banyak dari model kedua . Hitung banyak kain yang diperlukan utnuk masing-masing model itu


Untuk model kebaya pertama diperlukan 285 cm kain dasar, dan untuk   model kebaya kedua diperlukan 215 cm kain dasar. Nilai tersebut diperoleh dari perhitungan dengan metode subtitusi. Simak pembahasan berikut.

Pembahasan

Diketahui:

Untuk membuat 2 lembar kebaya yang modelnya berbeda diperlukan kain dasar 5 meter

Model pertama memerlukan kain 70 cm lebih banyak dari model kedua

Ditanya: Banyak kain yang diperlukan untuk masing-masing model kebaya tersebut

Jawab:

Misalkan:

Model pertama = x

Model kedua = y

Maka diperoleh persamaan sebagai berikut:

x + y = 5 meter

ingat! 1 meter = 100 cm

x + y = 500 .....................1)

Model pertama memerlukan kain 70 cm lebih banyak dari model kedua

x = y + 70 .......................2)

Subtitusikan persamaan 2) ke dalam persamaan 1)

x + y = 500

y + 70 + y = 500

2y + 70 = 500

2y = 500 - 70

2y = 430

y = 430/2

y = 215

Subtitusikan nilai y ke dalam persamaan 2)

x = y + 70

x = 215 + 70

x = 285

Jadi untuk model kebaya pertama diperlukan 285 cm kain dasar, dan untuk   model kebaya kedua diperlukan 215 cm kain dasar.
