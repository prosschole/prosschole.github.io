12. aplikasi untuk menyimpan kuota data internet wifi ke sim card/handphone untuk dijadikan cadangan?apabila tidak punya kuota/wifi?


1. Pertama, Anda harus memiliki Smartphone Android dengan sistem operasi       2.3 keatas.
2. Sim Card yang sudah memiliki kuota 10 MB, untuk pulsa lebih baik biarkan kosong. Terserah Anda, mau memakai Telkomsel, Indosat, XL, Three dan lain-lain.
3. Silahkan Anda untuk membuka Privatetunnel.com, Disinilah gunanya paketan 10 MB tadi yakni dipakai untuk proses daftar dan download aplikasi. Untuk lebih mudahnya, Anda bisa daftar melalui komputer terlebih dahulu, baru nanti bisa login melalui smartphone Android.
4. Klik Sign Up dan isi kotak pendaftaran.
5. Nanti akan muncul kotak bahwa Anda akan disuruh untuk memverifikasi email yang tadinya sudah Anda buat daftar.
6. Jika sudah selesai, Ambil Smartphone Anda dan buka lagi Privatetunnel.com dari browser Anda.
7. Gunakan akun email yang sudah didaftarkan tadi untuk login.
8. Selanjutnya Anda bisa mendownload aplikasi dari Google Play Store dengan kata kunci Private Tunnel
9. Jika sudah selesai mendownload, langsung buka aplikasi Private Tunnel tersebut dan pilih host, disini ada 7 pilihan host dan langsung klik Connect.
10. Jika Android VPN sudah konek, dibagian atas akan terlihat ada logo kunci Android
11. Selesai dan selamat bahwa Cara Internet Gratis di Android Tanpa Pulsa Anda sudah berhasil.

Note: Pada setiap harinya, Anda mendapatkan kuota gratis sebesar 100 MB, jadi jika selama sebulan Anda mendapatkan 3GB. Jika tiba-tiba saja Aplikasi PrivateTunnel sudah tidak mau konek, mungkin pihak operator sudah mengetahui dan memblockir Sim Card Anda :v.

18. Cara kirim data komputer ke hp android lewat wifi adalah .....


Cara Transfer File dari Android ke Laptop Melalui WiFi

- Pertama-tama download dulu aplikasi WiFi FTP Server langsung di Google Play Store.

- Sambungkan android dan laptop ke jaringan WiFi yang sama. Jika di tempat anda tidak tersedia jaringan WiFi, nyalakan WiFi Hotspot android anda dan sambungkan laptop ke hotspot.

- Selanjutnya buka aplikasi WiFi FTP Server yang sudah didownload tadi. Aktifkan FTP Server dengan menekan tombol Awal dan izinkan aplikasi untuk mengakses file anda.

- Setelah diaktifkan, akan muncul informasi berupa URL server ftp://192.168.43.136:2221, username, dan password.

- Langkah selanjutnya yaitu mengakses file android di komputer menggunakan Windows Explorer.

- Buka File Explorer di Windows, lalu ketikkan URL server beserta port-nya pada address bar seperti gambar di bawah ini.

- Kemudian tekan Enter,

- Untuk mentransfer file dari android ke laptop cukup dengan copy dan paste file seperti biasanya.

- Selain menyalin file dari android ke laptop, anda juga bisa menyalin file dari laptop ke android. Caranya kurang lebih sama, copy file di laptop dan akses direktori android menggunakan URL server, lalu paste di dalam direktori android yang anda inginkan.
