20. pertidaksamaan akar x pangkat 2 dikurang X lebih kecil daripada akar 2pertidaksamaan akar x pangkat 2 dikurang X lebih kecil daripada akar 2 memiliki himpunan penyelesaian a ​


Jawab:

{ x | –1 < x ≤ 0  atau  1 ≤ x < 2 }

Penjelasan dengan langkah-langkah:

Pertidaksamaan

[tex]\large\text{$\begin{aligned}&\sqrt{x^2-x}<\sqrt{2}\\&\quad.....\normalsize\textsf{ [ kuadratkan kedua ruas ]}\\&{\iff}\left(\sqrt{x^2-x}\right)^2<\left(\sqrt{2}\right)^2\\&{\iff}x^2-x<2\\&\quad.....\normalsize\textsf{ [ kurangi kedua ruas dengan 2 ]}\\&{\iff}x^2-x-2<2-2\\&{\iff}x^2-x-2<0\\&\quad.....\normalsize\textsf{ [ faktorkan ruas kiri ]}\\&{\iff}(x+1)(x-2)<0\end{aligned}$}[/tex]

[tex]\large\text{$\begin{aligned}&\textsf{Titik kritis: }\\&\quad x+1=0\quad\textsf{dan}\quad x-2=0\\&\quad {\iff}x=-1\quad\textsf{dan}\quad x=2\end{aligned}$}[/tex]

Pemeriksaan interval pertidaksamaan

[tex]\large\text{$\begin{aligned}&(x+1)(x-2) < 0\\&\begin{array}{llcl}(i)& x <-1 & \implies&(-)(-)=(+) \not < 0\\(ii)& -1 < x < 2&\implies&(+)(-)=(-) < 0\\(iii)& x > 2 &\implies&(+)(+)=(+) \not < 0\end{array}\\&\textsf{Sehingga, interval yang berlaku adalah:}\\&\quad -1 < x < 2\end{aligned}$}[/tex]

[tex]\large\text{$\begin{aligned}&\textsf{Syarat $\sqrt{x^2-x} \in \mathbb{R}$ adalah:}\\&\quad x^2 - x \ge 0\\&\quad{\iff}x(x-1)\ge 0\\&\textsf{Titik kritis: $x=0$ dan $x=1$.}\\&\begin{array}{llcl}(i)& x \le 0 &\implies&(-)(-)=(+) \ge 0\\(ii)& 0 < x < 1 &\implies&(+)(-)=(-) \not \ge0\\(iii)& x \ge 1 &\implies&(+)(+)=(+) \ge 0\end{array}\\&\textsf{Sehingga, interval yang berlaku adalah:}\\&\quad x \le 0\quad\textsf{atau}\quad x \ge 1\end{aligned}$}[/tex]

Penggabungan Interval

–1 < x < 2  dan  (x ≤ 0  atau  x ≥ 1)

⇔ (–1 < x < 2  dan  x ≤ 0)  atau  (–1 < x < 2  dan  x ≥ 1)

⇔ –1 < x ≤ 0  atau  1 ≤ x < 2

∴  Dengan demikian, himpunan penyelesaiannya adalah:

{ x | –1 < x ≤ 0  atau  1 ≤ x < 2 }

24. bentuk sederhana dari x pangkat -2(akar x + akar x pangkat 7) adalah?


[tex]\frac{1}{x\sqrt{x}} + x\sqrt{x}[/tex] atau

[tex]\frac{1 + x^3}{x\sqrt{x}}[/tex] atau

[tex]\sqrt{x} \big( \frac{1}{x^2} + x\big)[/tex] atau

[tex]\frac{\sqrt{x}}{x^2} + x\sqrt{x}[/tex]

Pembahasan

[tex]x^{-2} (\sqrt{x} + (\sqrt{x})^7)[/tex] memiliki bentuk yang sama dengan [tex]x^{-2} (\sqrt{x} + (\sqrt{x^7}))[/tex]

Sifat-sifat pangkat dan akar yang digunakan adalah sebagai berikut:

[tex]\sqrt[b]{x^a} \leftrightarrow x^{\frac{a}{b}}[/tex]

[tex](x^a)^b \leftrightarrow (x^b)^a[/tex]

[tex]x^a.x^b \leftrightarrow x^{a + b}[/tex]

√a · √a = a

Rasionalisasi (mengalikan bentuk akar sekawan): [tex]\frac{a}{\sqrt{b}} \rightarrow \frac{a}{\sqrt{b}} \times \frac{\sqrt{b}}{\sqrt{b}} = \frac{a}{b} \sqrt{b}[/tex]

[tex]x^{\frac{3}{2}} = x^{1 + \frac{1}{2}} = x.x^{\frac{1}{2}} = x\sqrt{x}[/tex]

Sifat distributif dari bentuk aljabar: A(B + C) = AB + AC

Mari kita olah kasus di atas.

[tex] = x^{-2} (x^{\frac{1}{2}} + (x^{\frac{1}{2}})^7)[/tex]

[tex] = x^{-2} (x^{\frac{1}{2}} + x^{\frac{7}{2}})[/tex]

[tex] = x^{-2 + \frac{1}{2}} + x^{-2 + \frac{7}{2}}[/tex]

[tex] = x^{-\frac{4}{2} + \frac{1}{2}} + x^{-\frac{4}{2} + \frac{7}{2}}[/tex]

[tex] = x^{-\frac{3}{2}} + x^{\frac{3}{2}}[/tex]

[tex] = \frac{1}{x^{\frac{3}{2}}} + x^{\frac{3}{2}}[/tex]

Bentuk pertama yang dihasilkan adalah

[tex]\boxed{~= \frac{1}{x\sqrt{x}} + x\sqrt{x}~}[/tex]

Kita lanjutkan dengan menyamakan penyebutnya agar diperoleh bentuk lainnya.

[tex]= \frac{1 + (x\sqrt{x}.x\sqrt{x})}{x\sqrt{x}}[/tex]

[tex]= \frac{1 + (x^2.x)}{x\sqrt{x}}[/tex]

Bentuk kedua yang dihasilkan adalah

[tex]\boxed{~= \frac{1 + x^3}{x\sqrt{x}}~}[/tex]

Dari bentuk pertama di atas, salah satu suku kita kalikan akar sekawan lalu diolah agar mendapatkan bentuk ketiga.

[tex]\frac{1}{x\sqrt{x}} \times \frac{\sqrt{x}}{\sqrt{x}} = \frac{1}{x^2} \sqrt{x}[/tex]

Dari bentuk pertama di atas, kita olah lebih lanjut.

[tex]\boxed{~= \frac{1}{x^2} \sqrt{x} + x\sqrt{x}~}[/tex]

Bentuk ketiga yang dihasilkan adalah

[tex]\boxed{~= \sqrt{x} \big( \frac{1}{x^2} + x \big)~}[/tex]

Terakhir, dari bentuk pertama masih dapat diolah dengan mengalikan akar sekawan dari √x.

[tex]= \big( \frac{1}{x\sqrt{x}} + x\sqrt{x} \big) \times \frac{\sqrt{x}}{\sqrt{x}}[/tex]

[tex]= \big( \frac{\sqrt{x}}{x.x} \big) + \big( x\sqrt{x} . \frac{\sqrt{x}}{\sqrt{x}}\big)[/tex]

Bentuk keempat yang dihasilkan adalah

[tex]\boxed{~= \frac{\sqrt{x}}{x^2} + x\sqrt{x}~}[/tex]
