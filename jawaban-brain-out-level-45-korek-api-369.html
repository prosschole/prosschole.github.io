2. berapakah angka paling besar yang anda dapatkan dari 369 (korek api) dengan memindahkan 1 korek api?


Angka paling besar yang bisa didapatkan dari 369 (korek api) dengan memindahkan 1 korek api adalah 965.

PembahasanMenyelesaikan Soal Cerita Tentang Bilangan Bulat

Bilangan bulat merupakan himpunan bilangan yang terdiri dari bilangan positif, nol, dan bilangan negatif.

Pada bilangan bulat positif, jika angkanya semakin besar maka akan semakin besar pula nilainya. Dan sebaliknya jika angkanya semakin kecil maka akan semakin kecil pula nilainya.

Pada bilangan bulat negatif, jika angkanya semakin kecil maka akan semakin besar nilainya. Dan sebaliknya jika angkanya semakin besar maka akan semakin kecil nilainya.

Pada kesempatan ini kita akan membahas soal yang berhubungan dengan bilangan bulat yang angkanya dibentuk dari beberapa batang korek api.

Diketahui :

Beberapa batang korek api disusun menjadi bilangan 369. (Bentuk bilangan dapat dilihat pada gambar lampiran).

Ditanya :

Angka paling besar yang didapatkan jika 1 batang korek api dipindahkan. Maksud dari dipindahkan disini adalah dipindahkan ke angka lain.

Penyelesaian :

Ada beberapa alternatif jawaban yang bisa didapatkan dengan memindahkan 1 batang korek api.

1. Memindahkan korek api pada angka 3 ke angka 3.

Jika batang korek api sebelah kanan bawah pada angka 3 dipindahkan ke sebelah kiri bawah angka 3 maka angka 3 tersebut akan berubah menjadi angka 2. Sehingga terbentuklah bilangan baru, yaitu 269 (lihat gambar nomor 1 : korek api yang dipindah ditandai dengan warna biru).

2. Memindahkan korek api pada angka 3 ke angka 3.

Jika batang korek api sebelah kanan atas pada angka 3 dipindahkan ke sebelah kiri bawah angka 3 maka angka 3 tersebut akan berubah menjadi angka 6. Sehingga terbentuklah bilangan baru, yaitu 669 (lihat gambar nomor 5 : korek api yang dipindah ditandai dengan warna biru).

3. Memindahkan korek api pada angka 6 ke angka 9.

Jika batang korek api pada angka 6 dipindahkan ke angka 9 maka angka 9 akan berubah menjadi 8 dan angka 6 akan berubah menjadi angka 5. Sehingga terbentuklah sebuah bilangan baru, yaitu 358 (lihat gambar nomor 2 : korek api yang dipindah ditandai dengan warna biru).

4. Memindahkan korek api pada angka 6 ke angka 3.

JIka batang korek api pada angka 6 dipindahkan ke angka 3 maka angka 3 akan berubah menjadi angka 9 dan angka 6 berubah menjadi angka 5. Sehingga terbentuklah sebuah bilangan baru, yaitu 959 (lihat gambar nomor 6 : korek api yang dipindah ditandai dengan warna biru).

5. Memindahkan korek api pada angka 9 ke angka 9.

JIka batang korek api sebelah kanan atas pada angka 9 dipindahkan ke angka 9 sebelah sebelah kanan bawah maka angka 9 akan berubah menjadi angka 6. Sehingga terbentuklah sebuah bilangan baru, yaitu 366 (lihat gambar nomor 3 : korek api yang dipindah ditandai dengan warna biru).

6. Memindahkan korek api pada angka 9 ke angka 6.

JIka batang korek api pada angka 9 dipindahkan ke angka 6 maka angka 6 akan berubah menjadi angka 8 dan angka 9 berubah menjadi angka 5. Sehingga terbentuklah sebuah bilangan baru, yaitu 385 (lihat gambar nomor 4 : korek api yang dipindah ditandai dengan warna biru).

7. Memindahkan korek api pada angka 9 ke angka 3.

JIka batang korek api pada angka 9 dipindahkan ke angka 3 maka angka 3 akan berubah menjadi angka 9 dan angka 9 berubah menjadi angka 5. Sehingga terbentuklah sebuah bilangan baru, yaitu 965 (lihat gambar nomor 7 : korek api yang dipindah ditandai dengan warna biru).

Dari keterangan di atas didapatkan beberapa bilangan baru, yaitu :

269, 669, 358, 959, 366, 385, dan 965.

Dari ketujuh bilangan baru tersebut, 965 adalah yang terbesar nilainya. Maka angka paling besar yang bisa didapatkan dari 369 (korek api) dengan memindahkan 1 korek api adalah 965.

Demikian penjelasan dan penyelesaian soal di atas.

22. Berapa angka terbesar dari 369 dengan memindahkan satu korek api ??jawaban 399,963,983,959 itu jawabannya salah.


Berapa angka terbesar dari 369 dengan memindahkan satu korek api ? Jawabannya adalah 965. Untuk penyelesaiannya bisa kita di lampiran dan pembahasan

Pembahasan

Angka 3 tersusun dari 5 batang korek api (sebagai ratusan)Angka 6 tersusun dar 6 batang korek api (sebagai puluhan)Angka 9 tersusun dari 6 batang korek api (angka satuan)

Angka 3 jika kita tambahkan 1 batang korek api, maka hanya akan terbentuk angka 9, sehingga angka ratusannya menjadi 9

Masalahnya 1 batang korek api tersebut berasal dari angka 6 (puluhan) ataukah angka 9 (satuan)

Jika angka tersebut berasal dari angka 6, berarti dari angka 6 tersebut jika dihilangkan 1 batang korek api, maka hanya akan terbentuk angka 5, sehingga angka puluhannya menjadi 5

Sedangkan kita tahu bahwa 50 lebih kecil dari 60.  

Jadi agar terbentuk bilangan terbesar, 1 batang korek api tersebut jangan kita ambil dari angka 6

Jika dari angka 9 kita ambil 1 batang korek api, hanya akan membentuk angka 3 dan angka 5

kita tahu bahwa angka 5 lebih besar dari 3, maka angka 9 kita ubah menjadi angka 5

Jadi bilangan terbesar yang terbentuk adalah angka 965
