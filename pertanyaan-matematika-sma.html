x² + nx + (n + 3) = 0

a = 1
b = n
c = n + 3

b²- 4ac > 0
(n)² - 4(1)(n + 3) > 0
n² - 4(n + 3) > 0
n² - 4n - 12 > 0
(n - 6)(n + 2) > 0

n = 6 atau n = -2

sehingga nilai n yang memenuhi adalah

n < -2 atau n > 6

karena akar-akarnya tidak positif, maka akar-akarnya negatif, sehingga

n > 0
(n + 3) > 0
n > -3

iris himpunannya

{ (-∞, -2) U (6, ∞) } n (0, ∞) n (-3, ∞)
= { (-∞, -2) U (6, ∞) } n (0, ∞)
= (6, ∞)

jadi, himpunan penyelesaiannya adalah : n > 6

Penjelasan dengan langkah-langkah:

integral tentu

=============

formula :

int (axⁿ) dx -----> axⁿ+¹/[n + 1] + C

-----> C = konstanta

=============

⁹ int 3 [6/x² + 1/x] dx

=============

int [6/x² + 1/x] dx

int [6x^-2 + x^-1] dx

int [6x^(-2 + 1)/(-2 + 1) + x^(-1 + 1)/(-1 + 1) ] dx

int [6x^-1/(-1) + x^(0)/(0) ] dx

= -6x^-1 + In|x| + C

= -6/x + In|x| + C

==============

batas atas = 9 , batas bawah = 3

sehingga,

= [-6/x + In|x|] - [-6/x + In|x| ]

= [-6/9 + In|9| ] - [-6/3 + In|3| ]

= [-2/3 + In|9| ] - [-2 + In|3| ]

= [-2/3 + 2 + In|9| - In|3|]

= [(-2 + 6)/3 + In (9/3) ]

= 4/3 + In(3)
