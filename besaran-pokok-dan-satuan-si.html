Satuan-satuan pokok SI tersebut di antaranya adalah detik sebagai satuan untuk waktu, meter untuk panjang, kilogram untuk massa, ampere untuk arus listrik, kelvin untuk suhu, mol untuk jumlah zat, dan kandela untuk intensitas cahaya.Besaran pokok adalah besaran yang satuannya telah didefinisikan terlebih dahulu dan tidak dapat dijabarkan dari besaran lain.

BESARAN DAN PENGUKURAN (kategori 6)

Besaran dibagi menjadi 2 macam yaitu Besaran Pokok dan Besaran Turunan.

-Panjang => meter
-Massa => kilogram
-Waktu => Sekon
-Suhu => Kelvin
-Intensitas Cahaya => Candela
-Kuat Arus Listrik => Ampere
-Jumlah Zat => mol
-Sudut datar => radian
-Sudut ruang => steradian
