3. Apa gunanya Corel Draw X4


Menciptakan desain logo atau simbol, yang mana ini adalah kegunaan Corel Draw yang paling banyak dimanfaatkan oleh penggunanya, terutama pembuatan logo dua dimensi karena kemudahannya dalam mengolah garis dan warna.Membuat desain undangan, brosur dan lain-lain juga menjadi suatu kegunaan dari program Corel Draw. Media publikasi offline lainnya juga menggunakan Corel Draw sebagai alat untuk mendesain. Corel Draw memiliki banyak jenis font yang dapat memudahkan desainer untuk mengeksplorasi imajinasi desain dan tulisan yang akan dibuat.Membuat cover buku juga dapat dilakukan di Corel Draw. Dengan Corel Draw maka tugas desain akan menjadi mudah karena dapat memanfaatkan desain sampul dan teknik pewarnaan yang lebih sempurna oleh Corel Draw. Detail gambar pun akan terlihat lebih jelas.Pembuatan gambar ilustrasi juga dapat dilakukan dengan Corel Draw. Gambar yang dihasilkan lebih berkualitas, terutama ketika berhubungan dengan lengkungan, garis atau sudut. Ukuran yang diperoleh dijamin sangat akurat.

11. Sebutkan Langkah untuk menyisipkan halaman di Corel Draw x4


Klik perintah menu Layout → Insert Page sehingga tampil kotak dialog Insert Page > Atur beberapa pengaturan pada kotak dialog Insert Page > Jika pengaturan telah selesai, klik tombol OK dan perhatikan bahwa akan terbentuk dua halaman baru yang di tambahkan.

Atau jg bisa :

1. Klik kanan pada bagian nama halaman (page 2) untuk menampilkan daftar pilihan menu perintah.

2. Klik perintah Insert Page After atau Insert Page Before untuk menyisipkan halaman baru setelah atau sebelum halaman aktif.

12. bagaimana cara mengatasi corel draw x4 yang tidak bisa di save, copy dan paste? mohon bantuanya. sejelas jelasnya. terimakasih.


1. Kamu download dulu file PSIKey_3.dll di: http://www.mediafire.com/download/kh5764u2ihb9u8p/PSIKey_3.dll.rar
2. Setelah di download kemudian di Ekstrak
3. Lalu Salin (Copy) ke: C → Program Files → Common Files → Protexis → License Service
4. Lalu pilih Replace atau Rewrite
5. Untuk memastikannya telah aktif kamu buka jendela Run, lalu ketik Services.msc, lalu klik OK
6. Lalu buka Corel Draw nya apa sudah bisa atau belum

14. apa yang harus dilakukan apabila saat menginstall corel draw x4 di PC tidak muncul perintah next dilayar monitor, terus harus klik apa????


Solusinya adalah dengan cara menghapus folder di windows yang melarang anda untuk menginstall program tersebut. caranya adalah

1. buka my computer pada pc atau laptop anda.

2. kedua buka directory C:

3. lalu pilih dan buka folder yang bernama "Windows".

3. kemudian cari folder yang bernama "System32" dan kemudian buka folder tersebut

4. setelah itu cari folder yang bernama "drivers"  kemudian hapus folder tersebut.

5. tidak perlu khawatir bila menghapus folder tersebut karena laptop anda akan update dan memperbaharui system operasi windows secara otomatis.

Kesimpulannya di dalam folder drivers terdapat program yang melarang anda untuk menginstall beberapa program dikarenakan pc atau laptop yang anda gunakan terlalu usang atau out of date. Jadi dengan menghapus folder tersebut pc atau laptop anda akan secara otomatis memperbaharui system operasi windows anda.
