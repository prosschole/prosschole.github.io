1. Tentukan persamaan fungsi kuadrat dari grafik fungsi kuadrat berikut​


Jawab:

Grafik A : y = [tex]-x^{2}[/tex]+ 5x -4

Penjelasan dengan langkah-langkah:

1. Grafik A = Diketahui titik potong dengan sumbu x adalah (1,0) dan (4,0), titik potong dengan sumbu y pada (0, -4), dan grafik terbalik menunjukan koefisien x^2 negatif,

y = ax^2 +bx + c

Jika x = 1 dan y = 0,

0 = a + b + c ...(1)

Jika x = 4 dan y = 0,

0 = 16a + 4b + c ...(2)

Jika x = 0 dan y = -4,

-4 = c ...(3)

Dari persamaan (3) dapat diketahui bahwa nilai c = -4, substitusikan nilai c = -4 ke persamaan (1) dan (2),

a + b = 4 ...(4)       x 4 --> 4a +4b = 16

16a + 4b = 4 ...(5) x 1 --> 16a +4b = 4

                                       _________ -

                                       -12a       = 12  --> a = -1

Substitusikan nilai a= -1 ke persamaan (4),

-1 + b = 4

b = 5

y = [tex]-x^{2}[/tex]+ 5x -4

Maka, untuk grafik A persamaan kuadratnya adalah y = [tex]-x^{2}[/tex]+ 5x -4

Untuk grafik B nya maaf saya gk jawab, saya udah capek ngetik. Semoga membantuu..


2. Tentukan persamaan fungsi kuadrat dari grafik fungsi kuadrat berikut​


Jawab:

a) [tex]y = -x^{2} + x + 2[/tex]

b) [tex]y = \frac{x^{2} }{y} - 3x + 4[/tex]

Penjelasan dengan langkah-langkah:

Diket:

a)

pembuat nol fungsi = (-1, 0) dan (2, 0)

titik potong sb. y = (0, 2)

b)

pembuat nol fungsi = (2, 0) dan (4, 0)

titik potong sb. y = (0, 4)

Ditanya:

PFK-nya = ?

Dijawab:

a)

[tex]f(x) = a(x - p)(x - q)\\f(0) = 2\\a(0 - (-1))(0-2) = 2\\a(0 + 1)(0-2) = 2\\a(1)(-2) = 2\\-2a = 2\\a = -1\\[/tex]

maka, PFK-nya adalah:

[tex]y = f(x) = a(x-p)(x-q)\\y = -1(x - (-1)(x -2)\\y = -1(x+1)(x-2)\\y = -1(x^{2} - x -2)\\y = -x^{2} + x + 2[/tex]

PFK-nya adalah -x² + x +2

b)

[tex]y = f(x) = a(x - p)(x - q)\\f(0) = 4\\a(0 - 2)(0 - 4) = 4\\a(-2)(-4) = 4\\8a = 4\\a = \frac{4}{8} = \frac{1}{2}[/tex]

maka, PFK-nya adalah

[tex]y = f(x) = a(x-p)(x-q)\\y = \frac{1}{2} (x - 2)(x - 4)\\y = \frac{1}{2} (x^{2} - 6x + 8)\\\\y = \frac{x^{2} }{2} - 3x + 8[/tex]

maka, PFK-nya adalah y = x²/2 - 3x + 4

[tex]y= -(x^2-2x+1) + 9[/tex]

10. dengan grafik fungsi kuadrat dibawah ini maka tentukanlah persamaan fungsi kuadratnya​


Dimisalkan : persamaan fungsi kuadrat tersebut adalah : f(x) = ax² + bx + c

Grafik f(x) melewati titik ( –6 , 0 ) :

f(–6) = 0

a.([–6]²) + b.(–6) + c = 0

36a - 6b + c = 0 ... ( i )

Grafik f(x) melewati titik ( 3 , 0 ) :

f(3) = 0

a.(3²) + b.(3) + c = 0

9a + 3b + c = 0 ... ( ii )

Grafik f(x) melewati titik ( 2 , 8 ) :

f(2) = 8

a.(2²) + b.(2) + c = 0

4a + 2b + c = 8 ... ( iii )

Eliminasi ( i ) dan ( ii ) :

( i )( × 1 ) : 36a - 6b + c = 0

( ii )( × 1 ): 9a + 3b + c = 0

___________________ —

27a - 9b = 0

« kedua ruas dibagi 9 »

3a - b = 0 ... ( iv )

Eliminasi ( i ) dan ( iii ) :

( i )( × 1 ) : 36a - 6b + c = 0

(iii )( × 1 ): 4a + 2b + c = 8

___________________ —

32a - 8b = –8

« kedua ruas dibagi 8 »

4a - b = –1 ... ( v )

Eliminasi ( iv ) dan ( v ) :

( v )( × 1 ) : 4a - b = –1

( iv )( × 1 ): 3a - b = 0

___________________ —

a = –1

Substitusikan nilai a ke ( v ) :

4a - b = –1

4.(–1) - b = –1

–4 - b = –1

–b = –1 + 4

–b = 3

b = –3

Substitusikan nilai a dan nilai b ke ( ii ) :

9a + 3b + c = 0

9.(–1) + 3.(–3) + c = 0

–9 - 9 + c = 0

–18 + c = 0

c = 18

Sehingga, didapatkan untuk persamaan grafik f(x) :

a = –1 ; b = –3 ; dan c = 18

Jadi, persamaan grafik fungsi kuadrat tersebut adalah :

f(x) = –x² - 3x + 18

17. Suatu fungsi kuadrat melalui titik (0,10),(1,4),dan (2,0) tentukan rumus fungsi kuadrat tersebut


Jawab dan Penjelasan:

Tanpa harus menghafal rumus sama sekali!

Fungsi kuadrat pasti berbentuk spt:

y = ax² + bx + c (harus ini gak boleh yg lain)

Titik (0, 10) artinya x = 0 dan y = 10

Titik (1, 4) artinya x = 1 dan y = 4

Titik (2, 0) artinya x = 2 dan y = 0

Masukkin aja semuanya ke fungsi kuadrat y = ax² + bx + c. Nanti dapatnya

10 = a(0)² + b(0) + c --> c = 10

4 = a(1)² + b(1) + c

4 = a + b + 10

a + b = -6 ...(1)

0 = a(2)² + b(2) + c

0 = 4a + 2b + 10

4a + 2b = -10

2a + b = -5 ...(2)

(1) dan (2) dieliminasi, kamu akan dapat a = 1, b = -7

Jadi, a = 1, b = -7, c = 10 sehingga fungsi kuadratnya

y = x² - 7x + 10
