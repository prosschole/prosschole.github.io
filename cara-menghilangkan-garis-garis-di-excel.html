15. Untuk menghilangkan atau menyembunyikan garis khayal pada microsoft excel 2007 , maka hilangkan centang pada menu....


garis khayal(Garis bantu) Hilangkanya centang pada menu  Klik ribbon Viewpada menu Show lalu buang ceklist Gridlines
1. Klik ribbon File lalu klik Options

2. Klik Advanced pada jendela yang muncul dan pada panel sebelah kanan scroll kebawah hingga sampai pada pengaturan Display options for this worksheetkemudian hilangkan tanda centang pada show gridlines

16. Jelaskanlah Langkah-langkah menghilangkan tampilan garis-garis grid pada Ms. excel​


Untuk menyembunyikan gridlines melalui Excel Options, klik pada menu File, di backstage Microsoft Excel yang muncul, klik Options. Di jendela Excel Options yang muncul, klik tab Advenced (1), gulung ke bawah area sebelah kanan sampai di kelompok Display options for this worksheet (2), hilangkan tanda centang di kotak centang (check box) Show gridlines (3) dan kemudian klik tombol OK (4).

Untuk menampilkan kembali gridline melalui Excel Options, ulangi langkah di atas, beri tanda centang di kotak centang Show gridlines dan kemudian klik tombol OK.

20. Cara untuk membuat garis tabel pada Microsoft Excel adalah ...

Cara untuk membuat garis tabel pada Microsoft Excel adalah dengan mengatur border

29. cara mewarnai garis tabel di microsoft excel​


tekan cut lalu pilihan warna sesuai yg diinginkan kalo ga salah

Jawaban:

Caranya adalah sebagai berikut:

- sorot/ blok tabel yg garisnya ingin diwarnai

- klik kanan di bagian yg tersorot & klik Format Cells

- saat tampil kotak dialog Format Cells, klik tab border

- klik Color dan pilih salah 1 warna yg tersedia

- klik tombol OK

Penjelasan:

Untuk tab border dapat dilihat gambar berikut ini (ditunjukkan tanda panah merah nomor 1). Sedangkan pilihan Color ditunjukkan tanda panah nomor 2.
