4. Cara merubah rom samsung s5 replika


1. rooting terlebih dahulu Smartphone tersebut ( model number, os version, build number diusahakaan sama )

2. download ODIN & file rom yang diinginkan
3. Buka odin lalu sambungkan smartphone, jangan lupa aktifkan debugging mode dengan cara menuju About Phone lalu tap 7x build number. lalu kembali dan klik Developer Options.
4. Turn Off Smartphone dan tahan tombol Power + Down + Home button. ( jangan lupa kabel harus tersambung ke pc )
5. di ODIN klik PDA lalu locate dimana anda menyimpan file ROM ( pastikan filenya dengan extension .tar )
6. lalu check Fast Boot, Reboot dan Re-Partition
7. lalu start
8. tunggy hingga berwarna hijau, lalu copot smartphone dari pc
9. tunggu beberapa saat karena proses boot up membutuhkan waktu sekitar 5-10 menit.

12. cara menjadikan sd card sebagai default penyimpanan gimana ya?, soalnya aku kalo download game/aplikasi dari google play selalu nyimpen nya ke internal memory, nah cara nya agar nyimpennya ke sd card gimana?


cara mengatur penyimpanan aplikasi android ke memori eksternal SD Card secara default :
√ Buka menu Settings atau Pengaturan, lalu pilih Penyimpanan.
√ Setelah masuk ke pilihan Penyimpanan, maka scrolling ke bagian bawah Penyimpanan, pilih Write Disk Default.
√ Pada Write Disk Default, atur pilihan penyimpanan perangkat ke memori eksternal atau SD Card. Selesai.

19. Bagaimana cara memformat memory card menjadi penyimpanan internal??


Masuk dengan aplikasi bertuliskan stroge/berkas lalu pindahkan memory card ke penyimpanan internal
Maaf klo salahCara mengubah MicroSD jadi penyimpanan internalUntuk mengubah MicroSD menjadi mediapenyimpanan internal sebenarnya sudah disiapkan (muncul dengan sendirinya) saat kamu pertama kali melakukan upgrade ke Android Marshmallow atau ketika kamu baru pertamakali memasukkan MicroSD ke smartphone.Disitu kamu akan diberikan opsi “Gunakan sebagai penyimpanan portabel”atau “Gunakan sebagai penyimpanan internal”. Bila ingin menjadikan sebagai memori internal, pilih opsti kedua yaitu “Gunakan sebagai penyimpanan internal”,setelah itu ikuti saja prosesnya hingga selesai (dalam prosesnya, migrasi data juga akan dilakukan).Bila opsi pengaturan diatas tidak muncul atau kamu ingin mengubah pengaturan yang sudah ditetapkan sebelumnya, cukup masuk saja ke “Pengaturan > Penyimpanan & USB > Pilih Kartu SD” setelah itu pilih “option (icon 3 titik dipojok kanan atas) > Format sebagai internal/portabel”.

26. apa kegunaan sd card?
