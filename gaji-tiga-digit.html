4. Sebuah bilangan terdiri dari 3 digit yang jumlah ketiga digitnya adalah 12. Jika digit pertama dan kedua ditukar, maka bilangan yang terjadi nilainya adalah 90 lebihnya dari bilangan semula. Sedangkan jika digit kedua dan ketiga ditukar, maka bilangan yang terjadi nilainya 9 lebihnya dari bilangan semula. Tentukanlah bilangan semula yang dimaksud.


misal bilangan ABC
maka
A + B + C = 12

"digit pertama dan kedua ditukar"
BAC = 90 + ABC
  100B + 10A + C = 90 + 100A + 10B + C
  90B - 90A = 90
  B - A = 1 
  A = B - 1 ........ (1)

"digit kedua dan jetiga ditukar"
ACB = 9 + ABC
 100A + 10C + B = 9 + 100A + 10B + C
  9C - 9B = 9
  C - B = 1
  C = B + 1

Jadi,
A + B + C = 12
  B - 1 + B + B + 1 = 12
  3B = 12
  B = 4

 A = B - 1
 A = 4 - 1
 A = 3

C = B + 1
C = 4 + 1
C = 5

Didapat : A = 3, B = 4, C = 5
ABC = 345

13. Digit tengah dari bilangan tiga digit adalah 0 dan jumlah dua digit lainnya adalah 7. Jika dua digit tersebut dibalik maka menjadi bilangan baru yang 99 lebih kecil dari bilangan awalnya. Carilah bilangan tersebut.


bilangan 3 angka, asumsikan urutan bilangannya adalah a,b, dan c, dengan angka tengah adalah 0 berarti b=0
bilangan awal: 100a + 10 + c
(ket : a + c = 7 (persamaan 1))
bilangan kedua (a dan c ditukar): 100c + 10 + a

bilangan awal - bilangan kedua = 99
(100a + 10 + c) - (100c + 10 + a) = 99
100a + 10 + c - 100c - 10 - a = 99
99a -99c = 99
-----------------------:99
a - c = 1 (persamaan 2)

eliminasi persamaan 1 dgn persamaan 2
a + c = 7
a - c = 1
________-
2c = 6
c = 3 (oke ketemu)

masukin c = 3, ke a + c = 7
a + 3 = 7
a = 4

jadi urutan angkanya adalah :
a = 4
b = 0
c = 3

jadi bilangan 3 digit pertama itu 403
dan bilangan 3 digit kedua itu 304
selisihnya 403 - 304 = 99

18. Saya nombor tiga digit.angka kedua saya empat kali lebih besar darioada digit ketiga.angka pertama saya adalah tiga kurang daripada digit kedua saya.siapakah saya?


Jawab:

141 atau 582.

Penjelasan dengan langkah-langkah:

Misalkan bilangan tiga digit itu adalah [tex]\overline{abc}[/tex], dengan [tex]a[/tex] adalah digit ratusan/pertama, [tex]b[/tex] adalah digit puluhan/kedua, dan [tex]c[/tex] adalah digit satuan/ketiga.

Dari soal, diketahui bahwa digit kedua empat kali lebih besar daripada digit ketiga, dan digit pertama tiga kurang dari digit kedua.

Maka, kita bisa tuliskan dua persamaan, yaitu :

[tex]b=4c \ \ldots(1)[/tex]

[tex]a=b-3 \ \ldots(2)[/tex]

Sekarang, perhatikan bahwa nilai [tex]c[/tex] yang mungkin hanya 1 atau 2. Kenapa begitu? Karena ketika [tex]c=3,4,\ldots ,9,[/tex] [tex]b[/tex] akan lebih dari 9, yang bukan merupakan bilangan 1 digit.

Sekarang, kita bagi kasus.

Kasus 1 : [tex]c=1.[/tex]

Substitusikan nilai [tex]c[/tex] ke persamaan (1), didapatkan bahwa [tex]b=4[/tex].

Substitusikan nilai [tex]b[/tex] ke persamaan (2), didapatkan bahwa [tex]a=1.[/tex]

Maka [tex]\overline{abc}=141.[/tex]

Kasus 2 : [tex]c=2.[/tex]

Substitusikan nilai [tex]c[/tex] ke persamaan (1), didapatkan bahwa [tex]b=8.[/tex]

Substitusikan nilai [tex]b[/tex] ke persamaan (2), didapatkan bahwa [tex]a=5.[/tex]

Maka [tex]\overline{abc}=582.[/tex]

Jadi, bilangan tiga digit yang mungkin adalah 141 atau 582.

19. Sebuah bilangan tiga digit mempunyai digit ratusan lebih besar dari digit puluhan dan digit ratusan sama dengan digit satuan. Diketahui bahwa selisih bilangan tersebut dengan bilangan yang diperoleh dengan menukar digit satuan dan puluhan bilangan tersebut adalah 72. Jika jumlah digit-digit bilangan tersebut adalah 19, tentukan bilangan tersebut.


karena angka satuan = angka ratusan maka bilangan tersebut adalah aba, dengan a > b.

selisih bilangan tersebut dan bilangan yang sudah ditukar adalah 72, maka:
aab - aba = 72
(100a + 10 a + b) - (100a + 10b + a) = 72
110a + b - 101a - 10b = 72
9a - 9b = 72
a - b = 8 ...(1)

tetapi 2a + b = 19 ...(2)

jumlahkan persamaan (2) dan (1) didapat :
(2a + b) + (a - b) = 19 + 8
3a = 27
a = 9

subtitusikan nilai a ke persamaan (1), didapat nilai b = 1

jadi, bilangan yang dimaksud adalah 919

23. suatu bilangan tersusun atas 3 digit. Jumlah ketiga digit adalah10. Dua kali digit ketiga dikurangi jumlah digit pertamadan kedua nilainya 5. Bilangan itu nilainya 20 kali jumlahketiga digit ditambah 215. Bilangan yang dimaksud adalah..kalo bisa sama contohnya yaa.. terimakasih​


Misal bilangan itu disusun oleh:

[tex]a[/tex] di tempat ratusan

[tex]b[/tex] di tempat puluhan

[tex]c[/tex] di tempat satuan

atau dinotasikan [tex](abc)_{10}[/tex]

Dari soal diperoleh informasi

[tex]\begin{aligned}a+b+c&=10\\ \text{selanjutnya,} (abc)_{10}&=20(a+b+c)+215\\&=20(10)+215\\&=200+215\\&=415\end{aligned}[/tex]

jadi bilangan itu adalah 415

25. Sebuah bilangan terdiri dari 3 digit yang jimlah ketiga digitnya adalah 12. Jika digit pertama dan kedua ditukar, maka bilangan yang terjadi nilainya adalah 90 lebihnya dari bilangan semula. Sedangkan jika digit kesua dan ketiga ditukar, maka bilangan yang terjadi nilainya 9 lebihnya dari bilangan semula. Tentukanlah bilangan semua yang di maksud.





Misal bilangan ABC
maka
A + B + C = 12

"digit pertama dan kedua ditukar"
BAC = 90 + ABC
⇒ 100B + 10A + C = 90 + 100A + 10B + C
⇒ 90B - 90A = 90
⇒ B - A = 1
⇒ A = B - 1 ........ (1)

"digit kedua dan jetiga ditukar"
ACB = 9 + ABC
⇒ 100A + 10C + B = 9 + 100A + 10B + C
⇒ 9C - 9B = 9
⇒ C - B = 1
⇒ C = B + 1

Jadi,
A + B + C = 12
⇒ B - 1 + B + B + 1 = 12
⇒ 3B = 12
⇒ B = 4

A = B - 1
⇒ A = 4 - 1
⇒ A = 3

C = B + 1
⇒ C = 4 + 1
⇒ C = 5

Didapat : A = 3, B = 4, C = 5
Jadi, bilangan semula adlah
ABC = 345

30. sebuah bilangan terdiri atas dua digit. nilai bilangan itu sama dengan tiga kali jumlah kedua digit itu di tambah 10.digit ke dua di kurang digit pertama sama dengan 5.Bilangan tersebut adalah.....


Digit kedua dikurang digit pertama = 5. Artinya bahwa digit kedua > digit pertama.
Kemungkinan itu terjadi ketika:
(i) 9 - 4 = 5 maka bilangannya 49
(ii) 8 - 3 = 5 maka bilangannya 38
(iii) 7 - 2 = 5 maka bilangannya 27
(iv) 6 - 1 = 5 maka bilangannya 15
(v) 5 - 0 = 5 maka bilangannya 05.

Untuk (v) tidak memenuhi karna bukan dua digit.
Untuk (iv) dan (iii) tidak memenuhi karna bilangannya kurang dari 30.
Pandang:
(i) misalkan A = digit pertama dan B = digit kedua maka;
A + B = 4 + 9 = 13
3.13 = 39
Lalu ditambahkan 10 menjadi 49.
Jadi benarlah 49 yg menjadi solusinya.
