2. pengertian input alat, proses proses,alat output dan penyimpanan​


Jawaban:

A.  Alat Input atau alat masukkan

      Alat Input/alat masukan adalah Alat yang fungsinya untuk memberi data atau perintah ke dalam komputer dari luar sistem yaitu hardwarenya sendiri agar menghasilkan informasi yang diperlukan untuk komputer.

   Contoh alat input:  

1. Mouse

Untuk menggerakkan dan menekan kursor/berbentuk panah pada monitor agar bisa digunakan untuk memilih objek.

2. Keyboard

Berbentuk papan panjang yang terdiri dari tombol angka, huruf, dan tombol lainnya untuk memasukkan data dari tombol tersebut ke dalam komputer.

3. Scanner

Bentuknya persegi panjang yang bisa dibuka tutup untuk memasukkan gambar dari luar komputer ke dalam komputer untuk diproses dan dilihat di monitor.

B. Alat Output atau Alat Keluaran

   Kebalikan dari alat input, yaitu Alat untuk mengeluarkan data atau informasi yang diperlukan dari dalam komputer.

Contoh alat Output :

1. Monitor

2. Printer

3. Speaker

4. Proyektor

C. Alat Proses

     

Suatu Alat untuk memproses atau mengolah data komputer.

1. Processor

2. RAM ( Random Access Memory )

3. VGA Card

D. Alat Penyimpanan

     

Dari namanya saja Sudah jelas, fungsinya untuk menyimpan data yang diperlukan.

1. Flashdisk

2. Harddisk

3. CD ( Compact Disc )

4. DVD ( Digital Versatile Disc )

4. perbedaan alat input langsung dengan alat input tidak langsung​


Jawaban:

Alat input langsung yaitu alat yang dapat digunakan untuk menginput data secara langsung ke komputer.

Alat input tidak langsung yaitu alat yang digunakan untuk menginput data ke komputer dengan melalui alat proses terlebih dahulu di komputer.

Penjelasan:

Contoh alat input langsung: keyboard, mouse, microphone

Contoh alat input tidak langsung: floppy disk, compact disk (cd), digital versatile disk (dvd).

Dalam hal ini, floppy disk butuh perangkat floppy disk drive untuk menginput datanya ke komputer sedangkan cd dan dvd butuh perangkat optical disk drive untuk menginput datanya ke komputer.

14. Jelaskan apa perbedaan antara alat input langsung dan alat input tidak pangsung


Jawaban:

alat input langsung yaitu alat yg dapat menginput data secara langsung ke dalam komputer tanpa perantara.

alat input tidak langsung yaitu alat yg menginput data ke dalam komputer dg bantuan alat pendukung (sebelum data benar-benar masuk ke komputer).

Penjelasan:

Contoh alat input langsung: mouse dan keyboard.

Contoh alat input tidak langsung: optical disk dan floppy disk.

Alat seperti mouse dan keyboard dapat langsung menginput data ke komputer sedangkan alat seperti optical disk dan floppy disk butuh bantuan alat pendukung:

- optical disk butuh perangkat optical disk drive

- floppy disk butuh perangkat floppy disk drive

Amati gambar berikut ini:

1. contoh perangkat optical disk drive

2. contoh perangkat floppy disk drive


15. apa yang dimaksud dengan alat input langsung dan alat input tidak langsung?


Alat input langsung yaitu input yang dimasukkan langsung diproses oleh alat pemroses

Alat input tidak langsung melalui media tertentu sebelum suatu input diproses oleh alat pemroses.

contoh alat input langsung:
1.mouse
2.keyboard
3.scener
4.ligh pen

contoh alat input tidak langsung:

·         keypunch yang dilakukan melalui media punched card(kartu plong)

·         key-to-tape yang merekam data ke media berbentuk pita (tape) sebelum diproses oleh alat pemroses

·         key-to-disk yang merekam data ke media magnetic disk (misalnya disket atau harddisk) sebelum diproses lebih lanjut.

19. Apa pengertian dari perangkat keras,perangkat lunak,alat input,alat output,dan alat proses?


Jawaban:

Perangkat keras komputer (bahasa Inggris: hardware atau disingkat HW) adalah semua bagian fisik komputer, dan dibedakan dengan data yang berada di dalamnya atau yang beroperasi di dalamnya.

Perangkat lunak atau peranti lunak (bahasa Inggris: software) adalah istilah khusus untuk data yang diformat, dan disimpan secara digital, termasuk program komputer, dokumentasinya, dan berbagai informasi yang bisa dibaca, dan ditulis oleh komputer.

Alat input adalah perangkat masukan yang memberikan perintah langsung kepada perangkat output(keluaran) pada komputer untuk memberikan sebuah data atau informasi yang dibutuhkan oleh user.

Alat Output adalah komponen komputer yang berfungsi menghasilkan informasi yang diperoleh dari hasil proses dari peralatan proses komputer.

Alat Proses adalah CPU (Central Prosesing Unit) yang merupakan unit proses utama dan terpenting dalam komputer yang mengendalikan seluruh proses pengolahan data mulai dari membaca data dari peralatan input, mengolah atau memproses sampai pada mengeluarkan informasi (Output) ke peralatan Output.

29. alat input suara? alat input video ? alat input game? apa saja ya minimal 5


Alat input atau perangkat input memiliki banyak jenis, diantaranya alat input suara, alat input video dan alat input game. Alat input merupakan mikroprosesor yang dipakai dengan tujuan untuk menghubungkan perangkat luar dan komputer. Benda-benda yang merupakan perangkat input diantaranya :

Alat input untuk suara adalah keyboard, mouse, joystick, touchpad, dan microphone.Alat input untuk video adalah webcam, flashdisk, keyboad, CD/DVD ROM, dan mouse.Alat input untuk game adalah modem, flashdisk, joystick, headset, digitizer, dan barcode scanner.Pembahasan :

Perangkat input atau alat input merupakan suatu sistem yang terdapat pada komputer. Komputer sendiri memiliki pengertian suatu sistem yang terdiri atas banyak komponen yang tersusun sedemikian rupa menjadi suatu kesatuan. Komponen-komponen itu nantinya akan saling terhubung dan bekerja sama untuk melakukan suatu proses sehingga mampu menghasilkan suatu informasi pada komputer yang akan ditampilkan pada layar.

Perangkat input memiliki 2 jenis, diantaranya adalah perangkat input langsung dan perangkat input tidak langsung. Perangkat input langsung adalah perangkat untuk memasukkan data secara langsung maksudnya, tanpa perlu diterjemahkan lagi oleh operator. Sehingga untuk perangkat input komputer yang satu ini akan memiliki resiko kesalahan yang sangat kecil dan waktu proses inputnya relatif lebih cepat.

Sedangkan perangkat input tidak langsung memerlukan bantuan operator  dengan cara manual diketik untuk pengerjaannya, sehingga resiko kesalahannya cukup tinggi.

Alat input untuk suara adalah keyboard, mouse, joystick, touchpad, dan microphone.Alat input untuk video adalah webcam, flashdisk, keyboad, CD/DVD ROM, dan mouse.Alat input untuk game adalah modem, flashdisk, joystick, headset, digitizer, dan barcode scanner.
