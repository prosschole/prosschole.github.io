13. Tolong jadikan nama dimas dalam bahasa jepang


ディマス 「でぃます」Dimas.

Pembahasan

• Nama dari luar Jepang Yang akan Masuk kedalam Bahasa Jepang , Kita tulis Menggunakan Sistem Katakana

Huruf Yang tidak Memiliki vokal Seperti "S" Akan Menjadi "Su"

Contoh:

Nama Saya Dimas.

Orang + の ( No ) + 名前 ( Namae ) + は ( Wa ) + [Nama]

私の名前はディマスです。 ( Watashi no namae Wa Dimasu desu. )

Watashi => SayaNo => PartikelNamae => NamaWa => PartikelDimasu => DimasDesu => Tidak Memiliki arti, Berfungsi sebagai Penghalus kalimat.

ディマス (Dimasu)

Karena Dimas bukan nama orang Jepang, maka ditulis menggunakan huruf Katakana. Katakana digunakan untuk menulis bahasa asing yang diserap kedalam bahasa Jepang.
