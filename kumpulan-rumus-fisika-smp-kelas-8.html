5. Tuliskan rumus rumus persamaan fisika yang pernah kalian pelajari di smp...?


Gerak Lurus.
x = vt
Δx = 1/2 (v + v₀)t
v = v₀ + at
Δx = v₀t + 1/2 at²
v² = v₀² + 2aΔx

Hukum Newton
ΣF = 0 ← Hukum Newton I.
ΣF = ma ← Hukum Newton II.
F aksi = -F reaksi ← Hukum Newton III.

Suhu dan Kalor
Q = mcΔT
Q = mL
Q = mU
c = Q / ΔT = mc
L = L₀(1 + αΔT)

Tekanan dan Fluida
P = F / A
P hidrostatis = ρgh

Usaha dan Energi
W = Fs
EP = mgh
EK = 1/2 mv²
EM = EP + EK
EM₁ = EM₂ ← Hukum kekekalan energi mekanik.

Optik
P = 1 / PR (untuk mata miopi) dan P = 4 - 1 / PP (untuk mata hipermetropi).
1 / f = 1 / s + 1 / s'
P = sn / f ← Kuat lensa pada lup.

Getaran, Gelombang dan Bunyi
d = 1/2 vt ← Jarak yang diukur menggunakan prinsip pantulan.
v = λf = λ / T
f = n / t = 1 / T
T = t / n = 1 / f

Listrik Statis dan Dinamis
F = kq₁q₂ / r²
E = F / q = kq / r²
V = kq / r
EP = kq₁q₂ / r

P = VI = V² / R = I²R
E = Pt = VIt = V²t / R = I²Rt
I = q / t
n = q / e ← banyaknya elektron yang mengalir.
