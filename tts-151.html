12. 111+111=6 121+121=8 151+151=?


111 + 111 = 6, 121 + 121 = 8, 151 + 151 = 14. Hasil tersebut diperoleh dengan melihat pola bilangan yang memungkinkan yaitu, terlihat polanya untuk mendapatkan hasil penjumlahannya adalah dengan menjumlahkan angka-angka penyusun pada setiap bilangan. Untuk lebih jelasnya bisa dilihat di pembahasan.


Pembahasan

Pola dari penjumlahan bilangan tersebut adalah


111 + 111

= (1 + 1 + 1) + (1 + 1 + 1)

= 3 + 3

= 6



121 + 121

= (1 + 2 + 1) + (1 + 2 + 1)

= 4 + 4  

= 8



Jadi hasil dari 151 + 151 adalah

= 151 + 151  

= (1 + 5 + 1) + (1 + 5 + 1)

= 7 + 7

= 14

•STATISTIKA•

Statistika adalah ilmu yang mempelajari mengumpulkan,menyajikan data,serta menganalisis data.

Data adalah sekumpulan ungkapan tentang fakta nyata berupa hasil temuan yang di peroleh dari suatu kegiatan penelitian. Data terbagi menjadi 2 yaitu :

Data Kualitatif, yaitu data yang variabel C-nya secara umum bukan angka (nonumerik). Misalnya, Data profesi/mata pencaharian data nomor urut peserta didik di suatu kelas dan lain lain.Data Kuantitatif, yaitu data berupa angka angka yang angkanya melambangkan/bermakna sebagai besaran tertentu.Besaran yang di maksud dapat berupa besaran cacahan (Buah,Lusin,Kodi,Gros,dan Rim) atau besaran ukuran (kg,m,inci,m²,m³,liter,barel,°C,°F, Detik dan menit)Ukuran Permusatan Data

1) Rata Rata (Mean) merupakan suatu ukuran yang memberikan gambaran yang lebih singkat dan jelas dari sekelompok data.Rata rata merupakan hasil yang cukup mewakili dari pengukuran sebenarnya.

Rumus :

Rata Rata = Jumlah Data / Banyak Data

x= x1 + x2 + x3 ... Xn / n

2) Median merupakan nilai tengah data setelah data diurutkan. Jadi Median adalah membagi data menjadi 2 sama besar.

Rumus Untuk Nilai n ganjil :

= Banyak data + 1 / 2

Rumus untuk nilai genap :

=X1 - X2 / 2
