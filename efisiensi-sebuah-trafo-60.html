Penjelasan Dengan Langkah-langkah :

Diketahui :

μ  = Efesiensi = 60%

WS =  Energi Sekunder = 300 J


Ditanyakan :

WP = Energi Primer = ...?


Penyelesaian :

μ  = WS/WP x 100%

60% = 300/WP x 100%

60WP = 300 x 100

   WP =  30.000/60

   WP =  500 J


Kesimpulan :Jadi Energi yang masuk Travo adalah 500 J

4. Efisiensi sebuah trafo 60. Energi listrik yang dikeluarkan sebuah trafo adalah 300J, berapakah energi listrik yang masuk trafo, jika efisiensinya 50%​


Transformator ( Trafo )

alat untuk mengubah tegangan arus bolak balik menjadi lebih rendah atau lebih tinggi.  

Effisiensi trafo :

η = ( Ps / Pp ) x 100%

Ps  =  Vs . Is

Pp  =  Vp . Ip

Ws =  Ps . t

Wp = Pp . t


dimana :

Ps  =  daya sekunder ( watt )

Pp  =  daya primer ( watt )

Ws =  Energi listrik keluar ( joule )

Wp = Energi listrik masuk ( joule )

________________________________________________________

diketahui :

η = 60%

Ws = 300 J

ditanya

Wp

jawab :

W = P x t

P =  W / t


              η  = ( Ps / Pp ) x 100%

                  = ( Ws/t / Wp/t ) x 100%    ( t keduanya di coret )

         60%  = ( 300 / Wp ) x 100%

60% / 100% = 300 / Wp

           0,6  = 300 / Wp

           Wp  = 300/0,6

                  = 500 J

jadi energi listrik yang masuk pada trafo sebesar 500 Joule

26. Efisiensi sebuah trafo 60%. Trafo ini dihubung-kan dengan tegangan primer 200 V. Jika dayayang keluar dari trafo sebesar 120 Watt, makakuat arus listrik primer trafo ini adalah ...​


Jawaban:

Kuat arus primer sebesar 1 A.

Penjelasan:

Diketahui :

η  =  Efesiensi = 60%

Vp = Tegangan Primer = 200Volt

Ps  = Daya sekunder = 120 Watt

Ditanyakan :

Ip = Kuat arus primer = ...

Penyelesaian :

[tex]\boxed{\eta=\frac{Ps}{Pp}\times100\%}[/tex]

[tex]\eta=\frac{Vs\times{Is}}{Vp\times{Ip}}\times100\%[/tex]

[tex]60\%=\frac{120}{200\times{IP}}\times100\%[/tex]

[tex]0,6=\frac{120}{200Ip}[/tex]

[tex]200Ip=\frac{120}{0,6}[/tex]

200Ip = 200

[tex]Ip=\frac{200}{200}[/tex]

IP = 1 A
