3. [179/200]616 x (-179) = (-179) x nnilai n​


Nilai n adalah 616

PENJELASAN

Diketahui:

616 × (-179) = (-179) × n

Ditanyakan:

Nilai n

Jawab:

616 × (-179) = (-179) × n

=> 616 × (-179)/(-179) = n

=> 616 × 1 = n

=> 616 = n

Sifat Komutatif

a × b = b × a

616 × (-179) = (-179) × n

616 × (-179) = (-179) × 616

Jadi, nilai n adalah 616

Penjelasan dengan langkah-langkah:

n = -110.264/-179

Pendahuluan

Bilangan bulat adalah bilangan yang tersusun dari bilangan cacah dan bilangan bulat negatif. Bilangan bulat juga bisa diartikan sebagai bilangan gabungan dari bilangan bulat positif, bilangan nol dan bilangan bulat negatif. Semakin ke kanan letak bilangan bulat pada garis bilangan, maka bilangan bulat tersebut akan semakin besar. Begitupun jika semakin ke kiri letak suatu bilangan bulat, maka nilai dari bilangan bulat tersebut akan semakin kecil.

Sifat-sifat operasi hitung bilangan bulat

Penjumlahan:

(+) + (+) = (+)

(+) + (-) = (-)

(-) + (-) = (-)

(-) + (+) = (+)

Pengurangan:

(+) - (+) = (-)

(+) - (-) = (+)

(-) - (-) = (+)

(-) - (+) = (-)

Perkalian:

(+) × (+) = (+)

(+) × (-) = (-)

(-) × (-) = (+)

(-) × (+) =(-)

Pembagian:

(+) ÷ (+) = (+)

(+) ÷ (-) = (-)

(-) ÷ (-) = (+)

(-) ÷ (+) = (-)

Macam-macam bilangan

1. Bilangan bulat

Bilangan ini terdiri dari bilangan negatif (–), bilangan nol (0) dan bilangan positif (+), contohnya

B = {…., –3, –2, –1, 0, 1, 2, 3, …}2. Bilangan cacah

Bilangan ini terdiri dari bilangan bulat nol dan bilangan bulat positif, contohnya

C = {0, 1, 2, 3, 4, 5, 6, ….}

3. Bilangan asli

Bilangan ini terdiri dari bilangan bulat positif, contohnya

A = {1, 2, 3, 4, 5, 6, 7, …}

4. Bilangan ganjil

Bilangan ini tidak habis dibagi 2, contohnya

G = {1, 3, 5, 7, 9, 11, 13, ….}

5. Bilangan genap

Bilangan ini habis dibagi 2 atau merupakan bilangan kelipatan 2, contohnya

G = {2, 4, 6, 8, 10, 12, 14, …}

6. Bilangan prima

Bilangan ini hanya memiliki dua faktor saja yaitu 1 dan bilangan itu sendiri, contohnya

P = {2, 3, 5, 7, 11, 13, 17, ….}

7. Bilangan komposit

Bilangan ini adalah bilangan asli selain bilangan prima yang lebih besar dari satu, contohnya

K = {4, 6, 8, 9, 10, 12, 14, ….}

8. Bilangan rasional

Bilangan ini adalah bilangan yang dapat diubah menjadi bentuk pecahan yaitu [tex] \frac{m}{n} [/tex], dengan m dan n adalah bilangan bulat serta n ≠ 0. Contohnya[tex] \frac{2}{3} \frac{7}{5} \frac{5}{8} \frac{1}{2} [/tex]dan sebagainya

9. Bilangan kuadrat

(Bilangan pangkat dua)

Bilangan ini disebut juga dengan bilangan persegi, diperoleh dengan cara dipangkatkan dua. Contohnya

{1, 4, 9, 16, 25, 36, 49, ….}

10. Bilangan irrasional

Bilangan ini adalah bilangan yang tidak dat diubah menjadi bentuk pecahan [tex] \frac{m}{n} [/tex],contohnya

√2, log 2[tex] \sqrt[3]{5} [/tex], π dan sebagainya
