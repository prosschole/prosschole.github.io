3. apa kang diarani dasanama?

➡️ tembung dasanama yaiku tembung sing sakpirang pirang utawa akeh, nanging duwe ateges padha.

→ tuladhane :

banyu = Toya, Tirta, lirbumi = jagad, Siti, lemahbapak = bapa, Yayah, rama

4. apa kang diarani tembung saroja?

➡️ tembang saroja yaiku tembung loro sing padha utawa meh padha tegese banjur digawe berbarengan.

→ tuladhane :

Abang mbranangayem tentrembabak bundhas

5. apa kang diarani tembung entar?

➡️ tembung entar yaiku tembung sing nduweni ateges kiasan.

→ tuladhane :

Abang kupinge = nesuadus kringet = nyambut gawegedhe endhase = sombong

6. apa kang diarani tembung garba?

➡️ tembung Garba yaiku tembung loro utawa luwih sing dirangkep dadi siji.

→ tuladhane :

Aneng = ana + ingjalwestri = Jalu + EstriNarendra = Nara + Indra-----------------------------------Pembahasan

Nanging, Sanadyan Endah wong liya isih bisa mangerteni apa kang tinulis.
