5. Tentukan luas permukaan prisma dibawah ini !


Diketahui:

a = 30 cm

t = 40 cm

t prisma = 20 cm

Ditanya:

Luas permukaan prisma

Jawab:

Jika alas prisma tersebut segitiga siku-siku, maka kita bisa cari sisi miring dgn pythagoras.

m = √(a² + t²) = √(30² + 40²) = √(900 + 1600) = √2500 = 50 cm

Luas permukaan:

L = (2 x Luas alas) + (Keliling alas x tinggi prisma)

L = (a x t) + ((a + t + m) x t prisma)

  = (30 x 40) + ((30 + 40 + 50) x 20)

  = 1200 + (120 x 20)

  = 1200 + 2400

  = 3600 cm²

L = [(a+b)×t] + [T×(a+b+c+t)]
