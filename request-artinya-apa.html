6. Apa artinya tulisan ini expired token. Please request a new password reset link


Expired token = token telah kadaluarsa.
Please request a new password = silahkan mengajukan sebuah password baru
Reset link = link disetel ulang

Token, mempunyai arti = a voucher that can be exchanged for goods or services (sebuah voucher yang dapat ditukarkan dengan barang atau jasa/pelayanan)

Password, dapat berupa kode rahasia.

Perintah ini biasanya muncul ketika akun yang akan kita buka telah kadaluarsa password/kode rahasia. Kita harus menginput kode rahasia tersebut sebelum batas waktu yang ditentukan. Apabila lebih dari wakyu yang ditentukan, kita harus mengset ulang password.

Detil tambahan:
Pelajaran : bahasa Inggris
Kelas : -
Kategori : instruction
Kata kunci : token, password, reset, translation
