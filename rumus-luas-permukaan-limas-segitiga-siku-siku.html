1. rumus luas permukaan limas segitiga siku siku ​


Rumus luas permukaan Limas

= Luas alas + jumlah luas sisi tegak.

= 1/2 x alas x tinggi alas + jumlah luas sisi tegak.

Penjelasan :

Diketahui:

Ditanya:

Rumus Permukaan Limas segitiga siku-siku

Jawab:

Untuk menyelesaikan soal ini, siswa harus faham bab limas.

Limas adalah bangun ruang yang alasnya berbentuk segi banyak, dan sisi tegaknya berbentuk segitiga.

Banyak sisi pada limas segi-n dapat dirumuskan sebagai n + 1.

Banyak rusuk pada limas segi-n dapat dirumuskan sebagai 2n.

Banyak titik sudut pada limas segi-n dapat dirimuskan sebagai n + 1.

Pada limas, namanya berdasarkan alasnya, jika alasnya segitiga, maka nama limas tersebut adalah limas segitiga, begitu seterusnya.

Rumus Volume Limas

= 1/3 x luas alas x tinggi

Rumus luas permukaan Limas

= Luas alas + jumlah luas sisi tegak

= 1/2 x alas x tinggi alas + jumlah luas sisi tegak.
