27. di koperasi sekolah tanti membeli 1 keripik pisang ,2 keripik singkong dengan total harga 6.750 sedangkan fika membeli 2 kripik pisang dan 3 kripik singkong dengan total harga 11.000. wahyu ingin membeli 4 kripik pisang dan 5 kripik singkong ,berapakah yang harus di bayar wahyu ?​


Penjelasan dengan langkah-langkah:

Misalkan :

x = harga 1 keripik pisang

y = harga 1 keripik singkong

x + 2y = 6750

2x + 3y = 11000

4x + 5y = .....?

Jawab

x + 2y = 6750

x = 6750 - 2y

2x + 3y = 11000

2 (6750 - 2y) + 3y = 11000

13500 - 4y + 3y = 11000

-y = 11000 - 13500

-y = -2500

y = 2500

x = 6750 - 2(2500)

x = 6750 - 5000

x = 1750

4x + 5y = 4(1750) + 5(2500)

4x + 5y = 7000 + 12500

4x + 5y = 19500

Uang yg harus dibayarkan Rp.19500,00.
