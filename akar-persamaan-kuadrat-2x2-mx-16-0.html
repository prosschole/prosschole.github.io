1. Akar-akar persamaan kuadrat 2x2 +mx +16=0 , alfa=2beta , nilai m ?


Akar akar persamaan kuadrat 2x² + mx + 16 = 0 adalah a dan β. Jika a = 2 β dan a, β positif, nilai m yang memenuhi adalah -12.

Pembahasan

Jika α dan β merupakan akar-akar persamaan ax² + bx + c = 0, maka berlaku:

1. α + β = -b/a

2. α x β = c/a

Diketahui :

persamaan 2x² + mx + 16 = 0

a = 2

b = m

c = 16

a = 2β dan a, β positif

Ditanya: nilai m?

Jawab :

Nilai β :

α x β = c/a

α x β = 16/2

2β x β = 8

2β² = 8

β² = 8/2

β² = 4

β = 2

α + β = -b/a

α + β = -m/2

2β + β = -m/2

3β = -m/2

Untuk β = 2, maka nilai m :

3β = -m/2

3(2) = -m/2

6 = -m/2

-m = 6 x 2

-m = 12

m = -12
