1. rumus mencari delta t


Dalam alfabet Yunani, delta merupakan huruf keempat. Delta menyatakan besar perubahan besaran tertentu. Rumus delta adalah nilai besaran akhir dikurangi nilai besaran awal.

Pembahasan

Delta (Δ)

Dalam alfabet Yunani, delta merupakan huruf keempat. Delta menyatakan besar perubahan besaran tertentu. Secara umum kita dapat menggunakan rumus delta sebagai berikut:

Δ = nilai akhir - nilai awal

Contoh :

Rumus perubahan suhu (ΔT)

ΔT = T₂ - T₁

dengan,

T₂ = suhu akhir

T₁ = suhu awal

Rumus perubahan waktu (Δt)

Δt = t₂ - t₁

dengan,

t₂ = waktu akhir

t₁ = waktu awal

No 1. rumusnya [tex] \alpha = \frac{lt - lo}{lo (T2 - T1)} [/tex]

27. Rumus percepatan tu deltav\delta t kah??


Percepatan :
a = (Vt-Vo) / (t2-t1)
   = ΔV/Δt

Keterangan :
a : percepatan
Vt = kecepatan akhir
Vo = kecepatan awal
t  = waktu

Rumus Percepatan = 
a = (v2-v1)/ t

a= percepatan
v = kecepatan
t = waktu

Contoh
Sebuah mobil bergerak dengan kecepatan 60 km/jam. Ketika mendekati lampu merah, direm selama 11 detik sehingga kecepatannya turun menjadi 5 km/jam. Berapakah percepatannya?

Jawab = 
a = (5-60)/ 11
a = -55 / 11
a = -5 m/s²

Jadi percepatannya adalah 5 m/s² diperlambat

29. cara mencari nilai delta t


Mencari Δt memakai rumus biasa yaitu

Q = M × C × Δt

Contoh soal =

Q = 31.500

M = 5

C = 4.200

Δt = ?


Jawabnya =

Q = M×C×Δt

31.500 = 5×4.200xΔt

31.500 = 21.000Δt

21.000Δt = 31.500

Δt = 31.500 : 21.000

Δt = 1,5


Akan tetapi Δt tidak selalu desimal kok :)
