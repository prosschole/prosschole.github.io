24. Sebutkan cara mengubah urutan nomor halaman di excel


Jawaban:

How to Provide Page Numbers in Microsoft Excel

As with other documents, documents created using Microsoft Excel should also be given a page number so that no errors occur in the order in which they are read. Numbering in Microsoft Excel is relatively more complicated than numbering in Microsoft Word but that does not mean that in Microsoft Excel numbering cannot be done. To create page numbers in Microsoft Excel you can use the following method.

Click the Page Layout ribbon menu, then click the page setup button, or

Press the keyboard keys ALT + P, S, P

Figure: How to display the Page Setup dialog in Microsoft Excel

Then in the Page Setup dialog select the Header / Footer tab

Decide where to place the page number whether in the header (the top of the document), or in the Footer (the bottom of the document). For example, select and click on the footer then click the Custom Footer button .

Next in the Footer dialog , select where to place the page number, whether on the left ( left section ), in the middle ( Center Section ), or on the right ( Right Section ). Suppose we want to put it in the middle of the page, then click an empty area in the Center Section .

Then click the Insert Page Number button.

In the center section area there will be a text & [Page] . (you can also type it directly)

Click the OK button to close the Footer dialog

Click the OK button again to close the page setup dialog .

The page number will now be used when printing a document.

besides giving the number, you can also add the excel page number directly to the footer / excel header .

30. jelaskan cara drag dan drop untuk memindahkan data pada ms excel​


1. Pilih Kolom atau Baris yang ingin Anda pindahkan

2. Gerakkan penunjuk mouse Anda ke tepi biru seleksi hingga berubah dari tanda silang ke panah atau tangan penunjuk biasa. Catatan: Pointer mouse harus berada di salah satu sel yang disorot dan tidak pada judul baris atau kolom.

3. Tahan tombol shift dan klik dan seret kolom / baris ke posisi baru. Bilah "I" samar yang membentang sepanjang baris atau kolom, bersama dengan kotak yang menunjukkan di mana baris atau kolom baru muncul.

4. Lepaskan tombol mouse dan kemudian tinggalkan SHIFTtombol, dan baris atau kolom Anda dipindahkan.
