18. Apabila 3 liter BBM cukup untuk menempuh jarak 60km. Maka banyak BBM yang dibutuhkan untuk menempuh jarak 90km adalah?


✏STUDI : MATEMATIKA
✏KELAS : VII
✏MATERI : PERBANDINGAN
↘↘↘PEMBAHASAN ↙↙↙

3 liter = 60km
x liter = 90km

Perbandingan senilai karena makin jauh jarak makin banyak liter bensin dibutuhkan

3/x = 60/90
3(90) = 60x
270 = 60x
4,5 liter = x

smoga membantuPenyelesaian
__________

Diketahui.
3 Liter cukup untuk menempuh jarak 60 km, artinya 60 ÷ 3 = 20 liter/km

Ditanya.
Banyak BBM untuk menempuh jarak 90 km ?

Jawab >>>
90 ÷ 20 = 4,5 liter

Jadi, banyak BBM yang dibutuhkan untuk menempuh jarak 90 km adalah 4,5 liter
