Jawaban:

1. Start pendek ( Bunch Start )

  Start pendek diawali dengan berdiri tegak dengan kedua kaki rapat. Kemudian, kedua tangan berada di samping badan. Adapun cara melakukannya adalah sebagai berikut:

a. Ambil ancang-ancang dengan posisi kaki kiri (kaki yang terkuat) diletakkan di depan dan ditekuk.

b. Posisikan lutut kaki kanan diletakkan di sebelah kaki kiri dan beri jarak sekitar satu kepal.

c. Posisikan kedua tangan diletakkan di belakang garis start dengan ibu jari terpisah dan empat jari lainnya merapat atau seperti membentuk huruf V.

d. Luruskan lengan kedua tangan dan fokuskan pandangan ke depan.

e. Angkat panggul hingga posisinya lebih tinggi dari pundak.

f. Lalu, tolak balok sekuat mungkin dengan kaki belakang. Mulailah berlari saat tanda sudah dimulai sudah dibunyikan.

2. Start menengah ( Medium Start )

    Seperti halnya start pendek, gerakan awal start menengah juga sama, yakni berdiri tegak lurus, kedua tangan berada di samping, dan fokuskan pandangan ke depan. Untuk cara melakukannya adalah sebagai berikut:

a. Ambil ancang-ancang posisi kaki kiri (kaki yang terkuat) diletakkan di depan dan ditekuk.

b. Posisikan lutut kaki kanan di sebelah kanan tumit kaki kiri dan beri jarak sekitar satu kepal lebih.

c. Kemudian, letakkan kedua tangan di belakang garis start dengan ibu jari terpisah dan empat jari lainnya merapat atau membentuk huruf V.

d. Luruskan lengan kedua tangan dan fokuskan pandangan ke depan.

e. Angkat panggul hingga posisinya lebih tinggi dari pundak.

f. Lalu tolak balok sekuat mungkin dengan kaki belakang. Mulailah berlari saat tanda sudah dimulai sudah dibunyikan.

3. Start panjang ( Long Start )

   Start panjang atau long start diawali dengan berdiri tegak, kedua tangan berada di samping badan, dan kaki dirapatkan. Cara melakukan start panjang adalah sebagai berikut:

a. Pertama, ambilah ancang-ancang dengan posisi kaki kiri (kaki yang terkuat) diletakkan di depan lutut kaki kanan dan ditekuk.

b. Terus posisikan lutut kaki kanan pada bagian belakang kaki kiri dan beri jarak sekitar satu kepal lebih.

c.  Kemudian, letakkan kedua tangan di belakang garis start dengan ibu jari terpisah dan empat jari lainnya merapat.

d. Luruskan lengan kedua tangan dan fokuskan pandangan ke depan.

e. Angkat panggul hingga posisinya lebih tinggi dari pundak.

f. Saat akan mulai berlari tolak balok sekuat mungkin dengan kaki belakang.

Start pendek (the short start / bunch start)

1) Sikap permulaan

Sikap permulaan start pendek yaitu : berdiri tegak kedua kaki rapat, lengan lurus di samping badan, dan pandangan lurus ke depan.

2) Cara melakukan

Cara melakukan start pendek yaitu :

a) Langkahkan kaki kiri ke depan dengan jari – jari kaki lurus ke depan.

b) Letakkan ujung jari kaki belakang sejajar dengan tumit kaki depan.

c) Jarak kedua kaki kira – kira satu kepalan tangan.

d) Letakkan kedua lengan lurus dengan bahu, jari – jari tangan rapat, ibu jari membuka membentuk huruf V tebalik di belakang garis start.

28. Start yang di gunakan untuk lari jarak pendek adalah start


Start yang digunakan untuk lari jarak pendek adalah start jongkok.

Pembahasan

Lari jarak pendek adalah salah satu olahraga lari yang termasuk dalam cabang atletik. Lari jarak pendek terdiri dari beberap nomor lari yakni 100 m, 200 m, dan 400 m. Dalam lari jarak pendek, terdapat beberapa teknik dasar yang harus dikuasai oleh setiap pemain. Teknik dasar dalam lari jarak pendek adalah.

1. Teknik start.

Teknik start pada lari jarak pendek yang digunakan adalah start jongkok. Start jongkok terdiri dari tiga macam yang dapat digunakan oleh para pemain, yakni start pendek, start menengah, dan start panjang.

2. Teknik berlari.

Teknik berlari pada lari jarak pendek dilakukan dengan posisi badan sedikit condong ke depan dan menggunakan langkah kaki yang lebar. Posisi kedua tangan mengayun dengan relaks dan seirama dengan langkah kaki.

3. Teknik memasuki garis finish.

Dalam lari jarak pendek, terdapat dua teknik yang dapat digunakan pada lari jarak pendek untuk memasuki garis finish. Teknik tersebut adalah membusungkan dada ke depan dan menjatuhkan salah satu bahu ke depan.

30. Jelaskan bagaimana awalan start pada lari jarak pendek ?


Jawaban:

1. Teknik Start

Dalam lari hal yang paling utama adalah awalan atau kita biasa menyebutnya dengan start. Seorang ahli yang bernama Purnomo mengatakan pentingnya awalan atau start untuk memaksimalkan lari cepat. Sebenarnya dalam lari pada umumnya terdapat tiga start yakni, start berdiri, start melayang, dan start jongkok.

Namun start yang digunakan untuk lari jarak pendek adalah start jongkok atau biasa disebut crouching start. Start jongkok sendiri mempunyai tiga teknik yakni,

a. Bunch Start (Start Pendek)

Posisi pelari mulanya berdiri tegak, lalu mulai bersiap dengan perlahan menurunkan diri. Taruh posisi kaki kiri di depan dan letakkan lutut kaki sebelah kanan di sisi kaki kiri dengan jarak kurang lebih 5cm atau sekepalan tangan. Posisikan kedua tangan di belakang garis awalan (start) dengan jari-jari yang rapat namun jempol terpisah.

b. Medium Start (Start Menengah)

Posisi awalan sama dengan start pendek. Namun yang berbeda hanya lutut kaki kanan berada di sisi tumit sebelah kiri. Jaraknya sama kurang lebih 5cm atau sekepalan tangan. Bentuk dan posisi tangan juga sama dengan start pendek.

c. Long Start (Start Panjang)

Start panjang ini memiliki posisi yang sama persis dengan start pendek. Dimana kaki kiri diletakkan di depan menjadi tumpuan dan lutut kaki kanan berada di sebelah tumit kaki kiri dengan jarak sekepalan tangan. Posisi tangan atlet diletakkan sama persis seperti start pendek dan menengah.

Sebenarnya kaki manapun yang menjadi tumpuan itu tidak masalah. Tumpuan bisa berupa kaki kanan atau kiri tergantung kaki mana yang nyaman untuk pelari digunakan sebagai tumpuan dan tolakan. Dalam start lari jarak pendek terdapat tiga aba-aba yakni,

Bersedia

Posisi atlet saat itu adalah bersiap dalam posisi start dengan pandangan mata pelari yang menatap kebawah.

Siap

Atlet mulai mengangkat tubuhnya dimulai dari lutut yang diposisikan 90°. Lalu bahu diturunkan sehingga pinggang posisinya lebih tinggi . Majukan bahu sedikit ke arah depan sehingga tubuh condong ke depan.

Yak

Selain menggunakan yak, banyak juga pertandingan yang menggantinya dengan suara pistol agar lebih jelas. Saat aba-aba ini pelari sudah memulai tolakan dan mulai berlari.
