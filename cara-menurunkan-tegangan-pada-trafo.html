12. Sebuah trafo memiliki efisiensi 75%. Trafo tersebut digunakan untuk menurunkan tegangan menjadi 40 volt. Jika arus input dan output yang mengalir pada trafo berturut-turut 1 A dan 3 A, tegangan masukan pada trafo tersebut sebsar .. V.


Tegangan masukan pada trafo tersebut sebesar .. 160 V.

Pembahasan:

Tegangan listrik dirubah (dinaikkan atau diturunkan) dengan menggunakan Transformator atau trafo.

Jika efisiensi sempurna, semua daya lilitan primer akan dilimpahkan ke lilitan sekunder. Namun jika efisiensi tidak sempurna, terjadi kehilangan daya, dan hanya daya sebesar efisiensi trafo yang berpindah dari tegangan primer ke tegangan sekunder.

Rumus efisiensi trafo adalah:

η = [(Vs x Is) / (Vp x Ip)] x 100%

dimana:

η = efisiensi trafo

Vp = tegangan primer dalam satuan Volt (masuk)

Vs = tegangan sekunder dalam satuan Volt (keluar)    

Ip = arus primer dalam satuan Ampere

Is = arus sekunder dalam satuan Ampere

Dalam soal diketahui:

Eefisiensi trafo (η) = 75%,

Tegangan sekunder (Vs) = 40 volt,  

Arus primer (Ip) = 1 A.

Arus sekunder (Is) = 3 A.  

Ditanua: Vp

Maka:

η = [(Vs x Is) / (Vp x Ip)] x 100%

75% = [(40 x 3) / (Vp x 1)] x 100%

75% = [120/Vp] x 100%

0,75 Vp = 120

Vp = 120 / 0,75

      = 160 Volt

22. sebuah trafo memiliki efisien 80% trafo tersebut digunakan untuk menurunkan tegangan menjadi 40 volt jika arus input dan output yang mengalir pada trafo berturut-turut 1 A dan 3A hitung nilai tegangan masukan pada trafo.​


Jawaban:

Untuk menghitung tegangan masukan pada trafo, kita dapat menggunakan rumus efisiensi trafo yaitu:

Efisiensi = (tegangan output x arus output) / (tegangan input x arus input)

Diketahui:

efisiensi = 0.8

tegangan output = 40 volt

arus output = 3 A

Maka:

tegangan input x arus input = (tegangan output x arus output) / efisiensi

tegangan input = (tegangan output x arus output) / (efisiensi x arus input)

tegangan input = (40 x 3) / (0.8 x 1)

tegangan input = 120 volt

Jadi tegangan masukan pada trafo adalah 120 volt.

Penjelasan:

Efisiensi trafo adalah rasio antara daya yang dihasilkan oleh trafo (daya output) dengan daya yang digunakan oleh trafo (daya input). Dalam hal ini, efisiensi trafo diketahui sebesar 80%, yang berarti bahwa 20% dari daya input akan hilang dalam bentuk panas.

Diketahui bahwa tegangan output trafo adalah 40 volt dan arus output adalah 3 ampere. Dengan menggunakan rumus efisiensi trafo, yaitu (tegangan output x arus output) / (tegangan input x arus input), kita dapat menghitung tegangan input trafo.

Jadi, tegangan input trafo dapat dihitung dengan mengalikan tegangan output dengan arus output dan membagi hasilnya dengan efisiensi trafo dikalikan arus input.

tegangan input = (tegangan output x arus output) / (efisiensi x arus input)

tegangan input = (40 x 3) / (0.8 x 1)

tegangan input = 120 volt

Jadi, tegangan masukan pada trafo adalah 120 volt.

Efisiensi trafo yang rendah dapat menyebabkan hilangnya daya yang cukup besar dalam bentuk panas. Oleh karena itu, desain trafo harus dibuat sedemikian rupa untuk mencapai efisiensi yang tinggi dan mengurangi hilangnya daya dalam bentuk panas.

30. berikut pernyataan tentang trafo 1. trafo menggunakan tegangan dc 2. trafo dapat menaikkan tegangan dc 3. trafo dapat menurunkan ac 4. trafo ideal daya primernya sama dengan sekunder yang benar adalah


Pernyataan yang benar tentang trafo adalah:

trafo dapat menurunkan AC trafo ideal daya primernya sama dengan sekunderPembahasan

Transformator (atau trafo) adalah alat yang digunakan untuk menaikkan atau menurunkan tegangan dari arus bolak-balik (AC). Transformator merupakan salah satu penerapan dari sifat induksi elektromagnetik.

Tipe trafo ada dua, yakni:

A. Trafo penaik tegangan (step up)

Jumlah lilitan sekunder lebih banyak dari jumlah lilitan primer (Ns > Np)NIlai tegangan sekunder lebih besar dari nilai tegangan primer (Vs > Vp)Kuat arus primer lebih besar dari kuat arus sekunder (Ip > Is)

B. Trafo penurun tegangan (step down)

Jumlah lilitan primer lebih banyak dari jumlah lilitan sekunder (Np > Ns)Nilai tegangan primer lebih besar dari nilai tegangan sekunder (Vp > Vs)Kuat arus sekunder lebih besar dari kuat arus primer (Is > Ip)

Hubungan antara jumlah lilitan kumparan, tegangan, dan kuat arus adalah sebagai berikut:

[tex]\boxed{~\frac{N_s}{N_p} = \frac{V_s}{V_p} = \frac{I_p}{I_s}~}[/tex]

Dengan demikian hanya jumlah lilitan kumparan yang berbanding lurus dengan tegangan.

Sebuah trafo ideal memiliki efisiensi sama dengan 1 atau 100%, yang artinya daya primer sama dengan daya sekunder. Namun pada kenyataannya hal tersebut tak mungkin tercapai, disebabkan oleh panas yang dilepas (Eddy current) sehingga daya sekunder akan selalu lebih kecil dari daya primer.

Efisiensi trafo non ideal adalah [tex]\boxed{~\eta = \frac{P_{out}}{P_{in}} \times 100 \%~}[/tex]

Pout = daya sekunderPin = daya primer

Ingat, P = V x I.

Contoh trafo step up dapat ditemui pada stasiun atau sebuah gardu listrik PLN yang akan mendistribusikan listrik ke pemukiman.

Contoh trafo step down adalah charger piranti (gadget) elektronik dan adaptor.
