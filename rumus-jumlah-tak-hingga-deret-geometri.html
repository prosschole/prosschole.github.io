29. diketahui jumlah deret geometri 3 + 9 + .... + 3n = 3.279 tentukan banyaknya suku pada deret geometri itu dan rumus jumlah n suku pertamanya


Diketahui :
Deret geometri
 a = U₁ = 3
U₂ = 9
Sn = 3.279

Ditanya :
n = ???

Dijawab :
⇔ r = U₂/U₁₁
       = 9/3
       = 3

⇔               Sn = (a*(r^n - 1))/(r - 1)
⇔          3.279 = (3*(3^n - 1))/(3 - 1)
⇔          3.279 = (3*(3^n - 1))/(2)
⇔       3.279*2 = (3(3^n - 1))/2)*2
⇔           6.558 = 3*(3^n - 1)
⇔    3*(3^n - 1) = 6.558 => dibagi 3
⇔ 3*(3^n - 1)/3 = 6.558/3
⇔          3^n - 1 = 2.186
⇔              3^n  = 2.186+1
⇔               3^n = 2.186
⇔               3^n = 3^7
⇔                   n = 7

Jadi n nya adalah sebesar 7
