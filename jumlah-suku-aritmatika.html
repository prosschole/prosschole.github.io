2. Sebutkan rumus banyak suku deret aritmatika dan jumlah suku deret aritmatika?


Rumus aritmetika

Un = a + ( n - 1)b

Sn = n/2( a + Un)

Contoh :

suatu barisan aritmetika 2 4 6 8 ..... Tentukan suku ke-12 dan jumlah suku ke-12.

diketahui :

a = 12

b = 2

ditanyakan : U12 dan jumlah suku ke-12

Jawaban :

U12 = 2 + ( 12-1 ) 2

U12 = 2 + (11).2

U12 = 2 + 22

U12 = 24

S12 = 12/2 ( 2 + 24)

S12 = 6. (26)

S12 = 156

23. Jumlah suku ke-3 dan ke-5 dari suatu deret aritmatika adalah –20 sedangkan suku ke-8 adalah 14. jika suku ke-n adalah 38, jumlah deret aritmatika tersebut adalah….?

Penjelasan dengan langkah-langkah:

﻿U3 + U5 = -20

﻿U8 = 14

﻿Un = 38

﻿Sn = ...?

﻿Un = a + (n - 1)b

﻿Sn = (n/2) (2a + (n - 1)b)

﻿U3 = a + (3 - 1)b = a + 2b

﻿U5 = a + (5 - 1)b = a + 4b

﻿U8 = a + (8 - 1)b = a + 7b

﻿a + 2b + a + 4b = -20

﻿a + 3b = -10

﻿a + 3b = -10

﻿a + 7b = 14 _

﻿-4b = -24

﻿b = -24/(-4) = 6

﻿a = -10 - 3b

﻿a = -10 - 3(6)

﻿a = -10 - 18

﻿a = -28

﻿Un = a + (n - 1)b

﻿38 = -28 + (n - 1)6

﻿38 + 28 = 6n - 6
