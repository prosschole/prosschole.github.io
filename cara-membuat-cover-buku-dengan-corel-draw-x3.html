6. bagaimana langkah langkah membuka program Corel draw X3 dari meu start​


Membuka program CoreldDRAW X3 pada komputer cukup mudah, berikut langkah-langkahnya:

Klik icon start yang terdapat pada bagian kiri bawah deskop. Kemudian akan muncul daftar program pada komputer.Carilah icon CoreldDRAW X3, kemudian klik 2 kali pada icon tersebut.Setelah itu, akan muncul logo CoreldDRAW X3 berbentuk kotak, tunggulah beberapa saat hingga program benar-benar terbuka.Saat program sudat terbuka, akan muncul kotak dialog "welcome to CorelDRAW X3. Untuk membuka lembar kerja baru, klik "new".Lembar kerja CorelDRAW siap untuk digunakan.

Pembahasan

CorelDraw X3 merupakan salah satu versi CorelDRAW. Program tersebut bergerak dalam editor grafik vektor. Perusahaan yang mengembangkan program ini bernama Corel. Perusahaan tersebut merupakan perusahaan perangkat lunak yang terletak di Ottawa, Kanada. Ada berbagai macam versi program Corel, versi terbarunya ialah CorelDRAW X8 yang dirilis pada tanggal 15 Maret 2016.

Program ini bisa kita akses melalui komputer. Langkah-langkah membukanya cukup mudah seperti yang sudah dijelaskan di atas. Program CorelDRAW X3 merupakan program untuk membuat gambar berbasis vektor, contohnya ilustrasi, kartun, logo dan lain sebagainya. Gambar yang dihasilkan program ini sangat baik, karenanya banyak orang yang menggunakan program ini.

￼

20. CARA MEMBUAT GAMBAR 3D DENGAN COREL DRAW


Jawaban:

Kita dapat memberikan efek 3 dimensi pada gambar dengan cara : Select gambar -> Effects -> 3D Effect -> pilih efek 3D yang ingin digunakan. Catatan : pada versi coreldraw tertentu pilihan 3D effect berada dalam menu bar Bitmaps.

Penjelasan:

semoga membantu yaayaya,,

Jawaban:

Efek 3D pada gambar akan memberikan kesan desain menjadi lebih hidup. Oleh sebab itu, pemberian efek 3 dimensi ini biasa digunakan para desainer ketika proses finishing desainnya.

Banyak cara yang dapat digunakan dalam memberikan efek 3D. Pada toolbox CorelDraw sudah disediakan berbagai pilihan, seperti :

Drop Shadow

Contour

Blend

Distort

Envelope

Extrude

Akan tetapi, tools diatas kita akan memerlukan lagi beberapa langkah untuk menyesuaikan. Lalu bagaimana dengan memberikan efek 3 dimensi pada gambar (Bitmap) dengan mudah?

Langsung saja kita akan mencoba memberikan efek 3 dimensi kedalam gambar dengan menggunakan opsi bitmaps pada menu bar.

Memberikan Efek 3D pada Gambar dengan Menu Bar

Dalam CorelDraw terdapat menu bar yang berisi banyak opsi, mulai dari opsi File, Edit, viev, Layout, hingga opsi Help. Sedangkan untuk memberikan efek 3 dimensi pada gambar (bitmap) kita dapat menggunakan opsi Effects pada menu bar. Pada beberapa versi coreldraw pilihan pemberian efek 3 dimensi dapat menggunakan opsi Bitmaps.

Kita dapat memberikan efek 3 dimensi pada gambar dengan cara : Select gambar -> Effects -> 3D Effect -> pilih efek 3D yang ingin digunakan.

22. Bagaimana cara membuat garis lengkung di corel draw ?


Yang pertama kita buat garis dahulu dengan menggunakan BEZIER.

Cara-nya cukup mudah tinggal klik Ikon BEZIER lalu klik pada objek kerja lalu tekan CTRL untuk membuat gari lurus lalu klik yang ke dua kali.

Setelah sudah membuat garis tinggal kita merubah-nya menjadi melengkung.

Cara-nya Klik di tengah-tengah garis dengan Tools SHAPE lalu pilih TO CURVE.

Jika sudah tinggal kita tarik pada bagian tengah hingga membentuk lengkungan
