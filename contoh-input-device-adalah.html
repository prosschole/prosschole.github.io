Jawaban:

1. Keyboard

Keyboard adalah perangkat yang digunakan untuk memasukkan data berupa huruf, angka, dan simbol serta melakukan perintah-perintah pada komputer. Tanpa adanya perangkat ini, pengguna tidak akan bisa mengetikkan data yang mereka inginkan, sehingga keberadaannya sangatlah penting.

2. Mouse

Mouse adalah perangkat keras yang berukuran kecil dan mudah digenggam yang umumnya memiliki dua tombol dan sebuah bola atau laser di bawahnya untuk menggerakkan kursor (pointer). Fungsi mouse adalah untuk menggerakkan kursor, melakukan scroll (pada mouse jenis tertentu), di mana keduanya akan membuat komputer bisa menjalankan perintah yang diinginkan pengguna.

3. Joystick

Joystick merupakan perangkat keras berbentuk seperti tuas gigi mobil atau seperti stick konsol yang memiliki banyak tombol di dalamnya. Perangkat ini digunakan untuk bermain game pada komputer.

4. Touchpad

Touchpad merupakan sebuah alat penunjuk yang menampilkan sensor pada permukaan khusus yang dapat menerjemahkan gerakan dan jari-jari pengguna ke posisi yang diinginkan pada layar monitor. Perangkat ini ada pada laptop dan notebook yang memiliki cara kerja seperti mouse.

5. CD (Compact Disc)

CD atau Compact Disc adalah perangkat yang berfungsi sebagai media penyimpanan dan pembacaan data menggunakan sistem optik yang terbuat dari bahan plastik dan berbentuk lingkaran pipih dengan lubang kecil pada bagian tengah.


5. jelaskan tentang Input device,Process device, Out put device beri masing² contoh beserta fungsinya​


Input device mengirimkan informasi ke sistem komputer untuk diproses, dan Output device mereproduksi atau menampilkan hasil pemrosesan tersebut. Input device hanya mengizinkan input data ke komputer dan Output device hanya menerima output data dari perangkat lain. Process device adalah semua perangkat di dalam komputer yang menafsirkan dan memanipulasi data yang masuk selama tahap ini.

Contoh Input device yaitu: Keyboard, mouse, microfon, dan webcam.

Contoh Output device yaitu: Monitor, proyektor, dan speaker.

Contoh Process device yaitu: CPU, memory and motherboard.

Pembahasan

Sebagian besar perangkat hanya merupakan Input device atau Output device, karena masing-masing peragkat hanya dapat menerima input data dari pengguna atau data output yang dihasilkan oleh komputer. Namun, beberapa perangkat dapat menerima input dan menampilkan output, dan disebut sebagai perangkat I/O (perangkat input/output).

Contohnya keyboard mengirimkan sinyal listrik, yang diterima sebagai input. Sinyal tersebut kemudian ditafsirkan oleh komputer dan ditampilkan atau dikeluarkan pada monitor sebagai teks atau gambar. Di bagian bawah gambar, komputer mengirim, atau mengeluarkan data ke printer. Kemudian, data tersebut dicetak ke selembar kertas, yang juga dianggap sebagai output.

Process device merupakan perangkat apa pun di dalam komputer yang menafsirkan dan memanipulasi data yang masuk selama tahap ini.

Process device adalah komponen yang bertanggung jawab untuk pemrosesan informasi di dalam sistem komputer. Ini termasuk perangkat seperti CPU, memori dan motherboard. Perangkat penyimpanan adalah komponen yang memungkinkan data disimpan dalam sistem komputer.

8. Jelaskan Perbedaan Input Device dan Output Device beserta contohnya !


Perbedaan INPUT DEVICE dan OUTPUT DEVICE

1.  Menurut Fungsi

Input device, sesuai namanya, fungsinya adalah memberikan masukan kepada komputer untuk dilakukan processingOutput device, fungsinya adalah untuk menghasilkan keluaran yang sudah diproses komputer bagi penggunanya

2.  Menurut urutan pengerjaan

Input dilakukan pertama kali lalu terjadi proses dan dikeluarkan oleh output device. Sehingga dapat disimpulkan output device baru mengeluarkan setelah adanya input dan proses.

3. Menurut Pemroses

Input device diproses oleh penggunanya sedangkan output device sebelumnya diproses oleh komputer

4. Contoh

Input Device : Microphone ( berupa input suara ), Scanner( berupa input dokumen), Keyboard ( berupa input karakter ), Mouse ( berupa input gerakan dan tanda ), dan lain - lainOutput Device : Monitor ( berupa output tampilan ), Speaker ( berupa output suara ), Printer ( berupa output kertas yang sudah diketik ),  dan lain - lain

Pembahasan

Input adalah proses memasukkan informasi atau data ke dalam komputer oleh input device yang kemudian akan masuk ke tahap processing yang dilakukan oleh processing device. Kemudian proses menghasilkan hasil kepada pengguna disebut dengan output yang ditunjang dengan output device.

Ketiga tahap ini saling berhubungan dan tidak dapat terpisah termasuk dalam implementasinya diperlukan input, outpur dan processing device
