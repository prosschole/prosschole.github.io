2. Dika berada di titik nol Dika mundur 8 satuan kemudian Dika maju sebanyak 7 satuan posisi Dika sekarang berada pada bilangan​


Dika berada di titik nol Dika mundur 8 satuan kemudian Dika maju sebanyak 7 satuan posisi Dika sekarang berada pada bilangan​ -1.

Pembahasan

Bilangan bulat ada 2, bilangan bulat positif dan bilangan bulat negatif.

Antara bilangan bulat positif dan bilangan bulat negatif dipisahkan oleh bilangan netral yaitu 0.

Bilangan disebelah kanan 0 atau disebelah atas 0 adalah bilangan bulat positif, dan bilangan disebelah kiri 0 atau disebelah bawah 0 adalah bilangan bulat negatif.

Lawan dari bilangan Negatif adalah bilangan Positif.

Kegiatan yang termasuk bilangan positif jika ditulis dengan lambang bilangan yaitu, Untung, Maju, Kekanan, Keatas, Diatas, Naik.

Kegiatan yang termasuk bilangan negatif jika ditulis dengan lambang bilangan yaitu, Rugi, Mundur, Kekiri, Kebawah, Dibawah, Turun.

Penyelesaian soal

Dika berada di titik nol = 0

Dika mundur 8 satuan = -8

Dika maju sebanyak 7 satuan = 7

Posisi Dika sekarang = 0 - 8 + 7 = -1.
