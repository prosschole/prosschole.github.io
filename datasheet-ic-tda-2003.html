7. Jika sistem tali setimbang, tentukan Tac Tab Tda Tbe Tbc


Jawaban:

Tda = Tbe = 25/3 N

Tac = Tbc = 25/4 N

Tab = 35/12 N

Penjelasan:

∑Fy = 0

10 = Tda sin α + Tbe sin α

simetris ( Tda = Tbe )

10 = 2 Tda sin α

Tda = Tbe = 5 / sin α = 5 (5/3) = 25/3 N

∑Fy = 0

Tda sin α = Tac sin β

Tac = Tda sin α / sin β = 25/3 (3/5) (5/4) = 25/4  N

simetris ( Tac = Tbc = 25/4 N )

∑Fx = 0

Tab = -Tac cos β + Tda cos α = -25/4 (3/5) + 25/3 (4/5) = 35/12 N

24. Apa yang kamu ketahui tentang seven segment? (Serta jelaskan dan gambarkan datasheet Seven segment jika kamu tahu)


Jawaban:

Seven Segment Display (7 Segment Display) dalam bahasa Indonesia disebut dengan Layar Tujuh Segmen adalah komponen Elektronika yang dapat menampilkan angka desimal melalui kombinasi-kombinasi segmennya. Seven Segment Display pada umumnya dipakai pada Jam Digital, Kalkulator, Penghitung atau Counter Digital, Multimeter Digital dan juga Panel Display Digital seperti pada Microwave Oven ataupun Pengatur Suhu Digital . Seven Segment Display pertama diperkenalkan dan dipatenkan pada tahun 1908 oleh Frank. W. Wood dan mulai dikenal luas pada tahun 1970-an setelah aplikasinya pada LED (Light Emitting Diode).

Multiplexer adalah rangkaian logika yang menerima beberapa input datadigitaldan menyeleksi salah satu dari input tersebut pada saat tertentu,untuk dikeluarkan pada sisi output. Seleksi data-data input dilakukan oleh selectorline, yang juga merupakan inputdari multiplexer tersebut. IC 74LS151 merupakan data selector/multiplekser yang memilih satu dari 8saluran masukan dengan menggunakan 3 sinyal kontrol. Komponen ini mempunyai 8masukan, yaitu I0 - I7 dan 2 luaran yaitu Z dan Z . Pemilihan data dilakukan oleh kaki A, B dan C. Untuk mengaktifkan fungsinya sebagai multiplekser, komponen ini jugamempunyai 1 masukan enable, yaitu E (aktifLOW). Sedangkan kaki 8 dihubungkanke GND dan kaki 16 dihubungkan ke +5V.IC 74138 adalah sebuah aplikasi demultiplexer. Demultiplexer adalah perangkat elektronik yang berfungsi untuk memilih salah satu data dari banyak data menggunakan suatu data input. Demultiplexer sering disebut sebagai perangkat dengan sedikit input dan banyak output ic ini cocok untuk pengguna mikrokontroler yang membutuhkan output. Demultiplexer 74LS138 berfungsi untuk memilih salah satu dari 8 jalur dengan memberikan data BCD 3 bit pada jalur masukan A0 – A2. Demultiplexer 74LS138 memiliki 8 jalur keluaran Q0 – Q7, 3 jalur masukan A0 – A2 dan 3 jalur kontrol expansi E1 – E3.Multiplexer adalah sebuah rangkaian logika yang terima beberapa masukan data secara otomatis dan pilih salah satu dari masukan tersebut pada waktu tertentu, untuk dikeluarkan pada keluaran. Digunakan untuk transmisi jaringan jarak jauh baik yangmenggunakan kabel maupun yang menggunakan media udara seperti wireless atauradio.Sebagai contoh satu helai optic Surabaya - Jakarta bisa dipakai untuk menyalurkanribuan percakapan pada telepon.Demultiplexer adalah sebuah rangkain logika yang menerima satu input datadan mendistribusikan input tersebut ke beberapa output yang tersedia, dan jugamerupakan kebalikkan dari multiplexer. seleksi data - data input dilakukan olehselektor line, yang juga merupakan input dari demultiplexer tersebut. Biasanya digunakan pada televisi karena cara televisiadalah menerima sinyal data yang kemudian akan dipisahkan berdasarkan chanel yangada cara ini disebut dengan teknik demultiplexing atau lebih jelasnya adalah prosese pnerimaan data dan kemudian akan dipisahkan sesuai dengan channel yang ada.Data routing atau perjalanan data, Multiplexer seringnya dapat memilih sebuah alur data dari satu asal sebuah data diantara beberapa asal ke satu tujuan data yang ada, pengurutan dari sebuah operasi, Konversi bilangan pada rangkaian dari rangkaian paralel ke rangkaian seri, Menghasilkan sebuah bentuk gelombang yang ada, Menghasilkan sebuah fungsi dari logika tersebut.Perangkat pengambil input tunggal, pemilih sinyal dari output mux yang kompatibel.

26. jelaskan update data dengan menggunakan datasheet dan query


Update data melalui query

a. Membuat query


    Sebelum memulai kegiatn ini,sebaiknya kamu tahu terlebih dahulu pengertian dari query.query bisa    diartikan sebagai pernyataan atu permintaan untuk memproses tabel yang ada.untuk mengupdate data melalui query,kamu harus memiliki query terlebih dahulu.namun jika kamu belum memiliki query,maka kamu harus membuatnya terlebih dahulu.terdapat dua cara membuat query

yaitu:   1.      Create query in design view

           2.      Create query by using wizard


Kita gunakan metode create query by using wizard.berikut adalah langkah-langkahnya. Pada jendela objects pilih queries,kemudian klik ganda pada create query by using wizard. Pilih nama pada tabel daftar.jika ingin ditampilkan seluruh field dalam query yang kamu buat,klik tombol >> namun jika hanya ingin dipilih beberapa field saja,klik tombol  > sekarang pilih seluruh field dan klik next selanjutnya,dalam pilihan yang diberikan,pilihlah detail dan klik next. Selanjtnya,kamu akan diberi pilihan untuk memberi nama query yang kamu buat.secara otomatis,ms access memberi nama sesuai nama tabel yang dirujuk,yaitu daftar query kemudian klik finish.


Kita dapat memilih untuk menampilkan field tertentu setelah query dibuat.bagaimana caranya?lakukanlah langkah-langkah berikut :

=> simpan hasil query yang kamu buat dan tutup query

=> klik kanan daftar query dan pilih design view

=> selanjutnya kamu akan mendapatkan jendela design view

=> jika kamu hanya ingin menampilkan field nama siswa,prog.pilihan,math,bhs inggris,maka berikan tanda   

     checklist pada bagian show hanya pada field ini saja

=> simpan dengan nama daftar query baru.selanjutnya,buka daftar query baru.
