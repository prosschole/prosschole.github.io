2. rumus luas permukaan prisma segiempat beserta cara menghitungnya


Jawaban:

Secara umum luas permukaan prisma segitiga dapat dicari dengan menggunakan sebuah rumus:

Luas = (2 x luas alas) + (luas seluruh bidang tegak)

atau Apabila sisi alasnya sama panjang, yaitu:

Luas = (2 x luas alas) + (3 x luas salah satu bidang tegak)

Dapat juga rumusnya :

Luas = (2 x luas alas) + (keliling alas x tinggi Prisma)

Contoh Soal :

Tentukan berapakah luas permukaan prisma tersebut apabila diketahui:

a.s = 10 cm

t.s = 12 cm

t = 20 cm

Yang ditanyakan:

Luas permukaan prisma … ?

Pembahasan:

Rumus: Luas permukaan Prisma Segitiga = t.s x [a.s + (3 x t)]

Masukkanlah nilai yang sudah diketahui ke dalam rumus,

Luas permukaan Prisma Segitiga = 12 x {10+(3 x 20)}

= 12 x (10+60)

= 12 x 70

= 840 cm2

Maka, di dapatkan luas permukaan prisma ialah 840 cm2

9. sebuah prisma segiempat dgn panjang sisi 125cm dan tinggi 247cm. hitunglah luas permukaan dan volume prisma tersebut?


Jawaban:

Untuk menghitung luas permukaan dan volume prisma segiempat, kita dapat menggunakan rumus sebagai berikut:

Luas permukaan = (luas alas + luas sisi x 2) x tinggi

luas alas = sisi x sisi

luas sisi = panjang sisi x tinggi

Luas permukaan:

luas alas = 125 x 125 = 15625 cm^2luas sisi = 125 x 247 = 30875 cm^2

Luas permukaan = (15625 + 30875 x 2) cm^2 = (15625 + 61750) cm^2 = 77375 cm^2

Volume:

Volume = luas alas x tinggi = 15625 x 247 = 3828125 cm^3

Jadi, luas permukaan prisma segiempat tersebut adalah 77375 cm^2 dan volume-nya adalah 3828125 cm^3.

16. rumus luas permukaan kubus,balok,prisma segitiga,tabung,limas segiempat,dan kerucut


Rumus luas permukaan kubus, balok, prisma segitiga, tabung, limas segiempat dan kerucut. Pada bangun ruang, ukuran yang biasa dihitung adalah volume dan luas permukaan. Berikut beberapa rumus luas permukaan dari bangun ruang


Pembahasan

Luas permukaan kubus = 6s²

dengan

s = rusuk kubus



Luas permukaan balok = 2(pl + pt + lt)

dengan

p = panjang balokl = lebar balokt = tinggi balok

Luas permukaan prisma segitiga

= 2 × luas alas + keliling alas × tinggi

= 2 × luas segitiga + keliling segitiga × tinggi

dengan

Luas segitiga = ½ × alas × tinggiKeliling segitiga = sisi + sisi + sisi

Luas permukaan tabung = 2πr (r + t)

dengan

r = jari-jari alas tabungt = tinggi tabungπ = 22/7 atau 3,14

Luas permukaan limas segiempat

= luas alas + jumlah luas sisi tegak

= luas segiempat + jumlah luas segitiga


Jika segiempat tersebut persegi, maka luas permukaan limas persegi adalah

= luas persegi + 4 × luas segitiga



Luas permukaan kerucut = πr(r + s)

dengan

s = √(t² + r²)r = jari-jari alas kerucuts = garis pelukist = tinggi kerucutπ = 22/7 atau 3,14
