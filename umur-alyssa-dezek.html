20. Alyssa swam from one end of a pool to the other end in 42 seconds at an average speed of 1.3 m/s Then, she swam back to the starting point in 40 seconds. What was the average speed at which Alyssa swam on the return lap?


Jawab:

1.365 m/s.

Penjelasan dengan langkah-langkah:

To find the average speed at which Alyssa swam on the return lap, we need to know the total distance she covered and the time it took.

Let's assume the length of the pool is x meters.

Alyssa swam from one end to the other end, a distance of x meters, in 42 seconds, so her average speed was:

x/42 = 1.3 m/s

We can solve for x by multiplying both sides of the equation:

x = 1.3 * 42 = 54.6 m

So the length of the pool is 54.6 meters.

On the return lap, Alyssa covered a distance of 54.6 meters in 40 seconds, so her average speed was:

54.6/40 = 1.365 m/s

So Alyssa swam on the return lap at an average speed of 1.365 m/s.
