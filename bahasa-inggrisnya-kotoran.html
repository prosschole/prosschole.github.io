28. Bahasa Inggris Nya Kelas Ini Kotor


this class is dirty

jangan lupa dilike ya

Jawaban:

This class is dirty

Penjelasan:

Kelas=class

Ini=this

Kotor=dirty

Karena kotor merupakan kata sifat, maka ditambah to be (is, am, are). Jumlah kelas itu kan satu maka to be -nya is.

Mengapa kelas ini seperti menjadi ini kelas?

Karena dalam Bahasa Inggris, misalkan kalau menyatakan kepunyaan, ini, atau itu biasanya seperti 'terbalik' begitu, makanya kelas ini dalam Bahasa Inggris menjadi THIS CLASS

Maafkan saya jika ada kesalahan kalimat
