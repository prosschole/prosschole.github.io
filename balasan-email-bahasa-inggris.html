Surat Konfirmasi Pemesanan Hotel Bahasa Inggris adalah surat yang dikirimkan kepada calon pelanggan hotel atau restoran mengenai rencana booking pelanggan tersebut. Konfirmasi ini ditujukan untuk memastikan tentang spesifikasi atau detail pesanan sehingga tidak mengalami kesalahan karena adanya ketidaksepahaman. Selain itu, hal terpenting bagi pihak hotel pada khususnya, surat konfirmasi pemesanan atau booking hotel ini dimaksudkan untuk mengetahui perihal kepastian jadi atau tidaknya calon penginap datang menginap di hotel tersebut terutama jika model pembayarannya dilakukan secara cash.

cara membalas email pemesanan hotel dalam bahasa Inggris​:

1. Buatlah sedetail mungkin rencana transaksi dengan menyertakan tanggal check in dan check out beserta dengan type dan tarif kamar hotel.

2.Tulislah peraturan hotel yang perlu untuk disampaikan seperti aturan pembatalan booking, dsb.

3.Berikanlah nomor contact hotel untuk memudahkan tamu hotel dalam   melakukan koordinasi atau konfirmasi secara lebih lanjut.

4. Bila perlu tawarkanlah paket lainnya untuk memudahkan tamu hotel             dalam melakukan kegiatan di tempat yang dituju.

An email has three basic parts. First is the Header, which is a set of lines containing information about the transmission of a message, such as the sender's address, recipient address, or stamp indicating the time when the message was sent by an intermediary server to a transport agent (MTA), which acts as an electronic mail sorting office.



The header also records all transmission paths that the message passes until it reaches the destination server. By using the Header, you can see the path taken by email in the transmission process and how long it will take for each server to process the transmission.



The second is the message, which consists of the following two elements: Header is a set of lines that describe the settings of the message, such as sender, receiver, date, etc. An email has at least three of the following headers: from the one containing the sender's e-mail address, to the one containing the recipient's e-mail address, the date that contains the time when the e-mail was received is various information about the intermediate server and the date when the message was processed, reply to i.e. the reply address, the subject of the message subject, the Message ID which is the unique identification of a message, the body of the e-mail containing the message separate from the header by a line.
