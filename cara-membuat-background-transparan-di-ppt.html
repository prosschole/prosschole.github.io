2. Cara membuat logo agar backgroundnya bisa transparan, itu pakai tools apa ya ?


Pakai photoshop bisa gan tinggal seleksi dl trs dihapusBerikut ini adalah langkah-langkah membuat logo background menjadi transparan menggunakan corel draw X4 (dan editing bitmapnya dengan corel Photo-Paint). Langsung saja simak tutorialnya dibawah ini:

1. Buka Corel Draw X4 File > New;

2. Import gambar logo File > import > (cari gambar);

3. Agar gambar bisa diedit terlebih dahulu kita harus mengconvert gambar ke bitmaps caranya, Klik menu Bitmaps > Convert to Bitmaps;

4. Setelah itu muncul pop up window, beri tanda centang pada transparent
background;

5. Selanjutnya edit bitmap dengan cara klik menu Bitmaps > Edit
bitmaps;

6. Setelah itu kita akan dibawa ke Corel Photo-Paint;

7. Pada Corel Photo-Paint Klik menu Mask > Color Mask;

8. Selanjutnya edit color mask dengan cara mengambil sample warna putih
yang ingin dihapus, lalu klik OK;

9. Setelah itu hapus warna putih dengan eraser tool;

10. Setelah selesai editing, klik finish editing

Setelah diedit di corel photo-paint, kita lihat hasilnya di corel draw.
Maka background putihnya sudah hilang. Lalu kita Export file yang sudah
jadi, simpan dengan format .png.

Demikianlah tutorial membuat background transparan di CorelDraw.
Semoga Bermanfaat.

4. bagaimana cara membuat slime putih polos/transparan


menggunakan gom,bedak,lem,Alat dan bahan:
1. Lem cair yang bening{boleh merek apa aja}
2. Slime activator khusus clear slime {terbuat dari gom/borax,air,sabun sedikit saja/clean and clear}
3.Wadah dan Pengaduk
4. Air
Cara pembuatan:
Masukkan Lem ke dalam wadah ,lalu beri air sedikit saja agar tidak terlalu keras hasilnya,lalu aduk perlahan.
Kemudian masukkan slime act  untuk clear slime lalu aduk
Setelah jadi memang agak berbusa,lalu ditunggu kurang lebih 3-5 hari untuk menjadi clear.

14. Cara membuat background di html


Jawaban:

Untuk membuat background di HTML, Anda bisa menggunakan atribut "style" pada tag "body". Berikut adalah cara membuat background berwarna sederhana:

Penjelasan:

Buka file HTML Anda menggunakan editor teks seperti Notepad.Temukan tag "body" di halaman Anda.Tambahkan atribut "style" pada tag "body" seperti ini: <body style="background-color: #000000;">.Ganti warna "#000000" dengan warna yang Anda inginkan. Anda bisa menggunakan kode warna HTML atau nama warna seperti "red" atau "green".Simpan file HTML Anda dan buka dengan browser web untuk melihat perubahan.

Untuk membuat background dengan gambar, Anda bisa menambahkan atribut "background-image" ke tag "body". Berikut adalah contoh:

*css

<body style="background-image: url('background.jpg');">

!Catatan: Pastikan file gambar berada pada lokasi yang benar dan memiliki format yang dapat diterima oleh browser web seperti JPEG, PNG, atau GIF.

Apabila kakak terbantu dengan jawaban ini, Jangan Lupa Bintang 5 dan Tombol Hati nya ya kakak. Juga boleh di Follow ko kak :D

Jawaban:

Untuk membuat background pada HTML, gunakan atribut "style" dan "background-color" pada tag "body". Ini adalah contoh sederhana:

<body style="background-color:lightgray;">

<!-- Isi halaman HTML Anda di sini -->

</body>

Penjelasan:

Anda juga bisa menggunakan warna hexadecimal atau nama warna (seperti "red" atau "blue") untuk mendefinisikan warna latar belakang.

Untuk menambahkan gambar sebagai latar belakang, gunakan "background-image" dan spesifikasikan URL gambar yang ingin Anda gunakan:

</body>

17. Bagaimana cara membuat jumbotron dibelakang navbar transparan.​


Penjelasan:

Jika Anda menggunakan framework CSS seperti Bootstrap, Anda dapat menambahkan class .navbar-transparent ke elemen navbar untuk membuat navbar transparan. Kemudian, tambahkan class .mb-5 ke elemen jumbotron untuk membuat jarak antara navbar dan jumbotron. Contoh kode HTML yang dapat digunakan adalah seperti ini:

<!DOCTYPE html>

<html>

 <head>

   <title>Jumbotron Behind Navbar</title>

   <!-- Tambahkan link ke Bootstrap CSS -->

   <link rel="stylesheet" href="https://stackpath.bootstrapcdn.com/bootstrap/4.5.2/css/bootstrap.min.css" integrity="sha384-JcKb8q3iqJ61gNV9KGb8thSsNjpSL0n8PARn9HuZOnIxN0hoP+VmmDGMN5t9UJ0Z" crossorigin="anonymous">

 </head>

 <body>

   <!-- Buat elemen navbar dengan class navbar-transparent -->

   <nav class="navbar navbar-expand-lg navbar-light bg-light navbar-transparent">

     <a class="navbar-brand" href="#">Navbar</a>

     <button class="navbar-toggler" type="button" data-toggle="collapse" data-target="#navbarNav" aria-controls="navbarNav" aria-expanded="false" aria-label="Toggle navigation">

       <span class="navbar-toggler-icon"></span>

     </button>

     <div class="collapse navbar-collapse" id="navbarNav">

       <ul class="navbar-nav">

         <li class="nav-item active">

           <a class="nav-link" href="#">Home</a>

         </li>

         <li class="nav-item">

           <a class="nav-link" href="#">About</a>

         </li>

         <li class="nav-item">

           <a class="nav-link" href="#">Contact</a>

         </li>

       </ul>

     </div>

   </nav>

   <!-- Buat elemen jumbotron dengan class mb-5 -->

   <div class="jumbotron jumbotron-fluid mb-5">

     <div class="container">

       <h1 class="display-4">Jumbotron</h1>

       <p class="lead">This is a jumbotron behind navbar</p>

     </div>

   </div>

 </body>

</html>

Perhatikan bahwa kode HTML di atas telah menambahkan link ke Bootstrap CSS pada bagian <head> dokumen HTML. Ini akan memastikan bahwa class .navbar-transparent dan .mb-5 dapat digunakan pada elemen navbar dan jumbotron.
