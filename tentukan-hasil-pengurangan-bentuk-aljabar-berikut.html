12. Tentukan hasil pengurangan bentuk aljabar berikut

a. 62n + 85m - 2

b. 4x + 4

c. -2y + 15z + 8

Penjelasan dengan langkah-langkah:

1.tentukan hasil pengurangan bentuk aljabar berikut:

a.(42n+35m+7) - (-50m-20n+9)

= 42n + 35m + 7 + 50m + 20n - 9

= 42n + 20n + 35m + 50m + 7 - 9

= 62n + 85m - 2

b.(5x + 3) - (x - 1)

= 5x + 3 - x + 1

= 5x - x + 3 + 1

= 4x + 4

c.(2y + 15z) - (4y - 8)

= 2y + 15z - 4y + 8

= 2y - 4y + 15z + 8

= -2y + 15z + 8

[tex]\huge { \bold{ \rm{{ \star \: Jawaban : }}}}[/tex]

Diketahui :

[tex] \sf {4p}^{9} + {8p}^{9} [/tex]

Ditanyakan :

Hasil bentuk Aljabar?

Penyelesaian :

Kita pakai cara Aljabar yaitu menambahkan sejenis bentuk dalam Aljabar, contoh :

[tex] \sf \: ax + bx = (a + b)x[/tex][tex] \sf \: ax - bx = (a - b)x[/tex]

Maka tinggal masukkan seperti cara diatas.

[tex] \sf {4p}^{9} + {8}p^{9} = ...[/tex]

[tex] = \sf(4 + 8) {p}^{9} [/tex]

[tex] = \boxed{ \red{ \sf{12 {p}^{ 9}}}} [/tex]

Pelajari lebih rinci tentang materi Aljabar :

1) 4p^9 + 8p^9

= 4p^9 + 8p^9

= ( 4 + 8 )p^9

= 12p^9 ✔

2) 4p^9 - 8p^9

= 4p^9 - 8p^9

= ( 4 - 8 )p^9

= -(8 - 4)p^9

= -4p^9 ✔
