1. Apa fungsi password pada Edmodo dan apa akibatnya jika orang lain mengetahui password tersebut?


PASSWORD

Fungsi passoword pada Edmodo dan akibatnya jika orang lain mengetahui password tersebut.

Jawaban

Fungsi password pada Edmodo adalah untuk melindungi akun Edmodo kita sehingga tidak disalahgunakan oleh orang yang tidak bertanggung jawab. Jika orang lain mengetahui password kita maka orang tersebut dapat melakukan akses ke dalam akun Edmodo kita dan melakukan apapun terhadap akun tersebut tanpa seizin dari pemilik akun.  

Pembahasan

Password adalah serangkaian karakter yang merupakan kata sandi yang harus dimasukkan kedalam suatu sistem agar kita dapat melakukan akses ke dalam sistem tersebut. Password bersifat rahasia. Password pada dasarnya adalah pelindung agar orang lain tidak seenaknya melakukan akses ke dalam ranah privasi kita.

Password sangatlah penting dalam melindungi hal – hal privasi kita dari orang – orang yang tidak bertanggung jawab. Misalnya saja password email. Jika ada orang yang mengetahui password email kita, maka orang tersebut dapan melakukan akses ke dalam email yang kita miliki. Mereka dapat melihat, membaca, bahkan mendapatkan akses penuh yang sama seperti kita. Bayangkan jika ada data rahasia di dalam email kita dan dicuri oleh orang – orang tersebut. Hal itu akan sangat merugikan kita.

Saat ini, di era globalisasi dan semakin canggihnya teknologi, kebutuhan akan keamanan data sangat tinggi. Password yang pada awalnya hanya berupa karakter, saat ini semakin berkembang. Beberapa cotoh pengenal identitas yang canggih seperti penggunaan scanner sidik jari dan retina, atau face recognition, adalah beberapa contoh semakin majunya teknologi untuk mencegah pencurian data.

9. Bagaimana cara mengetahui password router netis jika lupa sandi, tanpa merestart router?


Cara mengetahui password router netis jika lupa sandi, tanpa merestart router adalah dengan membaca buku manual dari router tersebut. Selain itu, kita bisa mencari pada internet untuk pasword default dari router netis. Router bisa berupa perangkat jaringan komputer atau berupa sistem operasi seperti mikrotik.

Pembahasan

Router merupakan salah satu perangkat jaringan komputer. Router berfungsi untuk menghubungkan jaringan komputer yang satu dengan jaringan komputer lainnya. Router biasanya dilengkapi user dan password untuk mengamankan pengaturan dari router tersebut. Beberapa password default dari router netis adalah sebagai berikut

user admin password adminuser guest password guestuser netis password passworduser unknown password unknown

Dengan mengetahui user dan password default dari sebuah router kita bisa mengakses router dengan mudah. Oleh karena itu untuk seorang admin jaringan hendaknya merubah user dan password routernya sehingga tidak menggunakan password default lagi agar mudah diakses oleh orang lain.

Pelajari lebih lanjut

Demikian pembahasan mengenai cara untuk mengetahui password router netis, kita bisa membacanya pada buku manual atau mencarinya di internet. Untuk mempelajari materi lebih lanjut mengenai perangkat jaringan komputer lainnya dapat dibaca pada link berikut

1. Melihat Password Wifi Pada Windows 10

• Cara untuk melihat password Wifi pada Windows 10 sebenarnya sangat mudah asalkan kamu mengikuti setiap langkahnya di bawah ini:

Cari ikon Wifi di bagian taskbar yang letaknya di kanan bawah layarPilih opsi Open Network & Internet SettingsDi dalam jendela Settings, pilih Network and Sharing CenterAda jendela baru yang terbuka lalu kamu bisa klik nama WifiDalam jendela Wifi Status, pilih Wireless PropertiesKembali muncul jendela baru dan kamu bisa klik tab SecurityUntuk melihat password dari Wifi yang sedang kamu pakai, centang show charactersSekarang kamu sudah melihat password Wifi di bagian kolom Network Security Key

• Masih ada cara lain untuk melihat password Wifi pada Windows 10. Kali ini melalui Command Prompt:

Buka “RUN” dengan menekan tombol Windows + RKetik “cmd” di kolom yang tersedia dan klik OKMuncul jendela Command Prompt lalu ketik “netsh wlan show profiles” (tanpa tanda kutip) dan tekan EnterKamu akan melihat daftar Wifi yang ada di dalam laptop/PCPilih Wifi yang kamu ingin ketahui password-nyaKetik kembali netsh wlan show profiles name=”nama profil Wi-Fi” key=clearTetap pakai tanda petik jika nama Wifi mengandung spasiKamu akan melihat password Wifi di bagian Key Content

2. Melihat Password Wifi Pada Mac

Sekarang saatnya mengetahui melihat password Wifi pada Mac. Ada dua cara yang bisa kamu gunakan. Bisa melalui Keychain Access dan Terminal.

1.Pertama, Keychain Access:

Buka aplikasi Keychain Access di bagian FinderKetika nama Wifi di kolom search dalam jendela Keychain AccessPastikan kamu ada di dalam kategori PasswordsKlik nama Wifi yang sudah munculAda jendela baru yang muncul lalu centang Show PasswordKamu perlu masuk menggunakan username dan password Mac yang digunakan sebagai bentuk fitur keamanan dari AppleSekarang kamu bisa melihat password Wifi tersebut

2.Kemudian melalui Terminal:

Buka aplikasi Terminal melalui FinderKamu bisa memasukkan tulisan ini: security find-generic-password -ga NAMA_SSID | grep -i passwordNAMA_SSID diganti dengan Wifi yang ingin diketahui password-nyaKemudian tekan EnterKamu perlu masuk menggunakan username dan password Mac yang digunakan sebagai bentuk fitur keamanan dari AppleSekarang kamu bisa melihat password Wifi yang diinginkan pada baris kedua

3. Melihat Password Wifi Pada Android

Ada juga cara mudah melihat password Wifi yang digunakan pada Android:

Buka SettingsPilih menu WifiKetik nama Wifi yang ingin diketahui password-nya lalu akan muncul QR CodeKamu bisa melakukan scanning menggunakan ponsel lain agar password Wifi tersebut muncul

4. Melihat Password Wifi Pada iPhone

Untuk melihat password Wifi pada iPhone, kamu tetap membutuhkan aplikasi Keychain Access dalam Mac:
