3. ikon polygon digunakan untuk menggambar bentuk...​


Polygon Tool adalah tool yang digunakan untuk membuat berbagai bentuk objek pada lembar kerja CorelDRAW, yang berbentuk polygonal. Dalam Polygon Tool ini, tersembunyi 5 macam tool yang fungsinya saling berkaitan, yakni : Polygon, Star, Complex Star, Graph Paper, dan juga Spiral. Tombol shortcut untuk Polygon Tool adalah Y, sedangkan tombol yang dapat digunakan untuk mengakses Tool Graph Paper adalah D, serta Tombol A untuk Spiral Tool.

17. Flyout yang digunakan untuk mengakses tool Polygon, spiral dan graph adalah …


Flyout yang digunakan untuk mengakses tool Polygon, spiral dan graph adalahJawabanPendahuluan

Flyout merupakan salah satu menu yang ada pada aplikasi Corel Draw. Adapun flyout yang digunakan untuk mengakses tool polygon, spiral dan graph adalah Flyout Polygon. Flyout sendiri merupakan menu yang berfungsi untuk membuat suatu bentuk atau memodifikasi bentuk yang sudah kita buat.

Pembahasan

Corel Draw merupakan salah satu dari sekian banyak aplikasi yang bisa kita gunakan untuk membuat desain grafis. Pada Corel Draw, gambar yang akan kita hasilkan akan berupa format vektor.

Aplikasi Corel Draw memudahkan kita dalam membuat desain grafis, banyak menu yang bisa kita gunakan untuk membuat desain grafi kita. Selain menggunakan Flyout Polygon, pada aplikasi Corel Draw masih terdapat menu flyout lainnya seperti

1. Flyout Shape Tool

2. Flyout Zoom

3. Flyout Freehand

4. Flyout Rectangle

5. Flyout Ellipse

6. Flyout Basic Shapes

Kesimpulan

Demikian pembahasan mengenai Flyout Polygon yang berfungsi untuk mengakses tool Polygon, Spiral dan Graph. Untuk mempelajari lebih lanjut mengenai aplikasi Corel Draw atau menu lainnya pada Corel Draw dapat dibaca pada link berikut
