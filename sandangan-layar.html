1. Layar sandangan sigeg gsntine suara apa



ꦽ  disebut layar, pengganti ꦃ sigeg (konsonan r), ditulis di atas huruf dasar atau huruf pasangan (huruf pasangan ꦔ꧋ꦛ꧋ꦞ); bila huruf pasangan terletak di bawah huruf dasar, layar ditulis di atas huruf dasar yang diberi pasangan.

Contoh:

ꦇꦑꦽ꧋ꦄꦽꦇ꧋ꦐꦂꦽ꧋ꦆꦏꦞꦽ꧋ꦉꦎꦖꦽ꧋ꦓꦏꦥꦽ꧋ꦋꦊꦚꦽ꧋

 
Jadi, layar sandangan sigeg gantine suara ( r )

4. Sandangan kang kalebu sandangan payigeg wanda​


Sandhangan kang kalebu sandhangan panyigeg wanda, yaiku:

WignyanLayarCecakPangkon

➖➖➖➖➖➖➖➖➖➖➖

Pembahasan

Sandhangan aksara jawa kaperang dadi 3, yaiku: sandhangan swara, sandhangan panyigeg wanda lan sandhangan wyanjana.

● Sandhangan panyigeg wanda

Yaiku sandhangan sing digunakake kanggo nyigeg utawa mateni aksara sing disigeg. Cacahing sandhangan panyigeg wanda ana 4, yaiku:

Wignyan → kanggo sigeg h ( ꦃ ).Layar → kanggo sigeg r ( ꦂ ).Cecak → kanggo sigeg ng ( ꦁ ).Pangkon → kanggo sigeg sakliyane h, r lan ng ( ꧀ ).

Tuladha:

Wignyan → Kolah ( ꦏꦺꦴꦭꦃ ).Layar → Pasar ( ꦥꦱꦂ ).Cecak → Gendhang-gendhing ( ꦒꦼꦤ꧀ꦢꦁꦒꦼꦤ꧀ꦢꦶꦁ ).Pangkon → Nyacat ( ꦚꦕꦠ꧀ ).

17. Sandangan layar iku kanggo paten aksara​


Sandangan layar iku kanggo paten R

Pembahasan :

Nulis aksara Jawa iku sing kudu dimangerteni yaiku pasangan aksara lan sandhangane.

➡️ Sandhangan ing Ukara pitakon ing dhuwur kuwi yaiku sandhangan panyigeg wanda.

➡️ Sandhangan panyigeg wanda ana 4 :

Wignyan = paten H = ꦃLayar = paten R = ꦂCecak = paten Ng = ꦁPangkon = saliyane h, r lan Ng = ꧀

★ tuladha :

pasar = ꦥꦱꦂcapat = ꦕꦝꦂgancar = ꦒꦤ꧀ꦕꦂ
