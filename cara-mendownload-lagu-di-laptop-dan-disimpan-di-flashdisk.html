6. Tania membeli laptop dan flashdisk di sebuah toko yang sedang mengadakan promo, Laptop tersebut diberi diskon 18% dan flashdisk 10%. Jika harga laptop Rp3.400,000,00 dan flashdisk Rp150.000,00, maka uang yang harus dibayarkan Tania adalahTOLONG YA JAWABNYA PAKE CARA DAN YG SERIUS​


Jawaban singkat:

Uang yg harus dibayarkan Tania adalah Rp 2.923.000

Penjelasan dg langkah²:

Harga Diskon laptop = 18% x Rp3.400.000,00

Harga Diskon laptop = [tex]\frac{18}{100}[/tex] x Rp3.400.000,00

Harga Diskon laptop = Rp 612.000,00

Harga Diskon flashdisk = 10% x Rp 150.000,00

Harga Diskon flashdisk = [tex]\frac{10}{100}[/tex] x Rp 150.000,00

Harga Diskon flashdisk = Rp 15.000,00

Uang yg harus dibayarkan Tania untuk beli laptop adalah

(Rp3.400.000,00 - Rp 612.000,00)

= Rp 2.788.000,00

Uang yg harus dibayarkan Tania untuk beli flashdisk adalah

(Rp 150.000,00 - Rp 15.000,00)

= Rp 135.000,00

Total uang yg harus dibayarkan Tania ialah sebagai berikut

Rp 2.788.000,00 + Rp 135.000,00

= Rp 2.923.000


7. Ada yang tau gimana caranya agar flashdisk bisa kebaca di laptop ?


Uh... langsung ditancep ke usb port aja (slot usb yg ada di pinggiran laptop).
Kalau belum kebaca, coba digerak-gerakin, atau agak ditarik sedikit flashdisknya (beberapa komputer punya port yg agak rewel)
Kalau masih belum kebaca, coba deh cek port usb nya support usb 3.0 atau nggak, karena beberapa keluaran flashdisk terbaru suka ga kebaca di laptop lama. Kalau udah gitu, kamu bisa buka di komputer atau laptop lain.Sambungkan Flashdisk ke port USB di bagian samping laptop kamu/anda/engkau. Biasanya akan berlangsung lama dalam penginstallan kalau memang itu Flashdisk baru. Kalau Flashdisk tidak bermasalah, maka akan langsung terbaca oleh komputer secara otomatis.
Apabila tidak terbaca/tidak terdeteksi, bisa ikuti langkah berikut;
1. Pasangkan Flashdisk yang tak terbaca ke port USB laptop
2. Arahkan mouse ke Start - Run - ketik "cmd.exe" tanpa kutip - enter
3. ketik "CHKDSK/fH:" tanpa kutip - enter
catatan : "CHKDSK/"f"H f disini adalah format flashdisk. Jika flashdisk berformat G, maka rubah menjadi "CHKDSK/gH. Jika E, ubah jadi "CHKDSK/eH" dan seterusnya.

jika tidak bisa menggunakan Command Prompt, bisa menggunakan cara format;
1. Pasangkan Flashdisk yang tak terbaca ke port USB laptop
2. Klik Start - Run - "compmgmt.msc" tanpa kutip - enter
3. Klik program tersebut
4. Pilih storage yang ada di tengah kemudian pilih disk management.
5. Setelah Flashdisk terbaca, klik kanan, pilih create partition.
6. Lalu format flashdisk
(source: wantekno.com)

11. Bagaimana cara mendownload npm di laptop?


Untuk mendownload npm (Node Package Manager) di laptop, Anda perlu mengikuti beberapa langkah sederhana:

1. Pastikan Anda telah menginstal Node.js di laptop Anda. Node.js adalah platform yang digunakan untuk menjalankan JavaScript di luar browser, dan npm merupakan bagian dari Node.js. Jika Anda belum menginstal Node.js, Anda dapat mengunduhnya dari situs web resmi Node.js (https://nodejs.org/).

2. Buka Command Prompt atau terminal di laptop Anda.

3. Ketik "npm install -g npm" tanpa tanda kutip pada Command Prompt atau terminal. Parameter "-g" digunakan untuk menginstal npm secara global, sehingga Anda dapat menggunakannya dari mana saja di dalam sistem Anda.

4. Tekan Enter dan tunggu hingga proses instalasi selesai.

5. Cek apakah npm sudah terinstal dengan mengetik "npm -v" pada Command Prompt atau terminal. Jika terinstal, Anda akan melihat versi npm yang terinstal ditampilkan pada layar.

Sebagai catatan, ada beberapa sistem operasi yang tidak memerlukan command -g , dan hanya cukup dengan npm install npm untuk menginstall, pastikan untuk mengecek dokumentasi dari NodeJS yang sesuai dengan sistem operasi yang digunakan.

19. Berapa jumlah lagu mp3 (masing masing lagu 3 mb) yang dapat di simpan pada flashdisk 8 gb


Terdapat sebuah diska lepas (flash disk). Kapasitas diska tersebut adalah 8 GB. Misalkan terdapat lagu yang masing-masingnya berukuran 3 MB. Diska lepas tersebut dapat menyimpan 2730 lagu.

Penjelasan dengan langkah-langkah

Diketahui:

Kapasitas diska lepas = 8 GB

Ukuran lagu = 3 MB

Ditanya: banyaknya lagu yang dapat disimpan dalam diska lepas

Jawab:

Konversi satuan

Satuan dari kedua besaran yag diberikan berlainan. Salah satunya akan diubah sehingga satuannya sama. Agar proses perhitungannya masih melibatkan bilangan bulat, ubah nilai kapasitas diska lepas ke satuan MB. Ingat bahwa: 1 GB = 1024 MB.

Kapasitas diska lepas = 8 GB = 8×1024 = 8192 MB

Banyaknya lagu yang dapat disimpan

Banyaknya lagu = kapasitas diska lepas/ukuran lagu

= 8192/3

≈ 2730,67

Karena banyaknya lagu harus dalam bilangan bulat, maka dibulatkan ke bawah (apabila ke atas, akan melampaui kapasitas diska lepas). Jadi, lagu yang dapat disimpan ke dalam diska lepas tersebut sebanyak 2730 lagu.
