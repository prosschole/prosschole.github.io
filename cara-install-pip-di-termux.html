1. Bagaimana cara menginstall ArchLinux di Termux ??? Ayo poinnya gede nih


ketik apt update && apt upgrade -y (enter)ketik apt install wget (enter)ketik wget https://raw.githubusercontent.com/sdrausty/TermuxArch/master/setupTermuxArch.sh(enter)ketik bash setupTermuxArch.shkemudian tunggu hingga selesaijalankan dengan mengetik startarch

Note: perhatikan huruf besar - kecilnya

Semoga membantu guys :-)

Oke, langsung saja kita ke tutorialnya.

Bahan-bahan :

Android

Aplikasi Termux

Koneksi internet yang stabil

Kapasitas memori > 800 mb

Langkah-langkah :

Instal dependensi untuk menginstal chroot Arch Linux dengan cara mengetikkan :

apt install git wget proot

Lalu clone script untuk instal Arch Linux dengan cara mengetikkan :

git clone https://github.com/sdrausty/termux-archlinux.git

Lalu pindah ke direktori arch linux dan jalankan scriptnya dengan cara mengetikkan :

cd termux-archlinux && chmod +x setupTermuxArch.sh && sh setupTermuxArch.sh

Lalu tunggu sampai proses instalasi selesai

Jika sudah, keluar dari chroot Arch Linux dan perbaiki scriptnya dengan cara mengetikkan :

exit

cd ~/arch && termux-fix-shebang startarch

Selesai

Kamu bisa menjalankan chroot Arch Linux dengan cara mengetikkan startarch

23. Apa Kegunaan Termux, Dan Apa Keuntungan Dan Kerugiannya??​


Termux bukanlah aplikasi media sosial. bukan juga aplikasi yang digunakan untuk chatting apalagi untuk gaming. Dalam Google Playstore, Anda akan menemukan penjelasan bahwa Termux adalah terminal emulator dan environment Linux berbasis Linux Debian.

Anda yang baru mendengar istilah ini mungkin bingung, apa itu terminal emulator? Secara mudahnya, terminal emulator itu semacam Command Prompt (CMD) pada Windows atau Linux. Jika CMD adanya di Windows atau Linux, terminal emulator ini adanya di sistem operasi Android. Terminal ini digunakan untuk memasukkan, menampilkan, dan mencetak data.

Kelebihan dari Termux adalah aplikasi ini bisa langsung dipasang di smartphone Android Anda tanpa perlu proses rooting atau pengaturan lainnya. Termux tidak membutuhkan itu. Aplikasi ini juga mendukung perangkat Android berbasis 32bit ataupun 64bit.

Aplikasi ini dibuat oleh Fredik Fornwall. Sejak dirilis pertama kali pada 30 Juni 2015 di Google PlayStore, Termux telah diunduh oleh lebih dari 10 juta pengguna android. Hingga tulisan ini diturunkan, Termux telah mencapai versi 0.101 dan akan terus mengalami pembaharuan.

Kegunaan Termux

termux adalah

Anda sudah diajak mengenal sedikit hal mengenai Termux, pendirinya, dan tanggal rilis dari aplikasi ini. Mungkin Anda penasaran, apa kegunaan dari aplikasi terminal emulator ini? Tulisan ini juga akan membahas mengenai hal tersebut.

Sebagai terminal emulator, fungsi Termux hampir sama dengan Command Prompt di OS Windows dan Linux. Perbedaannya terletak pada perintah-perintah yang digunakan. Secara umum, berikut ini merupakan kegunaan dari Termux :
