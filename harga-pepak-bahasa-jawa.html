17. buatlah dialog tentang mau beli buku pepak basa jawa​


JAWABAN

PERTANYAAN :

Buatlah Dialog Tentang Mau Beli Buku Pepak Bahasa Jawa!

JAWABAN :

Dialog :

A : Saya ingin membeli Pepak Bahasa Jawa.

B : Buku Pepak Bahasa Jawa?

A : Ya.

B : Ini Buku Pepak Bahasa Jawa.

A : Bolehkah saya lihat dulu?

B : Ya silahkan.

A : Saya akan beli Pepak Ini. Berapa harganya?

B : Rp. 55.000,-

A : Bagaimana jika Rp. 50.000,- saja, uang saya tidak cukup.

B : Baiklah.

A : Ini uangnya.

B : Apakah perlu di bungkus, pak?

A : Ya silahkan.

B : Ini Buku Nya, pak.

A : Terima kasih.

B : Sama-sama.

PENJELASAN :

Keterangan :

A = Pembeli

B = Penjual

Dialog Adalah sebuah literatur dan teatrikal yang terdiri dari percakapan secara lisan atau tertulis antara dua orang atau lebih.
