4. langkah-langkah mengaktifkan Adobe Photoshop CS5


Jawaban:

Cara Install dan Aktivasi Adobe Photoshop CS5

1. Setelah anda selesai mendownload file Adobe Photoshop CS5, selanjutnya anda perlu mengekstraknya. Dengan cara klik kanan > pilih Extract to.

2. Kemudian klik 2X file Set-up.

3. Pilih Accept.

4. Selanjutnya anda akan disuruh memasukkan serial number. Kita bisa skip tahap ini dengan cara memilih Install this product as a trial kemudian pada kolom bahasa pilih English (International). Kemudian pilih Next.

5. Jika laptop anda menggunakan Windows 32 Bit, maka disarankan memilih Adobe Photoshop CS5 (tanpa embel-embel 64 Bit). Jika anda tidak tahu Windows anda 32 atau 64-bit. Selanjutnya pilih Install.

6. Jika muncul notifikasi seperti ini, tinggal pilih Yes.

7. Tunggu proses instalasi Adobe Photoshop CS5 hingga selesai.

8. Adobe Photoshop CS5 sudah berhasil diinstall, selanjutnya pilih Done.

Tahapan aktivasi yang perlu kita lakukan

Download file dibawah ini untuk keperluan aktivasi Adobe Photoshop CS5. Selanjutnya ekstrak dengan cara klik kanan > pilih Extract to.

Selanjutnya buka folder Aktivasi.

Pilih 32 atau 64 bit

Kemudian copy file yang bernama amtlib.dll.

Buka Local Disk (C:) kemudian masuk ke folder Program Files.

Masuk ke folder Adobe > Adobe Photoshop CS5 (tergantung bit)

Pastekan file tadi kesini. Jika muncul notifikasi seperti ini, tinggal pilih Replace the file in the destination.

Jika muncul warning, pilih saja Continoe.

Sekarang anda sudah bisa menggunakan Adobe Photoshop CS5 secara maksimal tanpa perlu takut muncul popup yang menyuruh anda untuk melakukan aktivasi.

7. bagaimana cara membuat layout desain web dengan adobe photoshop ?


1. Buka aplikasi Photoshop, buat layer baru dengan klik File => New atau Ctrl + N
2. Buat ukurannya dengan width 1000 pixel dan Height 600 pixel, Resolusi 72 pixel/inch
3. Klik toolbox dan ganti warna foreground color dengan biru (#0038c6) dan background color putih
4. Masih di toolbox, pilih Gradient Tool (G) dan pada keyboard tekan Shift, lalu tekan mouse mulai dari atas dan lepaskan mouse jika sudah berada paling bawah,
5. Buatlah garis bantu / Guide dengan, caranya pilih View => New Guide,
6. Lalu membuat Header, caranya pada layer klik Create a new layer lalu pada toolbox klik Rectangle Tool lalu klik kanan mouse dan ubah foreground color menjadi putih lalu pilih Paint Bucket Tool (G),
7. selesai
8. Buat garis bantu / Guide lagi untuk membuat kotak logo dan kotak iklan
9.  Lalu masukkan logo dan kotak iklan yang sudah anda buat terlebih dahulu,
10. Buat garis bantu untuk membuat menu navigasi
11. ntuk membuat kotak menu navigasi, caranya sama dengan nomor 5, ubah warna foreground color menjadi biru dan background color menjadi putih. Lalu klik Rectangle Tool, klik kanan mouse lalu pilih Make Selection dan berikan gradasi. Untuk membuat warna gradasi seperti pada nomor 3.
12. selesai
13.  Buat teks dengan cara pada pallet layer, klik Create a new layer dan pada toolbox klik Horizontal Type Tool (T), lihat gambar dibawah cara menempatkan teks di kotak menu navigasi
14. Buat garis bantu lagi untuk membuat kotak sidebar kiri, content / post blog dan kotak sidebar kanan, lihat caranya dibawah sesuai nomor urut. Untuk membuat kotaknya seperti pada nomor 5.
15. selesai
16. Buat garis bantu lagi untuk membuat Footer
17.  Copy kotak menu navigasi pada nomor 10, caranya klik kotak menu navigasi dan tekan Ctrl+J, lalu move ke bawah. Lalu pada menu diatas silahkan klik Edit => Transform => Rotate 180°
18. Dan hasil akhir dari desain website dengan photoshop
19. Selanjutnya potong dengan Slice Tool (K) bagian-bagian seperti body, header, logo, kotak iklan, menu navigasi, sidebar kiri, post blog, sidebar kanan dan footer.
20. Lalu pilih File => Save for Web and Devices

10. Sebutkan dan jelaskan tipe teks dalam Photoshop CS5


Tipe teks dalam Adobe Photoshop adalah Bevel dan Embos, Stroke, Inner Shadow, Satin, Color Overlay, Gradien Overlay, Drop Shadow, patern Overlay, Outer Glow. Teks padaAdobe Photoshop sendiri merupakan salah satu fitur yang mempermudah kita dalam melakukan penamnbahan dan modifikasi tulisan pada gambar yang sedang kita edit.

Pembahasan

Setelah mengetahui apa saja jenis teks pada Adobe Photoshop, mari kita bahas satu persatu agar kita bisa memilih menggunakan jenis teks yang akan kita masukkan ke dalam gambar kita.

Bevel dan Embos, jenis teks ini menyerupai suatu ukiranStroke jenis teks stroke memiliki garis tepi pada teksnya, kita bisa mnegatur ketebalannya.Inner Shadow, jenis teks ini memnyerupai benda yang memiliki bayangan ke sisi dalam.Satin, jenis teks ini sama dengan teks inner shadowColor Overlay, pada jenis teks ini kita bisa memberi warna tambahan.Gradien Overlay, pada teks ini kita bisa memberikan lebih dari satu warna pada teks kita.Drop Shadow memberikan efek pada teks kita berupa bayangan seperti bayangan dari sinar lampu.Patern overlay, pada efek ini teks kita akan memiliki tekstur.Outer Overlay, efek teks ini memberikan efek menyerupai sinar lampu.

Dengan mengetahui efek teks yang ada pada Adobe Photoshop, kita bisa memilih jenis teks yang cocok untuk gambar kita. Sehingga hasil akhir dari pengeditan gambar kita akan semakin bagus.

Dalam PhotoShop terdapat beberapa model-model pewarnaan antara lain :  


1.      RGB (Red,Green, Blue)

Dalam model warna ini terdapattiga macam warna dasar yaitu merah, hijau dan biru. Untuk mendapatkan warna lain dapat dilakukan dengan proses pencampuran warna dasar. GBB ini disebut juga dengan Additive Colour.

Dalam penggunaan warna RGB dugunakan skala 0 terendah sampai 255 tertinggi. Jika nilai RGB asam maka akan menghasilkan warna abu-abu. Ketika warna RGB diset pada nilai 255 (tertinggi) , maka akan mengasilkan warna putih dan jika warna RGB diset 0 (terendah) maka akan menghasilkan warna putih.
