17. mungsuhe para pandhawa ing perang baratayuda yaiku​


Mungsuhe para Pandhawa ing perang Baratayudha yaiku Kurawa

Pembahasan :

Ing ngisor iki ringkesan cerita ngenani perang Baratayudha

>> Ing cerita pewayangan, Perang Baratayudha iku perang antarane Pandawa mungsuh Kurawa. Pandawa cacahe ana lima lan kurawa cacahe ana satus. Perang Baratayudha iku perang kanggo ngrebutake takhta Ngastina. Para kurawa ambisi banget kanggo nyingkirake sedulure supaya Ngastina bisa dadi hak e para kurawa dewe. Sembarang cara wis dilakoni kurawa kanggo nyingkirake para Pandhawa, nanging carane kurawa panggah ora kasil. Banjur ana perang Baratayudha. Perang Baratayudha iku dumadine mung 18 dina. Akeh para ksatriya kang gugur, para putrane Pandhawa uga gugur kayata Raden Gatotkaca lan Raden Abimanyu. Nanging akhire perang Baratayudha dimenangake dening para Pandhawa. Pandhawa sing cacahe mung lima iso menang mungsuh kurawa sing cacahe ana satus

Kesimpulan :

Dadi kapethik saka wacan ing dhuwur mungsuhe para Pandhawa ing perang Baratayudha yaiku Kurawa
