8. sebutkan lima macam tool yang ada pada corel draw


Pick Tool adalah tools untuk seleksi dan mengubah ukuran objek pada lembar kerja CorelDRAW

Shape Tool adalah tool untuk mengubah bentuk objek pada lembar kerja CorelDRAW. Dalam tool ini, tersimpan 4 jenis tool berbeda, yang fungsinya saling berkaitan, yakni : Shape, Smudge Brush, Roughen Brush, dan Free Transform. Adapun tombol shortcut atau jalan pintas yang dapat digunakan untuk mengakses tool ini secara cepat adalah tombol F10.

Crop Tool adalah tool yang dapat digunakan untuk memotong sebagian dari objek pada lembar kerja CorelDRAW yang tidak di inginkan. Dalam tool ini, tersimpan 4 jenis tool berbeda, yang fungsinya saling berkaitan, yakni : Crop, Knife, Eraser, dan Virtual Segment Delete. Tool ini tidak memiliki tombol shortcut.

Zoom Tool adalah tool yang digunakan untuk memperbesar ataupun memperkecil tampilan objek pada lembar kerja CorelDRAW. Dalam tool ini, tersimpan 2 jenis tool berbeda, yang fungsinya saling berkaitan, yakni : Zoom (shortcut Z) dan Pan (shortcut H).

Freehand Tool adalah tool yang dapat digunakan untuk membuat berbagain macam garis dalam lembar kerja CorelDraw, baik garis bebas, point to point, maupun garis-garis yang lain. Terdapat banyak sekali tool yang tersimpan didalam tool freehand yang fungsinya saling berkaitan satu sama lain. Tool-tool tersebut adalah Freehand, 2-Point Line, Bézier, Artistic Media, Pen, B-Spline, Polyline, serta 3-Point Curve. Tombol yang dapat digunakan untuk mengakses tool Freeand secara cepat/shortcut tombolnya adalah tombol F5, sedangkan untuk mengakses tool Artistic Media, kita dapat menggunakan tombol I.

13. Apa fungsi Polygon tool di corel draw


Fungsi Polygon tool  yaitu untuk menggambar dengan bentuk persegi secara banyak, sekedar dengan melakukan atau mengaktifkan drag dan klik mouse. Memiliki turunan 4 macam tools, yaitu :

Star tool : berfungsi untuk menggambar berbagai macam bentuk bintang.

Complex Star tool : berfungsi untuk menggambar bentuk bintang yang mempunyai banyak potongan sisi.

Graph Paper tool : berfungsi  untuk menggambar bentuk rangkaian  susunan kotak-kotak seperti jala-jala.

Spiral tool : berfungsi  untuk menggambar bentuk lingkaran spiral secara simetris atau logaritmis

Interactive Tool letaknya di sebelah kiri lembar kerja corel draw, diatas eyedropper tool dan dibawah table tool. Interactive tool sendiri dibagi menjadi beberapa bagian lagi yaitu : Blend, Contour, Distort, Drop Shadow, Envelope, Extrude, dan Transparency.

30. Jelaskan perbedaan Pick Tool, Shape tool, Freehand Tool, Zoom Tool dan Text Tool !


Perbedaan tersebut dapat kita lihat melalui fungsinya.

1. Pick Tool
Pick Tool ini memiliki ikon seperti cursor mouse pada setting defaultnya. Berfungsi untuk menyeleksi atau memilih objek.

2. Shape Tool
Shape Tool ini berfungsi untuk mengedit garis atau objek dengan manipulasi titik.

3. Freehand Tool
Freehand Tool berfungsi untuk menggambar garis kurva atau garis lengkung dan juga garis lurus.

4. Zoom Tool
Zoom Tool ini berfungsi untuk mengubah tingkat pembesaran window document Corel yang sedang kita buka. Kita bisa mengedit objek yang terlalu besar atau objek yang terlalu kecil sehingga nantinya tammpilan tersebut akan sesuai di mata kita.

5. Text Tool
Text Tool ini berfungsi untuk menambahkan tulisan pada canvas Corel kita. Kita juga bisa mengatur jenis font, besar font, dan juga memperbesar atau memperkecil tulisan yang ada.
